# 03_商圈分析与客群调研

## 目录作用

本目录沉淀**商圈分析、竞品调研、客群画像、购买任务判断和门店位置分析**的标准方法，是社区超市精准定位和商品策略的前提。

商圈和客群是一切商品规划、陈列设计和营销活动的源头依据。这里存放的是通用方法框架，具体门店的调研素材和结论存放于 `09_门店案例与项目复盘` 中对应门店的 `01_商圈与客群` 子目录。

---

## 适合存放

- 商圈分析标准框架（半径划分、竞品梳理、人口估算）
- 门店周边竞品调研模板
- 客群画像调研模板（年龄、职业、消费习惯、购买任务）
- 顾客购买任务分析（"买了啥""为什么来""还缺什么"）
- 社区客群分层模型
- 竞品截流分析方法
- 门店位置与动线判断方法
- 地图截图采集规范
- 竞品照片采集规范
- 商圈调研报告模板

---

## 不适合存放

- 某家门店的具体竞品照片（原始素材先进 `90_素材暂存与待整理`）
- 某家门店的具体商圈调研报告（属于 `09_门店案例与项目复盘` 对应子目录）
- 商品诊断结果（属于 `04_商品诊断与商品力提升`）
- 客户交付报告（属于 `10_咨询交付模板`）
