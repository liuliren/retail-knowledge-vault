# 03_客户报告模板（占位）

## 目录定位

本子目录沉淀**最终交付给客户的报告级模板**——介于 02_交付SOP/（顾问执行手册）与 04_老板看板与凭证模板/（持续运营层）之间的"项目阶段成果交付物"层。

报告必须同时通过：
1. 反幻觉关（[[反幻觉检查清单]]）
2. client_safety 关（[[品牌资产手册_v0.1]] §4 三道关）
3. 品牌口径关（[[品牌资产手册_v0.1]] §2 对外口径统一表）

---

## 适合存放（Phase 1+ 充实）

- 商品诊断报告模板（客户版）
- 品类整改报告模板（客户版）
- 陈列整改报告模板（客户版）
- 商圈分析报告模板（客户版）
- 门店诊断综合报告模板
- 30天商品提效阶段交付报告模板
- 90天陪跑路线图交付版

---

## 不适合存放

- 顾问执行手册类 SOP（→ 02_交付SOP/）
- 老板月度看板 / 凭证类（→ 04_老板看板与凭证模板/）
- 内部分析过程稿（→ 09_/<门店>/对应业务子目录）
