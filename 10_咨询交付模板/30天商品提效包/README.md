## 2. 交付主链（四件套）

```
商品诊断报告 → SKU动作建议表 → 周复盘表 → 结项报告 →（承接）后续产品
```

## 3. 商业模式（内部参考）

三版：基础版≈3万 / 标准版≈5万 / 加强版≈8万；标准周期 30 天（可升级 4–8 周）。**内部报价参考，非对外固定价格**（详见主定义 §17）。

## 4. 边界与纪律

只引用不另起炉灶；不重写 RetailOS；花厅坊为样板验证案例不升级为主定义；旧 PDF/DOCX 为素材库非主定义，须经校准后用。
