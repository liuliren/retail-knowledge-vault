# 门店商品动作RACI_模板定义

## 0. 模板定位

- 本模板用于明确**店长 / 采购 / 主管 / 店员 / 顾问**在商品动作中的责任；
- 服务 30天商品提效包，是 [[组织力最小5模块总览_v0.1]] 模块 1（执行责任机制）+ 模块 2（RACI）的落地实例；
- **不是**完整 HR 管理体系、**不是**组织架构设计、**不是**绩效考核方案。

## 1. 使用对象

顾问 / 店长 / 采购 / 主管 / 店员。

## 2. RACI 说明

- **R = 负责执行**（动手做的人）；
- **A = 最终担责**（对结果负责、签字验收的人）；
- **C = 参与确认 / 提供意见**；
- **I = 知会**（需要知道结果）。

> 每个动作**必须至少有一个 R 和一个 A**；A 通常唯一。

## 3. 角色定义

| 角色 | 核心职责 | 不该做什么 | 验收标准 |
|---|---|---|---|
| 店长 | 确认动作清单、分派、追踪、主持周复盘 | 不亲自摆货（除非小店）| 7天追踪表全勾、周复盘有结论 |
| 采购 | 补货/引进/淘汰执行、对接供应商 | 不擅自改陈列规则 | 引进/汰换清单按时到位 |
| 主管 | 货架调整督导、现场检查、拍照 | 不改商品策略 | 货架按图调整、有拍照留证 |
| 店员 | 按执行卡做动作（摆货/贴签/整理/上架）| 不自行判断淘汰/引进 | 执行卡当日完成、现场达标 |
| 顾问 | 给判断、带教、退出后支持 | 不替门店长期执行 | 门店能自运行后退出日常 |

## 4. 商品动作 RACI 初版

（R=负责执行 A=最终担责 C=参与确认 I=知会）

| 动作 | 店长 | 采购 | 主管 | 店员 | 顾问 |
|---|---|---|---|---|---|
| 确认商品动作清单 | A | C | C | I | R(首轮) |
| 确认补货/引进/淘汰 | A | R | I | I | C |
| 调整货架 | A | I | R | R | C |
| 贴价签/整理陈列 | I | I | A | R | I |
| 拍照验收 | A | I | R | I | I |
| 记录未完成原因 | A | I | R | R | I |
| 周复盘 | R/A | C | C | I | C(首轮) |
| 下周动作确认 | A | C | C | I | C |

## 5. 7天动作追踪机制

- **每天看哪些动作完成**：店长/主管对照当日执行卡逐项核；
- **谁记录**：主管（日跟）；
- **谁检查**：店长（周收口）；
- **未完成如何标记**：在对应日格标"未完成"，并在"未完成原因"列写明；
- **7天后如何进入周复盘**：7天追踪表作为周复盘的"执行核对"输入。

> 示例（非真实数据）：

| 动作 | 责任人 | 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 | 未完成原因 | 店长确认 |
|---|---|---|---|---|---|---|---|---|---|---|
| 示例饼干A 多摆2面 | 店员 | ✔ | | | | | | | — | 已验 |
| 示例冲调B 补满 | 店员 | ✔ | ✔ | ✔ | | | | | — | 已验 |
| 示例薯片C 清掉 | 店员 | | 未 | | | | | | 货未卖完 | 待整改 |

## 6. 周复盘机制

- **会前准备**：店长收齐 7天追踪表 + 主管拍照 + 采购到货情况；周复盘表瘦身版填好数据栏（接 [[周复盘表_模板定义_v0.1]]）；
- **会议议程**（≤30 分钟）：①上周动作完成率 ②有效/无效判定（卖动了没）③未完成原因 ④下周动作清单 + 责任人；
- **会后动作**：店长出"下周动作清单"→分派→录入 7天追踪表；
- **谁追踪**：主管日跟、店长周收口；
- **下周如何复盘**：同一表滚动对比上周，连续 3 周无效的动作升级到顾问复诊。

## 7. 未完成处理规则

| 未完成情形 | 处理 |
|---|---|
| 缺货 | 采购处理 |
| 不会做 | 主管 / 顾问带教 |
| 没人做 | 店长重新分配 |
| 货架不适合 | 主管反馈、调整方案 |
| 商品未到 | 采购反馈供应商 |
| 连续两周未完成 | 店长升级处理（排查根因，必要时顾问复诊）|

## 8. 顾问退出后的门店自运行机制

- **第一周**：顾问主导（确认清单 R、主持周复盘 R）；
- **第二周**：店长主导、顾问辅助（顾问转 C）；
- **第三周**：门店自运行；
- **退出标准**：连续两周门店能自主完成「**动作 → 验收 → 复盘 → 下周动作**」闭环，顾问退出日常执行（保留远程支持）。

## 9. 不扩展边界

本模板**不处理**：招聘 / 薪酬 / 企业文化 / 完整绩效 / 组织架构 / 大型培训体系。
