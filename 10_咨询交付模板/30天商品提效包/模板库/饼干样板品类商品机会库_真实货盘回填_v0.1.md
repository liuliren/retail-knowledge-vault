# 饼干样板品类商品机会库_真实货盘回填

## 0. 文件定位

- 本文件是饼干样板商品机会库的**真实货盘回填实例**；
- 用于把「待核实品牌 / 占位 SKU」推进为「**采购可询价清单**」；
- **不是**正式采购单、**不是**已确认供货清单、**不是**价格确认单、**不是**供应商合同；
- **不承诺动销结果**。

## 1. 使用边界

- 品牌 / SKU 方向来自**公开市场常见品牌与社区超市可见商品**；
- **价格带为估区间**；
- **规格以采购现场核实为准**；
- **供应商状态为待采购确认，不代表已确认供货**；
- **只有采购确认「现有供应商可供」后，才能进入正式 SKU 动作「引进 / 放大」**；
- 本文件用于**采购问货，不用于直接下单**。

## 3. 采购可执行清单

### A. 可先问现有快消经销商的品牌
- 亿滋系：奥利奥 / 趣多多 / 乐之 Ritz（同渠道，可一次问全）；
- 嘉士利；好吃点（达利）；康师傅。
> **采购动作**：向现有快消 / 休食经销商核对——是否在供、当前可供 SKU、进价、起订量、到货周期。

### B. 需寻找进口 / 节令渠道的品牌
- 谷优 Gullon；和情 Lotus；Loacker 莱家；茱蒂丝 Julie's；丹麦蓝罐曲奇 Kjeldsens；皇冠丹麦曲奇 Danisa。
> **采购动作**：对接进口食品经销商 / 批发市场进口专柜 / 节令礼盒渠道，确认是否可供与节前到货周期。

### C. 区域品牌待核实
- 青食。
> **采购动作**：核实本地是否有区域经销商；若无货源，则用康师傅 / 嘉士利低价位替代。

## 5. 需要新找供应商的缺口

1. **进口线**（谷优 / Lotus / Loacker / 茱蒂丝 / 丹麦蓝罐 / 皇冠）：现有快消经销商通常无货，需新对接进口经销商——**未对接前无法进「引进」**。
2. **区域线**（青食等）：非全国流通，本地拿不到时无差异化替代（只能退全国低价位）——**可落地性因地而异**。
3. **健康无糖线**（思朗易拿 / 谷优需进口）：无糖品类供给偏薄，是采购需重点补的货源缺口——**供给不足前难稳定上架**。
4. **节令礼盒线**（丹麦蓝罐 / 皇冠）：依赖节令渠道与节前到货周期，**错过节令窗口即无法上量**。

## 6. 结构断点说明

1. **进口品类供给依赖外部渠道**：中高端 + 部分早餐/老人无糖现有经销商无货，必须新对接进口渠道，未对接前这些格无法转「引进」。
2. **低价引流区域品牌可落地性因地而异**：区域品牌非全国流通，本地无货时无明确差异化替代。
3. **价格 / 规格 / 起订量需采购现场核实**：本表为估区间与方向，真实进价、在售 SKU 名、起订量需采购向经销商核对后才能进采购单。

## 7. 采购下一步动作

| 动作 | 责任人 | 输出结果 | 是否进入 SKU 动作 |
|---|---|---|---|
| 问现有快消经销商（A 组）| 采购 | A 组品牌可供 SKU / 进价 / 起订量 | 确认可供后→是 |
| 问进口食品 / 节令渠道（B 组）| 采购 | 进口经销商对接结果 / 节前周期 | 确认可供后→是 |
| 核实区域品牌（C 组）| 采购 | 区域经销商有无 / 是否退替代 | 确认可供后→是 |
| 确认进价 / 起订量 / 到货周期 | 采购 | 可采购参数 | 是（采购参数齐后）|
| 将可供品牌状态更新为「现有供应商可供」| 采购 | 机会库状态更新 | 是 |
| 将不可供品牌留在机会库待跟进 | 采购 | 待跟进清单 | 否（留库待跟进）|
