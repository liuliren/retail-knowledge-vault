# 周复盘表 Excel工具版 使用说明 v0.1

> 配套文件：`周复盘表_Excel工具版_v0.1.xlsx`（同目录）。字段口径以 [[周复盘表_模板定义_v0.1]] 为准。
> ⚠ **internal_only，不发客户**；本工具是 Batch 1 实体化**试产首件**。

## 1. 文件定位

30天商品提效项目第 7/15/30 天复盘工具；承接 [[SKU动作建议表_模板定义_v0.1]] 与 [[DisplayMap调整表_模板定义_v0.1]]，记录动作执行、复盘数据、有效性判断、误判防线检查与下一步动作。

## 2. 使用前准备

- 已完成 SKU动作建议表（顾问/店长版）；
- 涉及陈列的已有 DisplayMap调整表；
- 复制本 xlsx 为项目副本后填写，**先删除 ⑤脱敏示例行/主表示例行**再填真实（真实数据不写入 git，见 §9）。

## 3. 各 Sheet 说明

| Sheet | 用途 |
|---|---|
| README_使用说明 | 使用步骤与纪律 |
| ①周复盘主表 | 主填写表（32列/6字段组/16下拉）|
| ②动作有效性字典 | 5 类有效性结论判断口径 |
| ③误判防线字典 | 11 类误判防线复盘检查要点 |
| ④字段说明 | 32 字段含义/必填/来源 |
| ⑤脱敏示例 | 脱敏规则与示例说明 |

## 4. 主表填写步骤

1. **A 基础信息**：门店/品类/周期/复盘日期/顾问/店长/复盘阶段(第7/15/30天)；
2. **B 动作来源**：从 SKU动作表带入 SKU/商品·SKU动作·商品角色·原建议动作；涉及陈列带入"是否来自DisplayMap"+"陈列动作"；
3. **C 执行状态**：是否执行/执行日期/偏差/未执行原因/现场反馈；
4. **D 复盘数据**：销售/库存/毛利变化方向（**分档表达**）+ 客户/店长反馈；
5. **E 有效性判断**：有效性结论(查②) + 判断依据 + 误判防线检查(查③) + 是否继续观察；
6. **F 下一步动作**：下一步动作（复盘处置）+ 是否进入结项 + 是否需要二次调整 + 备注。

## 5. 与 SKU动作建议表 Excel 的接口

- 主表 B 组（SKU/商品·SKU动作·商品角色·原建议动作）**从 [[SKU动作建议表_Excel工具版_v0.1]] 导入**；
- **不直接复制真实数据**——用脱敏或空模板方式；
- 接入方式：手动复制 / 导入脚本 / Power Query 均可，**本版不做自动化**；
- 字段口径与 SKU表 Excel 一致（分档表达、脱敏规则）。

## 6. 与 DisplayMap调整表的接口

- "是否来自DisplayMap"=是 时，"陈列动作"取 DisplayMap **12 类陈列动作**；
- 陈列缺数据时陈列动作填"待陈列数据"，不强行精确货位（同 DisplayMap 口径）。

## 7. 脱敏规则

门店=示例门店A；商品=SKU-DEMO-XXX/示例商品A；人员=顾问A/店长A；数值用分档（明显改善/小幅改善/持平/小幅恶化/明显恶化/待观察/数据不足）。**禁止**真实条码/SKU/商品名/销售额/客户名称/花厅坊真实信息。

## 8. 常见错误

- ❌ 在变化列填真实销售额 → ✅ 用分档；
- ❌ 自创 SKU动作/商品角色/误判防线 → ✅ 沿用 10/10/11 类；
- ❌ 把"下一步动作"当成新增 SKU动作 → ✅ 它是复盘处置（保持原动作/转入结项/待补数据等）；
- ❌ 缺货架数据硬填精确货位 → ✅ 陈列动作填"待陈列数据"。
