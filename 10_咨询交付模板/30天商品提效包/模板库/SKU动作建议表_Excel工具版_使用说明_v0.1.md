# SKU动作建议表 Excel工具版 使用说明 v0.1

> 配套文件：`SKU动作建议表_Excel工具版_v0.1.xlsx`（同目录）。
> 字段口径与规则**以 [[SKU动作建议表_模板定义_v0.1]]（v0.2）为唯一来源**；本 Excel 是其工具落地。
> ⚠ 本工具为**对内执行/分析工具**，非客户对外版；示例行为占位（SKU-A/B/C），不含真实数据。

## 1. 工作簿结构（6 sheet）

| Sheet | 名称 | 用途 |
|---|---|---|
| 01 | SKU动作建议主表 | 实际填写主表（57列，三段分区+下拉）|
| 02 | 动作枚举字典 | 10 类 SKU 动作定义/触发/禁用/复盘 |
| 03 | 商品角色字典 | 10 类商品角色（项目级非永久身份）|
| 04 | 误判防线字典 | 11 类误判防线（含⑪负毛利/特价清仓）|
| 05 | 字段说明 | 57 字段口径/必填/来源/填写责任 |
| 06 | 复盘记录 | 7/15/30 天复盘滚动记录 |

## 2. 主表四分区（颜色区分）

- **顾问预填区（深蓝表头，1–33 列）**：诊断派生列（ABC/PSD/价格带/季节属性/特价清仓/角色等）由顾问预填；
- **动作裁决区（金色，34–44 列）**：建议动作/优先级/原因/证据/误判风险/陈列建议；
- **店长执行区（绿色，45–48 列）**：责任人/期限/状态/备注，店长维护；
- **复盘区（橙色，49–57 列）**：7/15/30 天结果与有效性。

> 建议分工：**顾问预填诊断与裁决列 → 店长只填执行+复盘列**，降低店长负担（判断列已设下拉）。

## 3. 下拉枚举（13 列，主表）

季节属性 / 是否特价清仓·负毛利异常 / ABC·ABCZ / 是否停购 / 商品角色 / 是否待验证需求 / 建议动作 / 动作优先级 / 误判风险 / 是否需要人工确认 / 是否进入DisplayMap调整 / 当前状态 / 是否动作有效。复盘记录表另设 4 列下拉（原建议动作/当前状态/是否动作有效/下一步动作）。

## 4. 使用流程（接交付物链）

```
商品诊断报告 → 顾问预填主表(1-44列) → 店长执行(45-48) → 周复盘(49-57 + Sheet06) → 结项报告
                                   └→ 是否进DisplayMap=是 → DisplayMap调整表
```

## 5. 关键纪律（来自模板定义，必须遵守）

- **淘汰**不得只按销量末位一刀切，须过 M-DEC-009 三查（缺货/季节/售罄率）；
- **引进**须"上新必腾位·一对一置换"，不得净增 SKU；
- **补货**是修正"该有货却缺货"，高库存不得触发补货；
- **待确认**须设补证据责任人 + 7 天内复核，不得长期悬空；
- **负毛利/特价清仓**先过误判防线⑪辨清仓，不得当正常低毛利放大；
- 陈列字段缺数据时，"是否进DisplayMap"选"待陈列数据"（表E/M-DEC-011 待装表）。

## 6. 边界

- 不填真实 SKU 明细 / 不导入真实 POS / 不含真实金额；
- 不使用宏、不链接外部数据；
- 实际项目使用时**先删除示例占位行**再填写。
