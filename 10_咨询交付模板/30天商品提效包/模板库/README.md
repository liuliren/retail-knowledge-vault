## 2. 统一口径说明（Pack-001 收口）

- **分档表达**："高/中/低/极低/积压/负毛利异常/待确认/不适用" 仅为脱敏与快速判断用，**不替代真实数值**；正式交付以真实数据为准。
- **引进品未在架**：销售/库存/PSD/ABC 等列填"不适用/待确认"，**不得虚构**；"是否停购"可留空或填"待确认"。
- **陈列缺数据**：缺陈列数据（表E/M-DEC-011）时统一标 **"待陈列数据"**，不强行给精确货位。
- **误判防线**：四件套**统一沿用 11 类**（见 [[SKU动作建议表_模板定义_v0.1]] §8），**不新增**。
- **SKU动作/商品角色**：统一沿用 10 类，**不新增**。

## 3. 分级口径归属表（三套分级不合并）

> 三套分级分属**不同环节**，**不得合并、不新增枚举**：

| 分级体系 | 取值 | 所属环节 | 主定义位置 |
|---|---|---|---|
| **诊断分级** | P0 / P1 / P2 / 待确认 | 商品诊断（问题·机会优先级）| [[商品诊断报告_模板定义_v0.1]] §12 |
| **复盘有效性** | 有效 / 部分有效 / 无效 / 待观察 / 未复盘 | 周复盘（动作执行效果）| [[周复盘表_模板定义_v0.1]] §14.8 |
| **结项闭环状态** | 已闭环 / 部分闭环 / 未闭环 / 待观察 / 转下一阶段 | 结项（问题/项目收口状态）| [[结项报告_模板定义_v0.1]] 字典 |

> 说明：诊断分级管"先做什么"，复盘有效性管"做了有没有用"，结项闭环管"最终收没收口"——三者是流水线上的不同刻度，互不替代。

## 4. 边界

只导航与口径收口，不新增方法论/模板/枚举；不改 Excel；不动真实数据。
