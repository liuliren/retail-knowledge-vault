# 社区超市30天商品提效包_工具与模板总索引

## 0. 文件定位

- 本文件是社区超市30天商品提效包的**工具与模板总索引**，**补充** [[社区超市30天商品提效包_交付物总清单_v0.1]]；
- **交付物总清单回答"有哪些交付物"，本文件回答"工作时怎么调用工具"**；
- 服务**顾问、店长、采购、店员和内部团队**；
- **不是新方法论、不新增工具、不替代具体模板**。

## 1. 使用原则

1. **先按阶段找工具**（项目前/首访/诊断/动作/门店执行/采购问货/复盘/结项）；
2. **再按使用对象确认谁用**（顾问/店长/采购/主管/店员/客户老板）；
3. **最后看输入 / 输出**；
4. xlsx 工具如被 .gitignore，**只作为本地工具使用**（gitignored-local）；
5. **可询价清单不是正式采购单**；
6. **客户版和内部版不能混用**；
7. **没有真实门店反馈前，不继续扩模板**。

## 4. 30天商品提效包完整交付路径

1. **客户启动**：使用 `一页纸销售单页` / `对外销售页预览` / `客户首访介绍话术` / `30天项目启动确认单`
2. **资料采集**：使用 `商品问题初筛资料清单` → `商品问题初筛判断表`
3. **商品诊断**：使用 `商品诊断报告 模板定义 + 内部版Word`（陈列侧 `DisplayMap调整表`）
4. **SKU动作建议**：使用 `SKU动作建议表 模板定义 + Excel工具版`
5. **商品机会库**：使用 `商品机会库 最小字段设计` → `饼干/冲调样板品类机会库`
6. **采购问货**：使用 `饼干真实货盘回填`（可询价清单）+ 五态机制
7. **一线执行卡**：使用 `一线员工版执行卡` / `一线执行工具 Excel`
8. **RACI分派**：使用 `门店商品动作RACI`
9. **7天追踪**：使用 `一线执行工具 Excel·7天追踪sheet`
10. **周复盘**：使用 `周复盘表 模板定义 + Excel工具版`
11. **结项报告**：使用 `结项报告 模板定义 + 内部版Word`（客户版按 `报告类Word客户版PDF规划`）
12. **后续陪跑**：使用 `私域获客发布包 精修版`（复购话术待补）

## 5. 门店一线执行路径

1. **顾问生成动作**：`SKU动作建议表`（10类动作+商品角色+误判防线）
2. **店长拆成今日动作**：`一线执行工具 Excel·今日动作执行卡sheet`（门店语言对照辅助）
3. **店员执行**：`今日动作执行卡`（做什么/在哪/做完标准）
4. **主管/店长验收**：`门店商品动作RACI` + 执行卡验收状态 + 拍照
5. **7天追踪**：`一线执行工具 Excel·7天追踪sheet`（✔/未/暂缓）
6. **周复盘**：`周复盘表`（有效/无效判定）
7. **下周动作**：回到 `SKU动作建议表`

## 6. 商品供给路径

1. **诊断发现缺口**：`商品诊断报告` → `SKU动作"引进/替换"`
2. **商品机会库查候选**：`商品机会库通用字段` → `饼干/冲调样板`
3. **真实货盘回填**：`饼干真实货盘回填`（16条真实品牌）
4. **采购问货**：可询价清单 A组(现有经销商)/B组(进口节令)/C组(区域)
5. **状态回写**：五态机制 `待采购确认 → 现有供应商可供`
6. **进入SKU动作**：**仅"现有供应商可供"才进引进** → `SKU动作建议表`
7. **门店执行**：`执行卡"新货到了上架"`

> **铁律**：`饼干真实货盘回填`是**可询价清单，不是正式采购单**；**只有采购确认"现有供应商可供"后才能进入 SKU 动作**；**未确认项留在机会库，不下发给店员**。
