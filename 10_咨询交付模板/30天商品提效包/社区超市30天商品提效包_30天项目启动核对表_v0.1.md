# 社区超市30天商品提效包｜30天项目启动核对表（内部）

> ⚠ **顾问内部使用，不发客户。** 客户确认用 [[社区超市30天商品提效包_30天项目启动确认单_v0.1]]（client_shareable）。本表承接初筛判断表结论，做启动前最后把关。

## 1. 文件定位

正式启动 30 天项目前，核对客户是否具备进入条件，识别数据、执行、范围、预期、复盘风险。承接 [[社区超市30天商品提效包_商品问题初筛判断表_v0.1]] 结论，是"初筛通过 → 启动 → 商品诊断"的内部把关环节。

## 3. 风险核查
| 风险 | 有/无 | 应对 |
|---|---|---|
| 数据不足 | ☐ | 列必补项 |
| 店长不配合 | ☐ | 启动前明确执行人 |
| 老板预期过高 | ☐ | 重申边界/不承诺业绩 |
| 品类范围过大 | ☐ | 收敛到样板品类 |
| 项目周期过短 | ☐ | 评估延 4–8 周 |
| 价格 / 费用未确认 | ☐ | 启动前定 |
| 现场拍照受限 | ☐ | 评估对诊断影响 |
| 没有复盘时间 | ☐ | 约定固定复盘窗口 |
| 客户只想听课不想执行 | ☐ | 暂缓/婉拒 |
| 客户要求保证业绩 | ☐ | 重申不承诺，必要时暂缓 |
