# 04_老板看板与凭证模板（占位）

## 目录定位

本子目录沉淀**项目结束后持续给老板看的"经营仪表 + 凭证物"**——是花厅坊样板"对外可讲"的物理载体。与 08_经营分析与数据看板/ 联动（08_ 沉淀方法论与指标体系，本目录沉淀客户可见模板）。

属于"销售素材化 + 复购拉动"的关键凭证层：没有这一层，方法论就停留在"已交付"，无法升级到"持续陪跑"和"对外讲述"。

---

## 适合存放（Phase 1+ 充实）

- 老板月度经营看板模板（销售/毛利/客流/客单/库存/活动效果一页纸）
- 改造前后对比凭证模板（陈列前后照 / 数据前后对比 / 顾客反馈）
- 项目对外讲述凭证模板（六哥可发布的去标识化案例素材包）
- 周度运营仪表模板（店长用）
- 季度经营复盘模板（陪跑期）
- 阶段性 ROI 凭证模板（对外销售时的"证据物"）

---

## 不适合存放

- 内部诊断过程稿（→ 02_交付SOP/）
- 客户阶段性报告（→ 03_客户报告模板/）
- 自媒体成品（→ 12_自媒体内容与表达转化）
