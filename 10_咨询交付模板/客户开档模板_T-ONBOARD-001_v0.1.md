# 附：决策点 2 裁决单（客户档案落位·待六哥）

**问题**：客户档案正本放哪？当前 09_（案例与复盘）与 16_（客户与战役档案）并存，花厅坊两处都有件。

| 选项 | 方案 | 代价 |
|---|---|---|
| A（推荐） | **16_ = 战役指挥层**（章程/里程碑/进展/M-DEC），**09_ = 交付与复盘层**（诊断/数据/作业包/AAR）——现状即如此，把这个分工**写成规则**而不是搬家 | 零搬家成本；需在两目录 README 各写一行边界声明 |
| B | 16_ 为唯一正本，09_ 只留复盘 | 花厅坊 03_商品诊断 百余件要搬，断链风险大 |
| C | 09_ 为唯一正本，16_ 降指针 | 战役章程混入交付件堆，指挥层变厚 |

*推荐 A：按"指挥/交付"分层而非"客户"分目录，与既有实践零冲突。你回"决策点2 选A/B/C"即生效。*
