# 分时段巡店检查表·通用模板

## 核心逻辑：按"时段顾客需求"匹配检查重点

不同时段的核心客群和需求不同，检查重点也应不同：

| 时段 | 典型客群 | 核心需求 |
|---|---|---|
| 早市(约7:00-9:00) | 家庭采购者 | 新鲜足量、家庭适配、方便挑拣 |
| 午市(约11:00-13:30) | 备餐补充者 | 快速拿取、即时可用、无残次 |
| 晚市(约17:00-20:00) | 便捷小份/清货顾客 | 小份包装、合理折扣、新鲜可食 |

## 通用巡检结构（按品类填入具体判据）

| 检查维度 | 早市检查要点(到货初检·防源头问题) | 午市检查要点(中途维护·防品质下滑) | 晚市检查要点(清货把关·防变质售卖) | 实际情况 | 问题等级 | 整改动作 |
|---|---|---|---|---|---|---|
| 商品品质 | [到货是否有损/虫蛀/异味] | [货架陈列是否有压伤/腐坏迹象] | [剩余商品是否变质/异味，是否标记"次日不售"] | □达标 □不达标 | □紧急1h内 □重要当日 □一般3日内 | |
| 陈列量感 | [是否开满/堆满] | [是否有明显空缺需补货] | [是否已清理不合格品] | | | |
| 价格标识 | [价签是否齐全清晰] | [价签是否与实际商品一致] | [特价商品是否已收尾/更新] | | | |
| 卫生 | [操作台/地面/通道是否清洁] | [同上] | [收档前是否已清洁] | | | |

## 生鲜品类具体判据示例（水果区）

- **早班8:00前共同任务**：全体人员协作把台面商品加满，量感陈列型商品(如西瓜)按"开满"标准——覆盖整个堆头区域，排列紧密无明显空隙。
- **风幕柜/冷藏区**：逐件检查生产日期/保质期，临期商品(距过期不足1/3保质期)单独标识；变质/损坏品(霉点/软烂/异味)立即下架并记录数量。
- **堆头维护**：整理堆头并及时加货，呈金字塔形或阶梯状陈列，顶层商品倾斜角度一致(约30°)，底层稳固不晃动，加货后堆头高度不低于1.2米(特殊堆头除外)。
- **果切台卫生**：台面无果汁残留/无果肉碎屑，刀具砧板每日消毒(75%酒精擦拭)，操作区地面无积水。
- **12:00前二次上货**：确保11点后货架满货，根据生意状况灵活确定上货数量。

## 生鲜品类具体判据示例（蔬菜区）

- **叶菜类**：到货无黄叶/虫蛀(如上海青单株完整率≥98%)；货架无压伤(堆叠高度≤8cm，避免中层菜压烂)；补水后无烂叶(雾化区喷水后10分钟查叶片状态)；剩余叶菜无黏连/异味，未售完标记"次日不售"。
- **根茎类**：到货无发芽/霉变(抽样10个，不达标≤1个)；表皮无泥土结块；剩余根茎无软化(捏着硬实)。
- **损耗目标参考**：叶菜≤7%、根茎≤3%、瓜茄≤4%(需按门店实际客群/商圈调整，不可直接套用)。
- **陈列量感判据**：是否切根/是否摘黄/是否剥皮/是否露底(商品堆得像小山看不见筐底)/陈列颜色是否跳色(绿-白-绿-紫错开摆放，避免一片全绿分不清品类)。

## 使用说明

1. 本表为**通用结构+生鲜示例**，交付新客户时需按其实际经营品类(如休食/日化/水饮)重新填入对应的具体判据。
2. 损耗率/达标率等数值目标仅为参考区间，需结合门店实际商圈客群、历史数据校准，不可直接套用。
3. 建议与[[月度门店复盘表_红绿灯制_v0.1]]配合使用——巡店表管"每日执行是否到位"，月度复盘表管"整体经营指标是否达标"。
