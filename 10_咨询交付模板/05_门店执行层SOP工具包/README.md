# 05_门店执行层SOP工具包

## 定位

`00_战役SOP` 和 `02_交付SOP` 都是**咨询顾问侧**的工作流程(怎么诊断、怎么调改、怎么复盘)。本文件夹是**门店员工侧**的日常操作工具——巡店检查/日检/月度复盘，交给客户店长和一线员工每天用，不是给顾问自己用的。

这块此前是vault的一个明确空白：诊断方法论很厚，但"诊断做完了，日常怎么维护"这层执行工具几乎没有。电脑备份深度精读工程中反复发现六哥手上有真实在用的版本(水果组/蔬菜组/休食区各自的巡店表，多个版本并存)，本次整理去重，提炼成可复用的标准化模板。

## 文件清单

- `分时段巡店检查表_通用模板_v0.1.md` —— 按早/中/晚市分时段的门店巡检工具，含生鲜(水果/蔬菜)具体判据示例
- `月度门店复盘表_红绿灯制_v0.1.md` —— 三大核心指标(销售额/毛利率/损耗率)达标判定 + 亮点/问题结构化记录
- `月度采购商品规划表_角色分工法_v0.1.md` —— 按"引流款/赚钱款/稳基本盘"三分角色法做品类月度企划

## 使用说明

这三份模板均为**通用化提炼**，不是照搬某一家门店的原始文件——原始文件是特定品类(水果/蔬菜/休食)的具体版本，本工具包提炼了背后的共同结构，交付给新客户时需按其实际品类结构做适配。
