# 月度采购商品规划表·角色分工法

## 用途

诊断做完了，采购/店长具体每月该怎么定单品策略？本表提供一个可直接套用的**三分角色企划法**，与[[01_科学零售方法论/价格带管理]]的红绿灯定价法是同一套逻辑在"采购月度企划"层面的具体应用。

## 核心逻辑：三分角色法

| 角色 | 定价策略 | 考核标准 | 战术要点 |
|---|---|---|---|
| **引流的**——"主要看气质" | 负毛利/零毛利，低于竞对/农贸市场20%以上 | 不考毛利额，只考销售吨位和客单量 | 当季量大、价格敏感度极高的商品；须用来带动连带消费(如买萝卜带牛腩) |
| **赚钱的**——"主要看产出" | 高毛利(30%-50%) | 考核毛利额和库存周转 | 顾客对价格不敏感的小众/特色商品；毛利要赚足 |
| **提升销售额与销量的**——"稳住基本盘" | 混合毛利(15%-20%)，随行就市略低于竞对即可 | 考核售罄率(避免过夜报损) | 时令刚需商品，重点在新鲜度 |

> 与[[01_科学零售方法论/价格带管理]]的对应关系：引流款≈红灯区(KVI)，赚钱款≈绿灯区(利润品)，稳基本盘款≈黄灯区(结构品)——采购月度企划是价格带三色分区在"具体挑哪些单品、怎么卖"层面的落地。

## 表格结构

### 第一部分：本月主推商品(核心商品企划)

> 由店长、主管和采购联合制定，采购填写

| 角色 | 选定商品(具体到单品) | 怎么卖？(一句话战术) |
|---|---|---|
| 引流款 | 商品名：__________（选择理由：____） | |
| 赚钱款 | 商品名：__________（选择理由：____） | |
| 稳基本盘款 | 商品名：__________（选择理由：____） | |

### 第二部分：陈列与动线配套建议

- 主堆布局：不同角色商品的视觉搭配（如色彩对撞、关联陈列）
- 动态调整机制：早市/晚市双峰陈列——早市(约7:00-10:00)量感陈列主推引流款满足计划性采购客群；晚市(约17:00-19:00)切换净菜/快手菜类满足下班便捷采购客群

## 实战参照案例（沙埔大道店蔬菜规划·六哥教练案例）

2026年1月，六哥对沙埔大道店某员工的蔬菜品类规划做出教练反馈，把手写经验方案升级为本表结构：
- 引流款：白萝卜/莴笋(当季量大、价格敏感度极高)——"负毛利"或"零毛利"，不控量上货，用于带动连带消费(买萝卜带牛腩、买莴笋带腊肉)
- 赚钱款：本地菜(小众特色)/配菜(姜蒜/小米辣/香菜)——顾客对价格不敏感(差5毛无所谓)，毛利率30%-50%
- 稳基本盘款：红菜心/白菜苔/辣妹子椒/茼蒿/宁夏菜心(火锅季刚需)——混合毛利15%-20%，随行就市略低于竞对，重点在新鲜度、考核售罄率

## 使用说明

本表需与采购/店长/主管三方联合填写，不是采购单方面拍板——引流款/赚钱款的选择理由需体现对当地客群和商圈的理解，"一句话战术"栏是本表最容易被简化跳过、但实际最重要的部分。
