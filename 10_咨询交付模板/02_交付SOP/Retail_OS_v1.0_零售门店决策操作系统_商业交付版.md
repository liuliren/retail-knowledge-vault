# 1. 一页门店执行手册（打印·贴墙版）

## 📋 商品决策 5 步（照做）
| 步 | 做什么 | 判断 |
|---|---|---|
| **1 扫数据** | 看库存/SKU/价格 | 滞销(有货>60天没卖)? 同类挤太多? 中间价位>70%? |
| **2 看位置** | 看陈列 | 卖最好的在黄金位吗? 黄金位有卖不动的吗? |
| **3 分ABC** | 按销售额排 | 累计≤60%=A / 60-90%=B / >90%=C |
| **4 问4问** | 砍之前必答 | (见下4问,任一"是"=不准砍) |
| **5 出动作** | 执行 | 上/下架 · 上/下移 · 补货/清仓 · 保留/替换 |

## 🅰️ ABC 规则
- **A类(累计销额≤60%)** → 保库存 + 上黄金位,**不准断货**;
- **B类(60-90%)** → 观察,挑卖得越来越好的往A推;
- **C类(>90%)** → 进"误判4问",过了才能动。

## ⚠️ 误判 4 问（砍 SKU 前必答 · 任一"是"=禁止淘汰）
1. 是**需求**吗?（儿童/健康/送礼/节日 = 低频有需求 → 留）
2. 是**结构**吗?（中价挤压/同类太多 = 调结构不砍）
3. 是**位置**吗?（在死角没人看见 → 先上移）
4. 是**缺货**吗?（货架空导致没销量 → 补货不砍）
> **4问全"否" + 真有货积压 = 才能砍。**

## 🏆 陈列黄金位规则（D）
- A类(卖最好)→ 货架**右上黄金位**(视线 0.6-1.8米),给更多排面;
- C类滞销 → **下移底层**,排面 5→3→2→1 递减,仍差让位;
- 同类多口味 → **集中放一起**,留卖最好的 1-2 个;
- 死角里的A类 → **立即上移**主通道/端架。

## 💰 价格带规则（C）
- 中间价位 SKU 占比 **>70% → 必须重构**（砍中价冗余/补两端）;
- 每个小类**必须有 ≥1 个超低价走量品**;
- 高价品**必须有用途锚点**(送礼/形象),否则不留;
- **低价补不了结构缺失**(缺强单品 ≠ 靠低价凑)。

# 2. SKU 决策工具包（可执行工具）

## 🛠 Tool 1｜SKU 裁决表（填表执行）
| SKU(品名) | ABC | 是否保留 | 是否淘汰 | 误判类型(需求/结构/位置/缺货) |
|---|---|---|---|---|
| | | | | |
| | | | | |
> 填法:先填ABC→A全保留;C填误判类型,4问全否才填"淘汰"。

## 🛠 Tool 2｜滞销处理流程卡（5 道闸）
```
滞销品(有货 >60天 没卖出)
 ① 货架空吗?(缺货) ──是→ 补货,不淘汰 ✗
 ② 售价<进价?(负毛利) ──是→ 降价清尾(特价清仓)
 ③ 儿童/健康/送礼/节日? ──是→ 保留(需求点)
 ④ 在死角没被看见? ──是→ 先上移黄金位观察
 ⑤ 以上全否 + 真积压 ──→ 淘汰/清仓
```
**缺货 vs 滞销**:货架空=缺货(补货);有货没卖=滞销(走5闸)。
**误杀防护**:任何C类砍前过①③④三闸+误判4问。

## 🛠 Tool 3｜陈列调整 SOP 卡
| 规则 | 动作 |
|---|---|
| 黄金位(A类) | A类必须上右上黄金位(0.6-1.8米),多排面 |
| 下移(C类) | C类滞销禁占黄金位→下移底层(验证期除外) |
| 同质压缩 | 同类多口味集中,留动销最好1-2,其余撤 |
| 动线优先 | 高频走量品前置端架/主通道;死角A类上移 |

# 3. 门店复制模型（如何复制到任意门店）

## 🚀 导入步骤（1-2-3）
1. **拉数据**:门店导出近90天商品销售明细 + 当前库存;
2. **跑诊断**:按"初次诊断30分钟流程"出ABC+滞销+价格带;
3. **落动作**:按 SKU裁决表 + 陈列SOP卡 执行,贴出一页纸手册。

## 📊 数据要求（最低 POS 数据）
- 必需:商品名 · 销售数量 · 销售金额 · 销售日期 · 现有库存;
- 选填:进价(算毛利) · 货位(算陈列) · 类别(分小类);
- **够用底线**:有"销售数量+销售日期+现有库存"即可开跑。

## ⏱ 执行节奏（7天落地）
| 天 | 动作 |
|---|---|
| D1 | 拉数据 + 30分钟初诊 |
| D2 | 出ABC + 滞销清单 + 价格带诊断 |
| D3 | 滞销走5闸 + 误判4问,定淘汰/保留清单 |
| D4 | 陈列调整(A上黄金位/C下移/同质压缩) |
| D5 | 价格带校正(砍中价冗余/补超低价走量) |
| D6 | 贴一页纸手册 + 培训店员照做 |
| D7 | 复盘:动销/缺货/黄金位是否改善,微调 |

## 🔍 初次诊断 30 分钟流程
1. (5min) 算ABC:销售额降序,累计≤60%=A/60-90%=B/>90%=C;
2. (5min) 拉滞销:有货 >60天没卖的清单;
3. (5min) 算中价占比:中间价位SKU数÷总数,看是否>70%;
4. (5min) 扫黄金位:A类在不在黄金位/黄金位有没有卖不动的;
5. (10min) 滞销走5闸+误判4问→出"淘汰/保留/上移/清尾"四清单。
> 产出:一张ABC表 + 一张四清单,当场可执行。
