# SOP_6机位拍摄法_v0.5

## 1. SOP 定位

- 本文件是"6机位拍摄法"的独立主定义。
- 用于门店陈列、货架、调改前后对比、DisplayMap 复核等场景。
- 本 SOP 从花厅坊项目二访 checklist §6 抽出，当前为 v0.5 验证版。
- 后续经多个项目验证后可升级为 v1.0（晋级路径参 [[2026-05-12_vault_v3.0顶层结构设计稿_v0.1]] §7.3）。

## 2. 适用场景

- 门店陈列现状采集
- 调改前拍摄
- 调改后 24h / 72h / 7 天复拍
- DisplayMap 还原
- 货架组位复核
- 商品层位诊断
- 客户交付报告取证

## 3. 6 机位定义

必须采用以下定义，**不要改写成其他编号**。

| 机位 | 编号 | 位置 | 距离 | 高度 / 对准 |
|---|---|---|---|---|
| 1 | M1 全幅 | 距货架 ≥ 2m | 远 | 正对中心 |
| 2 | M2 左段 | 距约 1m | 近 | 正对左 1/3 |
| 3 | M3 中段 | 距约 1m | 近 | 正对中 1/3 |
| 4 | M4 右段 | 距约 1m | 近 | 正对右 1/3 |
| 5 | M5 第 2 层 | 距约 1m | 近 | 平视约 1.2m，黄金手拿层 |
| 6 | M6 第 3 层 | 距约 1m | 近 | 平视约 1.5m，黄金视线层 |

> **层号方向**：本表 M5/M6 的"第 2 层 / 第 3 层"采用 [[2026-05-06_门店陈列照片命名规范_v0.3]] §6 新层号体系（**1 = 顶 / 末 = 底**）。注：来源文件 `2026-05-05_二访出门checklist_v0.1.md` §6 使用旧层号（1 = 底 / 末 = 顶 / v0.2 体系），方向相反。详参 §9 待确认。

## 4. 拍摄动作标准

- **M1**：用于完整货架 / 组位还原。距 ≥ 2m / 正对中心 / 整面入框 / 不允许局部裁切。
- **M2 / M3 / M4**：用于局部商品、价签、陈列细节。距约 1m / 分别正对左 1/3 / 中 1/3 / 右 1/3。
- **M5 / M6**：用于黄金层位、主推商品、视觉陈列判断。距约 1m / 平视高度 / 黄金手拿层 + 黄金视线层。
- **同一组货架应尽量保持站位、角度、高度一致**：以胶布在地面标记 6 机位为准。
- **复拍时必须尽量复用同一机位**：调改前 + 调改后 + 24h / 72h / 7 天复拍 / 全部走同 6 机位 / 同高度 / 同距离。

## 5. 与照片命名规范的关系

- **6 机位编号使用 M1-M6**（不允许 M01 / M-1 / 机位 1 等变体）。
- **M1-M6 是拍摄机位 / 不是货架编号**。货架编号属 G 字段（如 G3-A）。
- **G 货架编号如 G3-A，不得与 M 机位混用**。
- **照片命名仍以 [[2026-05-06_门店陈列照片命名规范_v0.3]] 为准**。
- **现场照片命名遵循**：字段内部用 `-`，字段之间用 `_`（参照命名规范 v0.4 §7.2）。

可写示例：

```text
HS-G3-A-04-3-L冲调-黄金视线_商品价签_20260513_常规.jpg
```

> **注意**：这个示例体现的是照片命名规范，不代表 M1-M6 要写入每一张照片名。是否把 M1-M6 写入内容字段或备注字段，后续需要在照片命名规范 v1.0 中统一确认，**当前标记为"待确认"**（参 §9）。

## 6. 现场执行清单

- [ ] 每组货架至少拍 M1 全幅
- [ ] 重点货架补拍 M2 / M3 / M4
- [ ] 黄金层位补拍 M5 / M6
- [ ] 调改前后复拍尽量保持同一站位
- [ ] 同一货架照片命名必须能回到 G 货架、组位、层位
- [ ] 不确定机位或层位时，先标记待确认，不自行编造
- [ ] 现场用胶布标记 6 机位地面位置（一组货架一组胶布）
- [ ] 拍摄前先确认时段锁定值（夜灯前 / 夜灯后 / 次晨白光 / 常规）
- [ ] 拍摄前先确认 L 主品类锁定值（参命名规范 v0.3 §4）

## 7. 常见错误

- 把 M1-M6 当成货架编号；
- 只拍局部，不拍全幅；
- 只拍商品，不拍价签；
- 调改前后站位不同，无法对比；
- 货架编号 G 与机位编号 M 混用；
- 用 AM / PM 等非锁定时段；
- 自创内容字段、层位语义或主品类字段；
- 漏拍 M5 / M6 黄金层位（最高商业价值层）；
- 调改后未在 24h 内复拍（参 命名规范 v0.3 §10 4 时段）；
- M5 / M6 高度与命名规范 v0.3 §6.4 物理高度不一致（黄金视线 1.6m / 黄金手拿 1.2m）。
