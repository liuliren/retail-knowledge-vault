# Retail OS v2.0｜选品/引进层（商业交付版·加法层）

> v1.0 答"**砍谁**"，本层答"**上谁**"。两层合成 **"汰旧上新"攻防完整系统**。
> 拿到即用 · 无需解释 · 直接执行。交付方：晟果新零售。
> 验证基础：花厅坊饼干(M3) + G4-A麦片货架全门槛回测 + 沙埔大道店跨店验证（规则跨两店重现专家决策）。

# 1. 一页门店执行手册（打印·贴墙版）

## 📋 选品 5 步（照做）
| 步 | 做什么 | 判断 |
|---|---|---|
| **1 找缺口** | 看缺什么 | 缺需求? 缺价格带(空档)? 缺强单品? 竞品有我没有? |
| **2 过门槛** | 4 闸必答 | 真需求? 有位置(汰谁腾)? 算得过来(PSD+毛利)? 供得上? |
| **3 定数量** | 算合理SKU数 | A品数×系数;头部弱/尾部厚→**补缺必腾位,净数不增** |
| **4 定位置价** | 放哪+定价 | 先观察位;只补空档档,**不补已密的档** |
| **5 上+观察** | 执行 | 上新+汰旧腾位;**30天观察**→达标转常规/不达标即汰 |

## 🚦 引进门槛 4 闸（上新前必答 · 任一"否"=先别上）
1. **真需求?**（填补需求单元,不跟风;先比对A类已有,别补已经强的）
2. **有位置?**（货架有空或换掉一个"双低"SKU → **上新必腾位**）
3. **算得过来?**（双信号：**PSD 定该不该补 · 毛利定补什么款**；高效率低毛利的档→补品挑高毛利,毛利红线 **≥25%**）
4. **供得上?**（可得性=**在购 + 近期有进货 + 采购规模 + 是否本地** 四代理判高/中/低；**不依赖采购周期**——多数门店管理水平不足、该字段不可靠；小众特色品重点核）

## ⚖️ 选品 5 条铁律
- **选品-01 缺口驱动**：先看缺什么再上,不跟风。
- **选品-02 上新必腾位**：上一个先汰一个双低,货架空间守恒。
- **选品-03 试销定生死**：小批试 + 30 天观察,达标转常规,不达标即汰。
- **选品-04 补空档不补过密**：只补价格带空档与缺失强单品;已密的档不补。
- **选品-05 双信号补品**：PSD 定"该不该补",毛利定"补什么款";高效率低毛利档要挑高毛利款。

# 3. 与 v1.0 的闭环（汰旧上新）
```
选品层(加法)·补缺 ──上新──▶ 货架
   ▲ 汰旧腾位                  │ 30天观察
   │(v1.0砍双低/清滞提供腾位)   ▼
v1.0汰换(减法) ◀──不达标──── ABC体检(v1.1双维)
```
> 加法 + 减法 = **门店 SKU 自我更新循环**：补缺靠汰旧腾位,新品 30 天后进 ABC 体检。

# 4. 门店复制模型

## 🚀 导入步骤（1-2-3）
1. **拉数据**：销售明细 + 库存 + **进价/毛利** + **主供应商**;
2. **找缺口**：按 Tool 2 出每小类缺口清单;
3. **过门槛上新**：按 Tool 1 打分,P0 先上 30 天试销,挂腾位。

## 📊 数据要求（选品层最低数据）
- 必需：商品名 · 销量 · 销售额 · 库存 · **进价/毛利率**(算得过来闸);
- **关键**：**主供应商 + 近90天进货 + 停购状态 + 供应商区域**(供得上闸·四代理判可得性);
- ⚠ **不依赖采购周期/MOQ**——低管理成熟度门店给不准，用上述代理替代;
- 选填：竞品对标数据。

## ⏱ 执行节奏
- P0 方向先上（30 天试销）→ 达标转常规进 ABC / 不达标走 v1.0 汰换；
- 每补 1 项必挂 1 个腾位 SKU（空间守恒）。

# 5. 产品总结

**Retail OS v2.0｜选品/引进层（加法层）**
- 把选品决策压成 **5步 + 门槛4闸 + 5铁律**;三工具填表即执行;
- 核心防线：**上新必腾位**(空间守恒) + **双信号补品**(PSD该不该补/毛利补什么款);
- 配 v1.0 减法层 = **"汰旧上新"攻防完整系统**;
- 验证：**三轮实跑**(花厅坊饼干 + G4-A麦片全门槛回测 + 沙埔大道店跨店),规则跨两店重现专家决策;
- 数据门槛：须补**进价/毛利 + 主供应商**;
- 适用：社区/便利/生鲜超市的商品力"加法"提升;可培训/可复制/可咨询交付/可售卖;
- 交付方：晟果新零售。

## 治理
- 选品 Cluster 与 M-DEC 编号待六哥分配（candidate）;不擅扩 v1.0 的 A-G;
- "供得上"闸须门店补主供应商数据后全闭环;
- 候选具体单品由采购按方向 + 毛利目标填实,本交付件给规则与方向,不指定品牌。
