## SOP 通用结构（A-G 7 段标准 + §H/§I/§J 附加段）

每个 S/G 系列 SOP 固定如下骨架（v0.1 占位仅含 §0 元数据 + §1 战役定位 + §C DoD 占位 + 各段标题骨架；其余段在 Phase 1+ 实战后回流）：

```
§0  SOP 元数据（含 quadrant / client_safety / 关联仪表）
§1  战役定位
§A  入场条件 (Entry Criteria)
§B  关键里程碑 (Milestones)
§C  必产出物与 DoD (Deliverables & Definition of Done)
§D  风险红线 (Risk Lines)
§E  人决策点 (Human Judgment Gates)
§F  出场条件 (Exit Criteria)
§G  责任分工 (RACI)
§H  客户可见性 (client_safety)
§I  已知陷阱（历史翻车记录）
§J  关联（上游 / 下游 / 案例 / 复盘）
```

详见 [[99_归档/2026-05-07_新基线封存/90_素材_全量/2026-05-04_知识库重构方案_v0.2_完整作战方案|2026-05-04_知识库重构方案_v0.2_完整作战方案（已归档）]] §5.1。
