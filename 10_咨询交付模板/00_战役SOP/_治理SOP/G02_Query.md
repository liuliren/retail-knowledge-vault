## 1. 战役定位

vault 与 Claude Code 协作的**第二个一等公民操作**——"知识被用"。本 SOP 把 [[CLAUDE.md]] §18.2 的简明流程落地为可执行步骤。

**核心约束**：
- 回答必须可追溯到具体依据文件 / 行号 / 段落
- 不得只凭"我记得"
- 正式结论前必须列依据（[[CLAUDE.md]] §13.10）
- 探索结果如有沉淀价值，**归档为综合主页**（不是堆 90_）
