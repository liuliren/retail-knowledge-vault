## 2. T/CCFA 039-2023《便利店商品分类》六位码分类体系详解

### 2.1 分类逻辑与编码规则（标准3.1节原文）

- 商品按属性分为**大类、中类、小类**三级。
- 小类编号为**六位数字**：前两位=大类编号(00-99)，中间两位=中类编号(00-99)，末两位=小类编号(00-99)。
- 编码是"大类前缀累加"式结构，如 `010101`＝01(个人洗护用品)+01(洗发沐浴)+01(洗发水)，中类编号本身写作四位（如0101），小类编号写作六位（如010101）——**编码位数与层级强绑定，无预留扩展位**（大类下中类若超过99个即无法编码，实务中不会触及但结构上无弹性空间）。

### 2.2 大类框架（共11个大类）

| 大类 | 名称 | 中类数（略举） |
|---|---|---|
| 01 | 个人洗护用品 | 洗发沐浴/洁面护肤/口腔护理/个人用品/婴儿用品/其他 |
| 02 | 生活用品 | 家居用品/电子产品/文娱用品/应急用品/其他用品 |
| 03 | 酒类 | 啤酒/白酒/葡萄酒/其他(果酒黄酒洋酒) |
| 04 | 烟及相关品 | 香烟/电子烟/打火机/其他 |
| 05 | 休闲食品 | 糖果巧克力/饼干/休闲零食/其他 |
| 06 | 便利食品 | 罐头/方便食品/食品杂货/冷冻品/婴儿食品/礼盒/其他 |
| 07 | 饮品 | 水&饮料/牛奶及制品/冲调品 |
| 08 | 快餐食品 | 面包/蛋糕甜点/面类粉类/米饭类/即食菜肴/现制现调饮品/现场制售热餐/柜台商品(关东煮煮蛋蒸包烤制品炸品)/水果/其他 |
| 09 | 服务性商品 | 书刊杂志/宠物用品/礼品卡/季节性商品/其他 |
| 10 | 医药品 | 计生用品/非处方药品/医疗用品/其他 |
| 11 | 博彩 | 机打彩票/即开型彩票 |

### 2.3 起草单位画像

物美（大卖场/综合超市代表）+ 美宜佳（便利店代表，全国门店数最大的连锁便利店品牌之一）联合起草，中国连锁经营协会归口。这决定了这套分类体系天然是**"便利店场景优先"**设计：11大类中，08(快餐食品，含关东煮/煮蛋/蒸包/烤制品/热餐)、11(博彩)、04(烟及相关品含电子烟)是便利店的核心差异化品类，在社区生鲜超市里要么完全不存在（博彩），要么占比极低且形态不同（快餐现制热餐）。

## 3.《商品学基础》教材节选（第3章·商品质量）核心内容

节选覆盖 3.1-3.4 四节，核心命题：

1. **商品质量的两层含义**：狭义=自然质量（商品本身的物理/化学/生物属性，如外观/内在质量），广义=市场质量（自然质量+售前售中售后服务质量）。
2. **质量构成四维度**：内在质量（适用性/可靠性/寿命/安全卫生）、外在质量（构造/色彩/气味/手感/包装）、社会质量（是否违反社会道德、环保）、经济质量（性价比）。
3. **各品类质量基本要求**：食品(无毒无害/营养价值/色香味形)、日用工业品(适用耐用卫生安全/结构合理)、纺织品(材料适宜/机械性/服用性)、进出口商品(专项要求)。
4. **假冒伪劣商品**：概念特征+"三鹿事件""毒大米"等案例、危害与四种典型形式（假冒商标/仿冒商标/假冒包装装潢/以次充好等）。
5. **全面质量管理(TQM)**：质量第一顾客至上、以防为主、以数据为依据、PDCA循环；附相关法规清单（产品质量法/消费者权益保护法/食品卫生法/价格法/商标法/广告法，均为1990s法规版本，**已严重滞后**，如"食品卫生法"已被2009年《食品安全法》取代）。

**这一章与"商品分类"或"选品/品类管理"方法论几乎无交集**，是通识性的质量管理/消费者保护内容，价值定位应是"质量意识/合规意识背景读物"，而非品类系统的技术依据。

## 4. 对花厅坊/社区生鲜超市场景的适用性评估

### 4.1 团体标准（T/CCFA 039-2023 便利店六位码）

**可直接借用的部分**：
- 01 个人洗护用品、02 生活用品（去除应急用品/文娱用品等低频类）——社区超市非食品区与便利店高度重叠，六位码结构可直接套用或仅需微调小类。
- 03 酒类、04(烟)——酒类框架（啤酒/白酒/葡萄酒/其他）可直接借用；烟草因专营资质问题，多数社区生鲜超市不经营，此大类对花厅坊基本不适用。
- 07 饮品——水/饮料/牛奶及制品/冲调品框架可直接借用，中类粒度合适。
- 09 服务性商品（宠物用品/礼品卡）——可选择性借用，季节性商品/书刊杂志类目在社区生鲜超市里权重很低。

**需要改造的部分**：
- **05 休闲食品 + 06 便利食品**——这两大类是六位码体系里粒度最细、结构最完整的部分，但设计逻辑是"即时消费、小包装、高周转"的便利店逻辑；社区生鲜超市这两大类需要按"计划性囤货消费"逻辑重新切分（如米面粮油在社区超市常作为独立大类而非06下的一个小类"060303"）。
- **08 快餐食品**——柜台商品(关东煮/煮蛋/蒸包/烤制品/炸品)、现场制售热餐(盒饭/现煮面)是典型便利店现制餐饮品类，社区生鲜超市极少经营此类现制热食柜台，此大类**基本不适用**，需整体剔除或替换为熟食/卤味等生鲜延伸品类。
- **11 博彩**——完全不适用，社区生鲜超市不具备彩票销售资质/场景。

**完全缺失、需要另起炉灶的部分**（这是最核心的适用边界）：
- **生鲜四大件（蔬菜、水果、肉禽蛋、水产）**——这是社区生鲜超市的核心品类，在便利店六位码体系里**完全没有对应大类**（08大类里仅"水果、果切"一个小类080901，且是切好即食的便利店逻辑，不是社区超市论斤称重的生鲜品类逻辑）。这是团体标准对花厅坊场景最大的适用性缺口，**不能靠"扩展小类"解决，必须新增独立大类**。
- **散称/称重逻辑**——六位码体系里的商品单元隐含"单品定价、条码扫码"逻辑（便利店无称重台），而生鲜四大件的核心是散称、非标品、日日变价，编码体系需要额外挂载"计量单位/称重类型"字段，团体标准本身未覆盖。
- **熟食/中式餐饮延伸**（卤味/凉菜/现制面点等社区超市常见业态）与08大类的便利店柜台商品性质不同（社区超市熟食通常是差异化引流品类而非便利店的即食便当逻辑），品类归属和陈列逻辑都不同，不能直接套用08大类小类。

**结论性评估**：T/CCFA 039-2023的六位码框架**结构本身（三级、六位数字、大中小类编号规则）可以借用**，但**11大类的具体内容有约30-40%对花厅坊不适用或需要大改**（主要是08快餐食品、11博彩两个大类基本不用，05/06两个大类需重新切分），且**最大的结构性缺口是完全没有生鲜品类**——这恰好是社区生鲜超市的立身之本，意味着这套标准只能作为"非食品/预包装食品部分的编码参考"，无法覆盖花厅坊品类系统的主体。

### 4.2《专业零售店店长岗位要求》（T/CCFAGS 047-2023）

标准本身写明"本标准适用于专业零售店的店长岗位，其他零售业态的店长岗位要求可参考本标准"——已自我声明为**可参考而非强制适配**。内容是任职能力（职业素养/基本能力/专业能力）+ 岗位职能（销售管理/服务管理/人员管理/环境管理/资产资金管理/日常基础工作管理）的通用框架，未出现任何品类专属条款，**对社区生鲜超市店长岗位设计的适配度是"框架通用、内容需本地化"**——比如5.1.4商品管理条款需要按生鲜的"损耗/效期/温控"重新细化（标准原文5.1.4.7已提及"温度、光照、效期、除湿"，方向对但深度不够，生鲜场景需要专门补充"损耗率考核""冷链断链应急"等条款）。

### 4.3《商品学基础》第3章节选

与花厅坊品类系统/诊断实务关联度低。**唯一可能用得上的点**：3.2.1食品质量基本要求中列出的"卫生无毒害性"5类污染来源（自身产毒素/生物污染/加工混入/保管不善/环境化学品污染），可作为生鲜品类质量巡检的检查维度背景知识，但这是通识常识级别，不构成独特引用价值。

## 5. 引用建议

如果晟果新零售品类系统（如 `05_品类管理与商品规划/KB-CAT-ARCH-001_全品类双投影架构_v1.2.md` 或 V6.0 品类主数据）要引用团体标准作为权威依据：

1. **标注格式建议**：引用编码结构时注明"参考 T/CCFA 039—2023《便利店商品分类》（中国连锁经营协会团体标准，物美科技集团/美宜佳控股起草）"，并**在同一处明确注明业态差异**——例如："本表大类01/02/03/04/07/09的中小类结构参考T/CCFA 039-2023便利店商品分类标准并按社区生鲜超市场景删减调整；生鲜四大件、熟食等品类为晟果新零售自建，团体标准无对应条款"。
2. **不建议**笼统写"依据中国连锁经营协会团体标准编制"而不说明改造范围——团体标准本身是便利店业态标准，若不加区分地作为社区生鲜超市品类系统的权威依据，容易在客户交付材料中被专业审阅者（如同行顾问、供应商）指出业态错配。
3. **店长岗位标准**引用同理，建议标注"参考T/CCFAGS 047-2023《专业零售店店长岗位要求》通用框架，生鲜/损耗/温控相关条款为晟果新零售补充"。
4. **商品学基础教材**建议**不作为品类系统的引用来源**，因其节选内容（商品质量章）与分类/编码无实质关联；如需引用商品质量管理通识内容，应引用更新的国标或权威文献（教材援引的多部法规已被后续立法取代，如食品卫生法已失效，直接引用会有合规风险）。
