## 3. 节点④⑤新建卡片（9段·决策链格式）

### 卡 A — 31 安全库存建模（3）｜节点④ 安全库存(结构+呆滞)

| 段 | 内容 |
|---|---|
| **1 所属决策链** | ① 库存决策链(唯一) · 节点④安全库存 |
| **2 输入变量** | 库存结构清单、无需求时长(>1年)、提前期VP、采购周期/核查期、订货单位(最小包装)、综合值(销量×销额×毛利权重)、ABC分类 |
| **3 判断链拆解** | 抓重点(二八)=呆滞库存 → 库存ABC分类(A类SS高/BC类SS低) → 降提前期(SS与提前期平方根成正比;采购周期月→周,SS大降) → 库存三分结构(周转/安全/呆滞) → 健康库存占比越大越好 |
| **4 核心错误来源** | 防"安全库存一刀切"=不分ABC、不分波动,A品缺货+C品积压并存;防呆滞库存沉淀(>1年无需求仍占资金) |
| **5 修正机制** | 库存ABC定SS差异化;降采购周期/系统再订货点触发;简易SS=（预计最大需求−平均需求）×采购提前期;最大≈平均→SS=0(需求固定不用SS) |
| **6 可复用模型** | **库存结构三分模型**(周转良性/安全中性/呆滞红色)+**呆滞库存清理模型**(>一段时期无需求→清,但先排缺货) |
| **7 M-DEC/FC映射** | **FC10(库存积压)字段化**:呆滞=库存结构红色块,可按"无需求时长+ABC"触发;**增强 M-DEC-009**:呆滞清理须先排除缺货(防误杀) |
| **8 花厅坊映射(T03)** | T03 库存三分:**4个有货=可能"安全→呆滞"漂移段**,需查无销时长;**10个I=0=不在呆滞结构内**(无库存何来呆滞),不可清。给6/12库存核查一把结构尺 |
| **9 反误判风险** | 若不分ABC一刀切SS:畅销缺货+滞销积压双输;若把缺货当呆滞清:误砍真实有需求断档品 |

### 卡 B — 32 安全库存建模（4）｜节点④ 安全库存(量化公式)

| 段 | 内容 |
|---|---|
| **1 所属决策链** | ① 库存决策链(唯一) · 节点④安全库存(公式核心) |
| **2 输入变量** | 需求标准差σF、补货周期L、预测时间单位FP、服务水平Z、提前期LT、历史销量(8月/32周)、泊松λ |
| **3 判断链拆解** | SS取决三因子:供应稳定性(LT)+需求稳定性(D)+服务水平 → 三步量化:①量化需求不确定性(正态用STDEV/极低需求用泊松)②量化供应不确定性(交期标准差)③量化服务水平(Z=NORMSINV(有货率)) → 整合 **SS=σF·√(L/FP)·Z** |
| **4 核心错误来源** | 防"凭经验拍脑袋设SS"=高的高低的低、畅销缺货+库存过剩;防低需求品误用正态(应泊松) |
| **5 修正机制** | 三步量化建模;供应周期恒定时简化为 SS=σF·√(L/FP)·Z;低频品用泊松POISSON.DIST算再订货点;先算基准SS再按业务调 |
| **6 可复用模型** | **安全库存三因子量化模型**(供应LT+需求σ+服务Z);**低频品泊松再订货点模型** |
| **7 M-DEC/FC映射** | **FC06(售罄速度/安全库存)字段化达成**:σF/L/FP/Z 全字段就位,补上1-A欠的阈值;支持↑可落地 |
| **8 花厅坊映射(T03)** | T03 饼干**节令/送礼品σ大**(波动)→需更高SS,淡季低销≠滞销(是σ大被低估);**健康老人刚需σ小**→低SS稳定补货。直接区分"波动型假滞销"vs"真滞销" |
| **9 反误判风险** | 若对σ大的节令品按平均需求补货(不放SS):旺季断货;若把σ大淡季低销判滞销砍掉:旺季无货=误砍 |

### 卡 C — 数据化业务驱动（09-再聊安全库存）｜节点⑤ 售罄/有货率

| 段 | 内容 |
|---|---|
| **1 所属决策链** | ① 库存决策链(唯一) · 节点⑤售罄→有货率→需求满足 |
| **2 输入变量** | 有货率(服务水平)、Z值、安全库存、订货周期预测量、商品流转单据(要货/补货/订货/验货/入库…)、一进一出标志 |
| **3 判断链拆解** | 不放SS→有货率50%基线 → 放1σ(Z=1)→84.1% → 再放1σ(Z=2)→97.7%→趋100% → 但末尾零点几%需大量库存(高库存/低周转/现金流失) → 分阶梯服务水平(90%立即/95%在2-3天/100%在1订货周期) |
| **4 核心错误来源** | 防"不好卖的没有,好卖的一大堆"(SS茫然=根因);防盲目追100%立即满足→现金转无用库存 |
| **5 修正机制** | 服务水平分阶梯(非全部100%立即);**一进一出**:引新品则老品SS调低(更新换代);旺季(春节)调高SS,高进价/现金采购品调低SS;单据流程精细化(损耗从报货起) |
| **6 可复用模型** | **分阶梯服务水平模型**(90/95/100按时段);**一进一出SS联动模型**(新品入→老品SS降) |
| **7 M-DEC/FC映射** | **直接增强 M-DEC-009**:"一进一出+老品SS调低"=老品汰换的**具体可执行机制**(不是直接砍,是先降SS观察再汰);**FC06**补售罄→有货率换算 |
| **8 花厅坊映射(T03)** | T03 若要引饼干新品(高端/健康):按"一进一出",先把同位老品SS调低观察,不直接砍 → **给M-DEC-009一个落地动作**;10个I=0先判有货率而非滞销 |
| **9 反误判风险** | 若不用一进一出直接砍老品:断档+顾客流失;若全品追100%立即有货:库存爆+现金流失 |

## 4. 节点⑥ 滞销识别 + 门店决策规则（v3.0 落地铁律）

### 4.1 真滞销识别四关(全过才判滞销)

| 关 | 判据 | 不过则 |
|---|---|---|
| 关1 有货 | I>0(库存>0) | I=0→**缺货,不判滞销**(节点②) |
| 关2 积压 | 实际库存÷(DMS×1.5) 远>1 | 接近1→正常,非积压(节点③) |
| 关3 低售罄 | 售罄率持续低/有货率够但不动 | 售罄高→快卖完,非滞销(节点⑤) |
| 关4 非波动 | σ小(非节令/送礼大波动) | σ大→旺季会动,假滞销(节点④卡B) |

### 4.2 门店决策规则表（直接可执行）

| 现象 | 链路诊断 | **门店决策** |
|---|---|---|
| 库存=0 + 历史有DMS + 属刚需 | 节点②缺货 | **补货**(Q=d(T+L)+Zσ√(T+L)−I) |
| 库存=0 + 历史DMS≈0 | 本无需求 | 不补,观察是否本就不该配 |
| 有货 + σ大(节令/送礼) | 节点④波动 | **调高SS,淡季留货**,不砍 |
| 有货 + 四关全过 | 节点⑥真滞销 | **淘汰候选**→进六桶细判 |
| 引新品 替老品 | 节点⑤一进一出 | 老品**先调低SS观察**,再汰(M-DEC-009) |
| 单品库存≫DMS×1.5 | 节点③积压 | 停补/降价清,查订货过量 |

```

   ▼

   │
   ▼
[门店决策] 补货 ✓ | 调高SS留货 ✓ | 淘汰候选 ✓ | 老品先降SS再汰 ✓
```
