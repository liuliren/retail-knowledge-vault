## 2. 原文核心命题
- **小类价格带**:整店统一价格带是错的;每个小类应有自己的价格带;**核心价格区间(销额/毛利占比高)细分,非核心粗分**;诊断过密/断层。
- **毛利目标模型**:毛利是经营"晴雨表";从盈亏平衡毛利率出发 → 层级分解(大→中→小→货位→单品)→ 贡献度评估 → 动态调价 → 预警(主力品类毛利跌破X%触发)。
- **计量化(二八)**:品类结构占比最好近 20:80 或 78:22;实测 16.5-18% 商品产 70-78% 销售;预测准 1% → 后续每流程效率 +5%(1:5杠杆)。
- **定价逻辑**:售价由市场+商品价值决定;"进价定售价"是自我需求思维,应转向"提升性价比降成本"。

## 3. 关键公式/计算口径(每条带来源)
| 公式 | 表达式 | 来源 |
|---|---|---|
| 盈亏平衡毛利率 | **固定费用 / 销售额**(例 100/500=20%)| 40 |
| 含利润毛利率 | 盈亏毛利率 + 目标利润率(20%+3%=23%)| 40 |
| 售价(正向)| **售价 = 进价 /(1−目标毛利率)** | 39/40 |
| 成本(反向)| 成本 = 售价×(1−目标毛利率)| 39 |
| 贡献度 | **贡献度 = 销售额×毛利率 − 直接成本** | 39 |
| 行业对标 | TopN **25%分位**毛利率 | 40 |
| 毛利基准 | SKU级动态维持整体毛利 **±0.5%** | 40 |
| 二八/犹太 | 20:80 / **78:22**;实测16.5-18%产70-78% | 39 |
| 价格带分段 | 核心区间细分/非核心粗分;小类×价格段笛卡尔积 | 14 |
| 毛利预警三级 | 黄(单项超5%)/橙(连2月)/红(净利1%)| 40 |

## 4. 业务判断链
```
某小类价格带 → 实测各段 SKU%/销额%/毛利%
├ 单段 SKU% 畸高 → 过密 → 疏(汰同质,过M-DEC-009防误杀)
├ 相邻段销额跳空,中间段≈0 → 断档 → 补中间
├ 核心需求段 SKU=0 → 漏档 → 补整段
└ 补哪段先?→ 按段毛利贡献度排序(高毛利段优先)
毛利目标 → 盈亏平衡毛利率 → 层级分解到小类/货位 → 贡献度分高中低 → 调价/汰换
```

## 5. 反常识点
- 整店统一价格带是错的;**价格带必须按小类各自设**。
- **"进价定售价"违背定价逻辑**;售价由市场决定,毛利靠降成本/提价值。
- 二八是理想态;真实多为 **16.5-18%产70-78%**。
- 高毛利率商品 ≠ 最赚钱(未摊库存/陈列/场成本)→ 须用贡献度,非毛利率。

## 6. 对花厅坊的可用部分
- M-DEC-010 价格带三态(过密/断档/漏档)的**量化口径**(§3)直接可用。
- 品类表v1.0 已有三占比列(sku_count_pct/revenue_pct/margin_pct)→ 可立即做"价格段健康度"联动分析。
- T01薯片(需低中高3段全覆盖)/T07巧克力(≥20元档断)→ 用段销额%跳空判断"补到什么程度算够"。

## 7. 对 M-DEC 的增强点(M-DEC-010)
| 维度 | 二遍补强 |
|---|---|
| 过密 | 口径=段内SKU%畸高;阈值待实测 |
| 断档 | 口径=相邻段销额%跳空 + 中间段销额%≈0 |
| 漏档 | 口径=核心段SKU数=0 |
| 排序权重 | 毛利贡献度=销额×毛利率−直接成本(高贡献段优先补)|
| 三占比联动 | SKU%高+销额%低=过密低效;销额%高+SKU%低=可加宽 |
> **不改 M-DEC-010 临界值**,仅补"过密/断档/漏档怎么算"+毛利作排序权重的口径。

## 8. 对 POS 字段回填清单的影响
- **无需新增字段**。010 所需(price_band/band_sku_pct/band_sales_pct/band_gp_pct)清单已含。
- 仅补充口径说明:`band_gp_pct`(段毛利占比)用作排序权重而非主判据 → 与清单 §C.3 一致,无冲突。

## 9. 可转成 Excel 工具的部分
- **花厅坊小类价格带矩阵**:行=小类×价格段,列=SKU%/销额%/毛利%,条件格式标过密(红)/断档(黄)/漏档(灰)。
- **盈亏平衡毛利率速算**:输入固定费用+目标销售额 → 最低毛利率。
- 暂不做:线性规划组合毛利优化引擎(数据/工具不支撑)。

## 10. 与现有科学零售体系的映射
M3 价格带毛利簇 → 金额管理·毛利 + 品类管理·价格带。M3 已有"小类价格带建模/毛利贡献度模型/情景毛利",本簇补足其计算口径(盈亏平衡毛利率/售价正向式/贡献度式/三占比联动)。

## 11. 与既有 M-DEC/标准/表格的冲突风险
- ⚠️ **价格段划分 vs 品类表v1.0 §5 三段(低<5/中5-20/高>20)**:本簇主张"核心区间细分",可能比 v1.0 三段更细。**处理:沿用 v1.0 三段为基线**,细分仅作分析层,不改 v1.0 定义。→ M3 卡标注。
- 无重定义毛利/价格带管理(主定义在 [[金额管理]]/[[品类管理]])。

## 12. 仍需验证的数据字段

## 13. 不吸收 / 暂缓吸收

- 毛利±0.5%基准、TopN25%分位 → 作参考,不直接当花厅坊阈值。
