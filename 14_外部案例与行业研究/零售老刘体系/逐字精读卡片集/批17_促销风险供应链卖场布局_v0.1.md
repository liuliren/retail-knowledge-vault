# 逐字精读卡片集 · 批17 促销风险/供应链/卖场布局

## 97. 品类建模(95) 促销风险 ⭐
- **核心论点**:**促销5大风险**:①**利润侵蚀**(增量不增利·赔本赚吆喝)②**品牌损害**(顾客等打折→不促销卖不动恶性循环)③库存供应链(需求扭曲·缺货或滞销)④**顾客群偏移**(大促引错误客群·薅羊毛党低忠诚→稀释核心价值顾客→倒置服务对象)⑤**内部管理/数据失真**(促销频繁→运营混乱·销售数据失真要清1/3)。
- **关键词**:促销5风险 / 增量不增利 / 错误客群=薅羊毛党 / 促销数据失真

## 98. 品类建模(96) 门店供应链
- **核心论点**:供应链=采购(买)/物流(动)/供应链主管(全局信息流资金流物流);**供应链=大脑,采购物流仓库=手脚**(需协作·各自为政=做不好)。门店小供应链=后台系统;**系统流程须与实际流程一致**(常换软件不改流程=流通不畅)。
- **关键词**:供应链=大脑/采购物流=手脚 / 系统流程=实际流程一致

## 99. 品类建模(97) 卖场布局 ⭐
- **核心论点**:**卖场布局5原则=层次性/需求性/适应性/经济性/实用性**;布局(战略整体)vs陈列(策略作业)。陈列跨学科(营销/消费心理/人体工程/视觉/空间);数据驱动(热力图/货架空间管理)。陈列从"简单堆叠"→系统/科学/精细/场景化。
- **关键词**:布局5原则 / 布局vs陈列 / 陈列跨学科 / 简单堆叠→精细化

## 100. 品类建模(98) 字段标签化
- **核心论点**:**商品标签≈属性≈字段≈维度列**(业务上很少区分);万物皆维度万物皆标签;标签库=神经元大脑。数据分析顺序=商品属性→标签属性→需求→业务(属性▶标签▶需求▶业务再现)。
- **关键词**:标签≈属性≈字段≈维度 / 标签库=神经元 / 属性▶标签▶需求▶业务

## 101. 品类建模(99) 卖场式陈列 ⭐
- **核心论点**:**陈列的是一个个"需求",不是商品**(陈列=需求的表达方式)。陈列=供应链在需求端的集中展示(掺杂供应商意愿)。**陈列单元(货位/BAY)=一个顾客需求**(小类↔BAY);基于需求的陈列=陈列进化趋势。
- **关键词**:陈列=需求的表达 / 陈列单元=需求单元 / BAY=一需求

## 102. 品类建模(100) 零售度量系
- **核心论点**:度量vs指标;**零售度量系(灵活/动态/可组合)vs 度量体系(整体/战略)**;万物皆维度万事可量化;系统性研究零售运营(统一语义/诊断问题/提效/驱动增长)。
- **关键词**:零售度量系 / 度量vs指标 / 万物皆维度万事可量化
