# 逐字精读卡片集 · 批50 以顾客为中心·假A类·陈列时空系数

> 📌 **本批 = 分析建模05-07**(2022年10月)。理念(05精细化零售)+定量(06陈列系数/07货位看板)。**双金矿:① 假A类商品(防误杀核心机制)② 以顾客为中心三步(客群方法论)**——都直击六哥诊断内核。
> ⚠️ 下一批续读 分析建模 08-09(绩效建模/卖场经营指数)等。

## 1. 分析建模(05) 以顾客为中心的精细化零售 ⭐⭐⭐
- **核心论点(理念金矿)**:粗犷→精细化=以顾客为中心的更清晰管理;关键不是服务好所有顾客,而是**维系带利润的价值顾客**(含潜在);从**商品主导型→顾客需求型企业**。
  - **顾客需求=买「什么」+「为什么」买**(两者结合才高绩效)。
  - **经营2大黑洞之一=对顾客不了解**(流失多少/哪类/为何流失·全是黑洞)→只能拍脑袋。
  - **⭐以顾客为中心三步**:① **找出真正价值顾客**(商圈研究+从销售与边际利润评估每类顾客潜在价值;99%企业不做商圈分析)② **以价值顾客需求为依据的改变**(小类=一类顾客需求·**至少每半年重新调整小类**·两问判据:**为的是不是目标价值顾客?是否让他们更愿来、买更多?**)③ **目标顾客价值量化**(小类销售占比/动销率/客流/客单·行为数据反推顾客价值·"**赶在顾客之前改变**"·半年或从不调价值类别=危险)。
  - 落地问题清单:多少小类合适/一货位陈列多少小类/增减哪些小类/大类A中类·中类A小类·小类A单品/品牌多少/主推商品监控/畅销缺货损耗量化——**最后都回到价值顾客的量化**。
- **关键词**:价值顾客(带利润) / 商品主导型→顾客需求型 / 买什么+为什么买 / 经营黑洞(不懂顾客) / **三步(找价值顾客→每半年调小类→量化)** / 两问判据 / 赶在顾客之前改变

## 2. 分析建模(06) 货位陈列时空系数·假A类商品 ⭐⭐⭐
- **核心论点(防误杀金矿)**:数据分析只看商品本身(销量/毛利/客流)→**忽略人(主导)+场(卖场空间)+顾客需求**。
  - **⭐假A类商品(防误杀核心)**:A前2天卖20后3天断货(本可5天卖100),B 5天卖50→分析显示B好,但**顾客真正想买的是A**(因A断货被迫买B)→"**分析中会产生大量假A类商品**"。我们认的A类=**卖家思维**;**顾客思维:不关心你的动销率,只想买想要的**。**经典名言:畅销的才是缺货的**(销售不佳反可能是畅销缺货)。采购的"自己觉得/个人经验"不一定好。
  - **每件商品付出的成本和服务不同**:净利/毛利按整体平均分摊成本=不公平(占库存成本/黄金位置不同·相同毛利但占黄金位vs角落不一样)。
  - **空间系数(陈列系数)**:每货位/层按动线位置好坏赋系数(黄金位说法的精细化);**时间系数**:货位调整随时间变化(开始/结束→扩展每天)。时空系数→判陈列位是否合适/陈列面增减(例:某货号销售占比19.3%却在第2层第2位→应移到第5层黄金位中间)。
- **关键词**:人货场分离(忽略人/场/顾客) / **假A类(断货致伪畅销·卖家vs顾客思维)** / **畅销的才是缺货的** / 每件商品成本不同(黄金位vs角落) / 陈列空间系数+时间系数(黄金位精细化)

## 3. 分析建模(07) 货位数据地图AR动态看板 ⭐
- **核心论点**:海鼎价格管理模型(毛利分割N等份·按会员等级让利倍数=顾客思维)→同理货位:货位分割N编码,**主通道位置承担更多毛利/销售指标**。货位管理致命问题=**不能即时动态直观了解问题**(亡羊补牢"想到A听说B做成C")。
  - Power BI视图做**货位/区位动态看板**(棚格图·货号图·区位看板·鼠标悬停显详情)→**盲区管理**(区位货位销售·畅销品缺货即时可见);陈列面占位(按面数拆行);收银员认商品(货号图看板)。
- **关键词**:主通道承担更多指标 / 货位盲区管理(即时非及时) / PBI动态看板(棚格/区位/悬停详情) / report-export可视化思路
