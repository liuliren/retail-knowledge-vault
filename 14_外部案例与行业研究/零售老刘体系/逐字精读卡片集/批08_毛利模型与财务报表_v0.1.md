# 逐字精读卡片集 · 批08 毛利模型与财务报表（财务向·从紧)

## 43. 品类建模(40) 毛利目标模型
- **核心论点**:建毛利贡献度模型步骤=数据采集清洗(3-5年财务/运营/市场/变量)→关键变量敏感性(价格弹性/蒙特卡洛成本波动)→目标设定(对标行业TopN25%分位/ROIC反推/线性规划优化空间毛利)→**三情景(乐观/中性/悲观)+ 毛利关键转折点预警系统**(主力品类毛利跌破X%触发修正)。
- **关键词**:毛利贡献度模型 / 价格弹性 / 三情景 / 毛利转折点预警

## 44. 品类建模(41) 成本8类 ⭐(接保利期)
- **核心论点**:毛利=收入−支出;**8类成本**:采购/头程配送/推广/经营(租金水电人工)/FBA件费/**仓储(含超龄库存费>180天·常被忽视)**/退货/其他。**弃置费**(临期/报损/损耗·生鲜大·反映管理水平)。
- **关键词**:8类成本 / 超龄库存费>180天 / 弃置费=管理水平 / 持有成本

## 45. 品类建模(42) 毛利业务语义
- **核心论点**:利润表主线=**收入▶成本▶利润**。**盈利三把斧=毛利率/净利率/ROE**(三道防线·不同角度揭示盈利)。毛利率=(销售收入−销售成本)/销售收入;对比同行/同期判断竞争优势。
- **关键词**:收入▶成本▶利润 / 三把斧(毛利率/净利率/ROE)

## 46. 品类建模(43) 财务三大报表
- **核心论点**:**资产负债表(实力/家底)/利润表(能力/赚钱)/现金流量表(活力)**;三表关联(净利润→未分配利润→经营现金流)。零售现状:存货积压周转低(生鲜/快消)、毛利率受挤压(20%左右)需高周转弥补;ABC分类优化存货。
- **关键词**:三大报表=实力/能力/活力 / 存货积压 / 毛利率受挤压靠高周转 / ABC优化存货

## 47. 品类建模(44) 财务利润表
- **核心论点**:COGS(销售商品成本);固定费用(租金/工资)vs变动费用(采购/物流);**业财融合**;会计方法(先进先出/加权平均/日结成本)影响利润表;利润表局限=反映过去、不能预测未来(需配现金流/资产负债表)。
- **关键词**:COGS / 固定vs变动费用 / 业财融合 / 利润表局限=反映过去

## 48. 品类建模(45) 利润表语义
- **核心论点**:财务分析内容(三大报表/比率/成本费用/现金流/预算)。常用度量:毛利率/库存周转/**坪效/人效**/客单价客流/应付账款周转天数。运用:业绩诊断(同比环比识别升降原因=需求疲软vs竞争加剧)、成本控制(优化高占比可变费用/降库存损耗)。
- **关键词**:坪效·人效 / 业绩诊断(同比环比) / 成本控制 / 业财联合
