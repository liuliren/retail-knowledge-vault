## 1. 序309 关于4月份部分百货货位调整的建议方案(真实案例) ⭐⭐⭐
- **核心论点(诊断卡方法论范本·可直接套花厅坊单品类诊断)**:一个完整的百货课货位调整诊断:
  - **诊断链**:① 坪效=**22元/㎡**(50组货架·过低)② 动销率:11325 SKU·有销445→**在销动销率6.8%**;有库存1732→**在库动销率44.6%(低于60%正常值)** ③ A类约180 SKU ④ **货架数测算=有销售SKU÷每组25单品**(737÷25=29组·实际40+组→**可减20组**·"货架公平分配原则")⑤ 销售占比3.38%→2.2%(下滑)·单品占比44%·陈列占比41%(资源占用过多)。
  - **调整建议**:重梳结构淘汰差类别+减陈列面积;减疲惫类别(玩具文体)增好类别(软百货)货架;查销售差真实原因(缺单品/缺陈列→即时调);**先分析去年A类"畅销丢失"+结合实时库存对"有销无库"A类即时补货**。
  - **⭐范本提炼**:坪效→动销率(有销/在库·60%线)→A类畅销丢失→货架数测算(SKU÷25)→减货架+补有销无库A类。这正是 [[单品类诊断]] skill 的方法骨架。
- **关键词**:坪效22元/㎡ / **在库动销率<60%=问题线** / **货架数=有销SKU÷25** / 货架公平分配原则 / A类畅销丢失+有销无库即时补 / 诊断卡范本

## 2. 序312 品类度量(07)·购买率 ⭐⭐
- **核心论点**:品类核心指标=**占比(比率)类**(反映结构)。**六大顾客购物指标=客流量/进店率/成交率/客单价/连带率/复购率**(统计:76%经营者认为转化率关键/54%用客单价管控结构/41%用客单数连带率定促销)。
  - **⭐购买率(成交率)=有购物小票的顾客数÷进入卖场总顾客数**(进店有几个买东西)=复合综合指标+最终绩效指标(反映卖场综合经营)。**盲点=进店量不清楚**(无法对比真实成交)→客流统计系统(摄像头对接POS)解。进店后**87%关注商品取决于卖场环境与陈列科学性/显眼度**。
- **关键词**:品类核心指标=占比类 / **六大购物指标(客流/进店率/成交率/客单价/连带率/复购率)** / **购买率=购物小票数÷进店数** / 进店量盲点 / 87%关注取决陈列

## 3. 序314 数据化业务驱动(09)·再聊安全库存 ⭐⭐⭐
- **核心论点**:安全库存三量化(需求/供应/服务水平·重申批58/59)。服务水平=Z值(50%基线·1σ→84.1%·**2σ→97.7%**·95%→Z=1.64)。
  - **⭐分阶梯服务水平(精细化金句·解"100%满足"难题)**:不是立即100%满足,而是**分时段分阶梯**——**90%订单立即由库存满足(配送中心直发)·95%在2-3天内发货·100%在一个订货周期内发货**。层级式兼顾服务水平与库存优化。
  - 安全库存调整:**一进一出**(新品进→老品安全库存调低)·春节调高·高进货成本/现金采购少放。"安全库存=库存计划技术含量最高的一块"。
  - **商品流转单据流程**(要货/补货/订货/随货/验货/入库/退货/扣补/费用/对账/付款单·每流程有时间规定):**⭐商品损耗从报货就开始**(前位流程出问题后面全受影响·从报货精细化就能控损耗)。
- **关键词**:服务水平=Z值(50%/84.1%/97.7%/95%=1.64) / **分阶梯服务水平(90%立即/95%2-3天/100%订货周期)** / 一进一出 / **损耗从报货就开始** / 单据流程每环节有时间规定
