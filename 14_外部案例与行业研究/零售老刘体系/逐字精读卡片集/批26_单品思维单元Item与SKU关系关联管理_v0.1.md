# 逐字精读卡片集 · 批26 单品思维/Item与SKU/关系与关联/单品管理

## 151. 品类建模(167) 单品思维 ⭐⭐
- **核心论点**:**产品思维(生产商)vs 单品思维(零售商·拥有购物篮行为数据)**。沃尔玛中国4策略:**精选SKU/单品驱动(非品类驱动)、低毛利高效率、EDLP每日低价(非周期促销)、净成本合作**。"**宽类窄品**"=削减SKU(每小类只留最畅销·1万+→5000-8000·70%新品·腾笼换鸟:窄掉非畅销品→换生鲜/熟食/烘焙的宽类空间)。**窄品=电脑重启/重装**(治标不治根·须基于自身目标顾客才有效→同样窄品有的成功有的失败)。产品vs单品思维核心区别=**目标导向+时间维度**(长期沉淀 vs 短期周转)。"善打篮球自己当裁判"=从已有单品里选品的尴尬(缺以需求为起点的选品闭环)。
- **关键词**:单品思维vs产品思维 / 沃尔玛单品驱动+EDLP+净成本 / 宽类窄品=腾笼换鸟 / 窄品=重启治标 / 单品思维=拥有购物篮

## 152. 品类建模(168) 单品单元 ⭐⭐
- **核心论点**:**以单品为战略单元**(优于"以品类为战略单元")。单品配置≈做菜(先备食材=单品)。**单品=SKU=最小配置单元**(主动选择/精细挑选/匹配需求/动态执行)。单品=生产线×需求线交集(王老吉×罐装×380ml)。单品管理=维度+度量(销售/数量/品质/价格/价值);生鲜"**一日三价**"(早正价/午降价/晚打折=在最有价值时卖出)。**单品单元四逻辑层:①识别(商品→数据元)②分析(总体概览→微观归因·"哪几个单品贡献了增长")③决策(经验→数据驱动)④协同(部门墙→统一语义SKU)**。单品优化提净利 0.8-1%。
- **关键词**:以单品为战略单元 / 单品=SKU=最小配置单元 / 生产线×需求线 / 一日三价 / 单品四逻辑层

## 153. 品类建模(169) Item与SKU ⭐⭐
- **核心论点**:**Item(品项·顾客心智"一种商品"·商业概念·回答"卖什么"·交易语言)vs SKU(库存单元·物理管理·回答"管什么"·管理语言)**。**一个 Item 对应多个 SKU(一对多映射=精细化管理起点)**。Item=品牌+品类+品名;SKU=Item+所有可区分物理属性。**选品配置逻辑="先有需求(Item)再有选品(SKU)"**。单品=使用线=生产线×需求线 → **HS 矩阵**(生产线×需求线作横纵坐标·交叉格=一个SKU)。西瓜=Item(需求)/切块=不同SKU(满足多人/1-2人/便携不同需求选择)。
- **关键词**:Item(卖什么/交易语言)vs SKU(管什么/管理语言) / 一Item对多SKU / 先有需求再有选品 / HS矩阵=生产线×需求线

## 154. 品类建模(170) 生命性关系 ⭐⭐
- **核心论点**:**零售=生命体**(诞生/成长/成熟/衰退·非静态交易);**关系=零售生命的"经络"**;熵增=生命周期=脉搏节律。管理对象=动态关系流(顾客/信息/商品/资金/物流/管理);管理者=**园丁/管家/生态工程师**(非冰冷耕作员)。单品关系生命周期=**联→链→网**(导入期建"联"试销脆弱/成长期成"链"进购物篮/成熟期织"网"明星品/衰退期解体)。顾客关系=偶遇→共生;供应链关系=博弈→协同(买卖→信息共享→战略协同CPFR)。**鸡蛋引流品价值不在自身利润,而在生态位**(维系价格敏感客的初级关系 + 激活高毛利品关联)。**"品类管理=以单品为单元、以生命性关系为中心"**;选品=在原有单品关系网上加有意义节点("选品选不对,后续全白费")。
- **关键词**:零售=生命体 / 关系=经络 / 联→链→网 / 园丁非耕作员 / 鸡蛋引流品=生态位非利润 / 选品选不对后续全白费

## 155. 品类建模(171) 关系与关联 ⭐⭐
- **核心论点**:**零售经营本质=构建和维护正向、良性的多元性关系**(2017年起坚持的核心观点·与顾客/供应商/供应链/内部组织)。**关系 ≠ 关联**:关系决定关联的价值方向,关联=关系在运营层的投射/量化表达/落地抓手(关联陈列/捆绑促销/购物篮Apriori"啤酒尿布")。**铁律:先有关系再有关联(关系包含关联)**=先有 Item 需求再有 SKU 选择=陈列先定 Item 需求(落位)再配 SKU。万物皆维度+万事可量化(弥勒佛对联:大肚=维度容器/笑口=度量)。**杂乱陈列 vs 关联陈列的销售差异 = 关系价值**(关联极致→关系价值最大,反之为负)。
- **关键词**:零售本质=维护多元关系 / 关系≠关联·先关系后关联 / 先Item需求后SKU / 关联陈列=关系价值 / 万物皆维度

## 156. 品类建模(172) 单品管理 ⭐⭐
- **核心论点**:**单品管理=单品+管理=对 SKU 做全供应链/全生命周期/全业务流程精细化管理 + 数据驱动主动干预动态优化闭环**。四条价值逻辑线(属性线/生产线=属性×线数/使用线=生产线×需求线/**价值线=属性×生产×需求**·变量在需求线→选品=扩需求线第一步)。四递进业务逻辑(可识别可追溯/可度量可分析/可决策可干预/可迭代可优化)。**"管理=度量"推导**(单品管理=单品+管理=单品+度量 → 管理=度量)。两类指标(绩效品质类/经营关键指标)。**"数量"指标不可替代**(先有数量再有销售额·任何单品业务表必须有数量列)。四场景(配置/减购物篮损失/提订货准确度/控合理库存)+"一进一出"保结构活力。
- **关键词**:单品管理=全链全周期闭环 / 价值线=属性×生产×需求 / 管理=度量推导 / 数量指标不可替代 / 一进一出
