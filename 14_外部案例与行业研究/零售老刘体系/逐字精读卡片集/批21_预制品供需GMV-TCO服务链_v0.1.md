# 逐字精读卡片集 · 批21 预制品/供需主计划/GMV-TCO/服务链

## 121. 品类建模(137) 预制品标签 ⭐
- **核心论点**:商品最大隐性属性=业务分类(是否能吃食品/非食品·是否加工初级/初加工/深加工·是否经营可售/不可售)。**"是否可售"=选品最初筛选**(可售=新品/正常;不可售=停购/淘汰/删除)→减少无效商品信息。
- **关键词**:是否可售=选品最初筛选 / 食品非食品 / 初级/初加工/深加工

## 122. 品类建模(138) 一日三餐类
- **核心论点**:预制菜=一日三餐属性(非副食零食)。**"新鲜"认知=食品标签化认知**(预制菜被贴不新鲜/不卫生标签);**鲜度=新鲜程度指标**(冷冻锁鲜=封住环节和时间)。
- **关键词**:预制菜=一日三餐 / 新鲜=标签化认知 / 鲜度指标

## 123. 品类建模(139) 供需主计划 ⭐
- **核心论点**:商品管理两链(产品▶商品▶属性▶标签▶组合▶品类 / 产品▶商品▶好货▶对的好货▶畅销化▶价值品)。**真正的选品=经属性了解→商品组合→适配需求单元**(非直接拿单品配=外圈伪选品)。
- **关键词**:商品管理两链 / 真选品=属性→组合→需求单元 / 外圈vs内圈

## 124. 品类建模(140) GMV与TCO ⭐⭐
- **核心论点**:**GMV − TCO ≈ Gross Profit(总毛利)**;GMV=目标(过程)/TCO=结果。系统化供应链=一个需求计划+一个成本计划。GMV(全渠道交易总额)/TCO(全部成本)/"不成立三角"(GMV/TCO/效率)。
- **关键词**:GMV−TCO≈GP / GMV=目标·TCO=结果 / 一个需求计划+一个成本计划

## 125. 品类建模(141) 采购方式
- **核心论点**:采购4次演变(手工→中央集权→ECR/CM数据驱动→数字化AI)。**采购从后勤职能→核心战略能力**(采买→价值创造)。核心逻辑=效率/成本/体验极致平衡。
- **关键词**:采购4演变 / 采购=核心战略能力 / 采买→价值创造

## 126. 品类建模(142) 服务价值链 ⭐
- **核心论点**:**商品价值链(效率/成本·物理属性)vs 服务价值链(关系/个性/顾客体验·心智)**。零售两核心=商品+服务(服务常被忽略·缺结构体系);服务价值链=长期主义战略。
- **关键词**:商品价值链vs服务价值链 / 服务=被忽略的核心 / 关系个性体验
