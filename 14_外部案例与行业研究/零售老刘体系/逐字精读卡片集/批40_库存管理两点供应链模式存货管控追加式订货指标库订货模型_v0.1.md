# 逐字精读卡片集 · 批40 库存管理系列11-16（数量+位置/推拉供应链/存货管控/追加式订货/指标库/订货模型/ROI收口）

> 📌 本批 6 篇为**库存管理系列收官段（2022年2-3月）**，承批39（库存05-10 的安全库存定量底座）。主线：库存管理两个点（数量+位置·11）→存货管控落地（12）→**反误区核弹「追加式订货=慢性毒药」（13）**→库存指标库+商品状态（14）→订货模型与安全库存集成（15）→**ROI 一句话收口（16）**。本批是 [[数量管理]] 的**指标库 + 订货模型 + 对客户最有力的一句话**。

## 235. 库存管理系列(11) ⭐⭐⭐ 〔库存两个点/推拉供应链/库存3影响〕
- **核心论点**：**库存管理就两个点=数量 和 位置**（建模/量化只关注库存结构=位置、数量=不是销售额也不是毛利）；所有库存相关流程都围绕这两点铺开成线和面=系统性库存体系。**供应链两种分类标准=库存动力来源 / 库存需求特性**。按动力来源分三类：**推式**（总部配送主导·上游往下推·需求信息总部→门店；前提=需求稳定+需求可分析；本质=根据过去经营数据定未来库存策略·数据重数量非金额；工具=**52周系数法/每周系数法**：以最低值或平均值为基准系数1、其他周相除得系数→分解销售指标）；**拉式**（门店/需求部门主导·报货多少配多少·貌似"以顾客为导向"但易演变成**营采矛盾**：要的没有不要的一堆 vs 看你陈列成啥样）；**推拉结合**（理想丰满现实骨感·鲜有成功案例·根因=没把时间花在正道：知识能力/技术运用/组织效率）。**库存3个核心影响**：①**资金消耗（最优先）**——沃尔玛"账期盈利"模式；结账周期<销售周期=货没卖钱已付=现金转不良存货=现金流枯竭。②仓库资源占用（不良库存=2倍损耗·再多仓库也解决不了好卖缺货）。③配送及物资管理（最不重要但不可忽视）。
- **关键词**：库存就两个点=数量+位置 / 供应链三模式(推式=过去数据+52周系数法 / 拉式=营采矛盾 / 推拉结合鲜有成功) / 库存3影响(资金=沃尔玛账期·最优先 / 仓库2倍损耗 / 物资)

## 236. 库存管理系列(12) ⭐⭐ 〔存货管控5方面·满陈列+再订货点〕
- **核心论点**：财务视角下库存=一切有价值资产(设备/厂房/人/时间/加工/资源)，库存控制上升为"存货管控"。**存货管控5方面**：①**账实相符率=账务相符单证数÷库存货物总单证数×100%**（无企业100%相符·靠清仓盘点+实盘切账）；②出入库单据管控（盘点前先查单据流转·人为错误多）；③**存量管控**（目标=合理库存非零库存）：根本解决采销量异常(以销售需求为原则按需采购)+**实现卖场合理满陈列、内仓零库存**(周转库存取低尾数法：先购基本量再少量补安全数)→引出**周转库存**概念(比安全库存更合理、更易理解运用、更利现金流通)+必要控制设置=**满陈列量 + 再订货点**(库存管理基础·很多企业连最基础设置都没做)；④**库存异常管控**(7种/9种异常·以滞销为例：判定=库存>0 且 DMS未销天数 百货≥45天/杂货≥25天/生鲜百货≥7天·系统一般不含滞销功能)；⑤**供应链库存管控**：外部=**VMI供应商管理库存**(所有权归供应商·共建VMI专区·信息共享·企业只投放订单/提供需求预测要货申请)，内部=执行品类管理优化结构(减不必要品类单品·采购重点放畅销品)。
- **关键词**：存货管控5方面 / 账实相符率公式 / 存量管控=满陈列量+再订货点(库存管理基础) / 周转库存>安全库存更易用 / 滞销判定=库存>0+DMS天数标准 / VMI供应商管理库存

## 237. 库存管理系列(13) ⭐⭐⭐ 〔报损率反思/追加式订货=慢性毒药·全系列最警醒〕
- **核心论点**：**①报损率=报损金额/销售金额×100% = 库存管理中"最失败的东西"**——损耗与销售非强关系，**强关系应是成本**；销售好≠赚钱，良性销售好=库存管理好=损耗应更小，硬定高损耗指标=逼好店造损耗（又一个"皇帝的新装"）；须考虑成本+库存金额+批次；行业标准总盘亏率≤4‰但总指标本身就值得商榷。**②追加式订货(补充式订货)=库存管理中最可怕之处=慢性毒药**（不会马上致病·长期下去卖场一成不变→客流失→难逆转）。本质=库存动态需求问题的具体变现。**🔥7-11铃木敏文《巷口商学院》X/Y/Z例**：X进50卖30、Y进30卖20、Z进10卖10——只看销量X最多。但分析速度：**Z一小时卖完=真畅销(销量却最少)，X/Y可能是Z卖完后顾客无奈购买=实为滞销**。只看销量做追加式订货→真畅销Z订少缺货、X/Y过订报废(2倍误差)。**③畅滞销分析**：畅销品给"五优先"(优先采购/仓储配送/上架陈列/促销/结账)；畅销品具季节性须随时调整(季节规律性最强·调整准确性最高)；订货系数=畅销品1→1.8、一般品1、淘汰品1→0.2甚至不订=**该多订的多订、不该订的少订**。**④"以不变应万变"→"以万变应万变"才是订货目标**：**"昨天的顾客"≠"明天的顾客"**——以销定货下的是"过去需求"的订单，要更多思考"明天的顾客需求"；良好订货以**品类与商品结构优化为前提**(50个SKU只动销10→先优化结构再订货)；必需品≠畅销品(配置属性vs销售属性)。
- **关键词**：报损率强关系=成本非销售(销售好损耗应更小) / 追加式订货=慢性毒药(库存动态需求问题的具体变现) / 7-11的X/Y/Z(卖得多≠畅销·真畅销看速度) / 五优先 / 订货系数(畅1.8/常1/淘0.2) / 昨天顾客≠明天顾客·订货以结构优化为前提

## 238. 库存管理系列(14) ⭐⭐⭐ 〔库存上下限/商品状态/库销比与可销天数/平均库存/库存收缩·指标库上〕
- **核心论点**：两核心=①最优库存量(=总费用最低=进出货费用+各项库存维持费用)②最大限度解决需求变动。**库存上下限**(实操替代财务最优·但只从数量考虑未计毛利→高毛利X应比低毛利Y有更高上下限)：订货点=库存下限·补货数=上限−现有库存；**R=库存费用/订货费用**(R>1不存货·R≤1存货)；前提=**商品优化**(先控类别SKU数·马上淘汰的商品还设上下限会触发不合理订单)。**需求变化**：今天订单要考虑未来需求·畅销品最易缺货(消极选择=顾客价值损失)·商品生命周期(导入期3-5周试销/成熟期更多订货)·**服装类售罄率50%(尺码齐全期=真实销量)→还原真实排名**·其他类关注**适销率(有销售商品数/总单品数·适合新品分析/采购绩效)**·二八原则定A类。**🔑商品状态管理=提升商品管理效率的必由之路**：商品6状态(建档/新品/正常/暂停/淘汰/已删除)+销售状态(畅/平/滞)+ABC；库存2大属性=**位置和数量**；**合理的库存管理=商品位置和数量属性的合理**(位置广义=合理存放地点+合适时间+合适价格+美陈氛围=商品力·超市三驾马车之一)。**指标库**：**库销比率=月初库存量/月销售量**(过低缺货损顾客·过高库存大/结构不合理资金利用低·须细化到小类)；**库存可销天数=现有库存/日均销量**(还能卖几天·即时状态·标准30天而库存45天=多15天大库存)；**WOH远期可销售周数(卖完当前库存需多久) vs FWOS当前可销售周数(计入计划销量·基于时间非库存水平·订货点依交货期)**；强势商品一旦缺货势头难回升。**平均库存=(当前库存+上期库存)/2**(有缺陷但唯一可拍脑袋的指标)；**库存损耗=库存收缩Shrinkage**(应存而实际不存·盘损多少=相关人员管理水平及责任感·强防损机制预示其他经营也良好)。
- **关键词**：库存上下限(R=库存费用/订货费用·前提=商品优化) / 售罄率50%·适销率(有销品数/总单品数) / 商品6状态·合理库存=位置+数量属性的合理(=商品力) / 库销比·可销天数=库存÷日均销·WOH/FWOS / 平均库存·库存收缩Shrinkage

## 239. 库存管理系列(15) ⭐⭐⭐ 〔库存控制因素/ABC订货系数/EOQ/双堆三堆法/定量定期模型/确定安全库存·订货模型集成〕
- **核心论点**：**库存控制影响因素**=需求量(现有+未来动态)/订货周期(提前期)/订货费用/库存费用/缺货损失。**是否存货=R=库存费用/订货费用**(R>1不存货·R≤1存货·财务现金流角度)；库存费用易被忽略(人工/仓库/物流)；费用计算须用每天变化(点时计算误差极大·例两天129 vs 28)。**ABC订货系数**：A类1.5-1.8倍·B类1倍·C类0.2-0.5倍(一刀切正常订货→不好卖一堆好卖缺货)。订货系统三类=固定间隔期/固定量/最大最小系统；一次性订货(生命周期固定/时尚/易腐/一次性·按最长提前期+最大盈利)。**EOQ经济订货批量图三参数**：经济订货批量(订多少)/订货点(何时订)/订货周期(间隔多久)。**双堆法/三堆法**(定量订货简洁法·无须经常盘点)：分两堆(订货点量+其他量)先用第二堆·用完即达订货点补货；三堆=另加安全库存/周转库存。**定量模型(事件触发·数量定时间不定) vs 定期模型(时间周期定数量不定)**。**安全库存=随机型独立需求控制方式**(与当前销量/消耗/订货量无关)：EOQ解决平均需求·安全库存解决随机需求分布变动。**确定安全库存**：贵在安全非越大越好·大小由"缺断货造成的利润损失"决定(难获得→用服务水平概率值替代)；**安全库存R=需求量标准方差∂×缺货概率安全系数A**(∂=日均销量·A=服务水平系数·95%=1.65)；系数不能乱给(满足率80%不到却用高A→C类积压)。**🔑任何库存控制模型都建立在"允许缺货"前提**；核心=在库存费用增长成本 与 缺货损失成本 之间找平衡点(控制模型里最难设置的一点)。
- **关键词**：是否存货R=库存费用/订货费用 / ABC订货系数(A1.5-1.8/B1/C0.2-0.5) / EOQ三参数(订多少/何时订/间隔多久) / 双堆三堆法(无须常盘点) / 定量(数量定)vs定期(时间定)模型 / 安全库存R=∂×A / 任何模型都建立在允许缺货=找平衡点

## 240. 库存管理系列(16) ⭐⭐⭐ 〔库存管理 ROI 一句话·全系列收口〕
- **核心论点**：**库存管理价值的一句话总结：每一个单品，其良好库存管理的现实意义在于——商品账期内现金流产生的净利润能提高约 1%。** 超市毛利率基本13-25%、纯利约3个点，若每个部门/类别/单品都有良好库存管控→**能提高近 1/3 的利润来源**。反讽：我们在库存管理上用到了 1/3 的人力物力财力吗？没有——**每周一次的营采扯皮吵架的时间，都比做库存管理花的时间多**。
- **关键词**：库存ROI一句话=好库存管理使单品账期内净利润提高~1% / 纯利3点→提升近1/3利润 / 反讽:营采扯皮时间>做库存管理时间
