# 逐字精读卡片集 · 批27 可得性/SKU定量/VMD体系/3P

## 157. 品类建模(173) SKU可得性 ⭐⭐
- **核心论点**:可得性=**三重维度(时间/空间/状态·缺一则可得性为零甚至为负)** + **四层次递进:①物理可得(=有货·库存/陈列量·但有货≠可得,过量库存=占现金)②系统可得(=有数·账实相符·数据滞后/错误=虚假可得)③感知可得(=顾客看得见/找得到/拿得到·最隐蔽的损失·零售老大难)④即时可得(=动态体验层·高峰不断货/30分钟达)**。可得性=**连接效率(供应链成本端)与价值(顾客体验端)的利润枢纽**。顾客满意度≈Σ(需求强度×可得性);利润=销售机会−机会损失(机会损失=需求未被满足的潜在销售额)。
- **关键词**:可得性三维度(时间/空间/状态) / 四层次(物理/系统/感知/即时) / 有货≠可得 / 感知可得最隐蔽 / 连接效率×价值

## 158. 品类建模(174) SKU新逻辑 ⭐
- **核心论点**:**SKU 需求的宏观传导**(倒金字塔:底层经济结构→中层消费市场→顶层零售SKU结构)。**实体店真死穴=目标客群的收入/时间/压力约束("时间贫困")**——"花钱购物从来不是想不想,而是敢不敢、有无必要";年轻人"宅家玩游戏>逛店",家庭主妇/老人/牛马班占比升。SKU 优化新逻辑=**更精(减决策负担)/更准(柔性匹配新需求)/更活(动态响应)/更深(场景关联)**;从"大而全"→"精而准";从"被动响应"→"主动设计"。
- **关键词**:SKU宏观传导(倒金字塔) / 时间贫困=真死穴 / 敢不敢有无必要 / 更精/更准/更活/更深 / 大而全→精而准

## 159. 品类建模(175) SKU鄙视链 ⭐
- **核心论点**:零售业态隐性"鄙视链"(高端精品>会员仓储>标准社区>大众夫妻店)=**消费社会分层在零售业态上的投影**;核心维度=**效率/时间/身份(符号)/情绪**。每个业态在链上找自己的"**生态位**"(非优劣排序·"都是卖东西并无真鄙视链,只是需求不同的选择")。终点=**关系**(交易→关系:底层生存/中层效率/高层身份/**顶层尊重**);**胖东来="异端"**(以平等尊重关系去中心化连接·不靠最贵也不靠最低·卷"人本"关系)。"物以类聚"=逛哪类超市=自己当下生活状态的映射。
- **关键词**:鄙视链=社会分层投影 / 效率/时间/身份/情绪 / 生态位非优劣 / 顶层=尊重(胖东来异端) / 交易→关系

## 160. 品类建模(176) SKU的定量 ⭐⭐
- **核心论点**:合理 SKU 数=**多维约束下的最优解**("刚刚满足需求的单品数")。**选品三部曲=需求确定→品类定位→SKU 选择**(需求被发现→被体现→被选择)。定量 **5 方法:物理空间法(货架长/面宽)/资金效率法(周转×成本)/需求覆盖法(品类角色)/边际效益法(品效拐点)/综合优化法(ABC/规划求解)**——多法对比就能粗判"哪个小类 SKU 偏高"(如物理空间法值显著高于其他=占位偏大)。**边际效益平衡点=增 1 SKU 边际收益=边际成本**。6 运用(定总量/ABC优化结构/品效防臃肿/库存差异化/价格带/动态一进一出:**新品占比≤15%·每月强制淘汰 5-15%**)。
- **关键词**:合理SKU数=多维最优解 / 选品三部曲(需求→品类→SKU) / 定量5方法 / 边际效益平衡点 / 一进一出·新品≤15%·月汰5-15%

## 161. 品类建模(177) VMD体系 ⭐⭐
- **核心论点**:**VMD(Visual Merchandising 视觉营销)= 五适/六适原则(合适的时间/地点/价格/方式/商品/人)的实现手段**。结构:**VMD = SD(空间设计) + MD(商品企划) + MP(陈列规划);MP = VP + PP + IP(=3P)**;实践中 VMD ≈ MP(陈列)。三层比喻=**骨架(SD空间·舞台)/血肉(MD商品·核心内容)/神经(MP陈列·导演手法)**。购买心理 **A.I.D.C.A**(注意/兴趣/欲望/确认/行动)。**IP 的 I = Item(单品≠SKU/Product)**=单品排面设计。陈列目的=把无序杂乱→规范清晰(简单/标准/规范/清晰化)→提**坪效与时效**。磁石理论(第一~第四磁石商品)。
- **关键词**:VMD=五适原则的实现手段 / VMD=SD+MD+MP·MP=VP+PP+IP / 骨架血肉神经 / AIDCA / 陈列=无序→规范提坪效

## 162. 品类建模(178) 3P与三原则 ⭐⭐
- **核心论点**:陈列**三大变化原则**(外部因素本质=需求变化驱动):**①决策便捷性(从繁杂选择→快速决策·治"选择瘫痪")②场景化消费(从品类导向→场景导向·早餐区/火锅区)③体验与效率并重(线上线下融合)**。3P 映射:**VP≈决策便捷 / PP≈场景化 / IP≈体验效率**。陈列从"**以商品为中心**"静态摆放 → "**以顾客为中心**"动态响应系统。落地 5 场景(品类→场景导向/货架从丰满→易读/动线从强制延长→高效直达+磁石点/数字化电子价签/服务嵌入试吃)。⚠️ 陈列价值靠长期积累,非短时"模仿跟风"。
- **关键词**:陈列三大原则(决策便捷/场景化/体验效率) / VP/PP/IP映射 / 以商品为中心→以顾客为中心 / 5场景 / 反模仿跟风
