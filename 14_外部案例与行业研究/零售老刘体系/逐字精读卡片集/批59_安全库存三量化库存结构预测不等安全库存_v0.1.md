## 1. 分析建模(31) 安全库存建模(3)·库存结构 ⭐⭐
- **核心论点**:**呆滞库存=区别库存管理好坏的关键**(暂时不用或永远没机会用的风险库存);降呆滞=找超一年无需求的商品清掉。
  - **库存策略**:① 采购成本模型(**采购技术>采购技巧**·成本意识到小类/单品)② **双轨制优胜劣汰供货**(同单品两家供货·好的多订·**订货量=(质量÷价格)×关系**·关系系数0-1·结合供应商生命周期:淘汰率<10%/送货准确率>90%给高系数)。
  - 降呆滞两法:① 库存ABC分类(A类安全库存高/BC低)② **降提前期**(**安全库存∝√提前期·管道库存∝提前期**·缩短采购周期/系统实时触发订单/供应商协同)。
  - **安全库存简易Excel**:`安全库存=(预计最大需求量-平均需求量)×采购提前期`。最大=平均→安全库存0(需求固定不用);偏差越大安全库存越高。订货量取订货单位整数倍(CEILING.MATH/FLOOR.MATH)。
  - **⭐库存结构三类**:**周转库存(良性)+安全库存(中性·易倒向呆滞或良性)+呆滞库存(不良)**;**健康库存占比越大=库存健康度越高**(可持续提升周转率)。
- **关键词**:呆滞库存=管理好坏关键 / 采购技术>技巧 / 双轨制供货(质量÷价格×关系) / 降提前期(安全∝√提前期) / **库存结构三类(周转/安全/呆滞·健康占比=健康度)**

## 2. 分析建模(32-4) 安全库存建模(4)·三量化 ⭐⭐⭐
- **核心论点**:不可预测=供应波动+需求波动;安全库存与三因素:供应稳定性(LT)+需求稳定性(D)+服务水平。
  - **⭐安全库存建模三量化**:① **量化需求不确定性**(历史标准差STDEV·正态;不符正态用泊松)② **量化供应不确定性**(交货期标准差·订单发出→收货入库)③ **量化服务水平**(有货率=需求产生时立即有库存满足的概率)。
  - **⭐服务水平=Z值**:不放安全库存→**50%有货率**;放1个标准差(Z=1)→提高34.1%→**84.1%**;**95%有货率→Z=1.64**(`NORMSINV(0.95)`)。**100%满足始终是最高目标**(但要"合适的度"·最后零点几%要很多标准差→库存高周转低);可在流程不同环节设不同服务水平。
  - **泊松算再订货点**:`POISSON.DIST(X,平均值,True)`·X=再订货点。例:月销服从参数5泊松·0.999%不缺货→λ=5·0.0001对应16=库存。
  - **⭐安全库存计算**:综合需求供应标准差×Z值;简化(供应周期恒定):**`SS=Z×σF×√(L/FP)`**(σF需求标准差·L平均补货周期·FP预测时间单位)。
- **关键词**:**三量化(需求/供应/服务水平)** / **服务水平=Z值(不放50%·1σ→84.1%·95%→Z=1.64)** / 100%满足=最高目标但要度 / 泊松算再订货点 / **SS=Z×σF×√(L/FP)**

## 3. 分析建模(32-5) 安全库存建模(5)·提高预测≠安全库存(完) ⭐⭐⭐
- **核心论点(核心结论)**:**⭐提高预测≠安全库存**——把100安全库存分摊到10周(每周+10)→门店按110报货→库存多备10%利用率低;且分摊的安全库存会被系统当"过高预测"删掉→实际安全库存远低于应设值。安全库存=永久需求(低于水位触发补货);预测=可有可无(没用就删)。
  - **⭐需求预测对付平均需求·安全库存对付不确定性=两个概念应分开处理**(核心结论·呼应批39)。
  - **安全库存设置(趋势/季节性时)**:不能直接取DMS求标准差(会过高预估)→**摘取15-30周需求历史·算每周预测·围绕预测误差算标准差**。**⭐需求变动性=预测的误差**(预测总不准·安全库存就是应对预测误差)。删异常数据;预测分解算每期误差再叠加。
  - **⭐数据点要求:≥30个(一个月)最可靠·实操不少于14个(2周)**。简化:把平均值(日均销量)当预测值·预测误差=实际最大需求-平均值·围绕差异求标准差(简单直观不用保留预测历史)。
- **关键词**:**提高预测≠安全库存** / **需求预测对付平均·安全库存对付不确定性(分开处理)** / 需求变动性=预测误差 / 围绕预测误差求标准差 / 数据点≥30(≥14)
