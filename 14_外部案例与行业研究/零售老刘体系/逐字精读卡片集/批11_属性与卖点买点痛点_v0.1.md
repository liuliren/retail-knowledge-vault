# 逐字精读卡片集 · 批11 属性结构与卖点买点痛点

## 61. 品类建模(58) 属性结构层
- **核心论点**:商品组合维度单元链=**SKU→属性→商品组合→商品群(需求单元/品类)**;需求▶需求单元▶选择=闭环。**点线面逻辑**(点=方法/线=模式/面=经营层)。商品结构层关键=**"适合"**(过度高大上会夭折/过简丧失盈利机会)。
- **关键词**:SKU→属性→组合→商品群 / 需求-选择闭环 / 点线面 / 结构=适合

## 62. 品类建模(59) 属性匹配
- **核心论点**:商品组合须匹配当地消费群体(关联性越强意义越大)。**同一商品在不同店定位不同**(橄榄油:小超市=高档品/标超=结构性商品)。[作者自述:工程约200小节/70万字]
- **关键词**:匹配当地消费群 / 同品不同店定位 / 关联性

## 63. 品类建模(60) 表现类型
- **核心论点**:商品组合评估3指标=**SKU评估值/库存评估值/销售评估值**(同比环比层级占比)。组合商品系统状态:**捆绑/制单组合/制单拆分/自动转货/自动加工**;一品一码;正常品vs组合商品。
- **关键词**:评估3值(SKU/库存/销售) / 组合商品5状态 / 一品一码

## 64. 品类建模(61) 卖点与买点 ⭐
- **核心论点**:**卖点(USP·Unique Selling Proposition·卖方视角·商品独特优势)vs 买点(FAB·Feature-Advantage-Benefit·买方视角·顾客真正关注/决定购买的原因)**。卖点≠买点 = **门店视角vs顾客视角错位**(有机认证案例:门店强调有机/顾客只关心价格包装)。求同存异;卖点与买点切合时才有意义。
- **关键词**:卖点=USP / 买点=FAB / 错位=门店vs顾客视角 / 切合才有意义

## 65. 品类建模(62) 卖点提炼 ⭐
- **核心论点**:错把卖点当买点→顾客不买单(316不锈钢案例:门店强调材质/顾客关心保温24小时)。**卖点=商品独有(零售商视角·定位/品牌/创新)·买点=顾客感知的场景化价值利益(顾客视角·介绍/话术/陈列/沟通)**。零售经营一切关于"卖点"(门店/大类/货位/服务/促销卖点)。
- **关键词**:错把卖点当买点→不买单 / 卖点=独有·买点=感知利益 / 场景化价值

## 66. 品类建模(63) 痛点 ⭐
- **核心论点**:**痛点=顾客未被满足的刚性需求/迫切要解决的矛盾**("现在立刻马上"要解决);基于顾客角度痛点≈卖点。**找痛点诀窍=洞悉替代方案("转身层"配置=有替代痛点的其他商品)**;**不能把伪需求当痛点**(把梳子卖给和尚=伪需求·非刚需)。生鲜线上搞不起=没解决"最后一公里"刚需痛点。
- **关键词**:痛点=未满足刚需 / 现在立刻马上 / 转身层=替代痛点商品 / 伪需求警惕(梳子卖和尚)

## 回填候选(待签)

- **转身层(替代痛点商品)** → 接 [[SKU角色层]]:替代品/防缺货的连带角色(配可得性)。
- **伪需求警惕(梳子卖和尚)** → 选品纪律:先验证是不是真刚需痛点再选品。
