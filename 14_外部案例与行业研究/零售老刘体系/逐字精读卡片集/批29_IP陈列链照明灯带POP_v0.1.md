# 逐字精读卡片集 · 批29 IP陈列链/照明/灯带/POP

## 169. 品类建模(185) IP陈列链 ⭐⭐
- **核心论点**:商品组合双闭环(商品→属性→标签→组合→品类 + 配置选品环)。**陈列两大基本元素=位置 + 数量**;现实是 **80% 精力花在"位置",而"数量"被漠视 → 陈列最多做到 50-60%**(基数低还自以为做好了)。**70% 左右购买=冲动性购买**(无清晰计划·购物篮在现场驻留中决定·**决策平均仅 19 秒**);购物篮两类(拿了就走 Grab-And-Go / 冲动 Impulse·**80% 陈列努力为冲动购买**)。陈列四层面(门店级平面布局/品类级货架布局/商品级货架组织/建设级追求)。**陈列="背包问题"**(有限空间 × 几乎无限需求·按商品价值最大化取舍·比背包多"位置"维度)。
- **关键词**:陈列两大元素=位置+数量 / 数量被忽视→陈列只做50-60% / 70%冲动购买·决策19秒 / 背包问题

## 170. 品类建模(186) 卖场照明 ⭐
- **核心论点**:**照明=超市设计第四大核心要素**(商品/陈列/动线/照明)。国标 GB50034-2013(**Ra≥80**·照度均匀度≥0.6);**生鲜灯禁用**(有色光改变真实色泽·侵犯知情权)。**三层照明:基础照明(环境光·均匀·卖场必备)/重点照明(商品光·射灯·制造视觉焦点·越来越重视)/装饰照明(氛围光·塑风格品牌)**。生鲜区=照明重中之重(高显色/恰当色温·**烘焙暖光、海鲜冷光**·堆头环境光:重点光≥1:3)。货架区=垂直照明(照立面非地面)。收银区(功能 600-800lx/引导 1000lx 灯塔/避眩光)。入口消"黑洞效应"。
- **关键词**:照明=第四大要素 / Ra≥80·生鲜灯禁用 / 三层照明(基础/重点/装饰) / 烘焙暖光海鲜冷光 / 货架垂直照明

## 171. 品类建模(187) 灯带纵横 ⭐
- **核心论点**:灯带两种布局:**平行于货架(精准照立面·效率高均匀·首选·但货架须固定)vs 垂直于货架(灵活调整·但货架顶阴影/立面照度仅200lx·权宜之计)**。**垂直照度 : 水平照度 ≥ 2:1**(地 300lx → 货架 600lx+)。配光:蝙蝠翼(通道中央)/洗墙(靠墙货架)。**垂直复合式布灯**(线性灯带基础 + 轨道射灯重点)。黄金区 70-160cm 加重点照明。**照明梯度防黑洞效应**(店外 500-800 → 入口 800-1000 → 主道 600-800·"先亮后平")。
- **关键词**:平行灯带优先vs垂直权宜 / 垂直:水平照度≥2:1 / 蝙蝠翼/洗墙配光 / 垂直复合式布灯 / 照明梯度防黑洞

## 172. 品类建模(188) 零售POP ⭐⭐
- **核心论点**:**POP = Point Of Purchase 购买点广告 = "与顾客最后沟通的桥梁"/"无声推销系统"/超市"神经末梢"**。核心=在正确位置/用正确信息/以正确时机/促成正确购买决策。**三大支柱:①决策漏斗"最后一米"**(进店100%→商品区60%→注意30%→购买欲15%→比较8%→放购物车5%·POP 把 5% 提到 8-15%)**②信息过载的"信号枪"**(POP信号强度=色彩对比×形状异质×位置优越×信息简洁)**③从"告知"到"说服"层次递进**。**极简主义**(一张POP一个核心信息·主语商品+谓语利益+宾语行动·吊旗≤12字/爆炸卡≤8字/跳跳卡≤6字·每远1米字号×1.5-2)。心理学(损失厌恶/社会证明/锚定/互惠)。距离法则(POP与商品≤30cm)。合规(禁"全网最低"等绝对化用语)。
- **关键词**:POP=购买点广告=神经末梢 / 决策漏斗最后一米 / 信号枪公式 / 极简主义(一POP一信息·字数限制) / POP与商品≤30cm

## 173. 品类建模(189) VMD的POP ⭐⭐
- **核心论点**:**POP = VMD 的"道具 + 台词"**(VP舞台/PP导游/IP景点手中的道具)。**VMD=无声导演 / POP=有声推销员**。POP 视觉三层次:**吸引(引·色彩/字体/动态)→ 传递(说·卖点/筛选标识)→ 促成(促·限时限量/社会证明)**。POP 按大中小对应 VP/PP/IP(大型POP→VP入口吊旗/中型→PP端架海报/小型→IP货架价签爆炸卡)。**三秒法则**(顾客扫 POP 仅 3 秒·字数≤8-12)。颜色心理学(红=促销·一卖场≤3种色)。一致性(与 VMD 调性统一)。**POP本质=让商品"开口说话"·无POP陈列=哑巴式陈列**。手写POP的亲切感仍有独特吸引力。
- **关键词**:POP=VMD的道具+台词 / 无声导演+有声推销员 / POP三层次(吸引/传递/促成) / 大中小POP对应VP/PP/IP / 三秒法则

## 174. 品类建模(190) 卖场POP ⭐
- **核心论点**:**卖场POP=全店级"视觉发声系统"**(品牌/主题/动线/氛围·不绑定单品)**vs 陈列POP=点位级"销售助攻工具"**(绑定单品/卖点)。卖场POP视觉规律(**中心偏上 + 从左向右**·左上部最易吸引视线)。卖场POP三类(商品维度类/引导指示类/企业宣传类)。运用步骤=**设计→统一→引导→营造→强化**(先统一后引导再强化)。五原则(一致性/清晰性/层次感由大到小由远至近/时效性过期撤/空间性吊旗≥1.8-2.2m)。**"让顾客懂得"=卖场POP最核心底层逻辑**(小白都能秒懂)。陈列POP=卖场POP的组成部分(直接与商品有关的=陈列POP)。
- **关键词**:卖场POP=全店面vs陈列POP=点位 / 中心偏上+从左向右 / 设计统一引导营造强化 / 让顾客懂得 / 五原则
