# 逐字精读卡片集 · 批28 VMD运用/3P引导/IP正侧面/入口

## 163. 品类建模(179) 3P黄金角 ⭐
- **核心论点**:VMD 执行层 3P 对应顾客"**吸引▶兴趣▶比较**":**VP(视觉展示·吸引第一眼·店头/橱窗/入口/端架/主通道)→ PP(要点陈列·唤起兴趣让顾客停留·通道/货架主题)→ IP(单品陈列·方便比较选择购买·货架主体)**。比喻:**SD=舞台 / VP=磁石(造氛围吸引)/ PP=导游(品类引导讲解)/ IP=景点(找到满意商品)**。3P=**点线面构成**(VP面/PP线/IP点),模拟顾客注意力从宏观到微观的自然衰减(视觉熵增)。3P 焦点=任意组合(顾客视觉无迹可循·不固定为维度单元更利运用)。
- **关键词**:3P=吸引▶兴趣▶比较 / VP磁石/PP导游/IP景点/SD舞台 / 点线面 / 注意力视觉熵增 / 3P任意组合

## 164. 品类建模(180) VMD运用 ⭐
- **核心论点**:成功陈列=VP(布景吸引入场)+PP(剧情抓注意)+IP(台词促共鸣)三者协同,**忽略任一环=视觉流中断+销售流失**(如三一律缺一不成戏)。**黄金陈列位(Golden Zone)**放 A 类畅销/主力品。视觉层次感+**倾斜向上规律**(低→中→高陈列线·增视觉触点/通透性)。3P 动态调整周期(全店 VP 半年-1年/区域季月/品类货位周)。**VMD 最难=长期动态维护**(专家指导一时有效·能长期维持的几无);某区域/品类销售恶化→先换 VP/PP(须基于需求分析,非凭感觉)。服装 VMD 标准化好(品类少/属性简单),商超可建自己的样板。
- **关键词**:VP/PP/IP协同·缺一断流 / 黄金陈列位放A类 / 倾斜向上规律 / 3P动态调整周期 / VMD最难=长期维护

## 165. 品类建模(181) 3P引导性 ⭐⭐
- **核心论点**:VMD 引导=**无声的导航指挥系统**(规划顾客"看什么/往哪看/往哪走/停多久/拿什么")。**三大引导层层递进**:**①视线引导(纵向陈列对抗水平扫描惯性/横向视线走廊/焦点磁石点"截流")②动线引导("堵疏"控速度·黄金区走最慢非黄金区走最快/回字之字端架转向·治右转倾向死角/强制vs自由动线)③心理引导(价格锚点/关联需求链"既然吃烧烤怎能没薯片"/稀缺紧迫)**。三层=看的入口→行的路径→买的落点。**VP/PP/IP 一致性原则**(风格/主题/卖点买点要清晰一致=评价 VMD 的核心标准)+ 关联性原则。
- **关键词**:VMD=无声导航系统 / 三大引导(视线/动线/心理) / 纵向陈列对抗扫描惯性 / 黄金区走最慢 / 一致性原则=VMD核心标准

## 166. 品类建模(182) IP正侧面 ⭐
- **核心论点**:**正面=品牌标识/卖点/最大识别面(朝客流方向)**;侧面=窄面/次要信息(增容量/量感/库存深度·割箱陈列)。原则=**总展示正面**。分工:**正面负责"卖与买"(吸引转化)/ 侧面负责"存与序"(容量+秩序)**。**黄金视线层(70-150cm)100% 正面朝外 + 向前对齐**(库存不足时前移,不可转侧面/背面充数);底层高层允许侧面(增容量)。端架"双向正面"(以曝光率为主·单位面积销售 1.5-5 倍)。正面=主语/侧面=状语。
- **关键词**:正面=识别面朝客流 / 正面卖买·侧面存序 / 黄金层100%正面+向前对齐 / 端架双向正面 / 正面主语侧面状语

## 167. 品类建模(183) IP的单品层 ⭐⭐
- **核心论点**:**IP(Item Presentation 单品陈列)=占 60-80% 陈列面积、约 70% 销售 = VMD 的"基本面/压舱石"**。IP 核心=**秩序与效率("用极致的秩序换极致的效率"·去故事化纯功能)**。遵循**分类树(大类→中类→小类)= 隐形信息架构**,让顾客自助"导航→筛选→决策"(买苹果:水果区→苹果区→某种苹果)。**"重复"陈列**=安全感(库存足)+价格透明+便于比较。依据(选择悖论/分类本能·视线 F型Z型扫描·补货盘点固定位置)。**格子式 IP**(同品牌纵向占整列·横向比较品牌/纵向选规格)。执行标准:**二指原则**(间距能插两指)/正面向前/色块管理。IP="量"(单品度量)、VP/PP="质"(维度)。
- **关键词**:IP占60-80%面积70%销售=压舱石 / 极致秩序换极致效率 / 分类树=隐形信息架构 / 重复陈列三信号 / 格子式IP·二指原则

## 168. 品类建模(184) 入口新语义 ⭐
- **核心论点**:**传统入口=固定/被动/物理"位置思维"(地段+动线)**;**现代入口=即时/主动/动态"关系连接思维"**(触点即入口·流量/时间/场景/社群/会员多维碎片化)。运营从"人找货"→"**货找人**"(内容/算法/社群主动触达)。**入口=商业漏斗第一道**:`销售额 = 进店率 × 转化率 × 购买率`(漏斗·比例依次减少)。入口精细化=店外1米缓冲层→店内3米吸引层→进门5米决策层(吸引层用感官拦截/缓冲层用过渡降槛/引导层用磁石导航/决策层用首单转化)。检测指标(进店率/停留时长/转化率/右转率)。
- **关键词**:传统入口=位置思维vs现代=连接思维 / 货找人 / 入口=商业漏斗 / 销售额=进店率×转化率×购买率 / 缓冲/吸引/决策层
