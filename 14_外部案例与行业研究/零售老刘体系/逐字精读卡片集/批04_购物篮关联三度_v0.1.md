# 逐字精读卡片集 · 批04 购物篮思维与关联三度

## 19. 品类建模(16) 购物篮思维
- **核心论点**:购物篮=门店"脉搏心跳"。**销售额 = 客流 × 转化率 × 复购率 × 客单价**;客流趋常量(分蛋糕)、转化率/复购率也趋稳→**客单价是唯一变量(但靠涨价危险)**。CBA(顾客消费行为分析)切入点=购物篮;购物篮分析反哺良性循环。TopN排行解决不了"单品在多少篮出现/连带买了什么/为什么没买"。
- **关键词**:购物篮=脉搏 / 销售额=客流×转化率×复购率×客单价 / 客单价唯一变量 / CBA / 反哺良性循环

## 20. 品类建模(17) 购物篮思维(目标客群)
- **核心论点**:**单品式思维(爆品引爆)= 不知树上什么鸟 = 对需求茫然**;购物篮式思维=识别目标客群。**7-11 一天四次改陈列**(早上班族/中白领/傍晚下班/夜归各不同)。低峰时段(中午)往往是高消费"有钱一族"却被怠慢=思维失误。市调找客群案例(买菜老龄/带娃/退休溜达三客群)→渗透率/购物频次/客单价/品单数。
- **关键词**:单品式vs购物篮式思维 / 目标客群 / 7-11时段客群 / 低峰高客价 / 渗透率

## 21. 品类建模(18) 单品式vs场景思维
- **核心论点**:**生鲜引流失败案例**:整改了(动线/选品/促销)但**思维没变=还是单品式**→爆品昙花一现(车厘子/帝王蟹终将大众化)。关键=**让顾客往篮里多放单品(关联规则)**,做**场景化陈列**;每种关联=一种顾客场景。"菜市场超市化"=只把菜市场搬进超市(同质弊病);称重落脚点应在出口处。
- **关键词**:思维没变=整改失败 / 爆品大众化 / 场景化陈列 / 关联=顾客场景 / 菜市场超市化 / 称重出口

## 22. 品类建模(19) 购物篮关联三度 ⭐
- **核心论点**:购物篮分析(MBA)=研究让顾客往篮多放商品,主度量=**品单数(连带率)**。**购物篮5指标:客流/客单价/客单数/品单数(连带率)/品单价**。**关联三度**:**支持度(Support=A∩B/总订单)、置信度(Confidence=A∩B/A)、提升度(Lift)**;例可乐+方便面同现5W/10W=支持度50%。维度=SKU,度量=关联度。
- **关键词**:购物篮5指标 / 品单数=连带率 / 关联三度=支持度·置信度·提升度 / A∩B

## 23. 品类建模(20) 相关性
- **核心论点**:关联须**同场景分析**(同时段/同店/甚至同员工陈列)。三种相关:正/无/负。**相关性分析=零售最有效统计工具**(把杂乱数据建成简洁量化规律);相关性维度(内部布局/促销/选品/陈列+外部竞争/天气)+度量。AI替代不了"业务常识/商业逻辑解释能力"(没关键词引导AI不知场景)。
- **关键词**:同场景分析 / 正·无·负相关 / 相关性分析 / 维度+度量 / AI替代不了业务解释力

## 24. 品类建模(21) 零/负关联 ⭐
- **核心论点**:**被忽略的零/负关联藏机会**。①**孤品篮子**(单商品购物篮·占约1/3=缺失机会,顾客想买更多没买)②**孤儿商品**(放入篮但最后没买/丢角落·应建"孤儿商品登记表"找不买原因)③**零事务关系**(无销售/零毛利/零购物会员/周转0=隐藏业绩空间)④稀少/异常(中医式切脉=真需求切入点)⑤**长尾**(篮中无关联、无人问津的商品)。
- **关键词**:零·负关联藏机会 / 孤品篮子=缺失机会 / 孤儿商品登记表 / 零事务=隐藏空间 / 中医式切脉 / 长尾
