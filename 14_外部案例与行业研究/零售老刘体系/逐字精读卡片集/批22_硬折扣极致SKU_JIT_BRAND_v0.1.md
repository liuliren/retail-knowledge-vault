# 逐字精读卡片集 · 批22 硬折扣/极致SKU/JIT/BRAND

## 127. 品类建模(143) 供应链履约 ⭐
- **核心论点**:**供应链履约=从下单(结账)到交付的端到端全过程**(库存/拣货/打包/配送/售后);**端对端=供应链最大特征**。履约成本影响定价(硬折扣靠极致供应链效率降履约成本);前置仓/社区团购/半小时达=速度与成本平衡。
- **关键词**:供应链履约=端到端 / 履约成本影响定价 / 前置仓/社区团购

## 128. 品类建模(144) 商品硬折扣 ⭐⭐
- **核心论点**:**极致SKU=主做"短尾"非"长尾"**(奥乐齐<2000·甚至<1000;一般店3500-5000)。逻辑=减少选择/提升周转/压缩成本/**高动销→供应链反应快**(二八法则)。自有品牌OEM覆盖80%需求·便宜2-3成。**硬折扣(系统性重构供应链)vs软折扣**;简约不简单=最优配比。
- **关键词**:极致SKU=主做短尾 / 减选择提周转 / OEM自有品牌覆盖80% / 硬折扣vs软折扣

## 129. 品类建模(145) 供应链转型
- **核心论点**:零食量贩成功3因素(适应需求变化/传统商超迟缓/灵活销售供应)。**餐食(主食/副食/辅食·固定时间)vs零食(临时不固定·适应人群更多)**;90/00后对餐食依赖降→零食盛行;传统超市对零食需求反应迟钝。
- **关键词**:零食量贩3因素 / 餐食vs零食 / 90/00后零食化

## 130. 品类建模(146) JIT供货 ⭐⭐
- **核心论点**:**JIT(Just-In-Time准时制)=在需要的时间、按需要的量供应**(丰田TPS核心·消除浪费/提效/降库存);接近**零库存**;**小批量、高频次补货**(面包每日2-3配);与供应商深度协作(共享需求数据)。售完即补,非提前囤积。
- **关键词**:JIT准时制 / 接近零库存 / 小批量高频次补货 / 供应商深度协作

## 131. 品类建模(147) ALDI基因 ⭐
- **核心论点**:**选品=企业战略基因**(非短时策略·非偶尔能为)。ALDI无形因素(核心竞争力/价值观/不成文原则)难复制;"好品质、够低价";真正重要的是看不见的无形约束。
- **关键词**:选品=战略基因 / ALDI无形因素难复制 / 好品质够低价

## 132. 品类建模(148) BRAND原理 ⭐⭐
- **核心论点**:**四大品牌:NB(全国性/一线)/LB(区域性/地方)/SB(限定性/店内独有)/PB(自有/白牌)**。**BRAND=零售商主导的多品牌生态组合**(每品牌有战略角色/品类角色);BRAND配置图(最大化单位面积产出+强化品牌定位/忠诚度);品牌=品类的战略属性单元。
- **关键词**:四大品牌NB/LB/SB/PB / BRAND=多品牌生态组合 / 品牌=品类战略属性单元
