# 逐字精读卡片集 · 批47 目标拆分·盈亏平衡·维度系数(非财务预算)

> 📌 **本批 = 「非财务预算模式」01-04**(2023年·非财务预算=目标/指标/任务)。定量+管理理念混合。**金矿=盈亏平衡(保本销售额·客户诊断直接可用)+目标拆分维度系数法**。
> ⚠️ 下一批续读 非财务预算 05-07(动态参数/预测增长率/品类指标)。

## 1. 非财务预算(01) 目标制定:不存在"合理目标" ⭐⭐
- **核心论点**:**从来不存在所谓"合理的目标"**——N 种方法能算出 N 种"合理"目标(老板想要多少就能"分析"出多少),目标常沦为托辞/鸡肋。解法=**将抽象业务量化(度量+维度)**→让目标制定者与执行者**能互相说服**→才是可执行目标。
  - **6种目标制定法**:拍脑袋(专家意见)/同行标杆/**化零为整(单店单类加总·较科学)**/市场份额/时间序列(历史增长率)/总值拆分(总目标按维度分解·配52周系数)。
  - **三层角色**:决策层(完成目标的策略应先于目标·不爬高讨好老板)/中层执行者(**分解到可执行最小单位**)/基层实现者(分解到最小单位如每个SKU·清楚"我能完成多少")。
  - **目标拆分=维度系数**:总体=1,各维度影响=总体×(小于1的比值系数)。时间维度(**时段系数法**·12月系数/52周系数)/空间(区域门店占比)/属性(品类/渠道/会员)/维度结合(+新品权重)。
  - **两拆分原则**:① **MECE**(完整性不漏+独立性不重·各项相加=总任务)② **SMART**(S明确/M可衡量/**A可实现(最难)**/R相关/T有时限;难在 A→制定目标同时要给**达成路径和资源**)。
- **关键词**:不存在合理目标·量化才能互相说服 / 6种目标法(化零为整较科学) / 维度系数(总体1分解·时段52周) / MECE+SMART(难在A·要给路径资源) / 分解到最小单位SKU

## 2. 非财务预算(02) 盈亏平衡分析 ⭐⭐⭐
- **核心论点(客户诊断直接可用)**:**盈亏平衡点=收入刚好=总成本时的销售额**(收入=总成本+利润;盈亏平衡时利润=0)。
  - **Excel保本算法**:费用率=总费用÷总销售;**盈亏点销售额=费用÷毛利率**。实例:月均费用24039元÷费用率16.52%=**145833元/月保本销售额**;**经营毛利率必须>费用率16.52%才盈利**。
  - **Power BI 调节仪表盘**:把各项费用(固定成本+变动成本)做成滑杆参数,拖动看对盈亏平衡销售额的影响(如工资费用率不超6%)→各部门据测算协同改善利润。
  - **完整公式**:**盈亏平衡销售额 = 固定成本费用额 ÷ (1 − 变动成本率)**。
- **关键词**:**盈亏平衡=保本销售额** / **盈亏点销售=费用÷毛利率** / 毛利率必须>费用率才盈利 / 盈亏平衡销售额=固定成本÷(1-变动成本率) / 调节仪表盘

## 3. 非财务预算(03) 指标vs实际·达成率建模 ⭐⭐
- **核心论点**:指标建模运用——动态指标值(历史月数据×系数)可任意筛选月份/类别+**向下钻取**(课→大类→中类→小类→单品)="**预算指标的万年历**"(模型固定逻辑,改系数+选维度即出任意月指标)。
  - **指标vs实际对比+达成率**:VLOOKUP 合并指标表与实际表→透视"目标与实际对比"→**插入计算字段**(月达成率=毛利÷月毛利指标)→动态查任意时期/门店/类别的达成率,即时发现问题驱动业务。
- **关键词**:预算指标万年历(改系数+选维度) / 向下钻取(课→单品) / 达成率=实际÷指标(插入计算字段) / 指标做完要与实际对比

## 4. 非财务预算(04) 指标计算两建模法 ⭐⭐
- **核心论点**:两种指标计算建模:① **同比月系数法**(每月销售÷最小值=系数;某课某月指标=该课月系数÷该课年系数和×课总指标·例:生鲜蔬果1月2.14÷20.485×6505157=679656)② **同比维度系数法**(去年同比月销售×增长系数=目标;如销售系数1.10/毛利系数1.25)。**每日目标=目标值÷销售天数;每周目标=每日×7**。
  - **建模 vs Excel**:**一次建模永远可复用**(更新数据即可)+任意维度筛选钻取(Excel 每变一个维度就要重做表)。再次收束"维度×度量"。
- **关键词**:同比月系数法 / 同比维度系数法(去年同比×增长系数) / 每日目标=目标÷销售天数 / 一次建模永远复用
