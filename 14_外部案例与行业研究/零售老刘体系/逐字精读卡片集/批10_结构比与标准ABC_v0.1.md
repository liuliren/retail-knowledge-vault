# 逐字精读卡片集 · 批10 结构比与标准ABC

## 55. 品类建模(52) 结构比思维 ⭐
- **核心论点**:**品类结构占比 = 商品管理"水平"的指标**(经营好坏都反映在结构比上)。生鲜占比>60%=经营好超市标配;重点客群销售占比应65%+(聚焦1-3客群、放弃全客群)。**自下而上购物篮指标 vs 自嗨指标**:8000万指标该问"有没有这么多购物篮支撑"→区位购物篮指标上卷为课/门店指标,非老板拍8000万分下去。
- **关键词**:结构比=管理水平 / 生鲜>60% / 聚焦1-3客群 / 自下而上购物篮指标vs自嗨指标

## 56. 品类建模(53) 标准ABC比 ⭐⭐
- **核心论点**:**标准ABC结构(30:70递归推导)**:**品种数 A 10% : B 30% : C 60%**;**销售额 A 50% : B 40% : C 10%**。即:约10%的A品实现约50%销售;30%B品实现40%;60%C品实现10%。(单品数的A比=销售额的C比=10%)。实际不必完全符合,ABC占比和=100%即可。
- **关键词**:标准ABC / 品种数10:30:60 / 销售额50:40:10 / 30:70递归 / 10%A品=50%销售

## 57. 品类建模(54) 推动逻辑(失败底层)
- **核心论点**:**经营失败的底层逻辑=负价值积累**:顾客对零售商天然不信任("卖给我=赚我钱")→购物迷茫/犹豫/担心=负价值→长期沉淀→顾客价值流失。商品/服务/线上/管理问题都只是"因或果"片段,根因=偏离需求层+缺信任纽带。胖东来=无限接近底层逻辑(信任背书)。
- **关键词**:负价值积累=失败底层 / 顾客天然不信任 / 信任纽带 / 偏离需求层

## 58. 品类建模(55) 价值链 ⭐
- **核心论点**:**从"搬运工差价"→"价值链"转变**。实体店原来=供应商与顾客间搬运工(赚差价+前后台费),没创造附加价值→被电商比价+到家便利抛弃(冲动购买消失)。出路=找新价值链=对变化需求的满足+价值再分配(品质/健康/体验)。**商品组合=微需求**。
- **关键词**:搬运工差价→价值链 / 新价值链=需求满足+价值再分配 / 商品组合=微需求

## 59. 品类建模(56) 系统报表
- **核心论点**:商品组合系统:**常规品类 vs 特殊品类**(试销/促销/滞销/临期/异常组合·系统自动生成);新品入小类、老品淘汰反之。建独立BI商品组合分析系统。**门店组**(相同属性门店组合=管理单元,差异化运营)。
- **关键词**:常规vs特殊品类组合 / BI分析系统 / 门店组

## 60. 品类建模(57) 属性设置
- **核心论点**:商品组合前提=商品属性(固有+销售)=商品信息表(Product)。**商品生命周期属性**(新品/正常品=重点管理·反映健康度·用状态字段透视各小类SKU数);基础代码需规范(后台乱=没规范维护流程)。属性越详细→精细化管理越好。
- **关键词**:商品组合←属性←商品信息表 / 商品生命周期属性 / 基础代码规范 / 健康体检
