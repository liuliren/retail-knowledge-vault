# 逐字精读卡片集 · 批07 动态化调整与计量化

## 37. 品类建模(34) 动态化调整
- **核心论点**:商品规划(全店·系统全流程)vs 品类规划(单/多品类·灵活即时)区别。商品品项动态调整(季节宽深/品牌品种/价格带/品类角色)。**品类面积分配=经营定位+"购物篮价值比例"**(销售/销量/毛利/客流);面积表(蔬果15-20%等);不重视正比→"越调越乱"。
- **关键词**:商品规划vs品类规划 / 面积分配=购物篮价值比例 / 动态调整 / 越调越乱

## 38. 品类建模(35) 品项配置
- **核心论点**:品项配置以**小类/货位需求单元**为基础→单品(不重视需求直接配单品=做不好配置的主因)。**品项选品4类属性=必需品/首选品/必选品/待选品**。蓝彻斯特法则。品类规划灵活即时(可就单/多品类、点线面、单品)。
- **关键词**:品项配置→需求单元 / 选品4类(必需/首选/必选/待选) / 蓝彻斯特法则

## 39. 品类建模(36) 直接vs间接数据 ⭐
- **核心论点**:冲动性购买占60-70%(发现冲动需求=成功关键)。**直接数据(收银/进货/销售/库存/盘点单据=维度表)vs 间接数据(客流/项数/费用/毛利率=度量表+参数表)**。门店竞争实质=管理竞争;一切管理对象量化→业务宽表;上卷/钻取。
- **关键词**:冲动购买60-70% / 直接数据=维度表 / 间接数据=度量表 / 业务宽表 / 上卷钻取

## 40. 品类建模(37) 五个结构调整指标 ⭐
- **核心论点**:商品结构调整5指标:**①毛利率(综合+分解考核)②库存周转天数③订货到位率(服务率·90-95%·Z1.68-1.74·非越高越好:增1点到货率需2-3倍库存)④引进率与淘汰率(淘汰比引进低10-15%·月1-5%)⑤贡献率=品类结构占比×毛利率(识别主力品类)**。
- **关键词**:5调整指标 / 到位率非越高越好 / 引进率vs淘汰率 / 贡献率=结构占比×毛利率

## 41. 品类建模(38) 计量化分析 ⭐
- **核心论点**:商品组合计量化4维(关联性/效率性/结构性/购物篮行为)。关键指标:关联三度+**周转率、GMROI=毛利率×周转率(筛高毛利高周转核心品)、品类贡献度、价格带分布、宽度深度(平衡长尾与畅销)**。设计原则=**目标导向**(提利润→GMROI/清库存→周转滞销/吸客→客流占比)+动态调整。
- **关键词**:计量化4维 / GMROI=毛利率×周转率 / 宽度深度平衡长尾 / 目标导向指标

## 42. 品类建模(39) 二八与毛利模型
- **核心论点**:**二八定律(20:80,最优78:22宇宙法则)**;多数超市16.5-18%商品创造70-78%销售。**销售预测:其他指标=1:5杠杆**(预测提1%→库存降5%/采购省5%/损耗降5%/人效升5%);每增1个A品配5单品。**整体性影响**(一环错连带全链,不能只看单环)。毛利目标模型建立步骤。
- **关键词**:二八78:22 / 销售预测1:5杠杆 / 整体性影响 / 毛利目标模型
