# 逐字精读卡片集 · 批19 主动线量化/有效促销/Z定律/标签

## 109. 品类建模(107) 主动线量化
- **核心论点**:动线效率=占用单位动线短、销售大。主动线不能过长(除非销售底气足)。提效两方向:调分子(**高贡献商品放高曝光位**)/调分母(缩动线)。
- **关键词**:动线效率 / 主动线不过长 / 高贡献放高曝光

## 110. 品类建模(108) 有效促销
- **核心论点**:**最好的促销=不做促销**(靠日常精细化+长期主义定力)。有效促销=**以顾客需求为中心**(非以商品为中心);读懂顾客=读懂购物篮;促销回到小类/货位需求单元。
- **关键词**:不做促销 / 以需求为中心 / 读懂购物篮 / 需求单元

## 111. 品类建模(109) 静态标签
- **核心论点**:标签化=静态(产品/基础标签·固化物理属性)+动态(单品/业务标签·后天产生)。标签库=战略意义("没有数据的零售企业没有未来")。标签管理须制度化/流程化/标准化/系统化(防乱贴/混乱)。
- **关键词**:静态vs动态标签 / 标签库=战略 / 标签管理四化

## 112. 品类建模(110) 动态标签 ⭐
- **核心论点**:**标签 ≠ 标签化**(标签化=动态业务变现)。标签属性必须**一一对应(一对一/一对多)**。**小类与货位(BAY)不是一对一 → 需"标签化"才能落位**(落位类别=类别与货位一对一标签;否则=没陈列位=没经营)。
- **关键词**:标签≠标签化 / 一一对应 / 小类≠货位·需标签化落位

## 113. 品类建模(116) 视觉Z定律 ⭐⭐
- **核心论点**:**T型技能=纵向商品知识×横向陈列技术**(优秀陈列提部门业绩~25%)。货架分割=商品区域划分。**分类陈列法:按属性(用途/品牌/功能/尺寸/包装)+按顾客分层(生活场景/价格带/份量/品类角色/氛围色彩)**。
- **关键词**:T型技能 / 货架分割 / 分类陈列(按属性+按顾客分层)

## 114. 品类建模(117) Z定律纵横 ⭐⭐
- **核心论点**:**Z定律=单个SKU横向陈列、多个SKU纵向陈列**(基于视觉规律)。视觉两方式(纵向/横向)。**纵向陈列=垂直浏览**(160cm+高货架的**2-3层中段=黄金区域**·便于纵线比较)。横向陈列(视野横向椭圆)。
- **关键词**:Z定律=单SKU横向·多SKU纵向 / 纵向陈列 / 160cm货架2-3层=黄金区域
