# 逐字精读卡片集 · 批02 品类结构与需求单元

## 7. 品类建模(03) 8大步骤优化
- **核心论点**:品类管理三大维度(供应商/零售商/顾客)+8大实施步骤(MBA智库经典)。改进=按"维度+度量"归类8步骤;**品类管理与品类建模 = 双循环流程线**(各自闭环、关联又区别)。定义改名:**品类定义→品类语义;品类评估→品类洞悉**(新增"**顾客洞悉**"步骤=品类管理的新增内容)。维度+度量即时动态组合适应千变万化业务。
- **关键词**:三大维度 / 8大步骤 / 双循环流程线 / 品类定义→语义 / 品类评估→洞悉 / 新增顾客洞悉

## 8. 品类建模(04) 需求单元(一)
- **核心论点**:**品类管理=用"结构化"逻辑应对"非结构化、动态、不确定"的零售(结构性矛盾)**;品类建模引数据建模解此矛盾。单元两类:单位(度量衡·有进制)vs **类群**(层级分类·无进制,如域界门纲目科属种、单品→细分类→小类→中类→大类→课);**类群=基于需求的层级关系="结构"**;**品类管理的一切都关于"结构"**→品类定义改"品类结构"(含定义+维度度量广义·指代人货场)。**品类思维=系统有序的结构化思维**(层次/逻辑/系统/清晰);归类分组=金字塔原理核心。
- **关键词**:结构化矛盾 / 单位vs类群 / 品类管理=结构 / 品类定义→品类结构 / 结构化思维 / 金字塔原理·归类分组

## 9. 品类建模(05) 需求单元(二)
- **核心论点**:**任何品类结构必须是"需求结构"**;问题=**静态被动接受 vs 动态主动创造**(后台商品结构表多是拷贝别家)。**商品配置=主动+动态**搭建独有商品结构(站顾客视角)。**三线**:商品线(零售商·宽深广·**定编**)/制造线(供应商·材料工艺成本·**定品**)/需求线=使用线(顾客·功能味色款规格·**定需**);**品类=制造线×需求线**;**HS矩阵**(选品工具)。单品=商品配置由单品开始到单品结束(月饼例:商品线/制造线散盒/使用线口味→类目→单品)。
- **关键词**:需求结构 / 静态被动vs动态主动 / 商品配置 / 三线(商品线定编/制造线定品/需求线定需) / HS矩阵 / 制造线×需求线 / 单品

## 10. 品类建模(05) 需求单元(三)
- **核心论点**:**需求货位(需求BAY)**=一个货位=1个BAY(一组货架/端架/堆头);一小类=一需求单元→货位也是需求单元。BAY制作三步(以需定编→需求到选品到陈列→维护)。**一切商品问题总表现为"结构"问题**(糟糕结构影响效率利润);层级占比分析。**粒度单元**:零售数据分析处"低级"=粒度不够(到课组大类就分析不下去);**最细粒度=一天/一张小票/一个单品(条码)**;**下钻**;生鲜时段化管理(按小时粒度)。维度+度量复合结构。
- **关键词**:需求货位·BAY / 一小类一需求 / 商品问题=结构问题 / 层级占比 / 粒度单元 / 下钻 / 生鲜时段化 / 最细粒度=天-小票-单品

## 11. 品类建模(08) 购买率(2)
- **核心论点**:[量化/工具]购买率=成交率;客流统计一般超市不用;用 Excel 建简易"成交率(购买率)"分析系统(无客流设备时的替代法)。
- **关键词**:购买率=成交率 / Excel建模 / 客流统计替代
