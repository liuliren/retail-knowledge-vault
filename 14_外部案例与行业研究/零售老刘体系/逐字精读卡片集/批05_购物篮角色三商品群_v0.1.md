# 逐字精读卡片集 · 批05 购物篮属性与三商品群

## 25. 品类建模(22) 负关联
- **核心论点**:负关联=替代/互斥。**向上替代(Up-Selling)鼓励**(高质量篮子收益);**向下替代避免**→转交叉销售(Cross-Selling)。品牌间过度替代→顾客"困惑"→流失。购物篮分析**基石=含小票号的销售流水表**(非日汇总表)。
- **关键词**:负关联=替代 / Up-Selling鼓励·向下避免 / 交叉销售 / 品牌过度替代→困惑 / 流水表是基石

## 26. 品类建模(23) 篮子主义(长期主义)
- **核心论点**:**沃尔玛长期主义 = 篮子主义**(长期坚持对购物篮的重视)。坚持品类定位(最早最好的品类实践者)+天天平价+全球规模采购;长期主义vs短期试错折腾(没瞎折腾反成红利)。
- **关键词**:篮子主义=长期主义 / 品类定位定力 / 天天平价 / 少试错=红利

## 27. 品类建模(24) 商品正负零状态
- **核心论点**:商品三状态=**正常品/异常品/零状态**(7类异常+零状态单列)。**商品力=零售核心力**(人货场之"货");购物篮质量下降=商品力变弱。消费变化:一站式→碎片化+到家;**货挑人=精选**(顾客要零售商精选)。
- **关键词**:商品正负零状态 / 商品力=核心力 / 碎片化购物 / 货挑人=精选

## 28. 品类建模(25) 购物洞察+6P
- **核心论点**:**顾客洞悉=购物洞察**(Win Weber/HEB首提)=理解购物体验的见解(需求状态/场合/店内行为/货架决策);关键=因素(维度)+模式(宽表)。**4P→6P**(+公共关系=顾客关系/+政治权力=场环境)。**品类角色(经营定位)×购物篮角色(需求定位)一致→优质购物篮**。
- **关键词**:购物洞察 / 因素+模式 / 4P→6P / 品类角色×购物篮角色一致

## 29. 品类建模(26) 品类角色(雅斯案例)
- **核心论点**:**核心品类的长期主义**(雅斯食堂13年坚持现场加工/先考核客流非销售毛利·引流)。**一家超市须有自己的核心品类才能定位购物篮**。主力品类vs辅助品类分析(豆制品例:豆芽品种最少但贡献33.9%客流=核心);区位内串岗率=小类间关联度(48%)。
- **关键词**:核心品类长期主义 / 客流优先 / 主力vs辅助品类 / 区位串岗率=关联度

## 30. 品类建模(27) 三商品群 ⭐
- **核心论点**:**购物篮三角色(商品群)=主力/附属/关联**。主力=核心品类(印象/引流·高PI·**IOI兴趣项/目标商品**);附属=同品类补充(连带);关联=跨品类(随机冲动)。**比例:主力销60-70%但品种20-30%;附属销15-20%品种35-40%;关联销10-15%品种30-40%**。**不能只要主力("红花需绿叶衬")**——缺附属/关联,主力也卖不动。
- **关键词**:购物篮三角色=主力·附属·关联 / IOI目标商品 / 红花需绿叶 / 三群稳定比例 / 缺绿叶主力也垮

## 回填候选(待签)
- **购物篮三角色(主力/附属/关联)** 与我五角色(引流/利润/形象/补充/季节)**互补**:五角色=单品存在理由,三角色=购物篮关联定位。**红花需绿叶 = 角色闸防误杀的外部印证**(附属/关联不能因低销砍)。IOI目标商品=我的目的品保护对象。
