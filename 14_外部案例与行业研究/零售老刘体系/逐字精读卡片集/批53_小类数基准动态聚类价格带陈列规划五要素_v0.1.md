# 逐字精读卡片集 · 批53 小类数基准·动态聚类·价格带·陈列规划五要素

> 📌 **本批 = 分析建模13-15**(2022年11月)。小类经营属性(13)+价格带(14)+陈列规划(15)。**金矿:小类数基准(社区超市规模锚)+小类动态聚类三法+陈列规划五要素**。
> ⚠️ 下一批续读 分析建模16-17(可怕的订货量/需求趋势预测)→18-20库存周转率等。

## 1. 分析建模(13) 了解类别经营属性变化·小类动态聚类 ⭐⭐
- **核心论点**:**⭐小类数基准**(社区超市规模锚·呼应批49):**500-1000㎡标准超市≈250个小类;300㎡小店≈150个小类**;别拿大卖场1000+小类规划自己(Hold不住)。**对小类的不重视=对顾客需求的不重视=战略性问题**(小类混乱=经营对顾客需求的混乱)。
  - **三种动态聚类(分群)**:① **按销售金额分组**(如白银/白金·会随时间变)② **按年增长率聚类**(同比±10%内=稳定 / <-20%=下降 / >20%=增长·可设区间)③ **按最高销售额固定分配**(一旦定不随时间变·如年销>500=铂金VIP永久归类)。透视+切片器动态看每个小类销售变化。
- **关键词**:**小类数基准(标超250/小店150/别套大卖场)** / 不重视小类=不重视需求=战略问题 / 动态聚类三法(销售额/增长率/最高额固定)

## 2. 分析建模(14) 小类价格带建模 ⭐⭐
- **核心论点**:家乐福价格带管理理念(价格带合理→顾客感觉整体便宜)。**⭐精细化=不同类别用不同价格带**(休闲食品最低价≠厨房用品·甚至同大类下不同小类不同价格带)→**包含小类的价格带配置表**(小类×价格区间笛卡尔积·200小类×9段=1800)。
  - **⭐价格分段技巧**:**销售或毛利占比高的核心价格区间→细分(间隔小)**;销售不好的段→间隔大。输出每个小类的价格带分组销售/客流→判该小类价格带合不合理。
- **关键词**:家乐福价格带(合理→显便宜) / **不同小类不同价格带** / 核心价格区间细分·冷门区间粗分 / 小类价格带判合理性

## 3. 分析建模(15) 陈列规划与建模 ⭐⭐
- **核心论点**:**陈列规划(战略性陈列·林光涛《陈列规划》)五要素=时间/商品/空间/陈列手法/顾客**。一个陈列点(单品/货架)容易,点→线→面困难→需**整体规划**(缺规划=陈列随心所欲无目的/前瞻/逻辑)。
  - **⭐商品生命周期的正确定义=商品在其性能价值最大化上对顾客需求的占比**(非"建档到删除时间");打折/促销/变价/临期=让商品重获价值的手段;**先追求商品性能价值最大化**。生命周期决定不能期望永远畅销。
  - 陈列规划信息分析:区位货位销售占比→**销售/毛利占比大的货位/小类/单品给更多陈列米线/排面**+时间空间系数→完整陈列规划(即时动态)。
- **关键词**:**陈列规划五要素(时间/商品/空间/陈列手法/顾客)** / 点→线→面需整体规划 / **商品生命周期=性能价值对需求占比** / 销售占比大给更多排面
