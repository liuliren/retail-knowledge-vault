## 1. 分析建模(28) 欲说还休的安全库存 ⭐⭐⭐
- **核心论点**:安全库存=弥补不确定性的额外存货保客户满意;不确定性=**需求不确定+采购提前期不确定**。
  - **仅需求波动(Z值法)**:正态分布Z表。例:日均30·标准差6·90%满足→Z=1.29→**安全库存=ROUND(6×1.29)=8**→库存=30+8=38。**95%满足→Z=1.65(最常用)**(Excel NORMSDIST 自制正态表)。
  - **含提前期**:`SS = Z × σ × √VP`(VP=提前期天数·多期间不是简单倍数,而是标准差平方和再开方;VP含采购前期+检查期)。
  - **⭐根本问题:用静态订货周期决策动态需求量**——越畅销越难掌握安全库存→越易缺货(VP从5天变12天=安全库存缺失)。
  - 建模:静态参数(Z值表·缺货率表=1-动销率·VP参数表)+动态标准差(DAX·分析期需求变化);按客流/ABC给不同满足率(A类/高客流→95%+)。
- **关键词**:安全库存=不确定性缓冲(需求+提前期) / **Z值法(95%→Z=1.65)·SS=Z×σ×√VP** / 静态订货周期决策动态需求=根问题 / 越畅销越易缺货 / ABC/客流定满足率

## 2. 分析建模(29) 安全库存建模(1)·MRP订货算法 ⭐⭐⭐
- **核心论点**:MRP/DRP维度(货号SKU/时期/LT提前期/SS安全库存/BS配送单位量/需求量=顾客需求+安全库存/收货量/ATP可承诺量/期末库存/净需求/批次量/订货单)。
  - **⭐净需求=需求漏斗(订货协同的强命题)**:净需求=上一流程对下一流程的欠缺量。例:顾客需求20·报货15(净5)·订单10(净5)·收货5(净5)·入库0(净5)→整流程净需求=4×5=20(**一个都没满足**)。**订货=需求漏斗,每流程都可能吃掉需求,最考验团队整体协同**。
  - **⭐MRP公式**:净需求=需求量+(预计出-预计入)-期初库存+安全库存;零售简化=**净需求=常规量+安全库存**(两核心量)。批次量:净需求<最小批量→取最小批量;>则最小批量+增量整数倍。
  - **EOQ简化建模**:EOQ=√(2×D×i×S/V)分解=① d/i(年销365×DMS/年库存率·无库存成本则用费用率15-20%)② S/V(批次进价总成本/批次售价总金额)。
- **关键词**:MRP维度(SKU/LT/SS/ATP/净需求) / **净需求=需求漏斗(每流程吃需求·最考验协同)** / 净需求=常规量+安全库存 / EOQ简化建模

## 3. 分析建模(30) 安全库存建模(2)·黄金分割点15.87% ⭐⭐⭐
- **核心论点**:**预算的错**:① **销量≠需求**(历史销量记录不了未满足的需求·A库存200顾客要300只记200)→需从历史清除/单独记录偶然事件 ② 未来事件难预测→**预算管理转需求管理**(动态需求vs静态管理的矛盾);被忽略的数据反而重要(胖东来价签说明卡)。
  - **⭐⭐黄金分割点15.87%(万维钢《学习是什么》·六哥本人在读)**:学习最佳新旧配比=85%熟悉+15%意外。**零售实证:15%-16.5%单品创造60%-65%销量**(不是二八/三七·是黄金点15.87%·别被20-30%忽悠)。
  - **⭐15%黄金线统一应用**:缺货率15%内可接受;费用率15%内良性(人事3-7%);动销率负指标15%(85%+)允许;绩效完成率85%=增降分界;**安全库存可设±15%**(嫌麻烦不算安全库存,允许15%误差也好过没有)。黄金分割点三区:舒适区/学习区/危机区。
  - 安全库存设置五层:不管不问(最差)→凭经验/Excel→每次粗估用黄金点→体现在流程步骤→精细化建模(最好)。
- **关键词**:**销量≠需求**(历史数据陷阱) / 预算管理→需求管理 / **黄金分割点15.87%(15-16.5%单品创65%销量·非二八)** / 15%黄金线(缺货/费用/动销/绩效) / 安全库存±15% / 万维钢
