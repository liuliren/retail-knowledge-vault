# 逐字精读卡片集 · 批49 品类结构维护·ABC实践参数·货位商品状态

> 📌 **本批 = 非财务预算08-09(收官)+分析建模03-04(浅谈ABC/货位配置)**。技术定量+实战经验。**金矿:ABC实践参数(15-17%A类→60%销售·非理想二八)+品类结构维护纪律(每半年梳理·小店400-500小类)**——直接服务六哥社区超市诊断。
> ⚠️ 「非财务预算模式」09篇全清。下一批续读 分析建模 05-精细化零售/06-货位陈列系数/07-货位AR/08-绩效建模等。

## 1. 非财务预算(08) 系统品类结构调整实践 ⭐⭐
- **核心论点(实战经验)**:某3万SKU超市结构表竟有**3400+小类**(开业软件公司直接导大企业结构·用了5年没动)→**60%以上商品不在适当类别**(小类过多·制单建档搞蒙)。再乱也不影响系统运转和销售→很多企业从不更新("我存在即有理")。
  - **结构精简**:整理后2688小类→**需精简到400-500小类**(原小类重分配到新小类·手动关联最花时间);**5级分类不适合小型超市**。
  - **结构设计**:充分分析经营数据+商圈+场景营造(调味蔬菜/软百货等自定义类)。
  - **⭐结构维护纪律**:**每半年(至少1年)梳理优化一次**——"拿小学旧课本给中学生上课会出问题"(需求变了·结构必须跟着变·尤其小类)。
  - 教训:遇重复性大的Excel工作先想批量法(筛选排序1秒/透视表去重)。
- **关键词**:结构乱60%错类·从不更新 / **小店精简到400-500小类·5级不适合** / **每半年至少1年梳理一次** / 场景自定义类

## 2. 非财务预算(09) 多维度粒度预算 ⭐
- **核心论点**:最细粒度预算(天/单品SKU+**节日系数**)。① 带公历农历节日的时期表(VBA农历+手动节日列·春节前15天后几天)② 每天粒度销售汇总(PQ分组SUMMARIZE)+**星期&节日系数**(与平均值的比)③ 粒度目标分解:每天销售占比=该天系数÷整月系数和;**每天预估销售=占比×当月指标**。粒度系数不仅用于预算,也用于订货/库存/促销/配置。
- **关键词**:粒度预算(天+节日系数) / 星期&节日系数 / 每天预估=占比×月指标 / 节日权重(春节前15天)

## 3. 分析建模(03) 浅谈商品ABC ⭐⭐⭐
- **核心论点**:**ABC类型**:① 单ABC(每商品在全部商品中·销售/毛利/综合值/敏感度ABC)② **组合型ABC**(双ABC=两时段/两度量;3ABC=类别/课组/货位组合)。**ABC三步骤**:综合值降序→累加百分比→ABC公式;小类内ABC技巧=每到小类汇总行结果归0→下个小类从头算。
  - **⭐ABC实践参数值(关键·纠正二八幻想)**:标准理想=10%A类→50%销售/30%B→40%/60%C→10%;但**实践常见≈15-17%A类→约60%销售**(理想二八法则事实上很难存在)→定A类阈值要按实际数据特点设。
  - **双ABC运用**:`_A`=去年无今年A(新晋);`A_`/`B_`=去年好今年消失(选品/淘汰参考);**订货按双ABC修正订货量(如AA类×1.8倍系数)**。层级粒度ABC(部门→大类→中类→小类·DAX)。
- **关键词**:单ABC/双ABC/3ABC / **ABC三步骤(降序→累加→公式)** / **实践15-17%A类→60%销售(非理想二八)** / 双ABC(_A新晋·A_消失)选品·订货AA×1.8

## 4. 分析建模(04) 货位商品配置与维护 ⭐⭐
- **核心论点**:货位建模需要的表:① 商品结构表(小类与信息表关联·**需维护**)② 货位类别关联表(货位/区位/课组)③ 区位货位配置统计参数表。**⭐货位/BAY分配=销售占比×毛利占比(交叉比例)**的比值分配区位货位数+单品数(比批46纯综合权重更精细)。
  - **数量参数表**:商品总信息表/在销单品数/**在陈单品数(盘点机扫货位条码)**/已选单品数/**货位条码差异表**(预估配置数vs已配置数差异)。
  - **⭐货位商品状态判定**(结合ABC+数量属性):**有销无库 / A类无货 / 已选无库 / 有库未陈 / 畅销缺货 / 此货暂缺**——这是货位健康度的可落地诊断标签。
- **关键词**:**货位/BAY分配=销售占比×毛利占比(交叉比例)** / 数量参数表(在销/在陈/已选/差异表) / **货位商品状态(有销无库/A类无货/畅销缺货/有库未陈)** / 货位条码差异表
