# 逐字精读卡片集 · 批38 任务价值交换/信任成本 + 库存管理新系列

> 📌 本批含**两大主题切换**:222-223=任务系列收口(价值交换/信任成本);224-227=**全新「库存管理系列」开篇(2022年文·更早·更技术/定量)**。库存系列建议下轮放慢节奏(见进度表标注)。

## 222. 品类建模(239) 任务的交换 ⭐⭐⭐
- **核心论点**:**践行价值交换四步**:①重新定义你的"类/商品"(不问"我卖什么"·问"帮顾客完成什么任务"·任务类与小类建一对多)②找到愿意交换的顾客(三需求圈交集=理想目标顾客:共同痛点/渴望/价值观)③建一套"让交换变容易"的机制(降首次门槛·超预期交付·一次性→可持续会员社群)④把顾客变成"宣传员"(NPS·价值清晰/显著/体面)。**卖货 vs 任务关系链思维**(门店甲卖货→关门怪电商 / 门店乙做价值交换→开新店、口碑裂变)。**三种价值交换:认知交换(解决方案换决策时间)/情绪交换(仪式感共鸣换情感认同·价格非首要)/时间交换(替顾客做到极致换溢价毛利)**。**信任交换**(用坦诚承诺/试吃/无条件退款·替顾客承担试错成本→换信任成为常客)。本质:"把'卖'字从零售字典删掉"·利润=为顾客创造足够价值后的自然回响。
- **关键词**:价值交换四步(定义类/找愿换的人/降门槛机制/变宣传员) / 三种交换(认知/情绪/时间)+信任交换 / 删掉"卖"字·利润是回响

## 223. 品类建模(240) 任务成本层 ⭐⭐⭐
- **核心论点**:**关键等式:你越想绕开"信任"这个高成本(长期主义)的构建过程,就越被迫陷入功能任务层(暂时毛利)进行无休止的极致低价/高效率竞争**(成本最终最高昂·利润无限趋薄)→反之投情感/社会层成本=积累"信任资产"(降未来交易成本·创溢价)。**任务三层对应成本/信任**:功能层=契约信任(履约·动销率/评分)/情感层=关系信任(非促销复购率/响应率/好评)/社会层=认同信任(NPS/口碑/社群活跃)。可用强势层弥补劣势层(对冲关系)。**把"信任价值"变核心经营指标**(别只盯销售额毛利)。会员系统(算积分=功能 vs 记偏好/生日惊喜=情感·后者成本更低信任回报极高)。员工(理货收银工具=管控成本 vs 生活方案提供者=投资成本)。**对抗"管理者诅咒"三问**(功能:更省时还是更费解 / 情感:被尊重还是被当麻烦 / 社会:顾客会怎么向朋友描述"没办法它便宜"vs"我喜欢这里很懂我")。调研陷阱:问"你想不想/要不要"成本≈0=更快的马·该问"真掏过钱没"。"对牛弹琴=层级差·鸡同鸭讲=维度差"→出路不是硬沟通是"同频"(统一语言如SKU单元)。
- **关键词**:信任成本等式(绕开信任→被迫陷功能层低价竞争) / 任务三层对应三种信任(契约/关系/认同) / 信任价值=核心经营指标·NPS / 对抗诅咒三问 / 调研陷阱(想不想成本≈0)

## 224. 库存管理系列(01) ⭐⭐ 〔新主题·2022〕
- **核心论点**:库存=经营结果的检验(盘点)·公司综合经营能力的体现("存在即道理"·结果不合理=管理有短板)。老总永远会问:**究竟多少库存才算合理?**(画那道线=全系列主题)。**合理库存=动态适应经营需要·不是越少越好也不是越多越好**(合适才合理)。合理库存好处(降采购运输成本·供求缓冲器·应对突发)。**合理库存结构=库存管理关键**(优化结构降资金占用·提周转率·平衡服务与成本)。优化路径(研商圈需求定结构·提服务质量·ABC分析·资金总额控制·定期盘点·用供应商资源)。**离开需求谈库存合理=脱裤子放屁**;从相对增量讨论有价值(95%在规定时间内卖出又补回=合理库存);损耗率0.4%为合理参考值。
- **关键词**:库存=经营能力检验·多少才合理=主题 / 合理库存=动态适应·非越少越好 / 合理库存结构=关键 / 离开需求谈库存=无意义 / 0.4%损耗率

## 225. 库存管理系列(02) ⭐⭐⭐ 〔新主题〕
- **核心论点**:**优秀库存管理=组织+系统+流程三关键因素**(不能只奔结果·结果取决于能力·能力=组织/系统/流程有机结合)。系统弥补组织短板(石斧vs F35比喻·咨询时培训1-2个懂信息的人)。**销售对库存不负责=需求计划没闭环**:开口的需求计划→销售推卸/采购成替罪羊;**闭环需求计划=以销定采→达成共识后以采定销**(同时约束激励销售+采购两团队)。**库存管理两个核心**:①最优库存量(总费用最低·EOQ模型)②最大限度解决动态需求(安全库存·正态分布·满足90%/95%服务水平)。**进货量公式 Q=平均需求(d×订货周期)+安全库存Z·σ−现有库存I**(=超市订货公式来源·Z取1.64对应95%)。**零库存=库存管理最大谎言**(库存≠仓库管理·≠先进先出≠货位陈列)。库存管理=不直接产生价值但让其他流程产生更多价值·"前流程-库存-后流程"。
- **关键词**:优秀库存=组织+系统+流程三因素 / 闭环需求计划(以销定采→以采定销) / 库存两核心(最优量+解决动态需求) / 订货公式Q=d·LT+Zσ−I / 零库存=最大谎言

## 226. 库存管理系列(03) ⭐⭐⭐ 〔新主题〕
- **核心论点**:**传统周转率(销售成本÷期初期末平均库存)的优点恰是最大缺点**——点库存代表不了每天变化、且易造假(期初期末做到零→周转率无穷大=神仙企业)。**正确算法:一个月库存应含30天点库存变化**·**库存现金系数 IDSM=AVG(D1..D30)/AVG(S1..S30)**(D=每天库存·S=每天成本·反映几倍现金支撑销售·同样月销有人用2倍现金有人无数倍)。**库存多少合理**:配送超市卖场+内仓≈日销1.5倍((1.64+1.28)/2≈1.5)·应分配到大/中/小类乃至单品·每单品库存≈日均销1.5倍。**内仓零库存配置法**(日销12→陈列量18=1.5×12·订货单位12·剩6时补12个全上架不进内仓)。**边际周转率**(新增业务的新增库存须达到的周转率·业务增15%库存增量须<15%·否则失规模效益)。周转库存无边际效应·安全库存有(量越大动态变动越小、需的安全库存越少=规模效益)。
- **关键词**:传统周转率优点=最大缺点(点库存可造假) / 库存现金系数IDSM(含30天点库存) / 1.5倍日均销=安全保有 / 内仓零库存配置法 / 边际周转率(库存增速<业务增速)

## 227. 库存管理系列(04) ⭐⭐ 〔新主题〕
- **核心论点**:**JIT(Just in time·不早不晚只在需要时点采购)**源自丰田·又称"零库存"生产方式·演变成采购方式·要求供应商/采购/销售三方高度合拍(实操困难:供应商参差/不重视)。**"零库存"式采购=知识盲点甚至商业风险**(非JIT生产则零库存连概念都算不上·一定合理库存=规避市场风险的投资·非科学"零库存"≈慢性中毒)。JIT/VMI"让它们见鬼去"·新采购法应适合本企业、代表采购力、难复制。采购难不在后端下单·在前期采购决策与实力(依商圈需求做品类/商品配置优化)。**安全库存=库存计划中技术含量最高的一块**:设置安全库存=量化**需求/补货/服务水平**三种不确定性。①量化需求不确定性(`安全库存=需求历史×重复性参数`·标准差σ=需求变动性·正态/泊松分布)②量化补货不确定性(采购周期/交货期的标准差)③量化服务水平不确定性(最关键·下节专论)。
- **关键词**:JIT=不早不晚只在需要时采购(丰田) / 零库存式采购=商业风险·非科学零库存≈慢性中毒 / 安全库存=技术含量最高 / 量化三不确定性(需求/补货/服务水平)
