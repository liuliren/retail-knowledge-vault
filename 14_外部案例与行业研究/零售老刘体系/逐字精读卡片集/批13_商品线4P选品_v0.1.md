# 逐字精读卡片集 · 批13 商品线/品类4P/选品定生死

## 73. 品类建模(70) 战略与战术
- **核心论点**:**战略(长期/宏观/方向:业态/品类定位)vs 战术(具体执行:促销/陈列/定价/POP)**。毛主席"战略藐视/战术重视"。会员精细化分层(流量→**留量**·VIP专人制)。供应链战略战术(战略层/运营层/支撑层+信息/物/现金流)。
- **关键词**:战略vs战术 / 流量→留量 / 供应链三层

## 74. 品类建模(71) 价值与趋势
- **核心论点**:**商品组合4核心作用:①需求精准匹配(连带:麦片+牛奶)②资源效率最大化(ABC平衡长尾与爆款)③差异化竞争(自有品牌+网红+场景陈列)④利润结构优化(引流品低毛利高流量+利润品高毛利低频)**。未来=动态智能组合(AI/POS实时)。
- **关键词**:商品组合4作用 / 引流品+利润品搭配 / 动态智能组合

## 75. 品类建模(72) 核心商品线G ⭐⭐
- **核心论点**:**核心商品线(G)=高频刚需/高渗透/基石型**(支撑客流与现金流·不可或缺)。特征:低价格敏感/高复购/强引流。**判据:约占SKU 20%但贡献(销量/销售额/交叉比)65%+,渗透>90%,复购>50%,周转>10次/年**(普通商品3-5次)。入选标准:连续3-6月排名前5-10%/复购≥60%/缺货≤5%。例:鸡蛋牛奶粮油。
- **关键词**:核心商品线G / 占SKU20%贡献65% / 渗透90%·复购50%·周转>10次 / 基石型

## 76. 品类建模(73) 商品线结构·价格带语义 ⭐
- **核心论点**:商品线属性(固有/表现)。**价格带语义:价格线(Price Line=同类最低-最高区间)/价格带(Price Zone=集中区间如10-15元)/PP点(Price Point=最易接受价)**;**在PP点附近配更多单品 → 给顾客"价格便宜"的低价感**。商品二因素;黄金陈列线/VMD;情感价值语义(NLP提取:天然/性价比)。
- **关键词**:价格线/价格带/PP点 / PP点配多单品造低价感 / 商品线属性

## 77. 品类建模(74) 品类4P战术 ⭐
- **核心论点**:**品类四大战术=选品/定价/陈列/促销(4P·麦肯锡《营销学》)**;零售一切基本问题=4P。三大体系=维度体系/指标体系/场景体系。一切品类管理从**品类战略**开始(避免"战术勤快掩盖战略懒惰")。
- **关键词**:品类4P=选品/定价/陈列/促销 / 三大体系 / 从品类战略开始

## 78. 品类建模(75) 选品定生死 ⭐⭐
- **核心论点**:**选品=商品力中影响力最大的因素(权重高于所有基础实操)**;**选品是所有业务的起点**(选品错→补货/订单/定价/促销/陈列/库存全被影响→加速熵增衰败)。木桶原理(短板比优势更危险)。选品被忽略/边缘化是普遍问题。**选品=筛选引入商品组合(非单品思维)**。
- **关键词**:选品定生死 / 选品=商品力影响力最大 / 选品=所有业务起点 / 木桶短板 / 选品=组合非单品
