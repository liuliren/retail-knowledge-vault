## 1. 序315 有货率(设置) ⭐⭐
- **核心论点**:有货率=现货率=库存达成率=服务水平(需求来时有多大概率立即满足)。**有货率近似=1-缺货天数÷(总天数-无需求天数)**(例10天有2天无库存→75%·要扣无需求天数)。**目标四因素**(顾客期望/绩效驱动/企业综合/谁设定);谁设定=销售+业务+财务联合(销售评营运风险/业务知采购成本毛利/财务评成本风险)。**⭐有货率太低=损失销售机会·太高=高库存成本→两风险期望值平衡=合理有货率**。三库存量(安全/管道=DMS×提前期/最佳订购批量)→三库存线(安全库存线/**再订购点=安全+管道**/最大库存=再订购点+最佳批量)→**三业务线(库存<安全=触发紧急处理/<再订购点=提示采购单/>最大=建议清理·触发/提示/建议三词分紧迫度)**。
- **关键词**:有货率=1-缺货天数÷(总-无需求天数) / 两风险期望平衡=合理有货率 / 三库存量→三线→三业务线(触发/提示/建议)

## 2. 序316 库存属性架构 ⭐
- **核心论点**:库存优化四点(业务压力/策略/能力/技术)。**多维度库存属性**(门店/供应商/时期/类别/货位/**货龄**)+库存类度量(在销库存占比/动销率/库销比/呆滞数)。时点vs时段库存(基本是点库存)。**⭐关键库存指标:已N天未购(判畅滞销)+库存可销天数(判大小库存·订货参考)**。
- **关键词**:多维度库存属性(+货龄) / 库销比/呆滞数 / 已N天未购+库存可销天数

## 3. 序317 动态化分析 ⭐
- **核心论点**:动态ABC(Excel静态ABC痛点=每换维度/度量要重做→Power BI动态:任意时间/门店/类别/度量/维度自由组合)。**数据世界=维度+度量+输出**(一个数据模型=一或多个业务行为·多模型组合=分析系统)。
- **关键词**:动态ABC(任意维度度量组合) / 数据世界=维度+度量+输出

## 4. 序318 零售经营之"终极问题"探索 ⭐⭐⭐(全工程capstone)
- **核心论点(整个零售老刘体系的capstone)**:借《终极问题》(赖克霍德)——**终极问题="你是否愿意把本企业推荐给别人?"→量化出 NPS净推荐值(NetPromoter Score)**=衡量客户忠诚度。客户分:推荐型/消极满意型/贬低型。**⭐客户需求(顾客效用/口碑价值)才是一切零售问题的根源和基础**;NPS=客户口碑价值的量化(零售最难量化的)。
  - **三问(找根源)**:① 你知道客户是谁吗?(5W1H第一个W=Who)② 了解他的需求吗?③ 应提供什么商品服务?用户模型Persona。
  - **⭐陈列的终极定义(基于终极问题)**:**基于顾客易看到、易选择、易拿取的需要,处理商品在卖场的位置和数量**(陈列没有原则才是最大标准·位置+数量两维度·呼应批55数量为先+批50陈列)。"先进先出"在卖场违反终极问题(顾客不愿买被前置的临期品·实为企业自身利益)。
  - **⭐NPS标杆**:胖东来NPS≈80%·超市NPS标准值线**20%-85%**(中值45%以下=缺乏"创造好利润促进真成长"条件·低NPS下的利润增加=暂时的非真成长);8个顾客需求量化模型(RFM/NFC等)。
- **关键词**:**终极问题→NPS(愿意推荐?)** / **客户需求=一切零售问题根源** / 三问(谁/需求/提供什么) / **陈列终极定义=易看易选易拿+位置数量** / 胖东来NPS80%·标准线20-85%中值45%

## 5. 序319 数据分析那些事儿 ⭐⭐
- **核心论点**:90%良性超市有数据驱动系统。案例:乐购弃天天平价(分析价格弹性商品·只对最需要的低价);淘汰20%商品→销量+20-30%;12%利润由30%顾客带来;**预测准确性带来各方面5%提高**(利润/失误/损耗/效率)。
  - **⭐坪效从被动量变积极量**:坪效=销售额÷坪数=被动量;**变换为 销售额=坪效×坪数=积极量**→可在坪效前加维度(如供应商)判"有效坪数/无效坪数"。凡有销售额的度量都可转积极度量。中小企业Excel+插件解80%分析。
- **关键词**:数据驱动诊断问题清单 / 预测准确+5%→各方面+5% / **坪效积极量(销售额=坪效×坪数)** / 有效坪数vs无效坪数

## 6. 序320 零模02·从商品ABC浅谈参数与系数 ⭐⭐
- **核心论点**:参数表+**系数(权重系数)**=修正度量计算的值。系数五类(参数值/时间系数/交叉比/业务指标/经营系数)。**⭐ABC系数运用**:① 双ABC促销次序(_B/BC/BB/BA)② A类增2-3倍排面/C类减20-30% ③ **⭐库存订货系数(关键)**:大类总缺货率-15%·良好状态≠一刀切·而是**A类+20%(多备)/B类-10%/C类-25%**(加权仍=-15%·但业务逻辑完全不同·呼应批49 ABC修正订货)④ 促销选品系数(A类品种过多15%=缺核心商品→A类促销占30%+;AB都低→全从C选促销·期望转A/B)。
- **关键词**:参数vs系数 / **ABC库存订货系数(A+20%/B-10%/C-25%=总-15%)** / 双ABC促销陈列系数 / 促销选品占比系数
