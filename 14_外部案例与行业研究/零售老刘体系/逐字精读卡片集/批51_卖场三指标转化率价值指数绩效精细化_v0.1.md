# 逐字精读卡片集 · 批51 卖场三指标·转化率·价值指数·绩效精细化

> 📌 **本批 = 分析建模08-绩效一/09-卖场经营指数/09-绩效二**(2022年10月)。绩效建模(08·二)+卖场指标(09)。**双金矿:① 卖场三指标(转化率/驻停/价值指数)=卖场诊断核心 ② 价值指数RMF+7-11小票客群=以顾客为中心落地**。
> ⚠️ 下一批续读 分析建模10+ 或转安全库存建模系列。

## 1. 分析建模(08) 绩效建模一 ⭐⭐
- **核心论点**:绩效维度=商品/部门/人员+销售/毛利度量。**关键:绩效必须回到小类(每个单品)+每个员工+货位关联**(员工销售来自其辖管货位上的商品);否则=不完善不科学。
  - **⭐伪管理(粗糙绩效=伪管理员的伪管理)**:主管说"粮油大类下降了"=**说了个A**,员工**听了个B**(哪个小类/单品出问题?不知道)→**想成C→做出D→结果E**(再不是A)。
  - 精细化绩效需:类别+货位+人员三者关联表(加员工编码)+建模到小类货位+每天系数+坪效人效+经营费用→盈亏平衡点。绩效还能**检验各经营环节漏洞与流程衔接**(不只是数据规范)。
- **关键词**:绩效必回到小类+员工+货位 / **伪管理(说A听B想C做D结果E)** / 三者关联表 / 绩效=检验流程漏洞

## 2. 分析建模(09) 卖场经营指数 ⭐⭐⭐
- **核心论点(卖场诊断金矿)**:数据指标=参照物(NPS基准:超市24%-82%)。建模三方向:**历史数据**(同比·以史为鉴)/**即时结果**(实时非及时·"黄花菜都凉了")/**洞悉未来**。
  - **⭐经营指标最终回到卖场指标**;卖场指标(维度化量化后)=**转化率 / 驻停时间 / 价值指数**。
  - **⭐转化率=来店客流÷有效购物客流**(例1600/2000=88%·一般控制90%左右)=**超市数据分析最关键指标**(所有经营活动最终反映于此·国内卖场很少关注);可初判商品/货位/陈列/绩效/服务好坏。商圈三率:入店率/市场占有率/购买率(进店数÷购买数·小票数=客流)。**货位转化率**(坪效/库效率/陈列转化率/区位转化率)→"一品一功能,一品一市场,一品一顾客"。
  - **驻停时间**:首要条件=环境布局动线;次要=商品氛围陈列音乐气味(日常维护);等待(收银/投诉/找货)越短越好。
  - **⭐价值指数(顾客价值指数)**:RMF模型(R最近消费/M总金额/F频次)或主成分PCA;**矩阵法**(价值指数×流失指数→高价值高流失=积极唤回);**负价值指数**(顾客不喜欢来/找不到东西/不知怎么买/服务差→努力转化为正价值);本质=以顾客为中心="超市从顾客口袋掏钱的能力"。
  - **⭐7-11小票客群案例**:收银员结账前看顾客一眼→按收银键(少/中/青/壮/老)收集年龄段;**一张小票=一个顾客→背后一个用户标签维度**;10个按键折射精细化管理。
- **关键词**:建模三方向(历史/即时/未来) / **转化率=客流÷有效购物(90%·最关键)** / 卖场三指标(转化/驻停/价值) / 货位转化率·一品一功能 / **价值指数(RMF·矩阵·负价值)** / **7-11小票看一眼按年龄键=客群收集**

## 3. 分析建模(09二) 绩效建模二 ⭐⭐
- **核心论点**:绩效结果最终回到工资。**⭐倒金字塔绩效**(绩效压力应在上一级·不在最底层员工)。**员工岗位表**(岗位=评价员工表现的具体化·"能用手指头数过来"才可作分析维度;**抽象行为量化=绩效做不下去的关键卡点**)。经营岗位绩效表(岗位×考核标准·二维转一维)+员工计分表(完美1/稍差0.8=工分)。**人事费用率=薪酬额÷销售额**(半年算一次·模拟配人看费用率判人员多寡·**增人最大理由=销售提高**)。绩效层级(公司→门店→部门→课组→员工任务)。
- **关键词**:倒金字塔绩效(压力在上级) / 岗位=员工表现具体化(可数=可作维度) / 抽象行为量化=绩效卡点 / **人事费用率=薪酬÷销售(增人理由=销售提高)** / 绩效五层级
