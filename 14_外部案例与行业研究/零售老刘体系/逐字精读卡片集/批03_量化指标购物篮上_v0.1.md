# 逐字精读卡片集 · 批03 量化指标(会员/支持/串岗/购买次数/购物篮上)

## 13. 品类建模(10) 会员率
- **核心论点**:**家庭式购物率 ≈ 会员购买率**(会员数/客流数)。生效条件:会员制运行3-5年+、会员占客流**60-70%**以上、销售表有会员卡明细列。**会员购买率>70% = 明显家庭式购物 = 大众化商品(易缺断货)**。县乡级超市该指标尤有用。
- **关键词**:会员率 / 家庭式购物率 / 会员数÷客流数 / >70%=家庭式·大众化 / 易缺断货

## 14. 品类建模(11) 支持率
- **核心论点**:**购买率下降 = 顾客对门店支持下降(负势能)**——比"销售额降"更本质。**客流不变/降但销售升 = 危险**(客单价/品单价上升=透支体验与性价比,长期流失顾客)。健康增长看购买率(正势能),非短期销售额。口诀:**遇事不决,维度+度量解决**;下钻到需求单元找问题。
- **关键词**:支持率 / 购买率=顾客支持(势能) / 客流降销售升=危险 / 透支体验性价比 / 维度+度量解决 / 下钻需求单元

## 15. 品类建模(12) 支持率/串岗率
- **核心论点**:**串岗率 = 跨课客流**(一个顾客在多个课购物,各课客流相加>总客流)。**串岗支持率**(各课客流÷不含串岗总数,合>100%)vs **实际支持率**(÷含串岗总数,合=100%);**串岗率=串岗支持率−实际支持率**(例41.8%)。铁律:**支持率/串岗率须同粒度比较**(粒度越小结果越大)。
- **关键词**:串岗率=跨课客流 / 串岗支持率vs实际支持率 / 串岗率=两者差 / 同粒度比较 / 粒度越小值越大

## 16. 品类建模(13) 购买次数
- **核心论点**:会员管理=**商家与老客户的关系经营**(非办卡积分发礼品);购买次数→**复购率**(转化率关系指标)。会员数据多维分析(类别/库位/时期)→提高会员转化率;开业有客流(购买率)→围绕目标客保支持率→会员管理形成闭环。
- **关键词**:会员管理=关系经营 / 购买次数→复购率 / 多维会员分析 / 转化率 / 购买率→支持率→会员闭环

## 17. 品类建模(14) 购买次数(2)
- **核心论点**:[延续13]购买次数的建模实现;为"购物篮系列"引子;复购率=会员价值的核心量化。
- **关键词**:购买次数建模 / 复购率 / 购物篮引子

## 18. 品类建模(15) 购物篮·篮子语义
- **核心论点**:**购物篮分析=关联度分析=最重要的经营指标**(零售为顾客提供商品服务的本质决定)。**会员购物有ID(会员卡号);非会员只有小票号**(不知哪张小票对应哪个顾客)→客群分析的数据基础。
- **关键词**:购物篮=关联度=最重要指标 / 会员ID vs 小票号 / 客群分析基础
