# 逐字精读卡片集 · 批54 追加式订货·真实需求还原·库存周转率IR系数

> 📌 **本批 = 分析建模16-18**(2022年11月·硬核定量)。可怕的订货量(16)+需求趋势预测(17)+被错用的库存周转率(18·起周转率建模子系列)。**三金矿:① 追加式订货=A类≠顾客畅销(防误杀集大成) ② 需求趋势预测三法 ③ 库存周转率正解+IR系数四运用**。
> ⚠️ 下一批续读 分析建模19-20(库存周转率建模1-2)→21-27货位棚格/售罄率等。

## 1. 分析建模(16) 可怕的订货量·真实需求还原 ⭐⭐⭐
- **核心论点(防误杀集大成)**:**追加式订货**(按销量补卖出的量)=90%+超市在用→导致"**越好卖越缺断货,不好卖一大堆**"。
  - **⭐我们眼中的A类 ≠ 顾客眼中的畅销品**(一厢情愿)。例:03单品入库10、一天卖完(后续断货),按销量订货→03继续少订(**商机损失**)+01/02多订(库存积压/退货/折价=**双倍损失**)。**"此货暂缺"价签的大都是顾客眼里真正的畅销品**。订货失误=双倍损失+衍生企业肿瘤。
  - **真实需求还原**:03断货致销量扭曲(假需求)≠真实需求。理想态(内仓为零+满陈列):需还原**每个商品平均每天的真实销量**。建模法:以第一周销量为基础,理想第二周应=第一周×2(若实际43<70=订货过少缺货)。
  - 站在顾客角度(陈列"**易拿易看易选**"=最大原则·52周MD=先顾客之忧而忧);数据分析最大作用=**预测和指导决策(能否为业务服务=落地)**;从"发生了什么"→"为什么发生"。
- **关键词**:**追加式订货(90%在用·越好卖越断货)** / **我们的A类≠顾客畅销品** / 此货暂缺=真畅销 / 真实需求还原(断货致假需求·双倍损失) / 易拿易看易选

## 2. 分析建模(17) 需求趋势与预测建模 ⭐⭐
- **核心论点**:订货重在预测未来订货周期内顾客需要的量。回归分析(一元/多元·Excel数据分析·相关系数 Multiple R)。**三种预测建模**:
  - ① **预估未来单品销量**(第一周/订货周期为起点·预测后N周需求·与实际销量差异判订货合理性·**差异±5~10正常·A类允许大些**·差负=缺货/差正=库存过大)
  - ② **趋势查询**(小类/单品在月度的变化·分下降/稳定/增长/高增类·**一小类=一需求·小类趋势=需求变化推演**)
  - ③ **多维度趋势**(顾客/门店/课组/货位·用ABC或1/2/3表示·A到C=2级变化·会员去年D今年G=降3级=该维度业绩大滑要赶紧调)
- **关键词**:回归分析(相关系数R) / 预测三法(预估单品/趋势查询/多维度) / 差异±5~10正常 / 小类趋势=需求推演 / 趋势=危机或机会

## 3. 分析建模(18) 被错用的库存周转率·IR系数 ⭐⭐⭐
- **核心论点(周转率正解·批38-40库存定量底座的关键补全)**:
  - **库存周转率ITO=年化指标**(分子=销售库存成本/分母=平均库存)。**年化例**:大类10个月销100万·毛利率20%(成本率80%)·平均库存10万→**年化周转=(80÷10)×(12÷10)=9.6次;周转天数DOS=365÷9.6≈38天**。
  - **现金流周期=周转天数+ARD-APD**(应收-应付);例:付款60天−周转38天=**22天缓冲期**(一天10亿×20%毛利×利息=不少)。
  - **⭐IR库存系数公式**:`IR = INV÷S = 12×MC% ÷ ITO = 12×(1−毛利率) ÷ ITO`。**四运用**:① 库存规划简单化(IR=12×0.8÷35=0.27→月库存≤销售0.27倍;1.5倍销量作库存参考)② 库存管控(销售额×IR目标=洞察库存是否合理;S=INV÷IR预估销售)③ **库存结构合理性**(IR=库存÷销售应稳定·**0.9-1.5合理**·过低出问题)④ 库存建模(每日单品粒度)。
  - **⭐点库存(期初+期末)÷2 无意义**(需求每天动态变·点库存易造假转移)→应**每日单品粒度**建模。库存现金系数IDSM=每日平均库存÷每日平均成本(如2.95=需2.95倍现金支撑销售)。**库存系数标准:销售增20%,库存增量要<20%**(边际周转率·批40已回填)。
  - **库存计划第一原则=保证合适的商品交付时间**(所有库存流程/订单必须有时间属性);别落入"**错误的精确**"陷阱(参数越多≠越准·没解决第一原则再多参数枉然);根本=闭环需求计划(业务运营协调)。
- **关键词**:**库存周转率=年化(9.6次/DOS38天)** / 现金流=周转天数+应收-应付(22天缓冲) / **IR=12×(1-毛利率)÷ITO·四运用(0.9-1.5合理)** / 点库存无意义→每日单品粒度 / 库存计划第一原则=交付时间 / 错误的精确陷阱
