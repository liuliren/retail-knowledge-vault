# 逐字精读卡片集 · 批30 陈列POP/有用POP/注意力体系

## 175. 品类建模(191) 陈列的POP ⭐
- **核心论点**:**陈列POP=绑定具体陈列点位(货架/端架/堆头/挂钩)的微型售卖点广告**·只服务 1-3 个 SKU(vs 卖场POP服务全店)。运用三层逻辑:**先吸引(0.5-3秒·高对比色/异形/动态)→ 再说服(3秒·极简5-8字卖点)→ 后推动(5-9秒·限时限量/社会证明)**(时间依次增加)。核心原则:**点位绑定(一物一签)/信息极简(5-8字)/不遮商品/层级匹配**。陈列POP="商品的贴身推销员",在犹豫的几秒里说出最关键一句话(买点),把"看一看"变"就买这个"。三大目标=容易看到/找到/拿到。
- **关键词**:陈列POP=点位绑定·服务1-3SKU / 吸引说服推动三层(0.5→3→5-9秒) / 一物一签·不遮商品 / 贴身推销员

## 176. 品类建模(192) POP的运用 ⭐⭐
- **核心论点**:**POP 核心="角度"问题——站顾客角度,非自己角度**(鸭蛋"生熟/咸淡"该不该标:门店嫌"小题大作",顾客真有此疑虑)。**POP 目的=尽可能帮顾客解决购物疑虑、降低决策成本**(不是装饰、不是自己看着好看)。陈列POP三级(吸引点位拦截·停留5-10秒/传递信息解码/推动临门一脚)。按顾客决策类型配 POP(价格敏感→红爆炸卡省X元/品质导向→绿蓝插卡成分/冲动→摇摇跳跳卡/计划→价签清晰即可)。量化指标(POP覆盖率≥99%/良品率≥95%/准确率=100%/投诉率≤0.1%)。**检核法:蹲下 + 退2-3步 + 默数3秒 + 三问**(3秒看懂?帮我还是烦我?会因它买吗?·发现85%问题)。
- **关键词**:POP核心=角度问题(站顾客侧) / POP=降低决策成本非装饰 / 按决策类型配POP / 四量化指标 / 蹲退数三问检核法

## 177. 品类建模(193) POP与3P层 ⭐
- **核心论点**:POP 分三视觉层对应 3P:**VP(卖场POP·门头/吊旗)→ PP(卖场+陈列POP·端架海报+爆炸卡)→ IP(陈列POP·价签/跳跳卡)**。**卖场POP=脸面(乱了赶客)/ 陈列POP=喉咙(哑了丢单)**·不能"张冠李戴"。三种=VP/PP/IP 视觉POP。A/B测试(每周轮换POP话术·类MVP实验)。时空响应(冬暖色厚字/夏冷色流线)。顾客阅读POP平均仅5秒·最快购物19秒·POP内容须≤3-5秒阅读量。
- **关键词**:POP三视觉层对应VP/PP/IP / 卖场POP脸面+陈列POP喉咙·勿张冠李戴 / A/B测试 / POP阅读5秒

## 178. 品类建模(194) 有用的POP ⭐⭐
- **核心论点**:**有用POP 四原则=可看见 / 可读懂(3秒≤8字)/ 可信任(价格真实·过期即撤)/ 可行动**。一句话标准:顾客 3 秒内明白"是不是我需要的 / 对我什么好处 / 现在买为什么划算"并愿放入篮。**反例:"水果新鲜·天天低价"自嗨式口号 POP = 无用**(不传递具体信息/不提销量客流毛利/不提形象/与陈列商品无直接联系——新鲜是顾客长期体验出来的,POP 造不出;靠口号撑低价形象=无本之木)。胖东来服装价签标"进价"=**POP 须与经营定位匹配**。POP=陈列+POP 整体(POP 不独立存在)。检验法:遮住POP看顾客是否多问、销量是否降。
- **关键词**:有用POP四原则(可看见/读懂/信任/行动) / 反自嗨式口号POP / POP须与定位匹配 / POP=陈列+POP整体

## 179. 品类建模(195) POP整体性 ⭐⭐
- **核心论点**:POP 视觉管理五级架构(**VI企业→零售POP→卖场/陈列POP→细分类→现用各类POP**)。**POP 三步骤=常规使用 → 整体性(一致性进化)→ 体系化(VMD视觉体系)**。**POP 整体性=POP 不是孤立个体,必须与其他维度构成整体**:POP+陈列=视觉化推销系统 / POP+动线=引导体验系统 / POP+促销=整合营销 / POP+价格=价值塑造 / POP+人=无声销售助手。**VI = "灵魂/宪法/战略",POP = "骨形/代言人/战术"**(无 VI 约束的 POP=野广告·无 POP 落地的 VI=空架子;如"家乐福红+黄=促销"=长期VI沉淀)。
- **关键词**:POP五级架构(VI→现用POP) / POP三步骤(常规→整体性→体系化) / POP须与他维构成整体 / VI=宪法·POP=代言人

## 180. 品类建模(196) 注意力体系 ⭐⭐⭐
- **核心论点**:**注意力经济学 = 整个零售视觉体系的"引擎"/底层逻辑/"道"**(把 Z/F定律、3P、卖场/陈列POP 串成一个体系)。顾客注意力=**稀缺资源**(只能关注 <1% 信息);**一个好的零售视觉体系 = "精心设计过的基于注意力的漏斗系统"**(非装饰)。**八大法则**:① 3-30-3(3米色块/30cm信息/3秒诉求)② 行为触发 ③ 选择性注意(=显著性优先)④ **认知负荷**(工作记忆 3-5 块·文字超 3 行/色超 3 种 → 阅读率降 63%·=认知减负)⑤ 格式塔三律(相似/接近≤30cm/闭合·=整体感知)⑥ 3秒法则(三秒定生死)⑦ **边际效用递减/视觉疲劳**(VP 周期≤3-4周·**促销品占比≤20-35%**·否则爆炸卡失去突出意义)⑧ 希克定律(**选择即阻力·减少选择=帮顾客做选择**)。四句指令集:突破过滤(显著性)/降低负荷(极简)/引导感知(格式塔)/辅助决策(社会证明+减选择)。线上(信息流注意力·比较搜索型)vs 线下(空间注意力·冲动场景型)。
- **关键词**:注意力经济学=视觉体系引擎/道 / 注意力=稀缺资源(关注<1%) / 视觉体系=注意力漏斗系统 / 八大法则 / 促销品≤20-35% / 希克定律选择即阻力
