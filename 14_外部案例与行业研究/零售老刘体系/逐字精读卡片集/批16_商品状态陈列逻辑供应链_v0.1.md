# 逐字精读卡片集 · 批16 商品状态/陈列逻辑/供应链

## 91. 品类建模(89) 属性字段列
- **核心论点**:商品信息表(Item Master File单品 + Product Information Sheet产品)=核心数据中枢/底层引擎;万物皆维度;以商品为中心分解6大维度结构;属性▶标签▶需求▶业务四级层级链。商品根基不牢=战略基因缺失。
- **关键词**:商品信息表=中枢 / 单品表+产品表 / 6大维度 / 商品根基

## 92. 品类建模(90) 商品状态 ⭐
- **核心论点**:商品状态字段=进销存生命周期。三维度=基础销售状态(可售否)/基础业务状态(库存陈列·补货逻辑)/临时特殊状态(样品/召回)。**7大(或9大)商品状态:建档/新品/正常/停购/停售/淘汰**;采购系统自动屏蔽停购品。区分"商品本身状态"vs"商品表现状态"。
- **关键词**:商品7大状态(建档/新品/正常/停购/停售/淘汰) / 进销存生命周期 / 状态三维度

## 93. 品类建模(91) 陈列逻辑 ⭐⭐
- **核心论点**:陈列=商品力与消费心理(需求)的平衡博弈(非单纯视觉营销)。**陈列量化benchmark**:**黄金视线法则(90-150cm视线高度·转化率是顶底层3.2倍·尼尔森)**;**磁石点理论(每20米焦点/C位陈列·驻留+40%·7-11)**;**蛇形/S型动线(路径1.8倍·关联购买+27%)**;五感(暖光3000K烘焙+18%)。
- **关键词**:黄金视线90-150cm·3.2倍 / 磁石点20米·+40% / S型动线1.8倍·+27% / 暖光3000K·+18%

## 94. 品类建模(92) 促销现状 ⭐
- **核心论点**:促销=鸡肋(做不做都一样;每周1-2次→每月/季)。**大多超市60%时间精力花在管理促销**(促销失败=浪费人力物力→恶性循环;无效陈列连带订货/补货/撤货全白费)。促销已不是万能药,顾客越来越反感。
- **关键词**:促销=鸡肋 / 60%时间管促销 / 无效陈列连带白费

## 95. 品类建模(93) 供应链效率
- **核心论点**:大润发衰退=供应链结构性挑战(停在标准化与效率化之间·规模化只是GMV表象)。**降本之后如何长期增效=关键**。"时间永远站在大众化一边";**低价化/大众化=零售经营最高评判标准**(非销售额/规模/资本)。
- **关键词**:供应链结构性挑战 / 降本后长期增效 / 大众化=最高评判标准

## 96. 品类建模(94) 供应链PB化
- **核心论点**:未来零售比拼=**供应链控制力**(谁有强大高效供应链谁活)。消费"精明理性"非降级;穷鬼超市=实在接地气。**零售竞争力5:战略/人才/选址/高效供应链/差异化护城河**。需求供应链(零售=需求供应链左/右端)。
- **关键词**:供应链控制力 / 消费精明非降级 / 零售竞争力5要素 / 需求供应链
