# 逐字精读卡片集 · 批48 动态参数·同比预测·毛利率贡献值(非财务预算续)

> 📌 **本批 = 「非财务预算模式」05-07**(2023年4月·销售预测/品类指标·定量)。承批47 目标建模,深入动态预测。**金矿=毛利率贡献值=销售占比×毛利率(品类指标分解的关键诊断指标)**。
> ⚠️ 非财务预算系列实为 01-09。下一批续读 08-09(品类维护/粒度预算)收官本系列,再转新主题(浅谈商品ABC/货位商品配置等)。

## 1. 非财务预算(05) 动态参数 ⭐⭐
- **核心论点**:批47 的「历史数据同比法模型」(去年同期×固定增长系数·只需销售/毛利2系数)**缺陷=固定系数反映不出最新趋势变化**。解法=**动态参数**:在原系数上**加当前销售趋势**(本期vs上期升降比)→系数从直线变曲线、随趋势变化。配合 **52周MD管理法+周例会**(以周为单元·总结上周研判下周)。
  - **⭐动态权重分配逻辑(系数以1为平衡点三类)**:① **系数>1(正趋势)→偏本期**:目标=本期×0.7+上期×0.3;② **系数<1(负趋势)→偏上期**:目标=本期×0.6+上期×0.4;③ 系数≈1(1~1.1)→取平均。"不存在绝对正确的预测,能反映需求变化的更精细预测才实用。"
- **关键词**:历史同比法缺陷=固定系数 / **动态参数=原系数+当前趋势** / 权重(正趋势0.7/0.3·负趋势0.6/0.4) / 52周MD+周例会

## 2. 非财务预算(06) 同比预测增长率 ⭐⭐
- **核心论点**:周销售预测三模块:① 更新近一周数据(上周保持不变·循环)② 显示本周/上周+**下周预测** ③ **钻取**拆解二级指标(找差异具体原因)。
  - **⭐同比预测增长率(YOY%)**:取近7天(或14/21/28天·周天数倍数)同比增长率=(当前7天销售−去年同期)÷去年同期。**未来N天销售预测=参考销售×(1+预测增长率/YOY%)**。预算表建议用**≥21天同比**更稳。
  - **⭐金句:预算准确性每提高5%,各项业务能提高>5%的回报**(机构研究)。预算建模两部分:①客观预测未来业绩 ②主观业务场景调整;预测不理想/判不出问题→钻取分解到二三级指标采取匹配动作。
- **关键词**:预测三模块(更新/预测/钻取) / **YOY%·未来N天=参考×(1+YOY%)** / ≥21天同比更稳 / 预算准确+5%→业务+5%回报

## 3. 非财务预算(07) 品类指标分解·毛利率贡献值 ⭐⭐⭐
- **核心论点(本批金矿·采购/品类直接可用)**:采购担销售+毛利双指标(尤其毛利)。采购四项技术:①品类指标制定分解 ②新店品类制定优化 ③价格带/定价/促销/库存/ABC ④供应链优化。
  - **品类指标层层分解**:公司指标→**部门(生鲜/食品/非食三大部)→课→大类→中类→小类**。
  - **⭐⭐⭐ 毛利率贡献值 = 销售占比 × 毛利率**(关键诊断指标):判哪个品类该扛毛利。例:公司要10万毛利/月,占销售65%的生鲜只完成2万→毛利任务泡汤(头部销售类别理应扛大头毛利);销售仅10%的百货即便100%毛利率也帮不上。**两关键要素**:① 各部门毛利率设定(据历史+改进策略)② 高毛利品类销售占比越大→毛利贡献值越大。
  - **层级占比**(中类在大类占比·小类在中类占比·累加=100%);**⭐再次小类内ABC**(层级ABC:中类在大类/小类在中类/单品在小类·比占总单品ABC更有意义·ABC至少到小类)。
  - **重点品类**:月预算分解重点=销售占比+毛利率(参考毛利率贡献值);生鲜销售占比≥60%保客流;预测关键=关注重点品类。
- **关键词**:**毛利率贡献值=销售占比×毛利率** / 品类层层分解(公司→部门→课→大中小类) / 采购四技术 / 头部销售类扛大头毛利 / 层级ABC(再证小类内ABC) / 生鲜占比≥60%保客流
