# 逐字精读卡片集 · 批18 GMV/TCO·陈列定义·主动线效率

## 103. 品类建模(101) 系统供应链GMV/TCO ⭐
- **核心论点**:供应链管理(控制)≠供应链。供应链控制两核心:**GMV(商品交易总额=线下+线上+预售卡券+退货全部订单)vs TCO(总成本)**。降本之上求长期增效=GMV/TCO平衡。
- **关键词**:供应链控制≠供应链 / GMV(全渠道交易总额) / TCO总成本

## 104. 品类建模(102) 陈列原点 ⭐⭐
- **核心论点**:**陈列标准定义 = 解决商品在卖场的"位置(空间)+ 数量"**(维度=位置、度量=数量)。陈列目的=让顾客最容易看到/拿取/选择/购买。逛街行为=卖场陈列原点;陈列=鲜活的生命。
- **关键词**:陈列定义=位置+数量 / 最容易看到/拿取/选择/购买 / 逛街=原点

## 105. 品类建模(103) MD事件
- **核心论点**:[案例向]零售舆论事件(西贝预制菜/胖东来筷子/百果园"教育消费者"/海底捞)反映:消费者知情权、诚信企业保护心态、高管不当言论破坏力、危机处理价值观表达。
- **关键词**:零售舆论事件 / 知情权 / 危机处理价值观

## 106. 品类建模(104) 陈列主动线 ⭐
- **核心论点**:卖场动线设计=空间规划核心(关系体验/效率/成本)。理论(视觉接触点/购物路径/磁石点)。**主动线思维=先画主通道线,再两边画"方块"**(布局基本思路);以顾客需求为中心的主动线。
- **关键词**:主动线设计 / 先画主通道再画方块 / 磁石点

## 107-108. 品类建模(105/106) 主动线效率 ⭐⭐
- **核心论点**:**坪效效率公式 = Σ(商品位销售贡献率 × 曝光时长) / 动线长度**(空间/时间/商品三维度量化动线)。空间效率(经营者角度)vs成本效率(顾客角度)。**销售贡献率=货位"价值密度"**(高毛利/高频商品应占高曝光位);曝光时长=货位客流;动线靠迂回/磁石点延长高价值区停留。
- **关键词**:坪效效率公式=Σ(货位贡献率×曝光时长)/动线长度 / 货位价值密度 / 高毛利高频占高曝光
