# 逐字精读卡片集 · 批14 选品效率/PSD/5P/陈列调整

## 79. 品类建模(76) 选品效率化 ⭐
- **核心论点**:奥乐齐极致SKU(少而有章法·自有品牌+特价·普遍低10-30%·高度标准化闭环)。**零售底层逻辑="用结构化的方式,售卖非结构化的商品"**(商品非结构化/管理需结构化)=印证品类管理的结构化矛盾。**商品化5步=标准化▶效率化▶规模化▶...**;标准化=商品化第一步。
- **关键词**:奥乐齐极致SKU / 用结构化售卖非结构化 / 商品化5步 / 标准化第一步

## 80. 品类建模(78) PSD ⭐⭐
- **核心论点**:**PSD(Per Store Per Day=单店单品日均销售)= 单店单品效率指标**(因地制宜/因品施策/千店千面);连锁多门店精细化核心。**但PSD是"单品式"(失整体战略)→需配 PMF(商品市场匹配)+ MVP(最小可行商品)**做最小可行实验。例:冰饮城区店vs社区店夏季差异。
- **关键词**:PSD=单店单品日均效率 / 千店千面 / 配PMF+MVP / 跨店效率指标

## 81. 品类建模(79) 5P人员 ⭐
- **核心论点**:**5P = 4P(商品/价格/销售/促销)+ 人员(People)**;人员三指向(目标顾客/供应链从业/员工·分析中"人员"主指目标顾客)。**顾客五类型:忠诚购买者/竞品忠诚者/游离者(摇摆)/价格敏感者/非购买者**;**游离者=促销重点**(忠诚者在减少)。
- **关键词**:5P=4P+人员 / 顾客五类型 / 游离者=促销重点

## 82. 品类建模(80) 陈列调整案 ⭐⭐
- **核心论点**:**卖场调整未必正向**("越调越乱越差")。主通道堆头案例=**SKU堆叠/单品式思维误区**(以为堆头越多/单品越多越好=错);**通道宽度120cm标准**(减到90-100cm=压抑不畅)。要兼顾**整体性原则+场景式陈列**("一个萝卜一个坑"单体vs整体)。
- **关键词**:调整未必正向 / SKU堆叠误区 / 通道120cm标准 / 整体性vs单体 / 场景式陈列

## 83. 品类建模(81) 广义促销 ⭐
- **核心论点**:**"真正的促销,是不做促销"**(系列终极论点)。促销受多因素(选品/季节/供应链/门店配合);促销初期就应定销售目标(各部门共同努力+评估基线)。促销=商品管理+组织管理的结果(非孤立)。
- **关键词**:真正的促销是不做促销 / 促销先定目标 / 促销=商品+组织管理结果

## 84. 品类建模(82) 5P四级流程 ⭐
- **核心论点**:经典语录:**复杂事情简单化 → 简单事情标准化 → 标准事情流程化 → 流程事情模式化**。**复杂简单化首要法则=分解**(把复杂拆成简单)=维度+度量量化思维。吓到我们的常是想象的困难,非复杂本身。
- **关键词**:复杂→简单→标准→流程→模式化 / 分解 / 维度+度量
