# 逐字精读卡片集 · 批33 宽类窄品货位/可得性/生鲜

## 193. 品类建模(209) 用宽类窄品 ⭐⭐
- **核心论点**:宽类窄品**货位落地四步**:① 画"品类平面布局落位图"(按主动线分方块·入口端架放促销/黄金视线放高频冲动/中段放目的性如水/次通道放低频必需)② 每小品类内部黄金视线+面位分配(按ABC·不平均主义)③ 品类关联布局(啤酒+下酒零食/酸奶+麦片·让货架自己卖货)④ 动态调整(好品替差品·整类无人买就撤换)。**商品力本质=顾客感知价值,非"你觉得货好"**。改造案例(SKU 20→11·每米日销 80→125 元)。**"对宽类温柔一刀,对窄品狠心一刀"**。
- **关键词**:货位四步(落位/面位/关联/调整) / 商品力=顾客感知非你觉得 / 宽类温柔一刀窄品狠心一刀

## 194. 品类建模(210) 货位的宽类 ⭐⭐
- **核心论点**:**空间层级=卖场 ▶ 课位 ▶ 区位 ▶ BAY ▶ 排面 ▶ SKU**。BAY=单节货架(标准 1.2/0.9m·陈列最小独立单元·唯一编号)。**棚格图表**=BAY 正面网格(层×位=SKU 位·"场"结构最细粒度·难点·需专业软件)。落位=平面图打网格·每格规划 1 小品类(落位类≠小类/中类)。**铁律:一个 BAY 尽量只服务一个明确的"类"**(顾客按需求导航·混类=误导心智导航)。"**把商品当人看**"(尊重敬畏·SKU 臃肿要进 ICU=对商品不尊重)。
- **关键词**:空间层级(卖场→课→区→BAY→排面→SKU) / 棚格图表=层×位 / 一个BAY一个类 / 把商品当人看

## 195. 品类建模(211) 宽类的实践 ⭐⭐
- **核心论点**:完整宽类窄品案例(饮料区 6m→确定类→每类 1-2 SKU→腾空间增类→SKU 40→15·库存降 20%/GMV 持平略升)。落地三步走(搞出类→品做加减→时常维护闭环);**SKU 降 15-30% 而利润升 5-10%=成功**。**6 种不适合宽类窄品的场景**:① 商品极度非标/孤品(艺术品·宜单品管理)② 规模极小品项少 ③ 跨界组合且组合本身是核心类(套餐·宜场景包)④ 需求高度个性化(定制·宜ETO/参数化)⑤ 系统/组织能力不足 ⑥ 目标性深度品类(粮油=窄类宽品·占比≤5%)。高频引流品(鸡蛋米肉)可 2-3 品牌(超 3 纠结)。
- **关键词**:饮料案例SKU40→15 / 三步走+SKU降15-30%利润升5-10% / 6种不适合场景 / 目标性品类=窄类宽品≤5%

## 196. 品类建模(212) 窄品可得性 ⭐⭐
- **核心论点**:**属性继承(宽类定通用属性→Item 继承+特有→SKU 限定执行)= 可得性的前提与基础**。可得性四大误区:① 把宽类规则当死教条不敢微调窄品(解:区分硬继承 vs 软继承·给店长微调权限)② 只记初始值不监控动态属性如温度(解:动态监控关键条件·异常自动标不可售)③ **窄品(Item)可售 ≠ 每个 SKU 实物可售**(解:加 SKU 级【实物状态】标签·店长看的是"实物状态=良好的比例"非布尔值)④ 过度依赖系统忽略人工现场判断。**系统处理稳定规则,人留灰色地带容错**。"类不清晰=找不到东西"(白糖该摆调味类·不懂属性继承选品就被动)。
- **关键词**:属性继承=可得性前提 / 四误区(死教条/不监控动态/窄品≠SKU实物/缺人工) / 系统+人工容错 / 类不清晰=找不到

## 197. 品类建模(213) 可得性实例 ⭐⭐
- **核心论点**:窄品独有可得性价值(五常大米三标识/大单品爆品=独特属性)。**物理属性(继承)vs 非物理属性(独有·生命周期/动态业务)**。**临期 = 门店最大损失**(实质损失 + 机会损失·**两者只能同降、不可分别降**·故拼命保"新鲜")。**保质期顾客 vs 门店错位**:顾客=线性折旧("过半应半价"·剩余价值适配比例)/门店=风险曲线("未临期原价"·成本覆盖+止损);**临期阈值=两曲线相交=握手降价点**。调和=非价格促销+信息透明(标生产日期·分时段折扣)。**分级管理**(新鲜度 A 正常/B 打 7 折/C 下架·感官属性比保质期"死得更早":香蕉黑斑>20-30% 转次品)。保质期=可得性的"法律终点",非物理属性="事实起点"。
- **关键词**:临期=最大损失(实质+机会同降) / 保质期顾客线性折旧vs门店风险曲线·临期阈值握手 / 分级管理A/B/C / 法律终点vs事实起点

## 198. 品类建模(214) 生鲜的宽类 ⭐
- **核心论点**:生鲜宽类窄品=受时间品质约束的包含关系(**宽类=管家"管环境"**:温区/损耗/采购预算·**窄品=活品"管动作"**:品质标准/动态调价/保鲜·窄品="控品"·控=动词)。**"生鲜做得越累越没效果"=门店的类离顾客的类越来越远**(门店"觉得不好卖"砍掉整个类→顾客找不到·实为"你觉得"非"顾客觉得")。**烟火气 = 生鲜"宽类" = 最切近顾客心智的"类" = 极致关系**(五感综合体验·非单纯现场加工);"烟火气落在有人气的门店"(可信任度差再好吃也没用)。生鲜的类反而最好做(少而清晰),难在"控品/品控"。
- **关键词**:生鲜宽类=管环境·窄品=控品管动作 / 越累越没效果=类离顾客越来越远 / 烟火气=生鲜宽类=极致关系
