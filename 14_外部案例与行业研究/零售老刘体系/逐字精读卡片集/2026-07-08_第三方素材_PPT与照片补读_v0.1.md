### 2.2 核心论点提取（按四期结构）

**第一期：商品配置的定义与要点**
- 商品配置 = 各种商品之间的某种搭配；【种】=**目的**的种类。三分类：**核心商品(Destination/差异化/选这家店的理由) + 普遍商品(Routine/大众泛化) + 搭配商品(Convenience/低频但特定场合必买)**。
- 公式：**商品＝核心品＋普遍品＋搭配品；配置＝目标顾客＋业态＋周期性**。
- 业态决定选品方向：大型一站式(多核选品·做加法) / 单品大量售卖型(单核选品·做减法·Costco山姆) / 小商圈小店(安全第一·周转现金优先·无特点) / 高竞争中小店策略=**用核心品+搭配品彰显特点、提坪效**（OLE/BLT/SKP路线）。
- **选品之前，先选人**（82年拉菲案例：店定位平民→稀有品×搭配品格；定位富豪→偏好品×普遍品格）。禁忌：对品选人。
- 商品配置与时俱进三切入点：①**季节时令**（周期性：24节气/节日；突发性：疫情/核污水→海鲜减配河鲜增配。=52周MD思想的最初出发点）②**商品生命周期**（iPhone红利案例：识别生命周期阶段提前扩缩配置=买手一生的课题）③**个人/社会成长**（顾客需求水平在成长：电竞小白→发烧友设备完全不同；社会生产力变迁：BB机消亡）。

**第二期：商品配置的单位**
- 分类单位不是博物学式"事后归纳"，而是**经营者主动构造**："把自己放在上帝视角，主动决定哪些商品属于哪一类"。
- 分级链条：**①商品线 → ②制造线 → ③使用线；制造线×使用线＝品目(Item)；Item×CDT＝单品(SKU)**（羽绒服案例：羽绒服=商品线/棉绒羽芯=制造线/颜色款式=使用线/"男士蓝色鹅绒长款带帽"=品目/加S号=单品）。
- **Item与SKU之辨（西瓜案例）**：整瓜本可直接是SKU；但按需求细分出整瓜(多人)/半切(两人)/四分切(单人)后，"西瓜"就是Item含3个SKU。**Item是H×S的各种可能，SKU是TPOS对Item的再细分**。
- **商品配置是目的，品类管理是工具**——商品配置课讨论"一道菜怎样才算好吃"，品类管理/单品管理/HS矩阵/ABC是"做好菜的具体方法"。

**第三期：业态与商品配置（连锁店理论）**
- **业态四象限 = 商圈影响力 × 复购频率**：①冷淡(Cool)经营法=大商圈+高频实用品(绝活餐厅/老字号，难扩张) ②**集中(Dry)经营法=小商圈+高频实用品→密集开店→供应链规模→总成本领先**（连锁超市/便利店） ③炒作(Warm)经营法=小商圈+低频高价品(快闪店，打一枪换一地) ④**分散(Wet)经营法=大商圈+低频高价品→商品必须不可替代**（山姆/Costco/奥莱/迪士尼）。
- 判断：①③商圈策略与选品逻辑内在矛盾，长期不可持续；**能持续造血的只有②④，本质都是总成本领先**。山姆/Costco 按定义属分散法，但从商品配置角度看**本质仍是集中经营法**——只是用超大包装把低单价高频品改造成高单价低频品。
- **集中经营法的实现 = MMS营销系统**：①**MASS大众化**＝Popular Item（Everybody+Everyday goods）+ Popular Price（80%人群，不做头尾20%）②**Merchandising**＝站顾客立场，选品前想清 **TPOS**（Time时机/Position场合/Objective目的/Style生活方式）=完整顾客画像(LOOK) ③**System系统性＝4S**：Specialization专业化/差异化 → 前提是 Specific明确化（卖点到底是什么？"只有我卖"不是卖点）→ Simplification简单化（卖方操作简单=谁都能做+谁做都一样；顾客使用简单）→ Standard标准化。**因果链：明确化→简单化→标准化→才能密集连锁**（中餐馆难连锁的根本原因=工艺复杂无法简单化；破法=中央厨房/预制半成品）。
- MMS内部逻辑：**System → 才能 Merchandising → 才能 MASS**。
- 胖东来判断（六哥原话式锐评）："不提商品配置、价格体系、供应链系统的胖东来分析，都是忽悠"——擦地板/美女收银员/标进货价是**在商品配置做到4S前提下才有底气的表层动作**。

**第四期：选品与定价是一回事**
- 核心命题链：**选品=选人 → 选人=定价 → 定价=成本控制 → 成本控制=品质与售价的平衡**；"选品即定价、定价即成本、成本即战略"。
- 选人是为了确定做 Needs（基本需求）还是 Wants（高级需求）→ 定目标顾客能接受的价 → 倒推毛利率与成本区间 → 再找工厂/供应商。
- **品质4级×价格7档**：劣质品(bad price·任何业态都不碰) / **实用品(Popular price·合格品质·80%人接受的平价)** / **新生品(Moderate price·实用+附加值·中档价)** / 高档品(Better price·顶级品质·多数人不接受) / 奢侈品(Best price·社会价值>使用价值) / 收藏品(Priceless·无价)。
- **裁决**：高档/奢侈品毛利可观但周转太慢，密集连锁压货即断现金流，只适合分散开店；**真正适合集中经营法密集连锁的只有【实用品】和【新生品】两类**——这正是 P域(Popular)/M域(Moderate) 的完整推导原点。
- HS矩阵补充专题：**S=Style 需求类型(Soft)四细分角度＝Target使用人群/Time使用时机/Position使用场合/Objective使用目的**；**H=Handle 产品类型(Hard)四角度＝Product产品设计品质/Price定价成本/Position渠道销路/Promotion沟通表达**。案例：CBD外卖选品——Hard分 RTE即食/RTH加热即食/RTC简烹/OFC办公现做 × Soft分 充饥/好吃过瘾/营养均衡/专业饮食 = 16象限调研后**只选8个象限**做目标客群。
- TPOS→CDT 简便推导法（春节商品案例：疫情严重度×本地外地→过节方式→聚焦 宅家自娱/邮寄礼品/拜访聚会 三需求→直接画出春节CDT）；提醒：需建立在对TPOS/CDT深刻理解上，新手仍走传统CDT四步法（需求三角→选品漏斗→消费思维阶梯→画CDT）。

**星级：⭐⭐⭐**（这是商品配置理论的"厚源头"，含大量md未吸收的判断与案例）
