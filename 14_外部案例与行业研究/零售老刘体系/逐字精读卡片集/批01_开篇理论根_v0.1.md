# 逐字精读卡片集 · 批01 开篇理论根

> 逐字读全文后出卡(非skim)。进度见 [[_逐字精读进度_321]]。

## 1. 品类建模(一) 品类的昨天今天明天
- **核心论点**:本系列不讲品类管理理论,而是"**抽象业务 → 量化建模**"(数据分析来源于业务、驱动落地、以结果结束)。零售本质=**研究顾客=研究人性(关系)**。数据分析的一切=**维度**;人类最擅长=**建模能力**(把好方法总结成模型/模式)。模式可**部分复制**(胖东来:分钱/商品/服务/信任成本模式)。品类管理=一整套解决商品管理问题的方法与思维,**中心理念=以顾客需求为出发点**(市场营销学1939→品类管理1990传播)。
- **关键词**:抽象业务→量化建模 / 维度思维 / 建模能力 / 零售本质=人性关系 / 模式可部分复制 / 胖东来 / 以顾客需求为中心

## 2. 品类建模(一) 暂停的品类语义
- **核心论点**:架构思维=先有贯穿始终的方法论(底层逻辑·管很久),再围绕它建模;架构靠**维度/度量**体现;目的=抽象业务具体量化。三主维度(零售商/顾客/供应商)难共赢→缺顾客维度的建模"缺胳膊少腿"。零售最大难题=**顾客盲盒/需求盲盒**(不知顾客要什么/怎么选)。胖东来=模式维度更高→降低**信任成本**(顾客"闭眼入")。零售经营的是**关系/烟火味/温度**。品类语义围绕"**需求**"和"**选择**"两核心。
- **关键词**:架构方法论 / 顾客盲盒·需求盲盒 / 信任成本·闭眼入 / 关系温度 / 需求与选择

## 3. 品类建模(一) 弥足的品类模式
- **核心论点**:零售进步=门店吸引顾客的方式(**效果**)+传递产品的方法(**效率**);三推动力(消费需求/内部竞争/信息技术)=三维度。大众化需求稳定(一个家庭约**720个基本品**)→市场难创造,只能重新分配(**抢蛋糕**)。效果指标=销售额/毛利额/库存周转;效率=全渠道链条。零售两专业能力=商品选择与组合+商品传递。**PI值**=顾客被吸引程度。**陈列有对错**(基于需求与选择,非"陈列无对错")。两业态:品类杀手/折扣店。
- **关键词**:效果vs效率 / 三推动力 / 720基本品 / 抢蛋糕 / PI值 / 陈列有对错 / 品类杀手·折扣店

## 4. 品类建模(00) 分割线
- **核心论点**:[导航/低密度]度量指标系列框架介绍;系统化逻辑思维=大道(**大道至简,绝非简单**);认真做(完成它)vs用心做(做好它)。
- **关键词**:度量指标框架 / 大道至简

## 5. 品类建模(01) 品类开篇
- **核心论点**:**关键改动=把品类管理8步骤里的"品类指标/评估表"改为"品类度量"**(四两拨千斤)。因建模有且仅有两元素=**维度+度量**;**KPI < 指标(Metrics) < 度量(Measures)**。**品类管理 = 品类维度 + 品类度量**(8步骤里除度量步骤外其余7步都归维度)。品类管理"**四化建设**"=解决"用静态思维处理动态经营"→品类步骤**动态化**。
- **关键词**:品类指标→品类度量 / 维度+度量 / KPI<指标<度量 / 品类管理=维度+度量 / 四化建设·动态化

## 6. 品类建模(02) 品类语义-需求与选择
- **核心论点**:维度=看问题的角度(不可计数),度量=可计数。**零售供应链(零售商/供应商/顾客)是最大维度**(优先性);人货场三维度;**商品属性=最重要、最动态的维度=零售经营基本单元和核心基础**。品牌维度(**NB/LB/SB/PB**)。**一个小类=一个需求,一个品牌=一个需求**;品类单品=基于顾客需求配置的单品(非普通商品);**有效品vs无效品=优化加减品的底层逻辑**;单品优化三方式(减/增/维持)。开店第一件事=搞清能满足哪些需求(可经营品类)+确定商品范围。
- **关键词**:维度vs度量 / 供应链最大维度 / 人货场 / 商品属性=核心基础 / NB-LB-SB-PB / 一小类=一需求 / 有效品·无效品 / 加减品三方式
