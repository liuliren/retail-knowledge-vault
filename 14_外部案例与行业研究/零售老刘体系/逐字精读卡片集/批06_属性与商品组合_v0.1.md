# 逐字精读卡片集 · 批06 篮子属性与商品组合

## 31. 品类建模(28) 三层结构与两大属性 ⭐
- **核心论点**:商品群三层结构(全部品类/部门定位/商品群层)。**商品两大属性 = 固有属性(品牌/功能/产地/规格/口味)+ 经营属性(ABC/价格/促销/会员/库存)**;经营属性含**4大品类角色:目标性/常规性/时令性/便利性**(Nielsen经典品类角色)。**商品信息表=最详细属性源**(应记固有+销售属性,残缺不全=病)。刺激性商品群(主题陈列·不单列一类)。
- **关键词**:固有属性+经营属性 / 4大品类角色(目标/常规/时令/便利) / 商品信息表=属性源 / 商品群三层

## 32. 品类建模(29) 散零食包装属性
- **核心论点**:**零食商品群=包装属性**(小规格+独立条码)弱化内容物=迷你小食品;包装属性一变→售卖/经营策略全变;膨化/饼干/方便/冲泡只要小规格都可归零食群。小份量累加毛利+提高购买频率/冲动。**无商品群思维=无法建长期主义**。
- **关键词**:零食群=包装属性 / 迷你小食品 / 小规格累加毛利 / 商品群=长期主义前提

## 33. 品类建模(30) 生鲜重量与小包装
- **核心论点**:生鲜重量属性=数量;**加工配方表**(成品销量→原料库存冲减·利润归原料·决策树)。**小包装=消耗周期短=快消**(购买频率高、易关联:180ml面酱关联挂面,1.28kg不会);新品宜小规格试销。**加量不加价 → 减量提质**(少子老龄化/一人食/7-11一人食火锅);论斤称重反而涨销。
- **关键词**:生鲜重量=数量 / 加工配方表 / 小包装=快消易关联 / 减量提质 / 一人食

## 34. 品类建模(31) 商品群关系 ⭐
- **核心论点**:逻辑线=**商品▶商品属性▶组合商品▶商品群▶品类管理**。**真商品群判据(唯一标准)=基于顾客需求+符合经营角色定位**(说不清用途的商品组合≠商品群);商品群=品类=战略单元。商品组合主维度=**属性维度(非SKU本身)**。商品群组合法(季节/便利/节庆/场景/主题/供应商/价格档)。
- **关键词**:商品▶属性▶组合▶商品群▶品类管理 / 真商品群=需求+角色定位 / 组合按属性维度 / 组合法

## 35. 品类建模(32) 四级分类原则
- **核心论点**:4-6级分类(部门/大类/中类/小类/细分类/单品)。大类=特性(吃/用);中类=功能用途·**落位桥梁**;**小类=最小类别单元=一个需求=业务管理单元**(一切管理下沉小类才有意义);单品=配置与选择。**品类结构表=零售经营最基础结构表**。
- **关键词**:四级分类 / 中类=落位桥梁 / 小类=一需求=管理单元 / 下沉小类 / 品类结构表=最基础

## 36. 品类建模(33) 结构比
- **核心论点**:商品群=分类概念/战略单位(需求单元·可打破原分类成新管理单元)。**商品组合结构比=关键度量**(高中低品项的项数平衡)。竞争门店战斗力3:1则弱者难翻盘。案例:洗化毛利占比9%/销售20%=定位失败(低价促销/引流过度)。价格带×购物篮销售占比。
- **关键词**:结构比=关键度量 / 项数平衡 / 商品群=战略单位 / 定位失败诊断(毛利vs销售占比)
