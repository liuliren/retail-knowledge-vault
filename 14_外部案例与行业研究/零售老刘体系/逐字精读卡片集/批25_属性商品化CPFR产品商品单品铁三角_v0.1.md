# 逐字精读卡片集 · 批25 属性商品化/CPFR/产品到单品/铁三角

## 145. 品类建模(161) 属性商品化 ⭐
- **核心论点**:**商品化 = 大众化 = MD**。商品化路径=**标准化→效率化→规模化→低价化→大众化**(5大目标);本质=商品从最初属性(功能/进价)到更高价值层的标签化(毛利/信任)。零售两大商品能力=**商品的组合(选品/组合/品类)+ 商品的传递(信息/货品流转)**。大众品两类:畅销品(长期每期畅销+短期暴增热卖)+ **常规品(无差异大众化·占份额最大、却最难卖)**——破解靠"顾客非来不可的理由"(=零售经营难题)。社区超市经营目标=用大众销售体系经营那部分大众化商品(占60%+);非常规品(只一人买/几年买一次/需特殊保鲜)尽量避免,否则业态就变了。
- **关键词**:商品化=大众化=MD / 标准化→大众化5路径 / 组合+传递两大能力 / 常规品最难卖 / 非来不可的理由

## 146. 品类建模(162) 供应链CPFR ⭐
- **核心论点**:**CPFR = 协同计划/预测/补货/分析 四步**(零售商后端 × 供应商前端深度协同·数据共享·共担责任)。目标=破"**牛鞭效应**"(需求被逐级放大扭曲:门店多报20%→采购多定20%→工厂多40%);从"**推式**"(按订单生产推送)→"**拉式**"(需求/消费数据驱动)供应链。最终指向=门店订货系统。让门店"需求"与生产"供应"说同一种数据语言(共同目标=更高 GMV + 更低 TCO)。竞争从"门店间"升级为"供应链生态间"。
- **关键词**:CPFR四步 / 破牛鞭效应 / 推式→拉式 / 同一种数据语言 / 供应链生态竞争

## 147. 品类建模(163) 产品与单品 ⭐⭐
- **核心论点**:**产品(制造端/前端·功能物理形态·进价)vs 商品(流通/中后端·交易市场价值)vs 单品(零售终端·售价/SKU)**——三态对应生产商/供应商/零售商。完整价值链:**产品▶定价▶商品▶售价▶单品▶价值**。零售商任务=**二次商品化**(选品/定价/组合/陈列/促销/服务)。**金龙鱼等品牌商品 80% 价值在供应商端,只有"单品价值"真正属于零售商**(经营单品而非商品)。SCOR 模型(计划/采购/生产/交付/退货 5 环节=供应链标准流程框架)。
- **关键词**:产品/商品/单品三态 / 产品▶定价▶商品▶售价▶单品▶价值 / 二次商品化 / 单品价值才属于零售商 / SCOR

## 148. 品类建模(164) 产品到单品 ⭐⭐
- **核心论点**:商品化=**定义→表达→传递→体现** 四流程,实现产品(定价)→单品(价值)。**单品 = SKU = "元"(最小粒度)维度单元**。价值传递=**使用→交换→交易**(交换是过程·交易是结果/基本度量单元;明码标价/童叟无欺=交易约定)。单品底层逻辑=**离需求最近**·可识别/可管理/可交易/可量化。**SKU vs 条码**:一品多码/一码多品/捆绑时条码不能作唯一值→必须有自己的 SKU 码;一个西瓜→切半/四分=5 个 SKU(=5 个需求选择)。
- **关键词**:商品化四流程 / 单品=SKU=元维度 / 使用→交换→交易 / 离需求最近 / SKU≠条码·西瓜5SKU

## 149. 品类建模(165) 单品铁三角 ⭐⭐
- **核心论点**:**供应链铁三角=产品/商品/单品(生产商/供应商/零售商)**,三点确定平面=天然稳定;"**去中间化"=现象非模式**(两点直线 vs 三点平面稳定;国内需求碎片化需更多中间形态;7-11 靠强供应链批发体系而非去中间)。单品四场景:**①财务=可度量利润单元("零售总利润=所有单品利润之和"·单品=毛细血管,利润回不到单品则无闭环)②运营=空间(陈列货位)+时间(生命周期)承载者③效率=客流/周转/动销/坪效晴雨表④消费=决策最终触点(顾客买走的是单品)**。
- **关键词**:供应链铁三角(三点稳定) / 去中间化=现象非模式 / 总利润=所有单品利润和 / 单品=毛细血管 / 单品四场景

## 150. 品类建模(166) 产品思维 ⭐
- **核心论点**:**产品思维(供应端/前端·由内向外·"我有什么·酒香不怕巷子深")vs 单品思维(零售端)**。产品思维6要素(用户驱动/问题导向/价值创造/系统思考/迭代实验MVP/优先排序);误区=技术迷恋(多功能堆砌)/过度场景/价值空心化。**采购=产品"翻译"(火眼金睛)**:把供应商卖点("央视广告/某技术")转成顾客买点("包装简洁/环保安全")+POP。PB化=用顾客信任把需求反馈给生产商。**成功零售商=两种思维整合者:市场/需求思维=雷达定方向("卖什么")+ 产品思维=显微镜/过滤器(甄别"什么是好货")**。
- **关键词**:产品思维vs单品思维 / 产品思维6要素 / 采购=产品翻译火眼金睛 / 卖点→买点 / 雷达+显微镜整合
