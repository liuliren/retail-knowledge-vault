# 逐字精读卡片集 · 批09 保利期/账龄库龄/折扣

## 49. 品类建模(46) 利润表报表
- **核心论点**:利润表维度(时间/品类/区域/渠道/空间)。零售健康线:**毛利率>30%(覆盖费用率20-25%)、净利率3-8%、总费用<25%**;租金/人工=最大支出。**开源在前,节流在后**(不把控费当第一目标)。销售额与费用须同维度同方向。
- **关键词**:利润表5维度 / 毛利率>30%·净利率3-8%·费用<25% / 开源在前节流在后

## 50. 品类建模(47) 保利期 ⭐
- **核心论点**:**保利期=从购进到售出实现目标利润的最长允许库龄**(毛利被消耗完的天数·类保质期)。**保利期≠保质期**(保利期=利润保障/财务;保质期=质量)——质量没过期但超保利期照样亏。**4核心要素:成本结构/销售周期/需求波动/库存周转**。运用:接近保利期→打折预警;按保利期定采购频率/促销/定价。
- **关键词**:保利期≠保质期 / 保利期4要素 / 接近保利期→打折预警 / 利润保障非质量

## 51. 品类建模(48) 账龄vs库龄 ⭐
- **核心论点**:**账龄(应收账款时长·评估资金回流风险·销售+应收表)vs 库龄(库存存放时长·分析库存健康/周转·库存表)**——分属财务vs业务。区间划分(0-30/31-60/60+);账龄=DATEDIFF(销售日期,TODAY);**库龄涉批次/先进先出(更复杂)**。
- **关键词**:账龄=资金风险 / 库龄=库存健康 / 区间划分 / 库龄批次先进先出

## 52. 品类建模(49) 库龄计算
- **核心论点**:库龄=当前日期−入库日期;**批次拆分计算**(结存数与各批次入库数比对·先进先出);库龄涉调拨/批次跟踪。Excel(MAX/SUMIFS)或建模(DAX/参数表)实现。
- **关键词**:库龄=当前−入库 / 批次拆分 / 先进先出计算

## 53. 品类建模(50) 折扣逻辑 ⭐
- **核心论点**:折扣底层=消费趋势(反向消费/性价比)+库存管理(临期/保利期/库龄)。**保利期决定何时降价避损;库龄>保利期 → 触发折扣**;库龄越长折扣力度越大;牛鞭效应致积压→折扣清理。折扣=理性消费(有原价参考排除顾虑)。
- **关键词**:折扣=消费趋势+库存联动 / 库龄>保利期→折扣 / 库龄越长折扣越大 / 反向消费

## 54. 品类建模(51) 折扣价值链 ⭐⭐
- **核心论点**:**"售价合理性"被忽略**——案例:9个月保质期已过4个月却按原价买=**顾客花全周期价、只剩5个月使用=亏**。**保质期临近=商品价值下降 → 应按"剩余有效期"折扣定价**(全周期售价 vs 剩余使用期);临期分类管理+折扣=双赢(减损耗+利消费者,符《反食品浪费法》)。全店若都在保质期1/3-2/3=危机。
- **关键词**:售价合理性 / 剩余有效期定价 / 保质期临近=价值降 / 临期分类折扣双赢

## 回填候选(待签)
