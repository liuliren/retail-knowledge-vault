## 2. 系列02 架构图组 + 事实表/维度表建模根 ⭐⭐
- **核心论点**:五张架构图——① 数据分析内容架构图(2006年作,坦言多年未落地)② 超市数据化管理架构图(数据分析涉及多部门/流程/超市知识,不好玩)③ 数据分析项目设置④ **报表体系**(六大类100+报表:顾客服务/店面管理/业务流程/商品流转/部门专用/决策分析·BI时代"一次设置可更新可维护")⑤ **事实表+维度表属性统计**(数据建模根:分析表分两类=事实表/维度表及其关系→Power BI 数据池模型)。
  - **两句纲领**:所有商品问题→几乎都能在「商品流传分析系统模型」找答案;所有顾客问题→在「CRM 子系统」找答案。
- **关键词**:报表体系六大类100+ / **事实表+维度表=建模根** / Power BI 数据池 / 商品流传模型+CRM 两大系统

## 3. 系列03 分析内容体系=商品为核心 ⭐⭐
- **核心论点**:分析内容体系最重要的是**商品**(几乎任何分析都离不开商品→量化为属性)。① **8大顾客需求量化模型**(把看不见的顾客需求,经购物行为轨迹/习惯/停留/数量金额/频次量化;典型 RFM=频率/次数/金额分群定重点顾客)② 会员数字化运用③ **商品属性分析**(商品有生命·属性核心=顾客需求·小类=一种顾客需求;好商品三必要条件:多少小类合适/属性与需求关联决定每小类放多少商品/商品流转)。
  - **撬动警句**:**80%以上超市的商品组织架构是混乱的**(实例:500㎡店配 3000 小类=软件商直接导家乐福资料,实际只需 <500 小类;老员工记不住自己管多少小类)。"凡是一家超市信息系统一团糟的,经营也好不到哪里去——这是重不重视顾客需求的问题。"
- **关键词**:商品=分析核心 / 小类=一种需求 / RFM / **80%超市商品组织混乱(3000 vs <500小类)** / 系统乱=不重视需求

## 4. 系列04 了解性图组 + 引出交易费用 ⭐⭐
- **核心论点**:八张了解性图(全渠道架构/数字营销地铁图/人货场/经济学看转型/**顾客需求层次理论**/需求不同阶段表现/**消费层次与GDP**)。两个关键落点:① **顾客需求=消费升级=零售变化(甚至形态变化)的根本原因**;运用 GDP 做市场分析/预估销售。② **正式引出核心概念=交易费用(成本)**:科斯(Coase 1937)提出,含搜寻费用/谈判费用/执行费用三种——为 05 铺垫。
- **关键词**:顾客需求层次 / 需求=消费升级=零售变化根因 / 消费层次×GDP / **科斯交易费用(搜寻/谈判/执行)**

## 5. 系列05 交易费用论 + 零售核心规律 ⭐⭐⭐
- **核心论点**:交易费用=**价值获取成本**(获得顾客价值所付出的成本),存在于零售价值链(制造商→零售商→消费者)的商品流传环节,**双向**。消费者侧三费用实例:找又好又近又便宜的店=**搜寻费用**;讲价询问=**谈判费用**;称重验货=**执行费用**。**交易费用=买卖双方花费的时间总和+信息成本(非货币支出)**;商品/服务同质化的今天,**竞争很大程度上体现在交易费用的经营上**。
  - **⭐零售核心规律(背下来)**:`宏观经济发展(GDP)带来新消费需求 → 新技术实施降低交易费用 → 交易费用降到一定极限 → 形成新商业形态实现商品交换`。两次业态印证:大型超市(对百货在三费用上极大优势)、线上电商(电商直连降搜寻费用/标准化产品降谈判费用/在线支付+快递降执行费用)。"用1个机器人比5个员工费用低"=同一规律。
  - 零售进化立方体(横向上下游/纵向成本/黄轴产品服务/内部管理)对应微软四个数字化转型维度。
- **关键词**:**交易费用=价值获取成本** / 搜寻·谈判·执行 / 同质化→拼交易费用经营 / **零售核心规律(GDP→新需求→技术降交易费用→新业态)** / 业态变化通用分析法

## 6. 系列06 价值链关系 + 经营顾客化 ⭐⭐
- **核心论点**:① **博弈均衡**:零售价值链任何一方都形成价值链关系;社区店稳固的核心=容易拉近与社区顾客关系(自然的距离费用优势)。**"与其说零售超市在经营商品和服务,不如说在经营价值链中的关系"**;价值链=闭环循环关系,提取/回溯它的核心=**行为数据记录→数据分析**。② 新合作模式:业务系统从封闭走向开放(云共享);忠诚度从"围绕购买行为"→"各种行为(参加活动/读产品介绍)都可成输入"。③ **顾客冲动消费占70%+**→营造氛围(环境/低价/新鲜/文化)+五官诉求(视听嗅味触)诱发需求。④ **顾客体验的通俗定义=把企业经营成顾客喜欢的样子(而非自己喜欢的样子)=经营顾客化**;最大误区:对竞争对手的关注远超对顾客的关注。
- **关键词**:经营的是价值链中的**关系**(非商品服务) / 社区店=距离费用优势拉近关系 / 价值链=闭环·核心是行为数据 / 冲动消费70%→氛围+五官 / **顾客体验=经营成顾客喜欢的样子=经营顾客化**
