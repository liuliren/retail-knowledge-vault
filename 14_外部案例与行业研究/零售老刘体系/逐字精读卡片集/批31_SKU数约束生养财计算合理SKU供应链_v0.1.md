# 逐字精读卡片集 · 批31 SKU数约束/合理SKU数/供应链

## 181. 品类建模(197) SKU数约束 ⭐⭐
- **核心论点**:**筛选 vs 约束**(两个不同逻辑操作):**筛选=从全集选子集**(基于属性如价格带/品牌·关系运算 >/< 定范围·不改SKU属性/上限)·**约束=施加限制规则**(如"每小类≤30个"/"价格区间不重叠"·= / != 定边界·影响 SKU 增减决策)。**合理 SKU 数=内外两组约束下的平衡点**(外部=消费趋势/竞争/供应链基础·不可控;内部=空间/资金/系统/组织·可优化)·是**刚性、动态、需持续校准的值**(非固定静态)。三层逻辑:**①匹配约束(不逾红线:≤物理空间/≤资金/≥需求覆盖)②边际平衡(边际收益=边际成本)③动态适配(与环境共振)**。**不盲目追多(稀释效率)/不片面追少(流失顾客)·始终用约束求解**。
- **关键词**:筛选(选子集)vs约束(定规则) / 合理SKU=内外约束平衡 / 三层(匹配/边际/动态) / 不多不少·用约束求解

## 182. 品类建模(198) SKU生养财 ⭐⭐
- **核心论点**:**SKU 数 = 大家伙的事(跨部门墙)**——一个 SKU 问题牵动商品/采购/营运/供应链/财务全部门(扯皮)。**SKU 生养管算四端语义不同**:**生产端=成本的根**(每多一SKU→开模/换产/采购复杂度指数升)/**供应端=压力的源**(库存持有+缺货风险+物流复杂度)/**门店端=机会的门**(覆盖更细需求/连带/丰富感)/**财务端=要算的账**(ROI/盈亏平衡·"凭什么立项")。优秀做法:**财务"可行性立项"作门槛 + 供应链"效率"作调节阀 + 动态"需求"作驱动力**;跨部门统一指标=**GMROI(毛利率×周转率)**。早年财务深度参与"新品立项"→库存周转 25-30 天(远低行业)。
- **关键词**:SKU=跨部门墙的事 / 生养管算四端(成本根/压力源/机会门/要算的账) / 财务立项=门槛 / GMROI统一指标

## 183. 品类建模(199) 约束与计算 ⭐⭐
- **核心论点**:SKU 定量**七大内部约束**(各对应一种计算法,可单用/组合):**①目标客群需求(需求覆盖法)②物理空间承载(物理空间法:理论最大SKU=有效陈列面积/单品平均面积)③资金与库存效率(资金效率法:经济承载=平均库存金额×周转率/单品成本)④边际效益(品效拐点:边际品效<平均品效则精简)⑤供应链响应(宽品低库vs窄品高库)⑥综合分析(ABC/规划求解)⑦动态优化**。**品类深度系数=品类总SKU÷子品类数**(目标性品类取高 0.8-1.5/便利性取低 0.3-0.8;深度系数显著高于行业基准=可精简)。品类SKU=总SKU目标×(品类权重/Σ品类权重),权重=销售占比×深度系数。**选品两步=先配品类骨架(宽/窄判断)再配SKU数**。
- **关键词**:七大内部约束+对应计算法 / 物理空间法/资金效率法/边际效益法 / 品类深度系数(目标0.8-1.5/便利0.3-0.8) / 浅而宽vs深而窄

## 184. 品类建模(200) 合理SKU数 ⭐⭐
- **核心论点**:**合理 SKU = 在满足顾客基本需求 + 最低品类覆盖前提下,通过效率指标(每米货架毛利/周转率)和约束(空间/资金/供应商/品项下限),求整体经营目标最大化的 SKU 组合**。四原理(资源有限性/需求覆盖/盈利性/边际效用递减)。**可得性 = 一切 SKU 语义的前提**:`实际可用 SKU = 理论规划 × 仓库现货率 × 供应商履约率 × 经营良性率`(可得性差的 SKU 即使理论最优也要弃)。约束硬边界(物理/资金/品类角色:目标性品类≥15 SKU、便利性 2-3 SKU/供应商/法规)。**SKU 不是公式算出的固定值,而是多约束下的一套模型**(从刚性约束→柔性平衡逐层逼近最优·需数据驱动+经验校准)。"动销率永远到不了 100%·没有最合理只有更合理"。
- **关键词**:合理SKU=约束下求经营目标最大化 / 实际可用SKU=理论×现货率×履约率×良性率 / 可得性=SKU前提 / 没有最合理只有更合理

## 185. 品类建模(201) SKU数语义 ⭐
- **核心论点**:**合理 SKU 数=需求匹配度 × 单品宽度 × 单品深度**。"慢与快"矛盾(慢=沉淀信任·快=信任后的结果)。合理 SKU 管理 5 步(定量分析→查可得性→约束过滤→动态优化规划求解→执行维护)。五原则(数据驱动非经验/约束优先非无限丰富/动态调整非一劳永逸/需求导向非单纯精简/技术辅助非手工)。SKU 语义转换链:**宽类→Item→SKU→单元→新品→滞销→淘汰**(全生命周期)。**理论 SKU 数=相对固定值**(门店定位/商圈/客群/品类结构不变时,理论上限稳定,SKU 优化=在此上限内调整逼近理想)。**单品管理核心=【单品数】度量·合理SKU=求"需求单品数"**。
- **关键词**:合理SKU=需求匹配度×宽度×深度 / 五原则 / 理论SKU=相对固定值 / 合理SKU=求需求单品数

## 186. 品类建模(202) SKU供应链 ⭐⭐
- **核心论点**:**"一切零售问题就是 SKU 问题"**(进销存→财务核算全绕 SKU·遇费解问题先想 SKU 数:大问题=大数量/小问题=小数量/无问题=数量合理)。**合理 SKU 数(供应链视角)=使采购/库存/销售/退货/结算各类单据的流量与复杂度,与仓储/资金/系统/供应商协同能力相匹配,实现供应链成本最优的 SKU 水平**(过多→单据激增/拣货低效/盘点难;过少→缺货/资源闲置)。**SKU 数变化=全供应链单据流转**(增加经≥7类单据/减少经≥6类·"数字行为足迹")。反推法:二八分析(20%SKU贡献80%单据量·保核心精尾部)/边际拐点(SKU vs 订单行数曲线)/仓储效率临界点。
- **关键词**:一切零售问题=SKU问题 / 合理SKU=单据流量与协同能力匹配 / SKU变化=全供应链单据流转·数字足迹 / 二八反推
