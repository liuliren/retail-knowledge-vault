# 逐字精读卡片集 · 批12 痛点/卖点量化/战略/工具

## 67. 品类建模(64) 零售痛点
- **核心论点**:痛点=阻碍运营/影响体验/利润流失的关键问题(供需失衡/资源错配)。需求变化快+响应滞后(静态思维解动态需求);**"用战术勤勉掩盖战略懒惰"**;最大痛点=**对销售增长的执念**;从"无货可卖"→**"货多成害"**(供需失衡)。
- **关键词**:痛点=供需失衡/资源错配 / 战术勤勉掩盖战略懒惰 / 货多成害

## 68. 品类建模(65) 痛点属性 ⭐
- **核心论点**:**痛点万能公式:痛点=需求×情感²;痛点强度=(迫切程度+使用频率)×情感共鸣²**(情感平方级作用)。**痛点四层:国民大众/分众/小众/微众**;痛点强弱分级(一级痛快/二级痒变爽/三级)。预制菜击中"懒惰"大痛点。
- **关键词**:痛点=需求×情感² / 情感平方级 / 痛点四层(大众/分众/小众/微众) / 痛点强弱分级

## 69. 品类建模(66) 痛点组合 ⭐
- **核心论点**:**找痛点4法:①客群市调(典型顾客:高频/流失/潜在)②使用场景(情景模拟/假顾客)③情绪发现(负情绪=强烈诉求·隐性负情绪=暗处杀手)④顾客抱怨**。**差评占比多的门店反而满意度更高**(差评=痛点=改进良药·"爱之深责之切")。
- **关键词**:找痛点4法(市调/场景/情绪/抱怨) / 隐性负情绪=暗处杀手 / 差评=改进良药

## 70. 品类建模(67) 卖点量化 ⭐
- **核心论点**:多种点(卖点/买点/痛点/痒点/爆点/爽点)不必纠结。**卖点3级维度=功能层(基础性能)/情感层(身份认同)/社会层(价值观共鸣)**(=商品管理三层级)。**卖点动态化**(USP=流动的价值共识·非静态标签;经济下行→情绪价值权重升)。三大需求(显性/隐性/反常识);卖点量化(心动度)。
- **关键词**:卖点3级维度(功能/情感/社会) / 卖点动态化=流动价值共识 / 显性隐性反常识需求

## 71. 品类建模(68) 零售战略
- **核心论点**:战略三逻辑动作=规划/决策/执行。名人战略观(波特=独特定位+取舍+配称/明茨伯格=计划与灵活平衡)。**零售6战略特征**:目的=良性持续增长/全局/着眼未来(长期主义)/动态/决策规划行动过程/各业务平衡。生态位基因。
- **关键词**:战略=规划/决策/执行 / 波特配称 / 零售6战略特征 / 长期主义

## 72. 品类建模(69) 分析工具
- **核心论点**:BI工具(FineReport报表/Tableau/PowerBI/Qlik关联引擎)。**Excel+PowerBI最常用**(老刘全系列用此)。**动态ABC=销售额/毛利/库存三指标交叉验证**;五大维度(人货场供财);Centric Planning(AI需求预测)。
- **关键词**:BI工具 / Excel+PowerBI / 动态ABC三指标交叉 / 人货场供财五维
