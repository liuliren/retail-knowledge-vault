# 二遍批判卡 · W1-M6 商品ABC与类别动态聚类

## 03. 商品ABC的创建与运用 ⭐⭐
- **可搬骨架**:① ABC三部曲=降序→累计%→阈值判 A/B/C;② **标准比:品种数 A10:B30:C60,销售额 A50:B40:C10**(实践常见 15-17%A产60%);③ **小类内ABC**(各单品在本小类范围内排序,比全店ABC更实际);④ **组合ABC**:双ABC(去年_今年,`_A`=新晋、`A_`=衰退)、3ABC(类课位);⑤ 运用=订货系数(AA×1.8)/选品汰换(A_进候选)/层级粒度ABC(大→中→小)。
- **批判层(失效条件)**:
  - **生鲜A类天然高集中** → 标准 10%A产50% 会误判核心单品进B,削锚点。生鲜小类改用累计销售%阈值(A=60-70%),接受"A类=3-5个单品"。
  - **双ABC需跨年稳定主数据** → 生鲜散装/非标品无固定条码、汰换快,双ABC前提不成立,只做当季小类内ABC。
  - **AA×1.8订货系数不可照搬** → 生鲜A类受鲜度/损耗约束,放大订货=放大损耗;须以货架寿命/损耗率封顶(接 KB-RETAIL-KPI 命题9 进货频次×货架寿命)。
- **可调用性**:小类内ABC + 双ABC汰换信号 = 花厅坊包装盘季度汰换清单的量化抓手。

## 13. 了解类别经营属性变化 ⭐⭐
- **可搬骨架**:① **社区超市小类总数外部基准**:500-1000㎡≈250小类 / 300㎡≈150小类 / 大卖场1000+;"拿大卖场结构表规划自己=Hold不住";② **一个小类=一个顾客需求**,小类混乱=顾客需求管理混乱(战略问题);③ 三种动态聚类:按销售额分组(随时间变)/ **按增长率分组(>+20%/±10%/<−20%)** / 按最高档锁定(定后不变)。
- **批判层(失效条件)**:
  - **小类总数基准是天花板不是目标** → 花厅坊按面积套上限(如800㎡→约250),超出即审查"每个小类是否对应真实需求";新店应低于此数起步。
  - **增长率聚类需两期稳定数据** → 新店/改造期(<6-12月)无同期数据,聚类无效。
  - **"小类=顾客需求"对生鲜可能太粗** → 一个"叶菜"小类内需求已分化,生鲜须再下沉粒度。
- **可调用性**:增长率三色聚类=花厅坊小类健康度看板,衰退小类直接进 M-DEC 诊断队列。
