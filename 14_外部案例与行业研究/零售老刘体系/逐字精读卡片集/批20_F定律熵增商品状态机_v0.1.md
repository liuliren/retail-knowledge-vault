# 逐字精读卡片集 · 批20 F定律/熵增/商品状态机

## 115. 品类建模(118) 视觉F定律 ⭐⭐
- **核心论点**:**F定律=视线呈F形扫描(左侧优先)**(源眼动研究)。**Z/F互补:Z依赖整体布局(用于冲动型消费区);F依赖文字引导(用于信息密集/决策周期长商品:奶粉/化妆品/精品电器/场景陈列)**。F三视线(顶部水平/次顶水平/左侧垂直)。
- **关键词**:F定律=视线F形 / Z/F互补 / Z=冲动区·F=决策区文字密集 / 奶粉化妆品用F

## 116. 品类建模(119) 有效促销(2)
- **核心论点**:[重复108]最好促销=不做促销;有效促销=以需求单元(小类/货位)为中心;读懂购物篮(韦尔奇:更快更准读懂顾客+变行动)。
- **关键词**:不做促销 / 需求单元为中心

## 117. 品类建模(120) 需求与熵增 ⭐⭐
- **核心论点**:**熵增=零售唯一的"万能公式"/统一底层逻辑**(贯穿所有经营·无例外)。麦克斯韦妖(靠信息降熵但需消耗能量)。从开业到倒闭=熵增自然规律(90%企业80%精力在"求增长"而不愿谈衰亡)。
- **关键词**:熵增=万能公式/统一底层逻辑 / 麦克斯韦妖 / 开业到倒闭=熵增

## 118. 品类建模(121) 时间与熵增 ⭐
- **核心论点**:时间=真实维度(与空间构成时空)。**一切零售经营过程都是"熵"的过程**;陈列=货位空间(长宽高)+时间(何时);不告知陈列时间=无序=混乱=熵增。
- **关键词**:时空 / 一切经营=熵的过程 / 无序=熵增

## 119. 品类建模(122) 商品周期 ⭐⭐
- **核心论点**:**商品6状态(建档/新品/正常/暂停/淘汰/删除)或9状态**;**状态机只能往后转(正常←新品转正·不可逆)**。建档=仅录入/新品=有业务操作/正常=可任何操作/淘汰=保留信息不可业务/删除=永久删。**商品状态本身=熵增,抗熵增=提升商品管理效率的必由之路**。
- **关键词**:商品6-9状态机 / 不可逆转档 / 状态=熵增 / 抗熵增=管理必由

## 120. 品类建模(123) 非商品周期 ⭐
- **核心论点**:非商品生命周期(客户/顾客/供应商/服务)。**熵的人货场维度 → "零售熵增系统模型"**(熵结构+熵度量·与任何经营模型无缝连接)。康波周期(万物皆周期);何时做=时间维度(任何模型必有时期表)。
- **关键词**:非商品周期 / 人货场熵增系统模型 / 万物皆周期
