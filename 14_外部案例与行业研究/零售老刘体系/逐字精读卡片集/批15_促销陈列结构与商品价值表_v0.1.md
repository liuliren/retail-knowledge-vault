# 逐字精读卡片集 · 批15 促销/陈列结构/商品信息表

## 85. 品类建模(83) 促销购物篮
- **核心论点**:促销三层次(价格/价值/购物篮)。**"新"≠促销**(花样≠创新);促销创新2要素=效率更高成本更低/出于顾客需要;**真正的促销=不需要促销**(良性选品+动态需求匹配=每天都在促销)。购物篮关联度=有效促销关键。
- **关键词**:促销三层次 / 新≠促销 / 真正的促销不需要促销 / 关联度

## 86. 品类建模(84) 陈列结构 ⭐
- **核心论点**:**陈列三大维度=时间/空间/数量**;陈列=解决商品组合后的"位置(空间)+数量"问题。实践多关注位置、**忽略数量(数量维度=精细化陈列方向)**。**商品5大陈列维度=位置/类别/宽度/深度/关联**;陈列必须匹配需求单元(小类/BAY)。
- **关键词**:陈列三大维度(时间/空间/数量) / 数量维度=精细化方向 / 5大陈列维度 / 匹配需求单元

## 87. 品类建模(85) 商品价值层
- **核心论点**:**广义商品=满足需求欲望的一切**(实物+服务+卖场+陈列+POP+氛围+灯光+音乐+人);以商品为中心的经营体系。商品=满足目标顾客需求的承载体(不可能满足全部→必有选品/更新)。
- **关键词**:广义商品 / 以商品为中心 / 需求承载体

## 88. 品类建模(86) 两大信息表 ⭐
- **核心论点**:**商品信息表=零售管理核心载体/底层引擎**(串联采购/营运/销售/库存/财务全链路)。递进=字段属性▶属性维度▶维度业务;**经营层级链=属性▶标签▶需求▶业务**(标签=属性组合=商品组合;需求=卖点;业务=经营)。
- **关键词**:商品信息表=核心载体 / 属性▶标签▶需求▶业务

## 89. 品类建模(87) PIM
- **核心论点**:PIM=业务神经中枢;三层(业务价值ROI/技术架构MDM/实施路径)。**实体门店现在不必要PIM**(先做好商品信息表;商品结构混乱致PIM失败)。PIM与ERP/WMS集成=常踩坑;知识图谱/RDF三元组。
- **关键词**:PIM=神经中枢 / 现在不必要PIM / 先做好信息表 / ERP集成坑

## 90. 品类建模(88) 属性的价值 ⭐
- **核心论点**:**商品属性三层=基础属性(SKU/名称/产地=身份证)/业务属性(进价/成本/毛利=铜臭味)/动态属性(实时库存/促销价/各期ABC=每分钟变)**。属性从静态→动态智能;商品属性=战略基因;核心=物理属性→可量化可分析。
- **关键词**:商品属性三层(基础/业务/动态) / 静态→动态智能 / 属性=战略基因
