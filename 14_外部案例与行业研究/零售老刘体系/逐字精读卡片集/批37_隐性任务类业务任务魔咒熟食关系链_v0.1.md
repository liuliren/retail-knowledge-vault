# 逐字精读卡片集 · 批37 隐性任务类/管理者诅咒/任务关系链

## 217. 品类建模(234) 隐性任务类 ⭐⭐⭐
- **核心论点**:**顾客可能并不知道自己要什么——只知道有一件事没做好、一个问题没解决、一种感受没被满足**(=隐性"任务")。问卷越听越错(降价销量不动/加功能说太多/抄竞品说雷同)=你听到的全是"更快的马"(噪音)非"汽车"(任务)。给"汽车"取名=**"任务类"**(陈列层级链:属性组合→需求品类→Item宽类→任务类)。**业务=包装(显性·客观描述)·另一件事=待完成任务(隐性·顾客内心剧本)**。落位陈列逻辑**从"展示商品"转向"服务任务"**(传统思维"我要摆好看/动线拉长"→任务思维"3秒看到/30秒拿到/300秒走人")。火锅一站式案例(分散各区来回跑20min→任务区集中5-8min)。
- **关键词**:顾客不知要什么只知任务没完成 / 听到的是更快的马非汽车 / 落位陈列从展示商品→服务任务 / 3秒看到30秒拿到300秒走人

## 218. 品类建模(235) 任务类业务 ⭐⭐⭐
- **核心论点**:**任务三结构层**:①六层业务链(属性→组合→品类→需求→宽类→任务)②四级递进关系层③顾客任务三重维度(功能/情感/社会)。**"顾客不在乎你(门店/业务)、只在乎他的任务"=零售的解放**(不再把资源浪费在自我感动)→"**顾客不在乎你的业务,但你得在乎他的任务**"=信任与复购之源。**任务类=带场景的宽类**=为完成某任务备齐的所有窄品集合。**任务类两公式**:①`任务类=宽类+任务场景约束`(乘法·权重≤1·只选强相关窄品)②`任务类=宽类(商品集)+窄品(选择约束)+任务(情境+社会)`(无功能层=不建议"卖体验"的依据)。功能层=补偿性规则·货架数学题 / 情感社会层=非补偿性(一票否决)·人心选择题。**"物以类聚(宽类)·事以群分(任务)·任务是串起宽类的风筝线"**。建表:列任务→拆宽类→定窄品约束→数据表固定三层。
- **关键词**:顾客不在乎你只在乎任务(=解放) / 任务类=带场景的宽类 / 两公式(宽类+场景约束·宽类+窄品+情境社会) / 功能层=数学题vs情感社会层=选择题 / 物以类聚事以群分

## 219. 品类建模(236) 任务的魔咒 ⭐⭐⭐
- **核心论点**:**"零售管理者的诅咒"=知识诅咒的零售变体**——管理者太熟内部逻辑(供应链/成本/KPI),反丧失像普通顾客那样思考感受的能力。三表现层:①视角被内部业务遮盖(管理者眼中"宽敞明亮"=顾客眼中迷宫)②**把功能当成任务**(沉迷毛利/周转/效率·忘了门店只是被顾客"雇佣"完成任务的跳板)③价值评估脱离现场(看报表≠感知货架前的焦虑犹豫疲劳)。专业视角/效率工具/经验=与顾客真实世界之间的一堵墙。**"超级客服"陷阱**(碎片化系统·一将无能累死千军·管理者成事实瓶颈)。**工具思维vs员工思维致命偏差**:管理者用"工具思维"求解(人=可替换零件)·员工用"员工思维"求存(意义/成长/被尊重)→越工具化越抹杀主观能动性=死循环。**错把"提效工具"等同"能承接结果的员工"**(几十万陈列软件成摆设)。出路:接受诅咒、亲手打破它。
- **关键词**:零售管理者的诅咒=知识诅咒变体 / 把功能当任务·门店=被雇佣的跳板 / 超级客服陷阱·一将无能累死千军 / 工具思维vs员工思维·错把工具当员工

## 220. 品类建模(236b) 熟食任务例 ⭐⭐
- **核心论点**:熟食类**三层业务逻辑结构**:①**标准分类体系**(按工艺酱卤/烧烤/油炸+运营温度热柜/冷柜/常温·SB/T 10658-2012)②**品类关联**(熟食不能孤立·须与酒水/生鲜/调味品强关联落位形成连带)③**宽类与任务类**(核心·宽类=结构决策"覆盖多少子类SKU"·任务类=彻底转顾客视角"快速搞定一顿晚饭/追剧解馋/给孩子垫肚子")。**静态落位陈列 vs 动态任务需求的永恒矛盾**(用一段固定陈列适配即时任务)。熟食常被想当然归"目标品类"但业务行为没当目标品类做(组合广度深度不合理)。**最终检验=用任务/任务类划分商品(跨越传统品类)**,否则熟食难做好、难"长期"。
- **关键词**:熟食三层(标准分类/品类关联/宽类任务类) / 静态落位vs动态任务矛盾 / 想的目标品类≠做的目标品类 / 检验=用任务划商品跨越品类

## 221. 品类建模(238) 任务关系链 ⭐⭐⭐
> ⚠️ 注:批37读到234-238(238=任务关系链/价值交换上半);239任务的交换/240任务成本层归批38。本篇要点与239/240连读。
- **核心论点**:**"卖货思维 vs 任务关系链思维"=狩猎 vs 农耕**(门店甲只想"进什么货怎么卖快"→2-3年关门怪电商 / 门店乙想"顾客买了生活会变什么"→开几家店、老顾客主动介绍)。**"价值交换"是零售顶层艺术**(用我所长换你所需·使彼此更好·不是算计是创造、不是博弈是共赢)。**三种价值交换:认知交换(用解决方案换顾客决策时间)/情绪交换(用仪式感共鸣换情感认同)/时间交换(替顾客做到极致换时间成本解放)**。**践行三问**:①顾客付出的隐性成本是什么(时间/决策风险/选择困难)②我能给什么独特价值抵消甚至超越③如何让顾客清晰感知"赚到的不只是商品,更是省心舒坦被理解被尊重"。最高级价值交换=**NPS净推荐值**(顾客忍不住告诉别人的主动意愿)。
- **关键词**:卖货vs任务关系链=狩猎vs农耕 / 价值交换=顶层艺术(创造非算计) / 三种交换(认知/情绪/时间) / 践行三问 / 最高级=NPS
