# 零售老刘体系吸收 · P2 促销三层次簇

> 商品促销系列。/ingest 深读·转述非引用·从紧。

## 一、核心命题（转述)
| 篇 | 命题 |
|----|------|
| 83 促销购物篮 | **促销三层次 = 价格 / 价值 / 购物篮**(由低到高);"促销购物篮"(高维)靠**购物篮关联度**带连带,非单纯降价 |
| 108/119 有效促销 | 传统促销有问题/危机;有效促销=理想状态(不只让利,要带结构性价值) |
| 92/95 促销现状/风险 | 促销的现状问题与风险(单纯降价透支毛利) |

## 二、最大收获：促销分级 → 升级我的 campaign 诊断
> **三层次**:价格促销(单纯降价·低级)< 价值促销 < 购物篮促销(带连带·高级)。
- **回填我的 campaign 失真标签**([[2026-06-22_零售品类分析方法论_v2.0]] §六B):campaign 诊断可**按三层次分级**——单纯降价透支毛利=低级红灯;带购物篮连带=高级。
- **接 ABCZ ⑦关联销售 + E2 小票法**:购物篮促销的效果用小票(购物篮)数据衡量(连带率/关联度),不只看单品销量。

## 三、回填（So-what·留签字批次)
1. campaign 失真标签加"促销层次"维(价格/价值/购物篮),诊断时标"这个促销停在价格层=透支毛利"。
2. 购物篮促销评估 = 小票连带分析(接 E2 小票法 + ⑦关联维)。

## 四、暂缓
