# 零售老刘体系二遍吸收 A 库存订货与安全库存簇 v0.1

> 象限 IV 养料 → 喂 M1 骨架。对外引用须标来源「零售老刘」。

## 本簇必答(用户指定)
| 问 | 答(精读结论)|
|---|---|
| **售罄率怎么用才不误判?** | ① 必带时间维度(无时间的售罄率无意义);② 用"达50%售罄的**速度**"判畅销,不用绝对销量;③ 对比平衡点售罄率46.7%判盈亏;④ 断货 SKU 的售罄率要先排除缺货再判。单看一个时点的售罄率会把"缺货畅销品"误判成滞销。 |
| **订货/库存/销量正确判断顺序?** | **销量≠需求** → 先还原真实需求(查库存=0占比/周销量倍增预估)→ 再定常规量(日均×周期)→ 加安全库存(Z·σ·√VP)→ 用 IR 系数校验库存合理性。顺序错(直接按销量补货)=追加式订货之害。 |
| **安全库存能否成花厅坊 POS 字段?** | 能,但 SS 是**计算列**不是原始字段。需 POS 提供原料:日均销量、需求标准差σ(POS算)、提前期VP(供应商台账)、目标满足率(查Z)。建议表B加 `safety_stock`(计算列)。 |
| **M-DEC-009 三查闸门是否需补公式?** | 需要。查1 补"库存=0天数占比"算法;查3 补售罄率②=销量/订货量 + 平衡点46.7% + "达50%售罄天数"。详 §7。 |
| **POS 回填清单是否需增补字段?** | 需增补 2 项:`demand_std_sigma`(需求标准差)、`lead_time_vp`(提前期天数)。详 §8。 |

## 2. 原文核心命题
- **追加式订货之害**:90%+ 超市仅按销量补货 → "越好卖越缺货,不好卖一大堆";根因 **销量≠需求**(断货掩盖真实需求=假需求/假A类)。
- **满陈列+内仓为零**:销量达订货周期一半前能维持满陈列=真实销量。
- **安全库存**:唯一直接关联"顾客需求变化"的指标;为需求波动+提前期不确定准备。
- **需求管理 > 需求预测**:预测=静态假设必错;重点是即时解决实际与预测的偏差。
- **库存系数 IR**:把年化周转率转成可日更系数;点库存((期初+期末)/2)无意义且易造假。

## 3. 关键公式/计算口径(每条带来源)
| 公式 | 表达式 | 来源 |
|---|---|---|
| 安全库存(需求波动)| **SS = Z · σ** | 28 |
| 安全库存(含提前期)| **SS = Z · σ · √VP**(VP=提前期天数)| 28 |
| Z 系数 | 90%≈1.28~1.29 / **95%=1.65**;缺货率=1−满足率=1−动销率 | 28 |
| 售罄率② ⭐ | **销量 / 订货量 ×100%** | 25 |
| 平衡点售罄率 | **毛利率/(1−费用率)** = 0.35/0.85 = **46.7%** | 25 |
| 净需求(订货)| **净需求 = 常规量 + 安全库存** | 29 |
| 管道库存 | **= 提前期 × 日均需求** | 26/29 |
| EOQ | **√(2·D·i·S/V)**,D=365×DMS | 29 |
| IR 库存系数 | **IR = INV/S = 12×MC%/ITO**;MC%=1−毛利率 | 18 |
| 周转天数 | **365/周转次数**;现金流周期=DOS+ARD−APD | 18 |
| 周销量倍增预测 | 第N周应≈第1周×N;差异±5~10正常 | 16/17 |
| 黄金分割点 | **15.87%**:缺货率≤15%可接受/动销率≥85% | 30 |

## 4. 业务判断链
```
低销/无销 SKU
├ 期间库存=0占比高 → 缺货假滞销(假A类)→ 补货,不淘汰
├ 周销量 << 第1周×N 预估 → 订少了 → 加订货
├ 有货且售罄率长期 < 46.7% → 真滞销 → 走滞销标准
└ 达50%售罄极快但断货 → 真畅销 → SS 拉到95%满足率
订货量 = 日均×周期(常规量) + Z·σ·√VP(安全库存)
库存健康 = 现库存 vs 销售×IR目标;库存增速应 < 销售增速
```

## 5. 反常识点
- **销量≠需求**:POS 只记销量,断货需求不被记录 → 数据天然低估畅销。
- **点库存毫无意义**且可造假;须日·单品粒度。
- **多期安全库存 ≠ 2×单期** = √(标准差平方和)。
- 订货失误是 **双倍损失**(砍真需求+留真滞销)。
- 实测 **15-16.5% 单品产 60-65% 销售**(非二八)。

## 6. 对花厅坊的可用部分
- 休食 7139 滞销:用"库存=0占比"分离缺货假滞销(M-DEC-009 查1 的量化依据)。
- 休食 A 类(德芙/乐事):SS=Z·σ·√VP 保 95% 满足率,堵"越畅销越缺货"。
- 滞销与库存挤压标准 v1.0:可引入 **IR 库存系数**作补充维(与现"库存月数/周转天数"互补,**不替换**)。

## 9. 可转成 Excel 工具的部分
- **花厅坊安全库存计算表**:输入 日均销量+σ+VP+目标满足率(查Z)→ 输出 SS + 建议订货量。
- **IR 库存系数监控列**:表B 加 `ir_coefficient`=库存额/销售额,对比 0.9-1.5 亮灯。
- **售罄率速度榜**:各 SKU 达50%售罄天数排名 → 真畅销榜。
