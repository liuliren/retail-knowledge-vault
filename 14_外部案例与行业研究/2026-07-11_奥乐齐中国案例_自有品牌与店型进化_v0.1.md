# 奥乐齐中国案例：自有品牌矩阵与店型进化

> **来源说明**：本页内容整理自六哥本人已撰写的多篇深度研究文章（存于个人电脑备份，未收录进vault），非本次新研究。原文数据均为六哥调研测算及公开信息整理，标注为inferred。

## 一、核心数据：从"局部成功"到"网络逻辑"的跨越

- **2024年**：中国连锁经营协会披露，奥乐齐中国销售额达20亿元，**同比增长100%，但当年门店数仅55家**——增长不是靠多开店堆出来的，而是单店模型效率显著改善后的结果。
- **门店扩张节奏**：2025年前奥乐齐深耕上海本地多年，2025年4月19日首次走出上海同步开出苏州/无锡首店，随后进入昆山/南京/常州/镇江；2025年末达88家；2026-01-24南京四店同开达95家；2026-03-21门店总数突破**100家**。
- **判断依据**：六哥提出"判断零售企业有没有跑通看两点——单店效率是否显著改善+模型能否开始复制"，奥乐齐2024年后两个指标都出现明显拐点，2025年9月华东区域配送中心投用是比"新开几家店"更重要的信号(区域规模+物流效率+采购能力+复制速度=硬折扣真正底盘)。
- **扩张路径特点**：优先选择江苏(与上海消费结构/社区密度/商业成熟度接近的同构市场)，不盲目全国化——先啃最容易复制的区域。

## 二、自有品牌矩阵（90%+占比，成本与信任双轮）

**核心经济账**：自有品牌毛利率30%-40%，vs 外部品牌15%-20%；成本优势来自"无品牌溢价(省30-50%)+无进场费+无促销返利+大规模统一采购(单SKU采购量是品牌商品10倍以上)"。奥乐齐中国自有品牌占比**90%以上**——顾客走进店里90%商品是"奥乐齐监制"，对商品品质的信任直接等于对超市的信任，形成极度绑定的消费关系。

**8大自有品牌系列**（分工明确，各司其职）：

| 系列 | 定位 | 核心逻辑 |
|---|---|---|
| Marketplace | 日常必需品(面粉/油/酱油/纸品/洗涤)，销量最稳定 | 消费者每周必买、不需要品牌溢价的品类 |
| 美食家(Gourmet) | 进口特色食材(意面/橄榄油/进口奶酪) | "品质锚点"策略——建立"这家超市品质不差"的整体印象，带动其他自有品牌销售 |
| 好生活(Green Fields) | 有机健康(有机蔬菜/牛奶/全谷物) | 有机蔬菜比盒马有机区低20-30%，证明"有机不一定非要高价" |
| Lacura | 平价美妆个护，对标国际大牌1/3价格 | 英国市场曾在盲测中击败雅诗兰黛/欧莱雅；中国区洁面乳9.9元(对标20-40元同类) |
| Urban Eaters | 城市轻食(沙拉/即食麦片) | 针对不想做饭又想健康饮食的年轻上班族 |
| Specially Selected | 节日礼品/高端体验 | 证明自有品牌不只是"便宜货"，2025春节礼盒(99-299元)销量超预期300%出现缺货 |
| 奥家厨房 | 现制烘焙熟食，"最能留人的区域" | 开放式烘焙+地域特色(南京盐水鸭/苏州蟹粉拌面/上海本帮红烧肉)，扬州包子与当地作坊深度合作把价格压到市场同类70%仍保正宗口感 |

## 三、生鲜选品哲学："三大家族"理论 + 金标果2.0标准

**三大家族**（覆盖中国家庭日常购菜需求80%以上，集中资源打透而非贪多求全）：
- 水果家族：香蕉(补钾高频) + 橙子(维C季节性) + 苹果(日常刚需)
- 蔬菜家族：番茄(百搭刚需) + 萝卜(季节性/清洁力强) + 辣椒(烹饪必需)
- 蛋白质家族：猪肉(最高频) + 禽肉(低脂偏好) + 牛肉(改善型消费)

**金标果2.0品控标准**：糖度下限(苹果≥14°Brix/橙子≥10°Brix) + 质地硬度范围 + 赏味期标注 + **坏果率控制在5%以内**(行业平均10%+)。案例：2J级智利车厘子88元/2.5kg，比电商同规格低15-25%(直签智利出口商省去进口代理+国内经销商环节)，2026年1月南京四店开业时是最快售罄商品之一。

## 四、店型进化史：从"社区精品店"到"平价社区超市"

**2019年首店**：上海两家试点，约1300个SKU，"轻精品化"表达——进口感强，给人"体面"而非"便宜"的印象。

**2023年起的转向**：商品结构和空间功能向"高频刚需"靠拢——从"展示进口零食/酒水/个护差异化"转为"承接一日三餐"(生鲜/鲜食/烘焙/冷藏日配/熟食/基础家清个护)。六哥总结：**"不是让消费者觉得你新鲜，而是让消费者形成依赖；不是让消费者记住一款爆品，而是让消费者反复回来"**——这是社区零售的核心能力。

**门店空间四个变化维度**（六哥概括）：
1. 从"进口感"转向"民生感"——核心任务从"展示与众不同"变成"解决日常所需"
2. 从"精致陈列"转向"效率陈列"——SKU压缩至约2000个内+自有品牌90%+后，陈列追求"更快决策/更高效补货/主力价格带更易感知"，而非"看起来高级"
3. 从"小测试店"转向"标准社区店+大店样板"——主流500-1000㎡，2024年10月上海金山店1014㎡(当时中国最大)，探索维持硬折扣效率同时加强生鲜/烘焙/熟食/线上履约
4. 从"单一门店"转向"网络节点"——同时是社区补给点+平台履约节点+区域供应链终端接口+高频消费线下入口

## 五、对花厅坊/社区生鲜超市的启发（待验证，非六哥已下结论）

- **"三大家族"选品哲学**与花厅坊现有的品类角色分工(核心/普遍/搭配商品三分法)逻辑相通，但"每个家族只选3个高频品类打透"这种极致聚焦程度，值得对照花厅坊生鲜区SKU结构做一次对标诊断。
- **"从进口感到民生感"的转型逻辑**，对判断花厅坊/沙埔大道等门店的商品结构是否"有情怀但不解决日常"有参考价值——警示"差异化"不能脱离"高频刚需"根基。
- **金标果2.0的量化品控标准(糖度/硬度/坏果率5%)**，可作为花厅坊生鲜品控标准制定的参照模板。
- **本页"自有品牌30-40% vs 15-20%毛利率"数据**，与[[01_科学零售方法论/价格带管理]]的红绿灯定价法"绿灯区(利润品)高毛利40-60%+"逻辑互相印证，PB自有品牌是绿灯区的典型来源。
