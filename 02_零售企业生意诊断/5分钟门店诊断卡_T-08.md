## 一、为什么需要这张卡

```
顶尖商品规划师的标志:
  普通从业者进店,看到的是"商品的物理排列"
  顶尖从业者进店,看到的是"商品配置的健康度"

差异不在天赋,而在"判断框架"是否在脑子里成型

本卡片就是这个框架的固化工具
用 100 次后,卡片消失,框架内化
```

## 二、5 分钟诊断的标准流程

### 2.1 时间分配

```
0:00~0:30  入口印象(30 秒)
0:30~1:30  Destination 品类扫描(60 秒)
1:30~2:30  Routine 品类骨架扫描(60 秒)
2:30~3:30  IPC 价格曲线感知(60 秒)
3:30~4:30  陈列错配检查(60 秒)
4:30~5:00  3 个最大问题输出(30 秒)
```

### 2.2 5 分钟内必须回答的 5 个问题

```
Q1:这家店服务什么样的顾客?
Q2:它的核心竞争力(Destination 品类)是什么?
Q3:它的 IPC 形态是右下倾斜还是右上偏斜?
Q4:陈列资源分配有没有明显错配?
Q5:如果让我经营,我会先动哪 3 件事?
```

## 三、五段落诊断清单(可勾选)

### Part 1:入口印象(30 秒)

> **观察点**:进门第一眼看到什么?整体感觉是什么?

```
□ 第一眼看到的是什么品类?
   ⊙ 生鲜(社区超市的标准做法 ✓)
   ⊙ 工业食品(可能是大卖场思维)
   ⊙ 促销/特价(可能是折扣店思维)
   ⊙ 其他:_______

□ 整体氛围:
   ⊙ 干净明亮(基本合格)
   ⊙ 拥挤杂乱(管理粗放)
   ⊙ 冷清(客流问题)
   ⊙ 热闹(经营状态好)

□ 第一感受用一个词形容:_______
   (这个词通常就是这家店给顾客的"心智标签")
```

### Part 2:Destination 品类扫描(60 秒)

> **观察点**:走到生鲜区,5~10 秒判断这家店把什么当成"Destination"

```
□ 生鲜区位置:
   ⊙ 入口附近 / 主动线上(标准 ✓)
   ⊙ 在店内深处(管理者不重视生鲜)
   ⊙ 在角落(放弃了生鲜竞争)

□ 蔬菜区(大湾区 Destination 的典型):
   □ 叶菜种类 ≥ 10 种?(< 10 种 = 不及格)
   □ 散装为主?(预包装多 = 不符合大湾区习惯)
   □ 看上去新鲜?(蔫菜 ≥ 3 种 = 严重失分)
   □ 价格清晰?(标签缺失 ≥ 5 处 = 管理问题)

□ 猪肉区(大湾区第二关键品类):
   □ 热鲜肉为主?(冷鲜为主 = 不符合本地习惯)
   □ 部位齐全(煲汤/炒菜/红烧)?
   □ 当天到货?(看肉色和水分)

□ 一句话评价生鲜区:_______
```

### Part 3:Routine 品类骨架扫描(60 秒)

> **观察点**:工业食品区(调味品/米面油等)的骨架健康度

```
□ 调味品区(大湾区文化重镇):
   □ 海天 / 厨邦 系列齐全?
   □ 蒸鱼豉油有 ≥ 2 个 SKU?
   □ 蚝油有 ≥ 2 个品牌?
   □ 老抽 / 生抽分开陈列?

□ 米面油区:
   □ 主力 5kg/10kg 装齐全?
   □ 价格梯度清晰(P 域 + M 域)?
   □ 本地米(丝苗米)有专门陈列?

□ 调味品区一句话评价:_______
```

### Part 4:IPC 价格曲线感知(60 秒)

> **观察点**:走过 3~5 个核心货架,凭直觉感知价格分布

```
方法:随机看 3 个货架(调味品 / 零食 / 饮料),每个货架数 10 个 SKU,把价格按 4 段记录

价格段分布(按 4 段简化版):
  < 8 元:____ 个 SKU(目标 ≥ 25%)
  8~25 元:____ 个 SKU(目标 ≥ 40%)
  25~50 元:____ 个 SKU(目标 ≤ 20%)
  > 50 元:____ 个 SKU(目标 ≤ 10%)

凭直觉判断 IPC 形态:
  ⊙ 右下倾斜(健康 ✅)
  ⊙ 右上偏斜(危险 ⚠️)
  ⊙ 双峰(混乱 ❌)
  ⊙ 平坦(无重点 ⚠️)

P 域占比估计:____%
  ≥ 65% → 良好
  50~65% → 警告
  < 50% → 危险

→ IPC 评分(0~3 分):____ 分
```

### Part 5:陈列错配检查(60 秒)

> **观察点**:用"高陈列 vs 低销售"的视角,扫一遍主货架

```
□ 视线层(1.4~1.6m,黄金位)放的是 A 类商品吗?
   □ 看到 ≥ 2 个明显是 C 类的商品占视线层 → ❌ 严重错配

□ 弯腰层(< 1.0m)放的是什么?
   □ 大件包装 / 重物 → ✅ 合理
   □ 小包装且高单价 → ❌ 错配

□ 端架(主动线两端)放的是什么?
   □ 节庆主题 / A 类爆品 / 关联陈列 → ✅ 用对了
   □ 永远是同一组商品 → ❌ 端架资源浪费

□ 收银台前:
   □ 小件冲动品(口香糖/便携食品) → ✅ 标准
   □ 空着 / 杂物 → ❌ 浪费高价值位置

陈列错配数量:____ 处明显错配
→ 陈列评分(0~3 分):____ 分
```

## 五、不同业态的"5 分钟诊断要点差异"

### 5.1 社区超市(乐易购同业态)

```
重点关注:
  ✓ 生鲜区是否被当作 Destination
  ✓ IPC 是否右下倾斜(P 域 ≥ 65%)
  ✓ 大湾区文化品类(调味品/腊味/煲汤料)是否齐全

不必苛求:
  ✗ SKU 数极致精简(山姆/Aldi 模式)
  ✗ 高端进口商品深度
  ✗ 自有品牌占比
```

### 5.2 大卖场(永辉/大润发等)

```
重点关注:
  ✓ 全品类完备性(SKU 数应 ≥ 20000)
  ✓ 多个 Destination 品类共存(生鲜+家电+服饰)
  ✓ 大包装商品占比

不必苛求:
  ✗ 社区超市的本地化文化品类
  ✗ 极致 P 域占比
```

### 5.3 精品超市(Ole'/BLT 等)

```
重点关注:
  ✓ Preferred 品类深度(进口/有机/特色)
  ✓ M 域占比应 ≥ 35%(与社区超市相反)
  ✓ 陈列美感和品质感

不必苛求:
  ✗ P 域价格竞争力
  ✗ 本地大众品的完备性
```

### 5.4 折扣店(Aldi/钱大妈折扣区等)

```
重点关注:
  ✓ SKU 数极简(< 2000)
  ✓ 几乎全 P 域(无 M 域)
  ✓ 自有品牌占比 ≥ 50%
  ✓ 周转极快

不必苛求:
  ✗ M 域 SKU 配置
  ✗ 陈列美感
  ✗ 服务体验
```

### 5.5 便利店(7-Eleven/美宜佳等)

```
重点关注:
  ✓ 即食/速食占比
  ✓ 时段性陈列(早餐/晚餐切换)
  ✓ 收银台/动线旁的冲动购买区

不必苛求:
  ✗ 生鲜深度
  ✗ 大件包装
  ✗ 完整品类
```

## 六、5 分钟诊断的进阶训练路径

```
阶段 1(M1~M3):用满 5 分钟,逐项勾选
  目标:形成判断框架的肌肉记忆
  辅助工具:本卡片打印 + 拍照存档

阶段 2(M4~M6):压缩到 3 分钟
  目标:跳过明显项,直奔核心
  辅助工具:本卡片简版(只保留 IPC + Destination + 3 个问题)

阶段 3(M7~M12):压缩到 1 分钟
  目标:进店扫一眼就有初步判断,详细诊断在脑子里
  辅助工具:不用卡片,直接在手机笔记里写 3 个问题

阶段 4(Y2 之后):"扫一眼"式直觉
  目标:走过门口就有判断,无需进入也能初判
  辅助工具:无,本能反应

→ 训练频率:每周 ≥ 3 次,每月 ≥ 12 次
→ 训练目标:1 年内累计完成 ≥ 100 次 5 分钟诊断
```

## 八、注意事项

```
⚠️ 5 分钟诊断 ≠ 完整诊断
   只是建立"第一印象判断"
   真正的诊断需要数据 + 顾客访谈 + 内部访谈

⚠️ 不要现场对店员/老板宣布诊断结论
   外人 5 分钟看出的"问题"
   可能是经营者深思熟虑的取舍
   先记录,后求证

⚠️ 警惕"专业病"
   工具用熟后容易陷入"挑刺模式"
   忘了"这家店为什么至今还活着"
   每个生存下来的门店都有自己的智慧
   先理解,再诊断
```
