# 02_零售企业生意诊断

## 目录作用

本目录沉淀**零售企业和门店经营全局诊断方法**，是咨询项目启动阶段的核心工具库。

诊断是一切后续工作的前提。这里存放的是"如何判断一家门店的整体健康度"，而不是具体某家店的诊断结果（具体门店的诊断结果存放于 `09_门店案例与项目复盘` 对应门店的 `02_生意诊断` 子目录）。

---

## 适合存放

- 零售企业生意诊断框架（标准诊断流程与维度）
- 社区超市经营诊断清单
- 门店盈利模型分析方法
- 销售额拆解模型（客流 × 客单价 × 频次）
- 毛利额拆解模型
- 客流客单分析方法
- 人效坪效分析方法
- 费用结构分析方法
- 组织能力诊断框架
- 老板访谈提纲
- 店长访谈提纲
- 首访调研框架

---

## 不适合存放

- 某家门店的具体诊断结论（属于 `09_门店案例与项目复盘`）
- 品类结构诊断（属于 `05_品类管理与商品规划`）
- 商品 SKU 诊断（属于 `04_商品诊断与商品力提升`）
- 客户交付版的诊断报告（属于 `10_咨询交付模板`）
