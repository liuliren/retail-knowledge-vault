# 分层→陈列一致性模型 v1.0（#3 × #5 联动）

> **核心问题**:SKU 分层决策(L3/L4/SKU)能否**稳定投影到货架陈列(DisplayMap)而不崩塌**?——即"让 SKU 决策不会在货架上崩掉"。
> **分水岭意义**:这一步过了,系统才从**咨询方法论** → **零售决策操作系统(结构能驱动现实货架)**。
> 联动 [[复杂类分层计算引擎_v1.0]](#3 结构层)× [[零售品类分析方法论_v2.0]] §六B 陈列组(#5 视觉层)。

## ③ 陈列修复规则
| 规则 | 用于 | 动作 |
|------|------|------|
| **视觉合并** | 结构过密型(同Item多品牌/规格) | 合并为一个排面块,减面数不减选择(饼干夹心) |

## ④ DisplayMap 调整规则（源 G4-A 执行位定义 + 8条修正规则）
**货架层级**:
| 层 | 定位 | 放什么 |
|----|------|--------|
| 第1层 形象层 | 轻量形象款 1-2/组 | 形象款(不放礼盒) |
| **第2层 黄金视线层** | 主力 4-6/组 | **高动销主销** |
| **第3层 黄金手拿层** | 主销+利润 6-8/组 | **高转化/高毛利** |
| 第4层 补充层 | 中动销 | 补充/中销 |
| 第5层 大包装层 | 低位 | 大规格/囤量 |

**黄金位分配**:第2-3层 = 主销带主力 + 高毛利(利润带)。
**高/中/低动销映射**:高动销→黄金位(2-3层);中→第4补充;**低但保留(锚点/形象)→第1形象层或浅库存,不占黄金**;汰换→下架。

**纵向陈列规则(零售老刘吸收回填·六哥 2026-06-25 签)**:**同一"类"纵向成柱 + 物理分隔**,把"类优先"决策逻辑投影成顾客"视线垂直扫描"的生理习惯(实现"类要清晰"),让顾客按"选类→选品"自然顺序、每步选择数最小化。这是结构稳定驱动视觉的**投影机制**(零售老刘229 印证本模型)。
**黄金位再定义 = 注意力漏斗高捕获点**:黄金位不只是物理层位,是"基于注意力的漏斗系统"(零售老刘196)的高捕获点;视觉体系=精心设计的注意力漏斗,非装饰。来源见 [[2026-06-25_零售老刘吸收_方法论回填清单_v0.1]]。
**陈列量化 benchmark(零售老刘逐字批16回填·六哥 2026-06-25 签)**:① **黄金视线层 = 90-150cm**(成人自然视线·转化率是顶/底层 **3.2 倍**·尼尔森)→ 第2-3层"黄金位"对应此高度;② **磁石点**:每 ~20m 设焦点/C位陈列 → 驻留 +40%;③ **S型动线**:路径拉长至 1.8 倍 → 关联购买 +27%;④ 五感:暖光 3000K 烘焙 +18%。这些是"黄金位/注意力漏斗"的可量化落地参数。
**陈列标准定义 + 坪效效率公式(零售老刘逐字批18回填·六哥 2026-06-25 签)**:**陈列 = 解决商品在卖场的「位置(空间)+ 数量」**——本模型做透了"位置(分层→货位)",**"数量(排面/陈列量)"是下一步精细化方向**。量化落地:**坪效效率 = Σ(货位销售贡献率 × 曝光时长) / 动线长度**,其中**销售贡献率=货位"价值密度"** → **高毛利/高频商品占高曝光位**(=本模型"高动销+高毛利→黄金位"的公式化印证)。
**Z/F 视觉定律(零售老刘逐字批19/20回填·六哥 2026-06-25 签)**:① **Z定律 = 单个SKU横向陈列、多个SKU纵向陈列**(纵向便于同类比较·160cm+货架 2-3 层中段=黄金区域)——选哪向看SKU数;② **Z/F 互补按品类决策周期选**:**Z 用冲动型消费区(靠整体布局);F(视线F形·左侧优先·靠文字引导)用决策周期长/信息密集区**(奶粉/化妆品/精品电器——POP 文字标识要重)。这是"黄金位/纵向陈列"的视觉规律依据。
**品类面积分配规则(零售老刘逐字批07回填·六哥 2026-06-25 签)**:各品类陈列面积应正比于 **经营定位 + 购物篮价值比例**(销售/销量/毛利/客流综合),不凭感觉。**不按此正比 = "不调不乱、越调越乱"**。这是分层结论落到"分多少排面"的量化依据。
**陈列调整未必正向·铁律(零售老刘逐字批14回填·六哥 2026-06-25 签)**:**调整 ≠ 必然向上销售**("越调越乱越差"常见)。三防:① 防 **SKU 堆叠/单品式误区**(堆头越多、单品越多 ≠ 卖得越好)② 防**通道挤压**(主通道宽度有标准≈120cm,挤到 90-100cm = 压抑不畅、客流受阻)③ 守**整体性原则**(一个萝卜一个坑·单体服从整体·场景式陈列)。调改前后须按本一致性模型验证,不是加堆头就行。

**VMD 体系 = 本模型的陈列方法论根(零售老刘逐字批27/28回填·六哥 2026-06-25 签)**:**VMD(视觉营销)= SD(空间设计)+ MD(商品企划)+ MP(陈列规划);MP = VP + PP + IP(3P)**,实践中 VMD ≈ MP。三层比喻=骨架(SD·舞台)/血肉(MD·核心内容)/神经(MP·导演手法)。**3P 焦点对应"吸引▶兴趣▶比较":VP(磁石·店头/端架/主通道·吸引第一眼)→ PP(导游·品类区/主题·唤起兴趣停留)→ IP(景点·货架主体·方便比较选择)**;3P=点(IP)线(PP)面(VP)构成,模拟注意力从宏观到微观自然衰减(="注意力漏斗"的视觉语义)。**VMD 三大引导(本模型"结构→视觉投影"的执行机制)**:① **视线引导**(纵向陈列对抗水平扫描惯性/横向视线走廊/焦点磁石点截流)② **动线引导**("堵疏"控速度·黄金区走最慢/回字之字端架转向·治右转倾向死角)③ **心理引导**(价格锚点/关联需求链/稀缺紧迫)。**铁律:VP/PP/IP 一致性 = 评价 VMD 的核心标准**(卖点买点风格主题要一致,杂乱=表达不清)——这正是本"一致性模型"的视觉侧总纲。⚠️ VMD 最难=长期动态维护(专家指导一时有效·能长期维持几无);某区/品类销售恶化→先换 VP/PP(须基于需求分析)。

**IP 执行标准(零售老刘逐字批28回填·六哥 2026-06-25 签)**:**IP(单品陈列)占 60-80% 陈列面积、约 70% 销售 = VMD 的"基本面/压舱石"**;核心=**"用极致的秩序换极致的效率"**(去故事化纯功能)。① **分类树(大类→中类→小类)= 隐形信息架构**,让顾客自助"导航→筛选→决策"(买苹果:水果区→苹果区→某种苹果·须配清晰 POP 类别指示牌);② **格子式 IP**:同品牌**纵向占整列**(横向比品牌/纵向选规格)——印证上文「纵向陈列规则」;③ **正面陈列铁律**:黄金视线层 **100% 正面朝外 + 向前对齐**(库存不足前移,不可转侧面/背面充数);正面="卖与买"(吸引转化)、侧面="存与序"(增容量/量感);④ **二指原则**(商品间距能插两指=刚好够卖的饱满秩序感)；⑤ **货架分层正侧面**:黄金层(70-150cm)正面/底层高层侧面增容量/端架"双向正面"(以曝光率为主·单位面积销售 1.5-5 倍)。

**🧭 总纲:注意力经济学 = 整个视觉体系的"引擎"(零售老刘逐字批30回填·六哥 2026-06-25 签)**:本模型所有视觉规则(纵向陈列/黄金位/Z-F定律/VMD/3P/POP)的**共同底层逻辑 = 注意力经济学**。顾客注意力=**稀缺资源**(只能关注 <1% 信息);**一个好的视觉体系 = "精心设计的基于注意力的漏斗系统"(非装饰)**——这是上文"黄金位=注意力漏斗高捕获点"的体系化升级。**八大法则(诊断/设计的检核清单)**:① **3-30-3**(3米可见色块/30cm可读信息/3秒传核心诉求)② 行为触发 ③ **选择性注意=显著性优先**(每个视觉元素须"够不一样"才被注意到)④ **认知负荷=认知减负**(工作记忆仅 3-5 块·**文字超3行/颜色超3种 → 阅读率降 63%**)⑤ **格式塔三律**(相似=同档期统一VI/接近=POP距商品≤30cm/闭合=组合陈列脑补出场景)⑥ **3秒法则=三秒定生死** ⑦ **视觉疲劳/边际效用递减**(VP 周期≤3-4周·**促销品占比≤20-35%**·否则爆炸卡失去"突出"意义)⑧ **希克定律=选择即阻力**(减少选择=帮顾客做选择·呼应 SKU 精简/窄品)。四句指令集:突破过滤(显著性)/降低负荷(极简)/引导感知(格式塔)/辅助决策(社会证明+减选择)。

**陈列两大元素 = 位置 + 数量(零售老刘逐字批29回填·六哥 2026-06-25 签)**:陈列只有两个基本元素——**位置(空间)+ 数量(排面/陈列量)**。**现实坑:80% 精力花在"位置",而"数量"被漠视 → 陈列最多只做到 50-60%**(基数低还自以为做好)。本模型已做透"位置(分层→货位)",**"数量"是明确的待补一极**(对接 [[单品管理]] 可得性/陈列量)。背景:**约 70% 购买=冲动性·决策平均仅 19 秒 → 80% 陈列努力都是为冲动购买服务**;陈列=典型"背包问题"(有限空间 × 几乎无限需求·按商品价值最大化取舍)。

**类要清晰 = 宽类窄品在陈列端的落地铁律(零售老刘逐字批35回填·六哥 2026-06-25 签)**:**"类要清晰"= 门店陈列出的"类"无限接近、适配顾客心智的"类"**(理论根=类优先 + 希克定律 + 认知减负·见 [[商品配置理论]]§16C)。核心="**一眼能辨,三步能懂**"(3 秒判断当前是哪个类·1.5 米外看清类边界)。三层落地:**① 宏观**(大类空间区隔·主通道作天然分界·地面色/吊顶/灯光区分)**② 中观**(子类视觉标识·PP·分类 POP 牌·不同色价签)**③ 微观**(同类**纵向陈列成"类柱"**+ 分隔条/空排面 + POP 插条)。四注意:① **避免过度分类**(每层 1 个维度·复合属性越多类越不清晰)② 分类一致性(全店统一·优先按场景/功能分)③ 动态调整慢过渡("搬家告示"POP 留 1-2 周)④ **标识与位置精确对应**(一物一签·补货不得跨类乱放)。**类 POP 牌必须用顾客听懂的语言**("普通/精品猪肉"=门店管理语 → 换"骨头类/五花肉类/肉馅类"=顾客语·手绘 POP 更亲切)。验证=顾客问"X 在哪"的次数越来越少。⚠️ **90% 门店只有陈列、没有"落位"**(开业后落位图就不再被销售反馈驱动优化)——本一致性模型正是补"落位→陈列→反馈"闭环。详见 [[论类要清晰_类清晰原则_v0.1]]。

**POP = VMD 的"台词"·有用性判据(零售老刘逐字批29/30回填·六哥 2026-06-25 签)**:**卖场POP=全店级"视觉发声系统/脸面"(乱了赶客)vs 陈列POP=点位级"销售助攻/喉咙"(哑了丢单)**·勿张冠李戴;按大中小对应 VP/PP/IP。陈列POP 三层=**吸引(0.5-3秒)→说服(3秒·5-8字卖点)→推动(5-9秒·限时/社会证明)**;**极简主义**(一POP一信息·主语商品+谓语利益+宾语行动)。**有用POP 四原则=可看见 / 可读懂(3秒≤8字)/ 可信任(价真·过期即撤)/ 可行动**;**核心=站顾客角度,不是自嗨**(反例:"水果新鲜·天天低价"口号 POP=无用——不传具体信息、不与陈列商品挂钩;新鲜是顾客长期体验出来的,POP 造不出)。**VI=灵魂/宪法,POP=代言人/战术**(无 VI 约束=野广告)。**实操检核法:蹲下 + 退2-3步 + 默数3秒 + 三问**(3秒看懂?帮我还是烦我?会因它买吗?·发现 85% 问题)。可进 [[报告导出系统_report-export_v1.0]] 的陈列诊断项。

## ⑤ 系统一致性结论:**YES（规则稳定),但执行完整性需归位跑完**
- **结构层能稳定驱动视觉层**:决策组(L4)→ DisplayMap zones → 层位,有清晰映射规则;3 品类的失真都被"4 失真型 → 3 修复规则 + DisplayMap 规则"捕获并给出修复。规则不靠品类特例。
- **证据**:G4-A 真实数据上,结构(决策组重构)与视觉(执行位定义)映射规则一致;T03/T05 的失真型被规则正确预测。
- **诚实边界(caveat)**:G4-A 暴露"结构定义完整但 SKU 级归位未跑完(Z5锚点带空白/49未标层位)"——**规则稳定 ≠ 执行已完成**。结论是"结构**能**稳定驱动视觉",落地完整性依赖归位执行到位。
