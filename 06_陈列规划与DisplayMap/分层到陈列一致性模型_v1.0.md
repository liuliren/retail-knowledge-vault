# 分层→陈列一致性模型 v1.0（#3 × #5 联动）

> **核心问题**:SKU 分层决策(L3/L4/SKU)能否**稳定投影到货架陈列(DisplayMap)而不崩塌**?——即"让 SKU 决策不会在货架上崩掉"。
> **分水岭意义**:这一步过了,系统才从**咨询方法论** → **零售决策操作系统(结构能驱动现实货架)**。
> 联动 [[复杂类分层计算引擎_v1.0]](#3 结构层)× [[零售品类分析方法论_v2.0]] §六B 陈列组(#5 视觉层)。

## ③ 陈列修复规则
| 规则 | 用于 | 动作 |
|------|------|------|
| **视觉合并** | 结构过密型(同Item多品牌/规格) | 合并为一个排面块,减面数不减选择(饼干夹心) |
| **强拆货架** | 价格带遮蔽型(断层) | 高/中/低价格带**分区陈列**,高端单独区(果干高端) |
| **隐藏但保留** | 防误杀的低销锚点/形象款 | 不占黄金位,放形象层/浅库存保留(呼应 [[SKU角色层与目的品保护机制_v0.1]]) |

## ④ DisplayMap 调整规则（源 G4-A 执行位定义 + 8条修正规则）
**货架层级**:
| 层 | 定位 | 放什么 |
|----|------|--------|
| 第1层 形象层 | 轻量形象款 1-2/组 | 形象款(不放礼盒) |
| **第2层 黄金视线层** | 主力 4-6/组 | **高动销主销** |
| **第3层 黄金手拿层** | 主销+利润 6-8/组 | **高转化/高毛利** |
| 第4层 补充层 | 中动销 | 补充/中销 |
| 第5层 大包装层 | 低位 | 大规格/囤量 |

**黄金位分配**:第2-3层 = 主销带主力 + 高毛利(利润带)。
**高/中/低动销映射**:高动销→黄金位(2-3层);中→第4补充;**低但保留(锚点/形象)→第1形象层或浅库存,不占黄金**;汰换→下架。

**纵向陈列规则(零售老刘吸收回填·六哥 2026-06-25 签)**:**同一"类"纵向成柱 + 物理分隔**,把"类优先"决策逻辑投影成顾客"视线垂直扫描"的生理习惯(实现"类要清晰"),让顾客按"选类→选品"自然顺序、每步选择数最小化。这是结构稳定驱动视觉的**投影机制**(零售老刘229 印证本模型)。
**黄金位再定义 = 注意力漏斗高捕获点**:黄金位不只是物理层位,是"基于注意力的漏斗系统"(零售老刘196)的高捕获点;视觉体系=精心设计的注意力漏斗,非装饰。来源见 [[2026-06-25_零售老刘吸收_方法论回填清单_v0.1]]。
**陈列量化 benchmark(零售老刘逐字批16回填·六哥 2026-06-25 签)**:① **黄金视线层 = 90-150cm**(成人自然视线·转化率是顶/底层 **3.2 倍**·尼尔森)→ 第2-3层"黄金位"对应此高度;② **磁石点**:每 ~20m 设焦点/C位陈列 → 驻留 +40%;③ **S型动线**:路径拉长至 1.8 倍 → 关联购买 +27%;④ 五感:暖光 3000K 烘焙 +18%。这些是"黄金位/注意力漏斗"的可量化落地参数。
**陈列标准定义 + 坪效效率公式(零售老刘逐字批18回填·六哥 2026-06-25 签)**:**陈列 = 解决商品在卖场的「位置(空间)+ 数量」**——本模型做透了"位置(分层→货位)",**"数量(排面/陈列量)"是下一步精细化方向**。量化落地:**坪效效率 = Σ(货位销售贡献率 × 曝光时长) / 动线长度**,其中**销售贡献率=货位"价值密度"** → **高毛利/高频商品占高曝光位**(=本模型"高动销+高毛利→黄金位"的公式化印证)。
**品类面积分配规则(零售老刘逐字批07回填·六哥 2026-06-25 签)**:各品类陈列面积应正比于 **经营定位 + 购物篮价值比例**(销售/销量/毛利/客流综合),不凭感觉。**不按此正比 = "不调不乱、越调越乱"**。这是分层结论落到"分多少排面"的量化依据。
**陈列调整未必正向·铁律(零售老刘逐字批14回填·六哥 2026-06-25 签)**:**调整 ≠ 必然向上销售**("越调越乱越差"常见)。三防:① 防 **SKU 堆叠/单品式误区**(堆头越多、单品越多 ≠ 卖得越好)② 防**通道挤压**(主通道宽度有标准≈120cm,挤到 90-100cm = 压抑不畅、客流受阻)③ 守**整体性原则**(一个萝卜一个坑·单体服从整体·场景式陈列)。调改前后须按本一致性模型验证,不是加堆头就行。

## ⑤ 系统一致性结论:**YES（规则稳定),但执行完整性需归位跑完**
- **结构层能稳定驱动视觉层**:决策组(L4)→ DisplayMap zones → 层位,有清晰映射规则;3 品类的失真都被"4 失真型 → 3 修复规则 + DisplayMap 规则"捕获并给出修复。规则不靠品类特例。
- **证据**:G4-A 真实数据上,结构(决策组重构)与视觉(执行位定义)映射规则一致;T03/T05 的失真型被规则正确预测。
- **诚实边界(caveat)**:G4-A 暴露"结构定义完整但 SKU 级归位未跑完(Z5锚点带空白/49未标层位)"——**规则稳定 ≠ 执行已完成**。结论是"结构**能**稳定驱动视觉",落地完整性依赖归位执行到位。
