## 三、公式附录（编码唯一口径）

### 3.1 ABC 分类（T-01 / T-02）
```
单维(销额)：按销售额降序累计占比 → ≤60%=A / 60-90%=B / >90%=C
双维(毛利交叉 T-02)：
  销额ABC（同上） × 毛利ABC（按毛利额降序累计：≤60%=甲 / 60-90%=乙 / >90%=丙）
  裁决：
    销额A+毛利甲/乙 = 核心引擎（死保+黄金位+不准断货）
    销额A+毛利丙   = 流量品（保留控排面，找高毛利替代/搭售）
    销额C+毛利甲   = 利润品（不准砍！上移/提曝光/促销）
    销额C+毛利丙   = 双低（进 F 层误判4问，过了才砍）
```

### 3.2 IR 周转系数（T-07）
```
IR = 12 × (1 - 毛利率) / ITO
  毛利率：小数（如 0.25）
  ITO：年库存周转次数
```

### 3.3 安全库存 / A 类水位（T-09）+ 库龄分级（T-10）
```
A类最低水位 = 日均销量 × 到货天数 × 1.5
  → 货架库存低于水位即补货，永不等空（1.5 为通用起点，按业态/补货周期微调）

库龄分级(T-10)：0-30 正常 / 30-60 预警 / 60-90 滞销(进5闸) / >90 重滞(优先清)
售罄闸(T-08)：一个补货周期内售罄率>90% 或经常断货 = 卖太快 → 加订货量，毛利薄可试探提价
```

### 3.4 PSD 单品效率（T-15）
```
PSD = 该区/该档销售额 ÷ 该区/该档 SKU 数
  双信号：PSD 定"该不该补"（高=值得补），毛利率定"补什么款"
  背离时（PSD 高但毛利低）→ 该补，但补品用毛利率二次筛，挑高毛利款
```
