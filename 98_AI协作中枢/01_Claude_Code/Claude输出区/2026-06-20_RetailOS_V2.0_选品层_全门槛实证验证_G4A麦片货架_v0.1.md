## 2. 门槛 4 闸逐闸回测（对专家"补"决策 Z4补2 / Z5补5）

### 闸③ 算得过来 — 重大发现：PSD 与毛利率是**两个信号**
- **Z4 高端区 PSD ¥110.7**，是全货架最高（均值¥36 的 3 倍）、只有 3 SKU 占 12.4% 销售
  → **单品效率极高但供给严重不足 → 选品层独立判定"该补"，与专家"Z4补2"一致** ✅
- **但 Z4 毛利率 19.3% 全场最低**
  → 选品层抓出矛盾：**高端区高效率却低毛利 → 补的高端品必须挑高毛利款，否则越补越拉低毛利**
- 🔧 **新校准（→M2 v0.3）**：「算得过来」闸拆成**双信号**——
  **PSD 决定"该不该补"，毛利率决定"补什么款"**。两者背离时（如 Z4），补品须用毛利率二次筛选。

### 闸② 有位置 — 腾位来源充足，置换式成立
- 即时腾位：**8 个清退**（南国椰子粉840天滞销 / 温莎威尔停卖×2 / 阿华田黄金大麦停卖 等）；
- 候选腾位：**8 个积压>180天**；
- 共 16 腾位 ≥ 7 补 → **空间守恒满足，闭环成立** ✅

### 闸① 真需求 — 缺口与补向一致
- Z4 升级带（高价/进口/礼盒）、Z5 差异化锚点（白咖啡/抹茶/姜茶/燕窝）= 明确需求单元，非跟风；
- 与选品层 Step1"价格带空档(高价)+差异化强单品缺失"判定一致 ✅

### 闸④ 供得上 — 唯一残口
- 主供应商字段存在但样本未填；**Z5 特色风味（小众品牌如旧街场/小山弄/清滋峰）可得性风险真实存在**
  → 印证"供得上"闸的**实际拦截价值**：小众特色品最易断货，补前必须确认供货 ⚠
- 当前用品牌 proxy；**待回填主供应商数据后闭环**。

## 3. 反向回测结论：选品层 ≈ 专家决策 ✅

| 维度 | 专家实际决策 | 选品层独立判定 | 一致? |
|------|------------|--------------|:---:|
| 总体方向 | 净-11（置换式） | 头部足/尾部厚→置换式 | ✅ |
| 补哪个区 | Z4补2 / Z5补5 | Z4(PSD最高+供给不足) / Z5(差异化空档) | ✅ |
| 压哪个区 | Z2/Z3/Z6压缩18 | 低PSD+高积压区压缩 | ✅ |
| 腾位来源 | 隐含 | 8清退+8积压=16 | ✅ |
| 毛利风险 | 未显式标注 | **Z4高端低毛利需挑款**（选品层多抓出1点） | ➕选品层更细 |

> **回测通过**：选品层 5 步 + 4 闸 在真实数据上**独立重现了专家的压缩/补充决策**，
> 并额外抓出"Z4 高端低毛利"风险——规则不仅对齐专家，局部更细。

## 4. 反复验证（跨 zone / 跨子品类）
G4-A 横跨多个子品类，逐 zone 重复跑 4 闸：
- **麦片谷物（Z2主力/Z6袋装）**：高积压+低PSD → 压缩为主，与专家压6+7一致；
- **传统冲饮（芝麻糊/姜茶，散落Z5）**：差异化锚点 → 补，与Z5补5一致；
- **咖啡冲饮（散落Z2/Z5）**：雀巢三合一大量积压(180-330天) → 压缩；温莎威尔停卖→清退；
- **奶粉豆奶（Z3功能/Z4高端）**：功能区压缩、高端区补 → 与专家一致。
> 四个子品类**重复验证规则方向一致**，未见反例。

## 5. 验证结论
1. **全门槛数据已找全**：进价/毛利率/供应商字段均在销售统计汇总 schema 内；G4-A 提供 zone 毛利率，4 闸全部可跑（供应商待回填）。
2. **选品层回测通过**：在 74 SKU 真实货架上独立重现专家压缩18/补7的置换式决策。
3. **置换式 + 上新必腾位 = 硬验证**：净-11、补7腾16，空间守恒。
4. **新校准（→ M2 v0.3）**：「算得过来」闸拆**PSD（该不该补）+ 毛利率（补什么款）**双信号；背离时补品用毛利二次筛。
5. **供得上闸价值实证**：Z5 小众特色品可得性风险真实，闸有拦截意义；需回填主供应商数据。
