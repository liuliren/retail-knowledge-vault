# 核心客户门店店型 = 近全品类社区超市

**类型**:project · **定于** 2026-06-26

## 事实
核心客户门店——**花厅坊 + 乐易购沙浦大道店**(六哥曾口误"黄埔大道",实为沙浦大道;库内早期文件或作"沙埔大道")——是**近全品类社区超市**(非纯生鲜店),面积 **1500-2000㎡**。老板经营思路:**在此面积内囊括尽量多品类**(全品类打法),现行沿用,也是未来调整方向。

## Why / How to apply
- **勿把方法论/产品/诊断框死在"社区生鲜超市"**——目标是社区超市(多店型·当前两家均全品类)。文具/玩具/文体等在全品类店是**真品类**(非便民补充),诊断/选品要正经对待、可细分。
- **数据底座原则**:主数据保最细(细分到底·如玩具六分/文具三分),诊断层按需 rollup;但**当前两家店同为全品类型,暂无"生鲜型/综合型"真实数据,故不预建多店型 rollup 引擎**(防过早系统化·真数据驱动)。等真出现别的店型再长。
- "社区生鲜超市"措辞在方法论/宪章里宜泛化为"社区超市(多店型·含全品类)"。
