# 时间标准:一律北京时间(铁律)

**类型**:feedback · **定于** 2026-06-27(六哥立为铁律)

## 规则
**所有计划、任务、以及一切以时间计量/表述的事情,默认按【北京时间(UTC+8)】标准。** 除非六哥明确告诉我改,否则一律北京时间——这是铁律,不是默认值。

## How to apply
- **排期/定时**:任务几点跑、deadline、提醒、cron/launchd 定时(如"每天 7:30 跑")——全部北京时间。
- **复盘/日志日期**:复盘页、log、断点 RESUME 的日期与时点,按北京时间。
- **相对→绝对**:把"今天/明天/下周"等相对时间转成绝对日期时,按北京时间锚定。
- **跨时区内容**:若处理外部数据带其他时区,显式换算回北京时间再呈现。
- 系统环境是六哥本机(darwin·大湾区),本就北京时区;此律确保我任何"自造"的时间表述也对齐,不漂。
关联 [[开机先读RESUME断点]](断点日期按此律)。
