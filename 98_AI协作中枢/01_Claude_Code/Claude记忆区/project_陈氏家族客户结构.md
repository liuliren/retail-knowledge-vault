## 1. 核心事实（D014 + D017 最终版 / 2026-05-09 定）

- **乐易购 = 陈氏家族 4 店连锁**：市场店 / 沙埔大道店 / 平沙店 / 花厅坊店，共用"乐易购"招牌。花厅坊是 4 店中 1 店；服务陈氏家族未来集团整合 = 凤凰计划真实目标客户。
- **家族 3 支治理结构**：
  - 支 1 陈海鹏（叔叔 / 集团操盘）— 娜姐（其夫人 / 花厅坊牵头之一）— 高彬（娜姐表弟 / 花厅坊副店长+散称采购）；主管市场店+大道店。
  - 支 2 启明父亲（陈海鹏兄弟 / 大股东级）— 启明（二代 / 花厅坊老板 / 小股东 / 兼水果采购 / 数据·POS 协调关键人，F-09）；主管平沙店。
  - 支 3 千千姐=芊姐（陈海鹏堂妹 / D017 同一人确认）：市场店+大道店财务现金 + 沙埔大道店采购；阿健（陈健 / 堂外甥 / 5-10 入职花厅坊蔬菜区理货员）。
- **花厅坊门店 9 人**：启明（老板）→ 彭新华（执行店长 / 非家族）/ 高彬（副店长）/ 张力（水果蔬菜主管）/ 佳妮（财务+休食采购）/ 尧长莲（文员+饮料低温采购）/ 理货员组。
