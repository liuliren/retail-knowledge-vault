# 培训/对客材料制作三原则（六哥 2026-07-08 口述批评沉淀）

**背景**：采购训练第1课 PPT v1 被批"看着懵"——用了系统分析的结论性表述（"14条滞销10条冤案"），受众是没参与过系统分析的采购团队。

**Why**：一线团队没有分析上下文，加工过的结论对他们=黑话；他们需要从自己认识的具体商品进入。

**How to apply（三原则）**：
1. **具象优先**：讲案例必须落到具体条码/品名/实际情况，逐个拆解直接分析；配套真实清单（纸质+电子）先备好并在材料中体现。不用"个别编排的东西"（金句/概念页）替代真实数据
2. **黄金圈结构**：先 WHAT（思路是什么·从顾客消费任务出发要改变什么）→ HOW（怎么做）→ WHY（为什么）
3. **实操为王，删系统黑话**："诊断/冤案/翻案/hypothesis/M-DEC"等内部词不进一线材料，全部换大白话

**边界**：含真实品名/条码的训练件=client_confidential，仅存输出区（gitignored）供打印/现场用，不入 git 不上云。
