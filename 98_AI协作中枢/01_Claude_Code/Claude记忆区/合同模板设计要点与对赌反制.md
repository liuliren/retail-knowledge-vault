# 合同模板设计要点（2026-07-05 起草·待签）

**文件**：`输出区/2026-07-05_服务合同模板_988与陪跑_draft_v0.1_待签.md`（988确认单 + 2万陪跑协议）。

**三条设计不变量**（源自主文档 §5.1/5.1b 已验证的本土对赌翻车调研，后续任何改版不得破坏）：
1. **效果必须落纸**：进场 7 天内共签《效果基线确认单》（指标 3-5 项、基线、达标线、POS 数据口径、剔除项）；无附件A = 裸奔对赌。
2. **留痕举证在我方**：客户配合义务只列 3 条可验证行为，未履行须当时微信书面留痕；**没留痕不得拒退**——把翻车主因（服务方拿"客户不配合"当万能拒退借口）反向锁死在我方纪律上。
3. **无锁死条款**：客户任意月可停、退款 7 工作日原路退、不设违约金；退款对赌只锁首月，续约月用"随时可停"替代。

**待六哥 4 裁决点**：效果门槛 ≥2/≥3 项；首月中途终止结算；案例授权条款写法；签约主体（个体户字号 vs 个人，问代账/法务）。

**关联事实**：旧报价单 v0.1（2026-05-04·3-8千/6-12万）定价已被 07-04 拍板（988/2万）取代，签字后需加 superseded 注记，防销售场景两套价打架。
