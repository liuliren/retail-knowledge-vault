# 经营看板产品 · 装配框架 v1.0（draft·待六哥碰）

## 结论先行
- **这是什么**：把老板"看不懂自己店"变成"一屏看懂经营健康"——指标自动算、看板月度刷、复盘出动作。是诊断产品的**持续跟踪层**（首诊是一次性，看板是长期黏性）。
- **架构铁律**：装配层。指标口径 SSOT 全在 [[KB-METRIC-DICT-001_经营指标字典_v0.2]]，本页只装配"指标→看板→复盘"的产品流。

## 二、可跑的看板流（3 步·月度循环）
1. **指标采集**→ POS 导出 → [[KB-DX-MATH-001_零售数学诊断工具体系_v0.1]] 算指标，口径查 [[KB-METRIC-DICT-001_经营指标字典_v0.2]] → 出各维数值。
2. **看板呈现**→ 五组指标（业绩/毛利/周转/效率/客流）红绿灯化 → 一屏经营健康仪表。
3. **复盘决策**→ 按 [[S07_经营复盘与凭证沉淀]] 对比上月/目标 → 出"这月做对/做错/下月 3 动作"。
