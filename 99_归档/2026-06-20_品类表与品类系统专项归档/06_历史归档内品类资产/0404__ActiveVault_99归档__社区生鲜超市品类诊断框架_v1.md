# 社区生鲜超市品类诊断框架 v1.0

**适用范围：** 大湾区 1000-2000㎡ 全品类社区生鲜超市  
**诊断时长：** 半天（4小时）现场走访 + 半天报告输出  
**交付物：** 本诊断报告 + TOP3问题清单 + 优先改造建议  
**理论基础：** CDT顾客决策树 × 商品配置三层结构 × 零售数学分析框架  

---

## 诊断底层逻辑

> 零售竞争的本质是两件事：**卷价格** 和 **卷时间**。  
> 社区生鲜超市的核心竞争力，是让顾客用最短的时间，找到最想买的东西。  
> 品类诊断的目的，就是找出哪里让顾客"浪费了时间"或"找不到东西"。

**销售额底层公式：**
```
销售额 = 客数 × 客单价
客单价 = 购买件数 × 件均价
客数   = 老顾客 × 来店频率 + 新顾客
```
每一个诊断模块，都指向这个公式里的某个变量。

---

## 诊断前准备（走店前）

向老板索取或观察以下数据（有则看，无则估）：

- [ ] 日均客流量（大约多少人/天）
- [ ] 客单价大约多少元
- [ ] 生鲜占全店销售额的比例（是否超过40%）
- [ ] 门店最近一个月有没有做过调价或陈列调整

## 模块一：生鲜护城河诊断
> **对应公式变量：客数（来店频率）**  
> 生鲜是社区超市的护城河。顾客每天来不来，取决于生鲜好不好。

### 1.1 肉类诊断

| 诊断维度 | 检查要点 | 健康标准 |
|------|------|------|
| 猪肉品种结构 | 白猪/黑猪/高档品牌猪的陈列占比 | 白猪≥50%，黑猪次之，高档品牌有增长趋势 |
| 价格带完整性 | 低中高三档价格是否覆盖 | 三档齐全，不缺位 |
| 售卖形式 | 是否有现场加工、烟火气氛围、打包售卖结合 | 有现场加工感 ≥ 1处 |
| 牛肉区域 | 是否独立陈列，高档感和氛围是否到位 | 独立区域，有氛围感 |
| 三鸟类 | 品种是否齐全（大湾区：走地鸡/清远鸡/肉鸡等） | 按家庭烹饪场景配齐 |
| 库存管理 | 是否有隔夜肉，是否有阶梯式日清打折机制 | 无隔夜肉，有日清机制 |

**现场快速判断：** 下午4点后走肉类区，看有没有积压品、有没有打折标签处理机制。

### 1.2 蔬菜诊断

| 诊断维度 | 检查要点 | 健康标准 |
|------|------|------|
| 叶菜保鲜 | 是否有喷雾加湿设备 | 有喷雾，叶菜不蔫 |
| 分类陈列 | 叶菜/根茎/瓜果/菌菇是否分类清晰 | 参考奥乐齐分类逻辑 |
| 鲜度管理 | 是否有变质商品混杂在正常品中 | 零容忍 |
| 本地菜占比 | 当季本地菜是否有主推 | 有明显本地/当季主推 |

### 1.3 水果诊断

| 诊断维度 | 检查要点 | 健康标准 |
|------|------|------|
| 鲜度与品质 | 外观、硬度、颜色是否达标（品质要"上称"） | 无明显瑕疵品混入 |
| 陈列丰满度 | 陈列面是否饱满，有没有空洞感 | 无空洞，陈列满 |
| 主推品逻辑 | 是否有季节性主推，堆头逻辑是否清晰 | 有主推堆头 ≥ 1处 |

### 1.4 鸡蛋/日配

| 诊断维度 | 检查要点 | 健康标准 |
|------|------|------|
| 鸡蛋陈列 | 是否冷风柜售卖 | 冷风柜陈列 |
| 价格带 | 普通蛋/土鸡蛋/品牌蛋价格带是否齐全 | 三档价格带覆盖 |

**模块一评分：\_\_\_\_ / 25分**

## 模块二：品类结构诊断
> **对应公式变量：客单价（购买件数）**  
> 基于CDT（顾客决策树）——顾客买东西的决策路径决定了商品该怎么配。

### 2.1 CDT符合度检查

顾客买猪肉的决策路径：**部位 → 品种 → 价格**  
顾客买饮料的决策路径：**口感（碳酸/无糖/果汁）→ 品牌 → 规格**

**快速判断法：** 站在某个品类区域，假设自己是顾客，按照自然决策路径找商品——能在30秒内找到想要的，CDT匹配度合格。

### 2.2 商品三层结构检查

| 商品类型 | 定义 | 检查要点 | 常见问题 |
|------|------|------|------|
| **核心商品**（差异化/引流） | 顾客选择这家店的理由 | 是否有明确的"只有这里才有"或"这里最好" | 没有差异化，跟隔壁店一样 |
| **普遍商品**（基础必备） | 哪里都有，必须齐全 | 主流品牌是否缺位 | 缺品牌，顾客会去别处 |
| **便利商品**（冲动/连带） | 顺手买的 | 关联品类是否相邻陈列 | 调味料离肉类太远，连带丢失 |

### 2.3 品类结构问题清单

- [ ] **品类缺位**：该有的品类没有（例：大湾区必备的腊味、豉油香肠等）
- [ ] **品类冗余**：占空间但动销率极低的品类
- [ ] **引流品选择错误**：引流品价格偏高或非刚需
- [ ] **价格带断层**：某品类只有高价或只有低价，中间断了

### 2.4 休食品类检查

- [ ] 是否跟上季节趋势（节假日、当季流行）
- [ ] 消费潮流品类是否有引入（例：近年流行的低糖/0卡饮料、健康零食）
- [ ] 细分品类满足率：同一品类下，主流细分是否覆盖
- [ ] 品牌满足率：TOP3品牌是否在架

**模块二评分：\_\_\_\_ / 25分**

## 模块三：陈列空间诊断
> **对应公式变量：客单价（冲动购买 + 连带购买）**  
> 陈列是无声的销售员。顾客停下来看的地方，就是钱。

### 3.1 黄金陈列位检查

| 区域 | 黄金位置定义 | 是否给了高价值商品 |
|------|------|------|
| 入口区 | 进门5米内 | □ 是 □ 否 |
| 主通道两侧 | 顾客必经动线 | □ 是 □ 否 |
| 收银台周边 | 冲动购买区 | □ 是 □ 否 |
| 端架/堆头 | 视觉焦点 | □ 是 □ 否 |

**判断标准：** 黄金陈列位是否给了高毛利或高动销商品。若给了低毛利滞销品，是浪费。

### 3.2 关联陈列检查

- [ ] 肉类旁边：腌制料、烧烤酱、生粉是否在附近
- [ ] 蔬菜旁边：火锅蘸料、调味料是否有连带引导
- [ ] 饮料区：是否有零食连带陈列
- [ ] 泡面区：火腿肠、榨菜是否同区

**核心原则：** 顾客的购买场景决定关联逻辑，不是品类分类决定的。

### 3.3 死角与冷区识别

走一遍全店，标记以下区域：
- [ ] 灯光暗、顾客不驻足的区域
- [ ] 最里角、拐角处的商品动销情况
- [ ] 距离入口最远的区域是否有吸引力商品

### 3.4 陈列丰满度

- [ ] 有没有明显的空架、断货情况
- [ ] 商品是否整齐，有没有"乱"的感觉
- [ ] 价格标签是否清晰、准确

**模块三评分：\_\_\_\_ / 25分**

## 模块四：连带与客单价提升诊断
> **对应公式变量：客单价（1客点数）**  
> 客单价提升的本质：让顾客每次多买一样东西。

### 4.1 连带品类配置

| 核心品类 | 应配套的连带品类 | 现状 |
|------|------|------|
| 肉类 | 腌制料/生粉/酱油/蚝油 | □合理 □缺失 □错位 |
| 水产 | 姜葱蒜/柠檬/豉汁 | □合理 □缺失 □错位 |
| 蔬菜 | 沙拉酱/火锅底料/调味料 | □合理 □缺失 □错位 |
| 主食（米面） | 酱料/调味品/速食搭配 | □合理 □缺失 □错位 |

### 4.2 客单价健康度判断

**警示信号（需要重点关注）：**
- 客流没有增加，但客单价在上涨 → 可能在消耗老顾客，性价比感知下降
- 某部门支持率（该部门客数/全店客数）持续下降 → 该品类在失去顾客

**健康标准：**
- 客流增长 + 客单价增长 = 健康增长
- 件数增长（多买一件）优于单价增长（涨价）

### 4.3 场景化选品检查

大湾区特色场景需覆盖：
- [ ] 煲汤场景：猪骨/鸡架/汤料包/枸杞红枣
- [ ] 打边炉（火锅）场景：毛肚/肥牛/火锅蔬菜/底料
- [ ] 家庭快手菜场景：半成品/腌制品/料理包
- [ ] 节庆场景：腊味/年货/节日礼盒（按月份排查）

**模块四评分：\_\_\_\_ / 25分**
