# A2 五分法品类角色分配方法

**方法论作者**：六哥（柳力仁）
**版本**：v1.0
**适用场景**：单区域内多个 L2 品类的角色分配，是区域调改五步法的第二步核心工具
**复用性**：可直接套用到任何区域、任何品类组合

---

## 一、为什么用五分法

晟果方法论中存在三套品类角色口径：

- 总纲三分法：**核心 / 普遍 / 搭配**（适用于 L2 商品配置层）
- 品类经营四分法：**目的性 / 常规性 / 便利性 / 季节/场景性**（适用于 L3 品类经营层）
- v2 框架五分法：**目的性 / 偏好性 / 偶发性 / 便利性 / 季节性**（适用于 L3 中观层）

**单区域调改优先用五分法**，理由：

- 五分法多出"偏好性"——这是面对零食专业店等专业对手时最关键的角色（不能做目的性，但可以做偏好性）
- 五分法把"季节"独立出来——节令商品的经营节奏与其他类型完全不同
- 五分法的颗粒度恰好匹配区域级（不是过粗的全店三分，也不是过细的 SKU 级）

三分法可以保留在 L2 商品配置层用，不混用。

## 二、五分法的定义

### 1. 目的性

**定义**：能让顾客**专程为它来店**的品类。

**特征**：
- 顾客带着明确目的来
- 对品类完整度、价格带、选择效率要求高
- 容易形成"目的性来店理由"
- 在顾客心智中占有该品类的特定位置

**经营要求**：
- 做强选择效率（让顾客快速找到、快速决策）
- 做强价格与品质竞争力
- 做强陈列识别度（最好有专属品牌墙或形象层）

### 2. 偏好性

**定义**：顾客不会专程为它来，但**进店看到会买**的品类。

**特征**：
- 顾客对品类有偏好但不强到专程来店
- 容易被竞争对手以更专业、更便宜的方式截走
- 销售量级通常较大，但同质化高
- 是"基础盘"角色

**经营要求**：
- 做减法（不做全 SKU、保住主力品牌）
- 强化品类清晰度（让顾客一眼分清子品类）
- 不在这上面跟专业对手硬拼宽度和价格

### 3. 偶发性

**定义**：顾客需求**不可预测**、需求出现频率不高的品类。

**特征**：
- 不是日常需求
- 需求一旦出现，希望立刻解决
- 对到手效率要求高（不希望多走两步）
- 通常做"补位"角色

**经营要求**：
- 控制 SKU 量（不做深）
- 必须保留基础覆盖（不能完全不做）
- 陈列邻接相关需求（比如海味零食邻接肉类零食）

### 4. 便利性

**定义**：依附主需求存在的**搭配/补充**品类。

**特征**：
- 单独看需求不大
- 在场景组合中价值高
- 是"提升客单、完善场景"的工具
- 对包装规格要求高（小包装、独立装）

**经营要求**：
- 围绕场景补位
- 控制范围，防止无限扩张
- 主要做小包装、便携装、即食型

### 5. 季节性

**定义**：在特定时间窗口内有明显需求峰值的品类。

**特征**：
- 节令性强（如节庆糖巧、节令坚果礼盒）
- 平时需求微弱
- 窗口期销售爆发
- 库存风险高（窗口过后滞销）

**经营要求**：
- 抓节奏、抓窗口期
- 控制时效性库存风险
- 平时少量保留，窗口前集中铺货
- 通常用端架/季节位承接，不长期占主货架

## 三、五分法的分配判断流程

### 第一步：列出该区域所有 L2 模块

按现有商品分类列清单。

### 第二步：对每个 L2 回答 5 个问题

1. 顾客会专程为这个品类来店吗？（→ 是→目的性候选）
2. 这个品类有强势竞争对手吗？（→ 有→大概率不是目的性）
3. 这个品类是日常需求还是偶发需求？
4. 这个品类需要"独立来店"还是"搭配购买"？
5. 这个品类是常态需求还是节令需求？

### 第三步：用销售数据反向校验

- **目的性候选**——必须有数据支撑：要么销售集中度高（前 20% 占 60%+），要么 TOP20 单品中本品类占多席（具体看品类数据）
- **偏好性**——销售量级大但 TOP 集中度低；同品类有强对手
- **偶发性**——SKU 数量少，单 SKU 产出偏低
- **便利性**——以小包装/独立装为主
- **季节性**——历史销售曲线有明显峰谷

### 第四步：用任务视角终审

每个 L2 必须能用一句话回答："**它在区域里承担什么任务**"。如果说不出，角色还没定准。

## 四、不同角色的资源投放原则

| 角色 | 形象层资源 | 黄金层 SKU 数 | 长尾压缩 | 端架优先级 |
|---|---|---|---|---|
| 目的性 | ★ 设品牌墙 | 多（保深度） | 慎重（保选择空间） | 高 |
| 偏好性 | 一般形象 | 中（保主力） | 大胆（压长尾） | 中 |
| 偶发性 | 简单形象 | 少（保基础） | 大胆 | 低 |
| 便利性 | 不必形象 | 少（小包装为主） | 中等 | 中 |
| 季节性 | 季节形象 | 窗口期多、平时少 | 窗口后清 | 季节专属 |

## 五、常见错误与纠偏

### 错误 1：把"销售大"等同于"目的性"

- 错误判断：膨化销售最大 → 膨化是目的性
- 正确判断：膨化销售大但同质化严重，赵一鸣等专业店是更强对手 → 膨化是偏好性
- 经营动作差异：偏好性做减法，目的性做加法

### 错误 2：把"差异化想象"等同于"目的性"

- 错误判断：高端坚果是客单提升带 → 应该做强
- 正确判断：销售数据 1.3% 占比，顾客根本不买账 → 不是任何级别的差异化锚点
- 经营动作差异：先看数据，再下角色

### 错误 3：所有边界商品都塞进"便利性"

- 错误判断：什么都装不下的就放挂钩面（便利性兜底）
- 正确判断：便利性必须有明确的搭配主体（依附哪个主需求？）
- 经营动作差异：便利性也要"任务级写死"，不是兜底面

### 错误 4：忽略季节性的独立性

- 错误判断：把节令糖巧归到糖巧偏好性
- 正确判断：节令糖巧的销售曲线和经营节奏与日常糖巧完全不同
- 经营动作差异：季节性应独立角色、独立资源、独立窗口管理

## 六、案例：花厅坊休食区的五分法分配

### 6 个 L2 模块的角色

| L2 | 角色 | 核心理由 |
|---|---|---|
| 蜜饯果干/凉果 | **目的性** | 大湾区刚需 + TOP20 占 8 席 + 集中度 66%（最集中） |
| 儿童休食/糖玩 | **目的性** | 周末家庭客流核心 + TOP20 占 3 席 + 赵一鸣弱项 |
| 膨化零食 | **偏好性** | 销售第一但同质化高、赵一鸣强项 |
| 肉类零食 | **偏好性** | 销售第二、晚高峰核心连带 |
| 糖巧果冻 | **偏好性** | 销售盘大 + 已有专项成果 |
| 坚果炒货 | **偏好性** | 瓜子花生主导 8.9%，高端坚果只 1.3% |
| 海味零食 | **偶发性** | 鲜香差异化补充 |
| 豆干素食 | **偶发性** | 麻辣素食年轻客群补充 |
| 综合休闲零嘴 | **便利性** | 与凉果带可联动 |

### 角色分配带来的资源差异

- **目的性带（蜜饯果干 + 儿童）**：G1-A 整面，资源最高优先级，设品牌墙（顺宝）
- **偏好性带（膨化 + 肉类 + 糖巧 + 坚果）**：G1-B + G3-A + G2-B 部分格，做减法
- **偶发性带（海味 + 豆干）**：G3-A 部分格，控制 SKU 量
- **便利性带（综合零嘴）**：G1-B 第 5 格"办公室便携"重定位

## 七、五分法的局限

- **角色不是固定的**——某个品类的角色会随客群变化、时间变化、竞争变化而调整。建议每年重审一次。
- **角色之间存在边界模糊**——比如某个品类既有目的性属性又有便利性属性，需要按"区域里承担的主任务"取舍。
- **不能机械套用**——五分法是工具不是目的。最终判断仍要回到"该区域要解决什么问题"。
