# 02B_重点复杂品类决策稿 V2.2

## 一、文件目的

本文件用于把本轮已经确认的高复杂度板块，整理成可直接回灌到新版品类体系中的结构稿。  
当前范围包括：

- 饮料
- 休闲零食
- 乳品
- 酸奶/乳酸菌
- 个护
- 家清

---

## 二、饮料

### 2.1 结构结论
- **L2：饮料**
- 饮料保留单一 L2
- 复杂度主要放在 L3 / L4，不再继续上拆多个 L2

### 2.2 饮料结构表

| L2 | L3 | L4 决策组 | 备注 |
|---|---|---|---|
| 饮料 | 饮用水 | 纯净水 / 天然水矿泉 / 气泡水 / 家庭大包装 | 高复杂度 |
| 饮料 | 碳酸饮料 | 经典含糖 / 无糖代糖 / 果味气泡 | 高复杂度 |
| 饮料 | 茶饮料 | 无糖茶 / 传统甜味茶 / 果味茶 / 草本植物饮 | 高复杂度 |
| 饮料 | 果汁果味饮料 | 100%果汁 / 复合果汁 / 果味饮料 | 高复杂度 |
| 饮料 | 功能饮料 | 提神能量 / 电解质补给 / 维生素功能 | 高复杂度 |
| 饮料 | 咖啡饮料 | 黑咖 / 奶咖 / 功能咖啡 | 中高复杂度 |
| 饮料 | 植物蛋白/谷物饮品 | 豆奶 / 燕麦谷物 / 椰奶坚果 | 中高复杂度 |

### 2.3 边界说明
- 纯牛奶、酸奶、乳酸菌不归饮料
- 乳品与酸奶/乳酸菌单独并列管理
- 植物蛋白/谷物饮品保留在饮料

## 三、休闲零食

### 3.1 结构结论
- **L2：休闲零食**
- 零食必须预留 L4
- 不建议把儿童、健康、分享直接上升为 L2
- 零食的复杂度主要来自“决策任务并存”

### 3.2 零食结构表

| L2 | L3 | L4 决策组 | 备注 |
|---|---|---|---|
| 休闲零食 | 膨化/薯片 | 日常解馋 / 儿童友好 / 家庭分享 / 轻负担烘焙 | 高复杂度 |
| 休闲零食 | 坚果炒货 | 即食坚果 / 混合坚果 / 健康轻食 / 分享礼盒 | 高复杂度 |
| 休闲零食 | 糖巧果冻 | 儿童糖果 / 即食巧克力 / 节庆分享 / 果冻布丁 | 高复杂度 |
| 休闲零食 | 肉脯卤味 | 肉干肉脯 / 卤味即食 / 解馋高蛋白 / 分享装 | 高复杂度 |
| 休闲零食 | 海味素食/豆干 | 海味零食 / 豆干素食 / 轻辣解馋 / 佐餐休闲 | 中高复杂度 |

### 3.3 零食的 L4 逻辑
休闲零食的 L4 不以品牌拆，而以顾客决策拆，优先考虑：

- 解馋型
- 儿童型
- 健康轻食型
- 家庭分享型
- 节令礼赠型（局部）

## 八、本轮确认结论

本轮已确认的 6 个复杂板块，后续都应统一回灌到：

- `02_品类体系`
- `02A_3+1层级架构方案`
- 相关分析与执行文档

一句话总结：

> 饮料、零食、乳品、酸奶/乳酸菌保留“主 L2 + 深 L3/L4”；  
> 个护、家清保留“业务板块 + 正式多 L2”。
