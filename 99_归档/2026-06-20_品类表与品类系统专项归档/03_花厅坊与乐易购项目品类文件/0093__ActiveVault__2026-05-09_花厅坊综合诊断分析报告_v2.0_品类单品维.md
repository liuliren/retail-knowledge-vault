- **休闲速食 +13%** = 方便食品调改后效果（与之前 v1.1 一致）

### E.3 调改净增量推断（v2.0 / 缺人流维）

[推断] 净增量 = 实际增长 - 大盘溢出（30%）：

| 子组 | 实际 | 大盘 | 净增量 [推断] | 解读 |

| 方便食品 | +12.9% | +30% | **-17%** | 调改前期下滑 / 调改后未恢复到大盘 |

### F.2 L3 级深度诊断（按销额 vs 毛利率四象限）

```
                       高毛利率 (>22%)
                            │
          ⭐ 明星品类         │  ⭐ 现金牛
          (扩品+增排)         │  (维持+精化)
          - 基础调味          │  
          - 酱菜类            │  - 糖果巧克力
          - 蜜饯果干          │  - 果汁饮料
                            │  - 茶饮料
高销额  ──────────────────────┼──────────────────────  低销额
                            │
          ⚠️ 待诊断           │  🔴 待砍
          - 米面主食          │  - D 类长尾
          - 食用油            │  - 真流失 272
          - 低温白奶 ⚠️       │  - 0 销售 SKU
          - 常温白奶          │  
                            │
                       低毛利率 (<22%)
```

**关键诊断**：
- **明星品类**（粤盐 + 海天 + 千禾 + 甘竹 + 鹰金钱）= 增排 + 锁黄金层
- **现金牛**（果汁 + 茶饮料 + 糖巧）= 维持 / 季度评估升 A
- **待诊断**（米面+食用油+乳品）= 高销额低毛利 / 是引流款还是结构问题？
- **待砍**（D 类 1,012 + 流失 272）= 1,286 待清理候选
