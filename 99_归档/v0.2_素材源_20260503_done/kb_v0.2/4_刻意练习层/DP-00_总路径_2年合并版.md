## 一、刻意练习的底层逻辑(共识)

### 1.1 四要素(不可妥协)

```
① 明确目标   每次练习有可量化的结果(不是"分析门店",而是"用六分仪找出该门店毛利损失最大的 3 个品类")
② 高度专注   单次练习 ≥ 2 小时不切换上下文
③ 即时反馈   练完立刻知道好坏(数据 / 同行评议 / Claude 校验)
④ 走出舒适区 主动选不熟悉的品类、不熟悉的城市、不熟悉的客户
```

### 1.2 普通采购 vs 顶尖规划师的工作循环差异

```
普通采购员的循环(无累积效应):
  接到需求 → 凭经验选品 → 上架观察 → 凭直觉调整
  → 10 年经验 = 1 年经验重复 10 次

顶尖规划师的循环(有累积效应):
  接到需求 → 用框架分析 → 量化预测 → 上架对照
       → 偏差分析 → 修正框架 → 形成案例库
  → 每次循环都升级方法论,3 年抵 30 年
```

### 1.3 三大伪练习陷阱(必须警惕)

| 陷阱 | 症状 | 破解 |
|---|---|---|
| 只看书不动手 | 知识库越来越大,实操不增 | 每个新概念当周必须找 1 个乐易购实例验证 |
| 只动手不复盘 | 重复同一种错误 | 每周日 18:00 强制复盘 |
| 只复盘不读书 | 凭直觉做事,无法规模化 | 每月强制读完 1 本 + 输出 1 份笔记 |

### 3.2 Y1-A:知识内化期(M1~M3)

#### M1:知识库重写月

> **核心心法**:Feynman 法则——如果你不能用简单的话给 12 岁的孩子讲清楚一个概念,说明你没真懂。把知识库节点用自己的话重写一遍,是最有效的"假装懂→真懂"测试。

| 周次 | 任务 | 产出物 |
|---|---|---|
| W1 | 重写哲学层 P-01~P-06 | 6 份个人化笔记,每份配 1 个乐易购案例 |
| W2 | 重写商品配置 MC-01~MC-13 | 13 份笔记,每份至少 1 张原创图表 |
| W3 | 重写品类管理 CM-01~CM-09 | 9 份笔记 + 1 篇《品类管理 vs 单品管理对比》长文 |
| W4 | 重写单品管理 IM-01~IM-10 + 零售数学 RM-01~RM-08 | 18 份笔记 + 1 个 Excel 零售数学计算器 |

**纪律**:
- 每天固定 2 小时(早 6:30~8:30 或 晚 21:00~23:00)
- 不允许复制粘贴,每个字自己打
- 每个案例必须来自乐易购或实地观察,不允许编造

#### M2:乐易购全店第一次诊断(练手项目)

> **为什么是乐易购**:你已经是它的总顾问,有数据、有信任、有执行渠道,不需要额外社交成本,能专注于方法论本身。诊断结果可以直接被销售数据验证。

```
W1: 数据收集
  □ 全店 SKU 明细(价格 + 3 个月销售 + 毛利 + 库存)
  □ 商圈调查(半径 1.5km 客流 / 竞争对手 / 居民结构)
  □ 顾客访谈 ≥ 30 人(建立 TPOS 素材库)
  □ 内部访谈(店长 / 采购 / 陈列 / 收银 各 ≥ 2 人)

W2: 整体诊断(哲学层 + IPC 层)
  □ 4B×4P 诊断完备性
  □ 全店 IPC 图 + P/M 域分布判断
  □ TF 函数计算理论周转天数
  → 输出《全店商品结构健康报告》(≤10 页)

W3: 品类深度诊断
  □ 选 5 个最关键品类(建议:生鲜 + 猪肉 + 酱料 + 饮料 + 休食)
  □ 每品类做 HS 矩阵 + CDT + 六分仪
  □ 识别每品类 Top3 问题
  → 输出《5 大品类深度诊断报告》(每品类 5~8 页)

W4: 单品级诊断 + 行动方案
  □ 全店 SKU 做 ABC-Z 分析
  □ "立即删除清单" + "立即引进清单"
  □ 与店长共同制定 3 个月调整路线图
  → 输出《乐易购商品力提升路线图》(含 KPI)
```

**关键产出**:
- 一份 20~30 页完整诊断报告
- 一份可量化 KPI 清单(毛利率提升 X% / 客单价提升 Y 元 / 周转天数下降 Z 天)
- 一份 6 个月后回看用的"自评清单"

#### M3:执行 + 反馈(最关键的 1 个月)

```
每周固定动作:
  □ 1 次现场巡店(拍照、记录、对照诊断报告)
  □ 1 次数据分析(销售 / 毛利 / 动销)
  □ 建立"预测 vs 现实"对照本:
     - 我预测改善了什么?
     - 实际改善了什么?
     - 差距来自哪里?是诊断错了,还是执行错了?
```

**M3 末自评**(达标线):
- [ ] 能独立完成一份完整诊断报告
- [ ] 报告中 ≥80% 的判断被实际数据验证
- [ ] 至少积累 5 个"预测错误"案例(这才是最宝贵的)

### 3.3 Y1-B/C/D:品类深耕(M4~M12)

> **为什么单品类深耕**:广博 vs 精深的取舍——50 个品类各懂一点 = 浮于表面;3 个品类极致深入 = 至少 1 个成为业内权威。**顶尖规划师在 3~5 个品类有"无人能撼动"的认知深度**。

#### 三个核心品类的选择(已锁定)

| 品类 | 选择理由 |
|---|---|
| **生鲜**(M4~M6) | 大湾区最难做、最有竞争力的品类;社区超市的护城河 |
| **调味料**(M7~M9) | 大湾区文化特色品(海天/厨邦/蚝油/腊味腌制);最能体现本地化 |
| **休闲食品**(M10~M12) | 客单价提升关键;与你 5 月在乐易购的休食区调改强呼应 |

#### 每个品类深耕的 3 个月固定流程

```
M(N):    全国/本地业态调研 + 文化研究
  W1: 全国头部门店调研(线上+线下,如生鲜:永辉/盒马/钱大妈/百果园/朴朴)
  W2: 大湾区本地文化研究(如生鲜:粤菜系/煲汤/打边炉/早茶的食材逻辑)
  W3: 全谱系 CDT 绘制(如生鲜:叶菜/瓜茄/根茎/肉禽/水产/豆制品)
  W4: 输出《大湾区社区超市 X 品类规划手册》v1.0(≥30 页)

M(N+1): 在乐易购落地验证 + 数据反馈
  W1: 选 1~2 个该品类的子品类做实际调改
  W2~W3: 调改 + 数据采集
  W4: 数据回看 + 手册 v1.1 修订

M(N+2): 跨城市市调 + 品类内容输出
  W1: 跨城市市调(广州 → 深圳/上海/许昌中选 1)
  W2~W3: 公众号 / 视频号 该品类专题输出(2~4 篇)
  W4: 手册 v2.0 定稿 + 归档
```

#### 每月固定动作(M4~M12 通用)

```
□ ≥ 2 次跨城市/跨业态市调
□ ≥ 3 位品类相关从业者深度访谈(采购/品牌方/店长)
□ ≥ 3 本相关专业书籍(每本 ≥ 1 万字读书笔记)
□ 1 份品类规划手册迭代
□ 1 篇公众号原创深度文章(基于手册)
```

### 4.2 Y2-E:方法论 SOP 化(M13~M15)

**为什么从"做"转到"教"**:
- 能做事不能教别人 → 天花板是你自己的体力
- 能教别人 → 天花板是被你影响的人数
- 规划师的真正价值不在分析报告,而在让更多人按你的方法做出好结果

**M13~M15 输出物**:
- 《社区超市商品力诊断 SOP 手册》(每个工具有标准化模板)
- 标准化诊断报告模板(让接手者照着做也能做出 80 分)
- 每个工具(HS / CDT / IPC / 六分仪 / ABC)对应的 Excel 模板

### 4.3 Y2-F:对外服务验证(M16~M18)

**新客户选择标准**:
```
✅ 业态相近(社区超市 / 精品超市 / 便利店)
✅ 老板愿意配合数据共享
✅ 距离不超过 2 小时车程
❌ 不要选大卖场(业态差异大)
❌ 不要选硬要求"立竿见影"的客户(容易破坏方法论)
```

**服务方式**:
- 单次诊断:3~7 天,2~5 万元(试水)
- 持续顾问:按月/季度,可复制乐易购模式

**关键纪律**:
> 必须做"复盘报告"——这次服务暴露了方法论的哪些不足?这才是新客户对你最大的价值(不是钱)。

模块 3:诊断报告
  - 基于规则的问题识别

模块 4:建议库

## 七、避坑指南(三版陷阱清单合并去重后的 8 条)

> 三个输入版本各有一份"陷阱清单",合并去重后形成下面这 8 条。每条都按"症状 - 后果 - 破解"三段式给出。

### 7.1 知识仓鼠型(理论崇拜)

```
症状:每天读书读文章,知识库越来越大,但实操不增
后果:学了立刻忘 / 学了不会用
破解:每读 1 个新概念,当周必须找 1 个乐易购实例验证
     不应用 = 不算掌握
```

### 7.2 完美主义型

```
症状:等所有数据齐全才动手,看不上"不科学"的同行
后果:80 分方案永远不出门,客户跑掉
破解:80 分立刻执行 > 100 分永远不执行
     在不完美中找最优解
```

### 7.3 工具崇拜型

```
症状:任何场景都用 HS 矩阵 / 六分仪;一次诊断把所有工具用一遍
后果:分析报告冗长低效;客户看不懂
破解:根据问题选工具,不是为了用工具找问题
     一次诊断不超过 3 个核心工具
```

### 7.4 同质重复型

```
症状:反复做擅长的部分(如调味品),回避不熟悉的(如非食/家居)
后果:能力发展失衡,变成"调味品专家"而不是"商品规划师"
破解:每月强制做 1 个不擅长的品类
     不舒适 = 在成长
```

### 7.5 客户依赖型

```
症状:所有时间都在做客户项目,自我精进时间被挤占
后果:技能不进步,变成高级打工人
破解:每周 ≥ 20% 时间用于自我精进
     不接收"破坏方法论"的客户需求
```

### 7.6 闭门造车型

```
症状:只看自己的客户,不研究头部同行
后果:你以为的"创新"是别人玩剩下的
破解:每月强制研究 1 家头部超市
     每季度跨城市调研 1 次
```

### 7.7 内容焦虑型

```
症状:为了更新公众号而硬写,出水文
后果:文章质量下降,反而损害专业形象
破解:宁可一周不更,也不输出水文
     每篇文章必须能让 3 个同行说"学到了"
```

### 7.8 低价竞争型

```
症状:报价心虚,给折扣给得心痒,客户一压价就妥协
后果:你的方法论被廉价化,客户也不珍惜
破解:你的方法论独一无二,你的结果可量化
     不收高价 = 不尊重自己的专业
```

## 八、反馈系统(4 重)

刻意练习最难的不是练习本身,而是**如何获得高质量反馈**。

| 反馈类型 | 频率 | 形式 |
|---|---|---|
| **自我反馈** | 每周日 18:00~19:00 | 5 问复盘:做了什么 / 哪些对 / 哪些错 / 下周改 / 最大收获 |
| **对照反馈** | 每月 | 用同一框架重新诊断同一家店,对比上次差异。差异越大 = 进步越大;差异为零 = 这个月没进步 |
| **同行反馈** | 每季度 | 找 1~2 位水平相当或更高的同行,把诊断报告给他们看,请他们指出 3 个最大问题 |
| **客户反馈** | 每个项目 | 项目交付 3 个月后回访,看方案执行后的真实数据变化 |

## 九、2 年末预期画像

```
知识层面:
  □ 拥有自己消化整理的完整知识库(200+ 节点)
  □ 在生鲜 / 调味料 / 休食 3 个品类有"权威级"认知
  □ 形成可对外输出的方法论 SOP(5~10 个标准化工具)

实战层面:
  □ 服务过至少 5 家不同超市,有成功案例
  □ 在乐易购有可量化的业绩改善
  □ 拥有自己的诊断系统 1.0(软件工具)

影响层面:
  □ 公众号粉丝 5000+(高质量从业者)
  □ 至少 1 次行业演讲
  □ 行业内"大湾区社区超市商品规划专家"初步认知

收入层面(自然结果,不是目标):
  □ 顾问费率从当前提升到 3~5 万/月
  □ 出现"客户主动找上门"的状态
  □ 出版 / 课程 / 培训等多元收入路径打开

身份层面(最重要):
  □ 你不再追求"显得专业",而是"就是专业"
  □ 你看到任何门店都能立刻判断好坏
  □ 你给出建议时心里完全有底
```

## 十、何时调整计划

```
出现以下信号时,立即调整:

🟡 黄灯(优化即可):
  - 连续 4 周完不成周目标 → 计划过激进,降低强度
  - 连续 4 周轻松完成 → 计划过宽松,提高难度
  
🔴 红灯(暂停 + 重审):
  - 心态持续低迷 ≥ 2 周 → 暂停,找根因,可能需要调方向
  - 客户/同行反馈持续负面 → 方法论可能有缺陷,回到知识库补课
  - 连续 ≥ 3 个月没新增可量化产出 → 战术失败,重做季度计划
```

## 十二、最重要的一段话

```

继续。
```
