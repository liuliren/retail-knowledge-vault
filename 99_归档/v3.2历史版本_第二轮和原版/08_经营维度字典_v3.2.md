# 08_经营维度字典 v3.2

> 说明：
> 1. 本文件用于把“主品类树”与“经营维度”彻底拆开。
> 2. 主树回答“它是什么”；经营维度回答“它怎么卖、在哪卖、什么时候卖、是否区域增强”。
> 3. 同一商品只能有一个主归属类，但可以叠加多个经营维度标签。

---

## 一、经营维度总原则

### 1.1 主树与经营维度分离
- 主树：L1 / L2 / L3 / L4（决策组）
- 经营维度：售卖方式、经营区、节庆、礼盒、现制、区域模板等

### 1.2 一主归属，多经营标签
一个商品只能有一个主归属类，但可以同时拥有多个经营维度标签。

### 1.3 经营维度不替代主树
“散称”“礼盒”“节庆”“现制”都不是主树类名，而是经营维度。

---

## 二、经营维度总表

| 维度 | 字段建议名 | 含义 | 示例 |
|---|---|---|---|
| 售卖方式 | selling_mode | 商品以什么方式售卖 | 散称、预包装、现制、组合装 |
| 经营区 | trade_zone | 商品主要在哪个经营区售卖 | 散称区、礼盒区、现制区 |
| 节庆/时令 | seasonal_tag | 是否具有明显时令或节庆属性 | 中秋、春节、端午 |
| 包装/组合 | pack_business_type | 是否属于礼盒、组合包、家庭装等 | 礼盒、组合包、家庭装 |
| 区域模板 | regional_template_tag | 是否属于特定区域模板增强项 | 广东模板、校园模板 |
| 经营角色标签 | business_tag | 用于辅助经营判断的业务标签 | 引流、节庆、现制、区域特色 |

---

## 三、售卖方式字典

字段建议：`selling_mode`

| 枚举值 | 中文说明 |
|---|---|
| bulk | 散称 |
| prepacked | 预包装 |
| combo_pack | 组合装 |
| gift_box | 礼盒 |
| freshly_made | 现制 |
| made_to_order | 现点现做 |
| weighed_pack | 称重包装 |

### 使用规则
1. 售卖方式只描述卖法，不改变主归属类。
2. 同一商品可因时期不同切换售卖方式。
3. “散称零食”“散称冰糖”都不做主树类名，只做售卖方式。

---

## 四、经营区字典

字段建议：`trade_zone`

| 枚举值 | 中文说明 |
|---|---|
| bulk_zone | 散称经营区 |
| packaged_zone | 预包装区 |
| gift_zone | 礼盒区 |
| fresh_zone | 生鲜区 |
| frozen_zone | 冷冻区 |
| instant_food_zone | 现制即食区 |
| checkout_zone | 收银台冲动购区 |
| seasonal_display_zone | 节庆陈列区 |

### 使用规则
1. 经营区回答“主要在哪卖”，不回答“它是什么”。
2. 同一商品可跨经营区，但需保留一个主经营区。
3. 经营区可以为坪效、动线、陈列复盘提供维度支持。

---

## 五、节庆/时令字典

字段建议：`seasonal_tag`

| 枚举值 | 中文说明 |
|---|---|
| spring_festival | 春节 |
| lantern_festival | 元宵 |
| qingming | 清明 |
| dragon_boat | 端午 |
| mid_autumn | 中秋 |
| national_day | 国庆 |
| school_season | 开学季 |
| summer_heat | 夏季清凉 |
| winter_tonic | 冬季进补 |

### 使用规则
1. 节庆标签为可叠加标签，不改变主归属。
2. 节庆标签需要有开始/结束时间。
3. 节庆标签适合用于礼盒、糖玩、汤料、坚果炒货等。

---

## 六、包装/组合字典

字段建议：`pack_business_type`

| 枚举值 | 中文说明 |
|---|---|
| single_pack | 单品装 |
| family_pack | 家庭装 |
| trial_pack | 试吃装 |
| combo_pack | 组合包 |
| gift_box | 礼盒 |
| seasonal_pack | 节庆包 |
| refill_pack | 补充装 |

### 使用规则
1. 包装类型可影响价格带、陈列和补货节奏。
2. 礼盒不入主树，进入包装/组合维度。
3. 家庭装、试吃装可用于辅助 SKU角色判断。

---

## 七、区域模板字典

字段建议：`regional_template_tag`

| 枚举值 | 中文说明 |
|---|---|
| guangdong | 广东模板 |
| school_area | 校园模板 |
| family_community | 家庭社区模板 |
| elderly_community | 老年社区模板 |
| tourism_area | 旅游商圈模板 |

### 使用规则
1. 区域模板用于增强，不用于替代全国标准树。
2. 例：广东模板可启用“汤料滋补干货”“凉果强化”。
3. 区域模板必须有门店簇级别的启用开关。

---

## 八、经营角色标签字典

字段建议：`business_tag`

| 标签 | 含义 |
|---|---|
| traffic_driver | 引流 |
| margin_driver | 毛利驱动 |
| impulse_buy | 冲动购 |
| festive_item | 节庆商品 |
| regional_specialty | 区域特色 |
| freshly_made | 现制 |
| bulk_focused | 散称强经营 |
| checkout_focus | 收银台重点 |

### 使用规则
1. 经营角色标签用于辅助经营分析，不替代品类角色/SKU角色。
2. 同一商品可有多个经营标签，但应限制数量，避免泛化。

## 九、典型映射示例

### 示例 1：黄冰糖（散称）
- 主归属：粮油调味 > 调味品 > 糖/甜味料
- selling_mode：bulk
- trade_zone：bulk_zone
- business_tag：bulk_focused

### 示例 2：散称碧根果
- 主归属：食品饮料 > 坚果炒货 > 坚果类
- selling_mode：bulk
- trade_zone：bulk_zone
- business_tag：impulse_buy / bulk_focused

### 示例 3：中秋坚果礼盒
- 主归属：食品饮料 > 坚果炒货 > 礼盒炒货（可选）
- pack_business_type：gift_box
- seasonal_tag：mid_autumn
- trade_zone：gift_zone

### 示例 4：现炒盒饭
- 主归属：现制即食 > 现制餐食 > 盒饭便当
- selling_mode：freshly_made
- trade_zone：instant_food_zone
- business_tag：freshly_made

### 示例 5：广东汤料包
- 主归属：粮油调味 > 汤料滋补干货 > 配方汤包
- regional_template_tag：guangdong
- pack_business_type：combo_pack

### 示例 6：糖玩
- 主归属：玩具潮玩 > 糖玩
- business_tag：impulse_buy / festive_item
- seasonal_tag：spring_festival（可选）
- 备注：同时保留“食品属性”业务说明

## 十、当前版本结论

本字典的目的，是为后续系统设计提供独立经营层支持。

到 v3.2，经营维度至少应独立管理以下信息：
- 怎么卖
- 在哪卖
- 什么时期卖
- 是否礼盒/组合
- 是否区域增强
- 是否具有特殊经营任务
