# G4-A 冲调首战军师简报 v0.3（5/6 落地版）

> **v0.2 → v0.3 关键修正**：v0.2 误用 GPT 母版 C002 的"早餐麦片/谷物冲调/奶粉营养/咖啡奶茶/茶叶蜂蜜/端架主题"6 组分组（按 L4 类聚）。
> 实际六哥版本B 设计是**按 SKU 角色 / 顾客决策路径**分组：
> **低价引流 → 主力奶类麦片 → 功能细分 → 高端升级 → 特色风味 → 袋装囤量·趣味**
> 这是 CDT 顾客决策树横向逻辑（同一 L4 的 SKU **会分散到多组** / 不同价位定位不同角色）。

**6 组合计**：当前 74 → 目标 63（结构重组 + 净减 11）

**关键洞察**：
- **Z1 引流带毛利率 30.2% 最高** — 印证"引流不是亏本"，低价高频反而结构最健康
- **Z6 占当前销售 26.3% / 16 SKU** — 说明现有大包装在卖（不应全砍 / 但要砍掉 7 个不动销大包）
- **Z4 仅 3 SKU** — 升级带过瘦 / 客单价升级路径缺失
- **Z5 特色 12 SKU** — 已有底子 / 补 5 个做强差异化锚点

## 三、6 层垂直规则（来自 GPT 案例 C002 8 条修正规则）

适用于 Z1-Z5 主货架（Z6 自身就是"大包装 + 家庭装"组，纵向不再分大包装层）：

| 层 | 角色 | 硬规则 |
|---|---|---|
| 第 1 层 形象层 | 轻量形象款 | ❌ **不放礼盒**（修正规则 #1 / #3）✅ 大牌头部款 |
| 第 2 层 黄金视线 | 主力 SKU | ✅ 主销 + 高毛利 |
| 第 3 层 黄金手拿 | 高转化 | ✅ 主销 + 利润 |
| 第 4 层 补充层 | 二线 / 长尾补充 | ✅ 二线品牌 |
| 第 5 层 大包装/补充 | 大包装下移（Z1-Z5 内的少量大包）| ✅ 仅 Z1-Z5 内残留大包 |
| 第 6 层 围货层 | 整箱 / 库存缓冲 | ✅ 围货 |

**8 条核心修正规则**（C002 案例库）：

1. ❌ 第 1 层不放礼盒
2. ✅ 蜂蜜放在 Z5 特色风味区（蜂蜜+柠檬蜜归 Z5，与茶饮"相邻不混排"）
3. ❌ 礼盒不上第 1 层主角
4. ✅ 大包装/家庭装/整箱下移到第 5-6 层（或归 Z6 整组）
5. ✅ 奶粉按人群分（学生/成人/中老年），不按品牌混排（**Z3 功能区核心规则**）
6. ✅ 咖啡集中做品牌墙（Z2 雀巢系块状陈列）
7. ✅ 茶叶与蜂蜜相邻但不混排（Z5 内子分区）
8. ✅ 端架优先早餐 / 营养主题（不放弱关联）

### 4.2 Z2 主力奶类麦片区（21 → 15 / 砍 6 个）

**TOP 3**（保留-观察）：

**8 条修正规则适用**：#6 咖啡集中做品牌墙

### 4.3 Z3 功能细分区（14 → 9 / 砍 5 个）

**核心规则**：**奶粉按人群分**（学生/成人/中老年），**不按品牌混排**（修正规则 #5）

**当前问题**：14 SKU 含多品牌混杂 — 雀巢怡养羊奶粉 / 伊利欣活配方 / 维维高钙等。建议同人群同段陈列：
- 学生段：1-2 SKU
- 成人段：2-3 SKU
- 中老年段：2-3 SKU
- 高钙/高铁/无糖等功能特殊段：1-2 SKU

**清退**：
- 雀巢怡养配方羊奶粉 675g ¥258（**价位过高 / 不匹配社区**）
- 雀巢高钙营养奶粉 850g ¥99（同上）

### 4.4 Z4 高端升级区（3 → 5 / 补 2 个）

**当前唯一**：雀巢醇品咖啡玻璃瓶 90g ¥35（不动销 / 但符合升级带角色定位 / 留作形象）

**补 2 个建议**：中高端形象款（¥30-50 价位带 / 客单价升级路径）
- 候选：高端燕麦片（如桂格金装）/ 某品牌中高端礼盒款（节令）/ 进口奶粉

### 4.5 Z5 特色风味区（12 → 17 / 补 5 个 / 关键差异化锚点）

**TOP**：
- 南方原味黑芝麻糊 480g（日均 0.13）
- 南方红枣黑芝麻糊 480g（日均 0.10）

**含蜂蜜专项**（修正规则 #7 茶叶+蜂蜜相邻）：
- 当前蜂蜜 SKU：清滋峰枇杷蜂蜜 / 怀滋露洋槐蜂蜜（不动销）
- 建议补：冠生园 / 福事多 / 椴树蜜 / 柠檬蜜（差异化）

**补 5 个建议**：
- 大湾区特色：陈皮姜茶 / 红糖姜茶（与休食区凉果带联动）
- 风味咖啡：抹茶拿铁 / 港式奶茶 / 阿萨姆奶茶
- 蜂蜜：冠生园 + 椴树蜜（如果未在 Z5）
- 阿胶燕窝类（中高端补充）

### 4.6 Z6 袋装囤量·趣味区（16 → 9 / 砍 7 个）

**TOP**：
- 桂格快煮燕麦片 1000g ¥23（日均 0.10）

**砍 7 个**：
- 多个 800g+ 不动销大包（金味系列 / 桂格系列）
- 长期滞销大包（预可售>365 天）

**核心逻辑**：Z6 是周末家庭客流的核心承接（区域定位说明书 §3.3）— 不是"砍光"，是"留主力大包 9-10 个 + 装满第 5-6 层"
