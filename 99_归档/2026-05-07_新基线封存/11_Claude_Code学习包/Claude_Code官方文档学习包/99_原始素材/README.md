# 99_原始素材 目录说明

## 1. 目录定位

存放 Claude Code 官方文档学习与核验过程中产生的**选摘片段**、**外链 URL**、**抓取日期**、**核验记录截图**等。

## 2. 适合存放

- 抓取官方文档时的关键节选（带原文链接与抓取日期，**不得是整页全文**）
- 核验某条"待核验"事项时的实测命令与结果片段
- 官方 `whats-new` / changelog 中影响本包的更新摘要
- 第三方深度文章 / 视频中关于 Claude Code 的精华笔记（标注作者、来源、日期、立场）

## 3. 不适合存放

- 官方文档原文全量（以官方原页为准，避免版权与体量问题）
- 含密钥、token、客户隐私的截屏
- transcript 原文（应留在 `~/.claude/projects/` 本机即可，**不入 vault**）
- 主题成果（应放回对应主题文件 01–10）

## 4. 命名规范

`YYYY-MM-DD_主题_版本.md`，例如：

- `2026-05-04_compact后survives表节选_v0.1.md`
- `2026-05-15_btw命令核验结果_v0.1.md`

## 5. 当前状态

v0.1 创建时本目录为空。后续按"待核验事项核验进度"逐步沉淀。
