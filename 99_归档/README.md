# 99_归档

## 目录作用

本目录保存**旧版本文件、过期资料、废弃方案、历史报告和阶段完成但不常用的内容**。

**归档不是删除**，而是把不再活跃使用的内容从主目录移走，减少对当前工作的干扰，同时保留历史记录和追溯能力。

---

## 适合存放

- 核心文件的旧版本（例如 `科学零售总纲_v0.5.md`）
- 已完成项目的阶段性报告（不再更新的版本）
- 废弃的方案设计
- 被替代的模板旧版本
- 历史诊断结论（已被新结论更新的）
- 试验性内容（没有继续推进的探索）

---

## 不适合存放

- 当前仍在使用的文件（错误归档会造成断链）
- 从未建立过的内容

---

## 归档规则

1. 重要文件在归档前，在原文件中加注"已归档"标记。
2. 归档文件建议在文件名中加注版本号或日期，例如 `科学零售总纲_v0.5_20260101.md`。
3. 归档不等于废弃：旧版本内容有时可以追溯方法论演进。
4. 建议每季度整理一次，将 `90_素材暂存与待整理` 中长期未处理的内容决定是否归档。
