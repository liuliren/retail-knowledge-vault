# 2026-05-07 新商品引进表 v0.1（模板 / 待填实）

> **本表是 [[M-DEC-007]] §3 S5 调整动作"补新 SKU"分支的输出模板**。每个候选必须经过 [[单品管理]] 新品引进规则评估（候选商品池 → 试销 → 验证 → 长存）。
>
> **铁律**（[[CLAUDE.md]] §13.7 / §13.13）：
> - 不可凭"看起来好"引进新 SKU
> - 必须有客户需求 / 排面释放 / 候选商品池 三选一支撑
> - 必须先经"试销"（30-90 天）再决长存

---

## §1. 表格定位

**输入**：
- [[2026-05-07_休食SKU去留表_v0.1]] 释放的排面（淘汰 / 减排）
- 老板战略提议 / 店长顾客反馈
- 候选商品池（[[单品管理]] 新品引进规则）
- 竞品 SKU 截流（赵一鸣 / 百欢 / 山姆 等同位）

**输出**：本表 → 老板/店长 sign off → 引进试销实施

**Sign off 入口**：[[16_客户与战役档案/花厅坊样板/05_决策记录]]

## §3. 引进规则（[[单品管理]] 新品引进核心）

| 规则 | 内容 |
|---|---|
| **G-1 战略必引** | 老板战略品类（如精酿啤酒 / 大单品）/ 强需求未满（如儿童零食 / 健康零食）|
| **G-2 排面释放后引进** | 去留表释放排面 → 同品类同价位补 / 不可跨品类 |
| **G-3 候选商品池** | [[单品管理]] 新品引进规则 + 行业新品 + 供应商主推 |
| **G-4 竞品截流** | 赵一鸣 / 百欢 / 山姆 同位 SKU 反向核（[[2026-05-06_读图分析待确认分桶派发表_v0.1]] C-05）|
| **G-5 试销期** | 30-90 天 / 期间不淘汰 / 期满走完整 [[M-DEC-007]] 7 步评估 |

## §6. 反引进红线（不引进的情况）

- ❌ 老板冲动决策 / 无客户需求支撑
- ❌ 同品类同价位重复（造成 SKU 冗余 → 未来减排）
- ❌ 供应商压货 / 无销售保障
- ❌ 季节性已过 / 试销期跨越淡季
- ❌ 反幻觉违规：仅凭"看起来好"引进（[[CLAUDE.md]] §13.7）
