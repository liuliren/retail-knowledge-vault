## 一、深调标准模板（4 段结构）

> 来源：90_/2.1花厅坊 2/2.1.1休闲食品/肉类零食/花厅坊_肉类零食深调_V1.md（首套实战）

### 1.1 第一段：结论先行（必填）

| 项 | 必含 |
|---|---|
| 原 SKU 数 | （二轮调改前的总量）|
| 边界迁类 SKU + 销售额 | （归错类的迁出明细）|
| 净 SKU + 销售额 | （边界清完后的有效量）|
| 库存=0 停卖确认 SKU | （已停卖）|
| 负库存盘点 SKU | （收货差错 / 待盘）|
| 最终上架 SKU | 含主货架 / 收银冲动购 / 端头下层 三类合计 |

### 1.2 第二段：L4 决策组重构

每行：L4 / 决策组角色 / 决策组动作 / 当前 SKU / 目标 SKU / 建议压缩 / 销售额 / 销售占比 / 毛利率 / 库存天数

### 1.3 第三段：执行位定义

每行：执行位 / 类型（主货架/端头/收银位）/ 任务 / 建议 SKU 数 / 负责人 / 计划完成日 / 陈列说明

### 1.4 第四段：执行排期

每行：序号 / 动作任务 / 负责人 / 优先级 / 计划完成日 / 状态 / 说明
