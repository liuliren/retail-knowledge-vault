## §3. 治理硬约束（vault 后续不再使用旧版）

### 3.1 引用规则

- vault 任何品类相关文档 / 必须引用 `05_品类管理与商品规划/乐易购_品类系统V4.0_定稿_大湾区版.xlsx`
- ❌ 不允许引用旧版（已封存到 99_归档/2026-05-09_品类表封存/）
- ⚠️ 旧版仅作 audit log 保留 / 不参与新分析

### 3.2 命名/版本一致性

- V4.0 内 L1-L4 命名 = vault 内权威命名
- POS 数据 cat_name 与 V4.0 L4 mapping 时 / 以 V4.0 为准
- M-DEC-007 §S2 数据交叉 / 6 表 v1.0.1 的 cat_name 字段 → 都以 V4.0 为准

### 3.3 升级机制

- V4.0 → V4.1+ 只能由六哥（vault owner）签发
- 启明侧产出"百货 V4.0 大湾区版" → 进入 vault 后 / 与 V4.0 食品并列 / 六哥审核合并
- 任何"完整品类表"重新出现 → 必须先与 V4.0 比对 / 不一致即视为废弃

## §4. 数据分析里使用 V4.0 的接口

### 4.1 5/8 客户方 POS 数据 mapping

POS cat_name（如"调味" / "酱菜" / "酱油" / "醋" / "蚝油" / 等）→ V4.0 L4 mapping：

```
POS cat_name "调味"    → V4.0 L4 "食盐 / 鸡精 / 味精 / 等"（基础食品 / 调味品 / 基础调味）
POS cat_name "酱菜"    → V4.0 L4 "广式酸菜 / 梅菜 / 榨菜 / 等"（基础食品 / 腌制品）
POS cat_name "酱油"    → V4.0 L4 "酱油"（基础食品 / 调味品 / 基础调味）
POS cat_name "蚝油"    → V4.0 L4 "蚝油"（基础食品 / 调味品 / 基础调味）
POS cat_name "醋"      → V4.0 L4 "醋"（基础食品 / 调味品 / 基础调味）
POS cat_name "料酒"    → V4.0 L4 "料酒"（基础食品 / 调味品 / 基础调味）
POS cat_name "火锅底料" → V4.0 L4 "火锅底料"（基础食品 / 调味品 / 酱料）
```
