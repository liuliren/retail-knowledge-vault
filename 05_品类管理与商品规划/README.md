# 05_品类管理与商品规划

## 目录作用

本目录沉淀**品类结构、品类角色、价格带、商品组货、商品配置和品类规划**的标准方法，是"品类升级"类咨询项目的核心工具库。

品类管理是商品力建设的结构层，解决"卖什么品类、卖多少宽度、覆盖什么价格段、围绕哪种顾客决策树排布"的问题。

---

## 适合存放

- 品类管理总纲
- 品类角色定义（主定义采用6类：目标性品类、常规性品类、季节性/便利性品类、形象品类、流量品类、利润品类；门店执行可使用4类简化版：主角品类、流量品类、利润品类、形象品类）
- 品类结构规划方法
- 价格带规划规则（好、更好、最好三档）
- 顾客决策树 CDT 模板（分品类）
- 商品组货逻辑
- 标准品类表（社区超市通用版）
- 各品类的品类树（休闲食品、饮料、冲调饮品、方便速食、米面粮油、日用品）
- 品类复盘模板
- 年度品类规划框架
- 52周商品规划框架

---

## 不适合存放

- 具体门店的品类整改报告（属于 `10_咨询交付模板` 或门店案例）
- 单 SKU 诊断结论（属于 `04_商品诊断与商品力提升`）
- 陈列图纸（属于 `06_陈列规划与DisplayMap`）
- 品类结构相关的原始 Excel（先进 `90_素材暂存与待整理`）
