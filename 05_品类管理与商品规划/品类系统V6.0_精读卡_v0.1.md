## V4.0→V6.0 核心升级

1. **+5个L3**：果蔬干 / 糖果 / 巧克力 / 果冻布丁 / 面包烘焙 / 袋泡茶（注：部分与V4.0合并类拆分）
2. **品牌库**：每L4≥3个品牌，排序有据（原V4.0仅示例2-3个）
3. **蜜饯果干→蜜饯果脯**（收窄命名，凉果独立）
4. **新增9张sheet**：L1-L4完整结构提取 / 判断规则字典 / 品牌库分类
