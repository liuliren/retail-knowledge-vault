# 品类系统 V6.0 精读卡

> **结论**：V6.0 是当前食品侧 L3–L5 主数据权威（六哥 2026-06-28 三裁决确认）。V4.0 降归档。

---

## 核心规模

| 层级 | V4.0 | V6.0 | 变化 |
|---|---|---|---|
| L1 | 3 | 3 | 同 |
| L2 | 16 | 16 | 同 |
| L3 | 63 | **68** | +5 |
| L4 | 308 | **385** | +77 |
| Sheets | — | 21 | +品牌库/判断规则 |

---

## V4.0→V6.0 核心升级

1. **+5个L3**：果蔬干 / 糖果 / 巧克力 / 果冻布丁 / 面包烘焙 / 袋泡茶（注：部分与V4.0合并类拆分）
2. **品牌库**：每L4≥3个品牌，排序有据（原V4.0仅示例2-3个）
3. **蜜饯果干→蜜饯果脯**（收窄命名，凉果独立）
4. **新增9张sheet**：L1-L4完整结构提取 / 判断规则字典 / 品牌库分类

## 与 category_mapper.py 关系

category_mapper.py 使用**粒度更细的L3名**（巧克力/硬糖软糖独立），与V6.0 T码聚合层有映射关系：

| mapper L3 | V6.0 T码 |
|---|---|
| 膨化零食 | T01 薯片膨化 |
| 巧克力 + 硬糖软糖 | T02 糖果巧克力 |
| 饼干 + 威化夹心 | T03 饼干 |
| 蛋糕派类 + 中式糕点 | T04 蛋糕糕点 |
| 果干类 + 蜜饯果脯类 + 凉果类 | T05 果干蜜饯 |
| 坚果类 + 瓜子花生类 + 混合炒货 | T06 坚果炒货 |
| 肉类零食 | T07 肉干肉脯 |
| 海味零食 | T08 鱼干海味 |
| **袋泡茶（V6.0新增）** | 冲调区独立L3 |
