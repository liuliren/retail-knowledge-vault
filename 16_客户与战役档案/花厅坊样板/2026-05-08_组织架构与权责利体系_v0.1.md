**5/8 拟调整方向**（待启明 confirm + 家族 articulate）：
1. **执行店长**承担 4 件事：商品组织 + 人员调配 + 员工培训 + 制度流程
2. **高彬**作为副店长 = 协助店长 + 后勤 + 人资
3. **张力**作为主管 = 现场运营 + 树标杆（核心岗 / 涨工资 / 设目标奖金）
4. **佳妮**专注财务 + 休食采购（"专人专岗"减负）
5. **文员**拆分（5/8 拟招新文员做基础录入 / 现文员调前场 / 仍兼饮料低温采购+营运）

## §2. 决策权边界（5/8 共识）

### 2.1 老板 / 启明（最高决策权）

| 权类 | 范围 | 配套责任 |
|---|---|---|
| 利益分配权 | 薪酬体系 / 奖金 / 分成 / 合伙安排 | 公平 / 透明 / 利益相关 sign off |
| 重大决策审批权 | 年度预算 / 大额采购 / 跨店事务 / 战略转型 | 与陈海鹏 / 启明父亲家族协同 |
| 战略方向 | 标杆项目落地 / 家族 4 店复制可能性 | 客户对外可讲 / 7 月业绩验收 |
| 水果采购权 | 水果品类采购决策 | 启明本人兼任执行 |
| 任免权（关键岗）| 执行店长 / 副店长 / 主管层任免 | 与家族沟通 |

| 权类 | 范围 | 配套责任 |
|---|---|---|
| **商品组织** | 商品引进 / 淘汰 / 陈列 / 调价（在采购体系内）| 7 月 +30% 销售 + 毛利同比 KPI |
| **人员调配** | 排班 / 调岗 / 日常人员管理 | 团队效能 / 一线执行 |
| **指挥权** | 日常运营指挥 / 直线管理副店长+主管+理货员 | 标准执行 / 巡店机制 |
| **奖惩权** | 基于工作标准的日常奖惩 | 制度公平 / 不滥用 |
| **批假权** | 一线人员请假审批 | 排班连续性 |
