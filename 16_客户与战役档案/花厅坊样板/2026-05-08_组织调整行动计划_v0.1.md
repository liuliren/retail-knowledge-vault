## §4. 7 月 +30% 业绩验收倒推（关键节点）

### 4.1 验收口径（[推断] / 待启明 confirm）

```
[推断] 7 月 +30% 默认口径：
  2026 年 7 月销售额 vs 2025 年 7 月销售额 +30%
  + 2026 年 7 月毛利额 ≥ 2025 年 7 月毛利额（同比提升基础）
```

需要启明 confirm（pending）。

### 4.2 倒推时间表

| 时间 | 节点 | 协同动作 |
|---|---|---|
| 2026-07 | 陈海鹏参观 + 7 月销售验收 | 启明 + 全员 / 三端集体过 |
| 2026-06 月底 | G05 阶段门评审 | 是否 GO / NO-GO |
| 2026-05-30 | 启明 A2 articulate 陈海鹏完成 | 赏罚权+利益分配权明确 |
| 2026-05-22 | B1 执行店长权限明确 | 4 件事承担 ack |
| 2026-05-16 | 商品端首测 13 天调改后销售复测 | 数据支持 |
| 2026-05-15 | A1 启明 articulate 娜姐 完成 | 协同支持就位 |
| **2026-05-09** | **本计划建立** ✅ | 三端启动 |

### 4.3 7 月销售保障路径（差值法估算 / [推断]）

[推断] 实现 7 月 +30%销额 + 毛利同比的可能路径：

1. **方便食品**已实证 +30% 销额 / +23% 毛利（差值法 / 5/9 报告）✅
2. **调味组**待 confirm 是否同步调改（H1）/ 如已调改则 +30% 是合理预期
3. **休食组其他子组**（休闲 / 糕点 / 烘焙 / 茶叶 / 等）需 W19-W22 通读 + 调改
4. **生鲜组**（水果+蔬菜 / 张力主管） / 客流主驱动 / 不在本战役范围 / 但与商品调改协同
5. **整体**= 4 个子动作叠加 / 才能稳到 +30% 整体目标

→ **关键风险（R21）**：单纯调改可能不足 / 需调味+其他品类协同 / + 启明三端联调发挥作用。
