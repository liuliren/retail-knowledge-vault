# 糕点扩品清单 v0.1（启明 5/10 上午 sign off 候选）

> **本文件性质**：5/11 烘焙撤场后糕点扩品的**顾问建议清单** / 不是启明已决定 / 是 5/10 上午现场会议的 sign off 依据。
>
> **§13.17 推断标注**：所有具体 SKU 选品 / 数量 / 价格带都是 [推断] / 启明 sign off 后才生效。

---

## §1. 扩品总盘子（基于糕点目标 ~250 SKU）

| 当前（30T） | 扩品目标 | 净增 | 扩品来源 |
|---|---|---|---|
| 饼干 132 SKU | 保留 + 精筛 = ~100 | -32（清流失噪音）| 现有 |
| 面包 18 SKU | **扩到 60-80 SKU** | **+42-62** | 5 大方向 A |
| 糕点 1 SKU | **扩到 30-50 SKU** | **+29-49** | 5 大方向 B+C |
| 糕点其它 0 SKU | **扩到 10-20 SKU** | **+10-20** | 5 大方向 D+E |
| **合计 151** | **目标 ~220-250 SKU** | **+69-99** | 5 大方向 |

---

## §2. 5 大扩品方向（启明 sign off 候选）

### 方向 A：面包类（核心 / 净增 42-62 SKU）⭐⭐⭐

**承接客户**：原烘焙区现烤面包 34 + 吐司 28 + 桃李 26 = 88 SKU 的客户群

| 子类 | 推荐 SKU 数 | 说明 |
|---|---|---|
| **软面包**（长保 / 桃李 / 达利园 / 盼盼）| 20-25 | 主力 / 与原桃李用户无缝对接 |
| **吐司**（长保 / 桃李 / 曼可顿）| 10-15 | 早餐场景 / 中老年喜爱 |
| **小餐包 / 早餐杂粮包**| 10-15 | 早餐 / 高频低单价 |
| **三明治面包 / 法式面包**（短保）| 5-10 | ⚠️ 损耗风险 / 启明评估 |
| **节令面包**（含端午）| 待 5/15 评估 | 端午前 1-2 周备货 |

**注意**：
- 短保 vs 长保 → 启明 sign off 偏好（损耗 vs 品类丰富度）
- ⚠️ 现烤类**不上**（已 5/11 撤场 / 不再做现场烘焙 / 仅预包装）

### 方向 B：西式糕点（净增 15-25 SKU）

| 子类 | 推荐 SKU 数 | 说明 |
|---|---|---|
| **蛋黄派 / 蛋糕预包装**（达利园 / 桃李 / 米奇妙）| 8-12 | 长保 / 高频 / 中老年喜爱 |
| **法式糕点 / 玛德琳类** | 3-5 | ⚠️ 大湾区社区超市市场未必大 / 启明评估 |
| **巧克力糕点**（朗德姆 / 等） | 4-8 | 与糖巧子组互补 / 不与之冲突 |

### 方向 C：中式糕点（净增 15-30 SKU）⭐⭐

**关键战略**：大湾区**地方特色** + 中老年常备 + 与赵一鸣硬折扣错位

| 子类 | 推荐 SKU 数 | 说明 |
|---|---|---|
| **沙琪玛 / 老式糕点** | 8-12 | 中老年高频常备 / 价格 5-15 元 |
| **桃酥 / 手指饼**| 5-8 | 同上 |
| **凤梨酥 / 老婆饼 / 鸡蛋酥**（粤式 ⭐）| 5-10 | **大湾区地方特色** / 客群匹配 / 高毛利 |
| **沙拉酱+面包+蛋糕组合**| 3-5 | 早餐+下午茶场景 |
| **节令月饼**（5 月底 / 6 月起备）| 待 5 月底起 | 端午+中秋节令窗口 |

### 方向 D：早餐糕点 + 速冻类（净增 8-15 SKU）

| 子类 | 推荐 SKU 数 | 说明 |
|---|---|---|
| **早餐饼干**（康师傅 / 健达 / 盼盼）| 5-8 | 上班族+学生客群 |
| **燕麦糕 / 红枣糕 / 八珍糕**（养生）| 3-5 | 中老年送礼 / 高毛利 |
| **速冻烘焙类**（小餐包 / 速冻面包） | 0-2 | 启明评估冷链能力 |

### 方向 E：礼盒类（净增 8-15 SKU）⭐ 节令窗口

| 节令 | 推荐 SKU 数 | 时点 |
|---|---|---|
| **5/12 母亲节礼盒** | 3-5 | **紧贴 5/11 烘焙撤场 / 当日母亲节** |
| **5/31 端午粽子礼盒** | 3-5 | 5 月底起备货 |
| **中老年送礼礼盒**（粤式糕点合集 / 养生）| 3-5 | 全年常备 / 高毛利 |
| **6 月儿童节礼盒**（如适用）| 0-2 | 6/1 节令 |

## §3. 价格带配置建议（[推断] / 待启明 confirm）

```
低价带（2-5 元）        50-60% SKU   引流款 / 早餐饼干 / 小面包
                                     高频低单价 / 客流主驱动
                                     
中价带（5-15 元）       30-40% SKU   主力 / 软面包 / 吐司 / 老式糕点
                                     家庭日常 / 主毛利贡献
                                     
高价带（15-50 元）      8-15% SKU    礼盒 / 节令产品 / 高毛利尝鲜
                                     5/12 母亲节 + 5/31 端午
                                     
超高价（50+ 元）        ≤ 2% SKU     精选粤式礼盒 / 中老年送礼
```

## §4. 毛利率组合（目标 ≥ 25% / [推断] 待启明 confirm）

```
方向 A 面包类           毛利率 18-22%（承接桃李等大牌 / 偏低）
方向 B 西式糕点         毛利率 22-28%（蛋黄派+巧克力 / 中等）
方向 C 中式糕点         毛利率 28-35%（粤式特色 / 高 ⭐）
方向 D 早餐糕点         毛利率 22-28%（早餐饼干 / 中等）
方向 E 礼盒类           毛利率 30-40%（节令 + 中老年送礼 / 最高 ⭐⭐）

加权平均预期：25-28%（假设 A:B:C:D:E = 30:15:25:15:15）
→ 满足糕点目标毛利率 25%+ 要求 ✅
```

## §5. 进货渠道（[待补 / 启明 confirm）

| 选项 | 利 | 弊 |
|---|---|---|
| **原烘焙供应商保留**（部分品牌共用）| 关系延续 / 谈判成本低 | 烘焙取消后 / 供应商可能涨价 |
| **新增预包装供应商**（桃李/达利园/盼盼总代）| 大牌+稳定供货 | 新关系 / 谈判周期 |
| **本地中式糕点供应商**（粤式特色） | 地方差异化 / 高毛利 | 量小 / 配送难度 |
| **混合**（推荐 ⭐）| 主力大牌 + 本地特色双轨 | 管理复杂度增加 |
