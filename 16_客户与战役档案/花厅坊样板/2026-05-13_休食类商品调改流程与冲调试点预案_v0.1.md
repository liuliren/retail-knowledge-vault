## §6. Step 4 冲调 CDT（M-DEC-008 §9 应用 / 完整决策树）

### 6.1 冲调 CDT 5 层标准结构

```
第 1 层：购买任务
  早餐代餐 / 中老年营养 / 办公提神 / 家庭常备 / 节令送礼 / 儿童成长 / 价格敏感补充
  ↓
第 2 层：子品类
  麦片/谷物 / 蜂蜜 / 咖啡奶茶 / 茶叶 / 营养粉/代餐 / 黑芝麻糊/藕粉 / 礼盒
  ↓
第 3 层：价格带
  低价基础（< 10 元）/ 中价主力（10-30 元）/ 高价品质（30-100 元）/ 礼盒（100+）
  ↓
第 4 层：规格
  小包装（单次/试吃）/ 家庭装（多次/中量）/ 大包装（家庭囤货）/ 礼盒装
  ↓
第 5 层：品牌
  外资大牌（雀巢/家乐氏/麦斯威尔）/ 国产大牌（南方黑芝麻/汤臣倍健）/ 本地品牌 / 高毛利新品牌
  ↓
第 6 层：陈列位（最终落到货架）
  黄金层（眼平 / 高毛利 + 礼盒 + 主推）/ 中层（主流走量）/ 下层（家庭装大包装）/ 顶层（囤货）/ 端架（节令）
```

### 6.2 CDT 关键约束

- ⚠️ **第 1 层（购买任务）必须先于子类**：顾客不是先想"麦片"再想"早餐"/ 而是先想"早餐"再选"麦片或粥"
- ⚠️ **第 6 层是结果 / 不是起点**：陈列由 CDT 反推 / 不是先画陈列再倒推商品
- ⚠️ **大湾区中老年客群 CDT 可能与上班族不同**：启明 5/13-5/15 confirm（"中老年是花厅坊主力客群" / 用户原 prompt 强调）
