# 花厅坊样板战役 #1 — 里程碑与关键路径(v0.2 / 2026-05-06 D004 修订)

## 1. 关键路径概览(D004 修订:8 面 × 8 夜班)

```
5/5 二访 → 5/6 G4-A 夜班 → 5/7 G4-? 夜班 → 5/8 → 5/9(休)→ 5/10 → 5/11(休)→ 5/12 → 5/13 G4-H 夜班 → 5/14 8 面复拍 → 5/20 一周验证 → 5/27 14 天深度复盘+G05 → 5/31 对外可讲样板 → 6/30 第一个新客户签约
   M0       M1.0           M1.1            M1.2  休      M1.3  休      M1.4   M1.5            M1.8           M2          M3                  M4              M5
```

> **关键路径阻塞点**:
> - M1.0(5/6 G4-A 调改)失败 → 整个战役停摆
> - M1.1-M1.7 任何一面到货率连续 2 晚 < 50% → 当面延期 + 顺序顺延
> - 8 面累计延期 ≥ 3 面 → 触发 G05 紧急评审
> - M2(5/20)数据不支持 → M3-M4 重新校准

## 5. 偏离应对

| 偏离 | 应对 |
|---|---|
| 偏离 ≤ 7 天 | 黄灯,仪表标注,本周仪式必谈 |
| 偏离 > 7 天 | 红灯,立即升级风险登记 + 决策记录,必要时主持紧急 G05 |
| **当晚到货率 < 50%(M-DEC-003 夜班版)** | 当面延期 1 晚 / 连续 2 晚 < 50% → 当周不补 |
| **8 面累计延期 ≥ 3 面** | 触发 G05 阶段门紧急评审,重新评估 5/20 M2 节点 |
| **失误率 > 20%(R9 红线)** | 立即休战一晚 |
| 关键路径阻塞(M1.0 调改失败 / 5/6 当日到货率 <50%)| 战役进展直接停,先解阻塞 |
