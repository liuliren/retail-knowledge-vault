# M-DEC案例化第一批 v0.1

> 本轮不生产新方法，只把三个已有真实问题整理成统一格式的案例记录。三个案例进度不同——案例A刚建立验证框架(待执行)，案例B/C决策已下达但**均无执行后复盘**，这本身就是需要补的动作。

---

## 案例A：白酒库存效率改善

| 字段 | 内容 |
|---|---|
| **问题** | 白酒品类库存成本¥238,914，占全食品库存33.2%，GMROI=0.24(健康基准2.5的不到1/10) |
| **数据** | 192 SKU中32个高端礼品酒(单瓶成本>¥300)占库存成本80.95%；供应商集中度84.9%在"德誉酒业(名酒)"；生肖纪念茅台(2024龙年+2025蛇年)单独占¥31,200，跨年度未清 |
| **判断** | H1(节庆库存)证据链最强(生肖酒+高端礼品酒结构+专营名酒供应商三重印证)，但H2(订货惯性)/H3(真滞销)未被数据排除，**不下淘汰结论** |
| **动作** | ①核实德誉酒业供货条款(可退/换)；②调取32个SKU近90天SKU级销售数据；③若确认真滞销部分，先促销/渠道清货再考虑IM-09淘汰 |
| **负责人** | 六哥(供应商核实+最终判断)、启明(SKU级数据) |
| **验证指标** | 2周内数据补齐；30天后观察是否已启动清货动作(不设定具体GMROI达标值)；90天后GMROI方向性复核 |
| **详见** | [[2026-07-09_花厅坊白酒GMROI改善验证卡_v0.1]] |

## 案例B：方便食品23款汰换

| 字段 | 内容 |
|---|---|
| **问题** | 机器筛出23款净可汰候选(已排除引流/利润/形象/锁定款保护名单)，均为动销1-4天的滞销方便面/速食产品 |
| **数据** | 近3月销量2-4件、动销天数1-4天，单价3.5-13.7元区间，全部为"补充"角色 |
| **判断** | 六哥已裁决(2026-06-26)：**21款汰、2款观**(第8/9款康师傅好汤面老母鸡汤面、康师傅鲜Q面什锦海鲜面·动销4天/3天略高于其他候选，暂观察)，0款留 |
| **动作** | 店长版调改卡已出(`花厅坊方便食品_店长版调改卡_v0.1.md`)，理论上应已下架21款并补位——**但目前无任何执行确认记录** |
| **负责人** | 店长(执行下架+补位) |
| **验证指标** | ① 21款是否已实际下架(需店长确认，决策已下达13天未见复盘)；② 腾出排面补了什么(新品试销 or 畅销款加量)；③ 2款"观"是否已到期需转汰/留；④ 方便食品品类整体动销/坪效是否有变化 |

**⚠️ 本案例最大缺口不是判断，是执行确认——决策质量已经够了，现在缺的是"有没有真的做"。**

## 案例C：巧克力8款优化

| 字段 | 内容 |
|---|---|
| **问题** | 机器筛出8款净可汰候选，均为动销1-3天的滞销巧克力/糖果产品 |
| **数据** | 近3月销量1-4件、动销天数1-3天，单价2.0-13.9元区间，全部为"补充"角色 |
| **判断** | 六哥已裁决(2026-06-27)：**5款汰、3款观**，0款留 |
| **动作** | 店长版调改卡已出(`花厅坊巧克力_店长版调改卡_v0.1.md`)——**同样无任何执行后续记录，且比方便食品案例晚1天决策，目前已过去约13天，3款"观"的观察期(通常约1个月)接近到期** |
| **负责人** | 店长(执行下架)、六哥(1个月观察期到期后转汰/留判断) |
| **验证指标** | ① 5款是否已下架；② 3款"观"截止约2026-07-27，届时需明确转汰或转留；③ 汰换后巧克力品类整体动销/毛利变化 |

## 三案例横向观察

1. **案例A是新问题(库存资金效率)，案例B/C是执行缺口(决策已下、无人跟进)**——性质不同，但都指向同一个系统性问题：**花厅坊项目目前"诊断→决策"这一半闭环运转良好，"动作→验证"这一半基本空转**。
2. **建议**：与其继续开发新的诊断/决策工具，不如先把B/C两案例的执行确认做完——这是本阶段"少建设、多验证"原则下最该优先做的事，成本极低(一通电话/一次到店确认即可)，价值是把已经产生的判断真正兑现成经营结果。
