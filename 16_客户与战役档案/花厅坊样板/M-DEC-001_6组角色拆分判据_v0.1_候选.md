# M-DEC-001 6 组角色拆分判据 v0.1（候选）

> **本 SOP 解决**：单一货架面（如 G4-A 冲调正面 / 74 SKU）应拆分为多少角色组？每组多少 SKU？判据是什么？
>
> **战役验证状态**：v0.1 候选立项（2026-05-07）→ 5/13-5/20 战役 #1 G4-A 至 G4-H 8 面（如恢复）or 休食区其它品类调改 → 5/27 G05 阶段门 → 6/30 跨战役 #2 验证 → 升 v1.0 落 `04_/中层决策库/M-DEC-001_..._v1.0.md`

## §1. 决策定位

**决策名**：单一货架面 6 组角色拆分判据
**类型**：商品力 / 货架结构 / 中层方法论 L4

## §2. 适用场景

- 单一货架面 SKU 数 ≥ 50（如花厅坊 G4-A 冲调 74 SKU）
- 老板 / 顾问感觉"商品太多 / 重复"但说不出具体哪些（5/6 金句 #1 "太多条码 / 需细化筛选"）
- 调改前需角色重构

## §3. 推演链（5 字段核心）

### 3.1 6 组角色基础结构（来自 G4-A 冲调实战 / 待跨品类验证）

| 组 | 命名 | 价值定位 | SKU 数（参考 / 花厅坊冲调）| KPI |
|---|---|---|---|---|
| Z1 | 低价引流 | 价格敏感 / 客流引入 | 8 | 销量 / 客流 |
| Z2 | 主力 | 销售支柱 / 复购 | 15-21 | 销售额贡献 ≥ 30% |
| Z3 | 功能细分 | 客群细分需求 | 9-14 | 客群匹配度 |
| Z4 | 高端升级 | 形象 / 利润 | 3-5 | 毛利率 ≥ 35% |
| Z5 | 特色风味 | 差异化 / 网红 | 12-17 | 话题度 / 试销动销 |
| Z6 | 袋装囤量 | 大包装 / 家庭 | 9-16 | 客单价 / 件均 |

### 3.2 推演（why 6 组而不是 4 / 8 / 12）

- **4 组（A/B/C/D）** = ABCD 分类 / 仅按贡献 / 不反映角色差异
- **6 组（Z1-Z6）** = 引流 + 主力 + 功能 + 升级 + 特色 + 囤量 / **完整覆盖客户购买动机** + 与价格带 3 段（低/中/高）正交
- **8+ 组** = 过细 / 一线员工记不住 / 边界模糊（5/6 金句 #5 "需简单语言"）

### 3.3 角色拆分依据（多维交叉）

```
角色组 = f(价格带, 包装规格, 品牌定位, 客群任务)

Z1 低价引流: 价格带低 (<5 元) + 包装小 + 大众品牌 + 解馋独享
Z2 主力:     价格带中 (5-20 元) + 包装中 + 主流品牌 + 家庭时光
Z3 功能:     价格带中 + 功能性诉求 + 专业品牌 + 办公室零食
Z4 高端升级: 价格带高 (>20 元) + 包装精美 + 高端品牌 + 形象 / 礼品
Z5 特色风味: 价格带中 + 网红 / 地方特色 + 小众品牌 + 朋友分享
Z6 袋装囤量: 价格带中 + 大包装 + 主流品牌 + 家庭时光 / 聚会
```

## §4. 临界值（数字化阈值）

| 维度 | 临界值 | 触发动作 |
|---|---|---|
| 单组 SKU 数下限 | < 3 | 该角色组**取消**（合并到相邻组）|
| 单组 SKU 数上限 | > 25 | 该角色组**进一步细分**（如 Z2 拆为 Z2A 主力品牌 + Z2B 主力子品牌）|
| 单组销售额贡献 | < 5% | 该角色组**收缩**或淘汰候选 |
| Z2 主力销售额贡献 | < 30% | 主力区**重构**（M-DEC-001 重启动）|
| Z4 + Z5 合计 SKU 占比 | < 15% | **形象 / 差异化不足** / 引进新品 |
| 净增 SKU 数 | > +5 | 拒绝（避免膨胀 / 单兵带宽）|
| 净减 SKU 数 | > -25% | 警示（可能过激 / 需分阶段）|

## §5. 反事实（推翻条件）

本规则在以下情况**不适用**：
- 单一货架面 SKU 数 < 30（角色组无意义 / 自然按 ABCD 分类即可）
- 客群高度同质化（如纯白领商圈 / 只需 Z1+Z2+Z4 三组）
- 季节性极强品类（如冷饮 / 月饼 / 5 月不分 6 组 / 按时令滚动）
- 单一品牌主导（如某乳制品专架 / 角色拆分由品牌厂商定）
