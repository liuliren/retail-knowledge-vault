# 花厅坊样板战役 #1 — 战役章程

## 1. 战役定位

- **客户代号**:乐易购花厅坊店
- **客户类型**:社区超市(自家样板)
- **战役类型**:商品力提升(**G4-A 至 G4-H 8 面 × 8 夜班连续战**,2026-05-06 D004 修订)
- **战役期号**:W18 启动 → W22 完成(2026-05-05 → 2026-05-31)
- **首席顾问**:六哥
- **合伙顾问**:无(单兵期)
- **PM**:六哥兼任

## 2. 战役命题

> "这场战役要为这个客户解决什么问题,为我们沉淀什么资产?"

- **客户视角命题**:30 天内 G4-A 冲调区销售 / 毛利 / 30 秒找到率三维同步提升,数据 + 整改 + 复盘三件套可看可讲
- **我方视角命题**:
  - L1 数据:G4-A 冲调销售 / 毛利量化提升
  - L2 凭证:6 机位前后对比 + 4 天连续时段切片
  - L3 案例:M5 后去标识化案例包(花厅坊样板对外可讲)
  - **L4 中层方法论 ⭐**:M-DEC-001/002/003 三条候选 → 5/31 验证提炼到 `04_/中层决策库/`
  - **战略价值**:**生产 L4 中层方法论的工厂** — 解决六哥真瓶颈"中层 playbook 缺失"

## 3. 战役范围(Scope)— 2026-05-06 D004 修订

### 3.1 本战役 IN(必做)

- **G4-A 至 G4-H 8 面调改 × 8 天连续夜班**(5/6-5/13,每晚 22:00-凌晨 02:00)— D004 节奏调整
  - 5/6 G4-A 冲调正面 6 组 SKU 角色重构(74→63):Z1 低价引流(8)/ Z2 主力奶类麦片(21→15)/ Z3 功能细分(14→9)/ Z4 高端升级(3→5)/ Z5 特色风味(12→17)/ Z6 袋装囤量(16→9)
  - 5/7-5/13 其他 7 面顺序待六哥 22:00 上场前锁定(R11 缓解)
- **6 机位拍摄约定调整为 3 时段对比**(R10 缓解):① 调改前 22:00(夜灯)② 调改后 02:00(夜灯)③ 次晨 09:00(白天日光,同机位真对比)
- 时段切片连续 ≥4 天(5/14-5/17 一周复测,白天 + 夜晚双时段)
- 30 秒找到率 + 5 分钟采购完成比基线测试(5/5 调改前 + 5/14 起 8 面连测)
- **M-DEC-001/002/003 三条 L4 草稿**(5/8 前填实)+ **M-DEC-004 夜班作业 SOP**(本战役新增)
- L1 数据 / L2 凭证 / L3 案例 / L4 中层方法论 四层产出 DoD 全过

### 3.2 本战役 OUT(明确不做)

- 沙埔大道店或其他门店项目(已 M1 战时封存)
- 7D / 选品系统 / V3.2 任何软件开发动作(M4 后再启)
- 30-80 万体系化大项目(单兵交付不了)
- 第三方供应链全面重谈(本战役只动 G4 系列 8 面对应供应链)
- 其他通道(F 休食 / J 日百 / K 凉茶等)的调改 — 等 G4 8 面 5/13 验证后 W20+ 再启

> 不写 OUT 清单 = 范围爬升风险。OUT 必须显式列出,防"顺手就干"。

- 5/6-5/13 每晚 22:00 前 当晚到货率 < 80% 时:延期 vs 临时替换 vs 硬上(M-DEC-003 夜班版实证点)
- 5/6 22:00 上场前 5/7 调哪一面?(R11 缓解触发点)— A G4-B 同货架反面水饮 / B 邻排 G3-A / G5-A / C 其他

- 5/20 一周数据是否触发"调改成功"对外讲述(反幻觉红线 — 至少 4 天连续切片 + 1 周观察)

- M-DEC-001/002/003/004 草稿验证 → 提炼条件(✅ 客户认可 + ✅ 数字支持 + ✅ 现场看到 三选其一)
