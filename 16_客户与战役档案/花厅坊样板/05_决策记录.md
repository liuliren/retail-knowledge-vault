### D012 — 战役 #1 范围扩三端联调（商品+营运+制度）+ 标杆项目落地动机明确

- **决策日期**：2026-05-08（22:36-5/9 01:01 双人录音）

- **状态**：✅ 生效（候选 / 5/9-5/15 落地）
- **决策类型**：战役范围 / 战略升级 / 客户结构认知

#### 一、决策

战役 #1 范围**从"商品端单端"扩展为"商品+营运+制度三端联调"**：
- **商品端**（已在范围内）：6 表 v1.0 / M-DEC-001~007 / 5/10 调改 / 库存去化
- **营运端**（新增）：日常作业链条 SOP / 收货流程 / 仓库管理 / 巡店机制 / 理货员标准
- **制度流程端**（新增）：门店组织架构 / 权责利体系 / 核心人员盘点 / 奖惩制度 / 培训体系

**核心动机订正**：会议 / 战役动机 = **落地标杆项目** + **整个门店家族发展**，不是启明个人战略。启明个人能力升级是关联收益。

**客户结构升级认知**：花厅坊 = 陈氏家族 4 店连锁中的 1 店（市场店 / 大道店 / 平沙店 / 花厅坊）/ 启明任花厅坊老板（小股东 / 兼水果采购 / 二代）/ 服务陈氏家族未来集团化整合。

#### 二、可选方案

| 方案 | 描述 | 优势 | 劣势 |
|---|---|---|---|
| A | 维持单端（仅商品） | 战役更聚焦 | 5/8 启明已 take ownership 三端 / 不响应等于浪费动机窗口 |
| B | **三端联调** | 命中真实需求 / 标杆样板对外可讲度高 / 触达家族 4 店机会 | 工作量翻倍 / 需要新建 3 个组织文件 + R19+ 风险登记 |
| C | 三端但分两期（商品 → 营运 → 制度）| 节奏可控 | 错过 5/8 启明 ownership 拉起的高峰窗口 |
