# M-DEC-008｜门店多商品调改SOP v0.1 候选

## §0. 一句话定位

> **本 SOP 用于解决"数据分析之后、正式陈列调改之前"的中层决策问题。商品先于陈列，陈列是商品动作的空间表达。**

**M-DEC-008 vs M-DEC-005/006/007 的区别**：
- M-DEC-005：错品类货架重建（陈列纠错单点）
- M-DEC-006：陈列层数确认与区域落位（陈列设计单点）
- M-DEC-007：商品动销综合诊断与调整（数据诊断 + 7 步动作单点）
- **M-DEC-008（本）：把 005/006/007 串成完整中层决策流程**（数据 → TPOS → CDT → 品类归属 → SKU 动作 → 新品 → 陈列 → 复盘 闭环）

## §0.5 ⭐ 步骤 0：经营目标确认（v0.2 新增 / 不可跳过）

> **新增触发**：W20-Day2 专项审计发现"经营目标"在 v0.1 隐含在 §1 / 用户原 prompt §Step 1 明示需独立成段 / 是入口 / 不可跳过 / 否则 Step 1-10 会失焦。

```

   ↓

   ↓
Step 4 CDT 顾客决策树判断
   ↓
Step 5 品类 / 子品类归属修正（边界商品）
   ↓
Step 6 SKU 动作决策（10 动作）
   ↓
Step 7 新品候选池与替换映射（硬规则 9 项判断）
   ↓

   ↓

   ↓
Step 10 15 天小复盘 + 30 天复盘
```

## §6. Step 1 数据诊断

### 6.1 数据维度（11 维）

```
销售额 / 销量 / 毛利额 / 毛利率 / 库存量 / 库存金额 / 周转天数 / 缺货天数 / 动销天数 / 价格带 / ABCD 分级 + Z 异常标记
```

### 6.2 关键约束

- **不直接输出陈列图** / 仅输出商品表现事实
- **必须区分自然动销 vs 促销拉动**（参 [[CLAUDE.md]] §13.14）
- **必须扣除大盘客流溢出**（如 5/9 全店 +20-40% / 净增超 +10pp 才算调改净增量）
- ERP 进价虚高（烘焙类 90%+）→ 多维度交叉校准 / 不纯依赖毛利数据

## §7. Step 2 现场陈列诊断

### 7.1 现场看 9 维

```
黄金层使用 / 低位化（高毛利错位）/ 混层（大/礼/小三层包装错位）/
价签清晰度 / 缺货标记 / 顾客动线 / 排面占比 / 顾客 30 秒找到率 / 包装层级
```

### 7.2 关键判断

> **很多商品问题其实是陈列问题，不可只凭数据判断。**

例如：
- "销量低 → 直接淘汰"是错的 / 必须先查 Step 2 是否陈列位差
- "毛利率低 → 砍 SKU"是错的 / 必须先查 Step 1-2 是否引流款 + 陈列错位
- M-DEC-007 §S4 6 分支决策树（陈列位差? 活动期外? 价格虚高? 缺货? 客群不匹配? 真不需要?）

| 8 | 高毛利礼盒 | 节令礼盒 + 粤式糕点合集 + 茶叶礼盒 | 中老年送礼 |

### 8.3 TPOS 在共享通道的关键作用

> **冲调 + 方便速食共享通道**应承担：早餐 + 即食 + 家庭常备 + 节令补充
> 4 大任务 / 单子类无法独立服务 / 必须联动判断（不孤立调冲调）

## §9. Step 4 CDT 顾客决策树判断

### 9.1 CDT（Customer Decision Tree）原理

> 顾客在面对货架时的实际决策路径 / 不是逻辑顺序 / 是顾客眼睛 + 大脑实际经过的节点

### 9.2 花厅坊休食 4 子类 CDT 示例（[推断] / 5/13-5/15 启明 confirm 校准）

#### 9.2.1 冲调 CDT

```
用途（早餐 / 即食 / 滋补 / 礼赠）
  ↓
子类（麦片 / 咖啡 / 奶茶 / 蜂蜜 / 茶叶 / 谷物）
  ↓
品牌（外资 / 国产 / 本地）
  ↓
规格（家庭装 / 单次装 / 礼盒）
  ↓
价格带
  ↓
包装（袋 / 瓶 / 礼盒）
```

#### 9.2.2 糕点 CDT

```
场景（早餐 / 下午茶 / 送礼 / 儿童）
  ↓
子类（面包 / 中式 / 西式 / 早餐饼干）
  ↓
保质期（短保 / 长保）
  ↓
价格带
  ↓
品牌
```

#### 9.2.3 方便速食 CDT

```
即食需求（充饥 / 早餐 / 滋补）
  ↓
子类（方便面 / 方便粉 / 即食类 / 方便食品）
  ↓
口味（红烧 / 酸辣 / 螺蛳粉等）
  ↓
品牌（统一 / 康师傅 / 白象 / 网红）
  ↓
价格
  ↓
规格（桶 / 袋 / 礼盒装）
```

#### 9.2.4 饼干 CDT（参考）

```
场景（早餐 / 茶歇 / 儿童 / 送礼）
  ↓
子类（咸味 / 甜味 / 夹心 / 苏打）
  ↓
包装（小份 / 家庭装 / 礼盒）
  ↓
品牌
  ↓
价格带
```

### 9.3 CDT 校准约束

- ⚠️ CDT 必须 5/13-5/15 启明 confirm（启明对花厅坊客群最了解）
- ⚠️ CDT 不可凭顾问推断 / 必须基于现场观察 + 客户访谈
- ⚠️ CDT 不同子类不同 / 不可统一套用

## §11. Step 6 SKU 动作决策（10 动作）

### 11.1 10 动作触发条件
