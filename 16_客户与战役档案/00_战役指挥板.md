## 五、本目录档案纪律

1. **不填空段**：战役未开案就不要先填 00_战役章程模板（避免"假忙碌沉淀"）
2. **不抹历史**：风险登记表 / 变更控制日志只能追加，不能删除（用 `~~删除线~~` + 理由）
3. **不混业务条线**：商品诊断 / 陈列方案等业务过程不进 16_，进 09_
4. **不绕开 G04 / G05 / G06**：开案 / 阶段门 / 复盘三个仪式不能省（参 v0.2 §6.5）
5. **frontmatter 必有 quadrant + client_safety**：未标 = Lint 红灯（参 [[Vault健康仪表]] 维度 #4 / #5）
