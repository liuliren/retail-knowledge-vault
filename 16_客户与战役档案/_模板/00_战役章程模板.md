## §D. 风险红线（Risk Lines）

战役级 5 条不可触红线：

- [ ] 不编造门店数据（参 [[CLAUDE.md]] §13.1-§13.3）
- [ ] 不在客户付款前承诺超出销售包范围的交付
- [ ] 不在未签 NDA 时对外讲述客户案例
- [ ] 不绕开 G05 阶段门评审进入下一阶段
- [ ] 战役超 7 天偏离 → 升级 [[战役指挥板]] 健康灯 🔴

> 详细风险登记见同目录 `04_风险登记表.md`。

## §E. 人决策点（Human Judgment Gates）

以下决策**必须人决策、不可委托 LLM**：

- 客户是否签约 / 续约
- 是否对外讲述本战役案例（含视频号 / 长文 / 直播）
- 商品力诊断结论（淘汰 / 保留 / 新增 SKU 清单）—— LLM 给依据，人拍板
- 陈列整改方案 —— 必须现场判断
- 战役暂停 / 终止 / 失败定性

---

## §F. 出场条件（Exit Criteria）

战役完成判据（满足任一即出场）：

- [ ] 全部交付物 DoD 通过且客户书面确认
- [ ] 已回款（或回款进入合同约定流程）
- [ ] G06 战后复盘 AAR 已主持并入库（同目录 `08_战后复盘AAR.md`）

战役暂停判据：客户原因或我方原因暂停 ≥7 天 → 走暂停流程。
战役失败判据：核心命题被推翻或客户单方面终止 → 走 G06 失败 AAR。
