# 2026-05-13_命名规范冲突清单_v0.1

## 1. 文件定位

本文件记录 vault 当前已知的**命名冲突 / 缺失 / 重复 / 待统一**事项 + 修复路线图。

**配套**：
- [[2026-05-13_命名规范总纲_v0.1]] — 跨场景总原则
- [[2026-05-13_命名规范索引_v0.1]] — 命名规则查询入口

**用途**：
- vault 治理周仪式（每周）扫一次 / 标记新冲突
- Phase 1 / Phase 2 治理任务的输入清单
- 历史问题归档 audit-log

---

## 2. 冲突分级

| 级别 | 含义 | 处理策略 |
|---|---|---|
| **P0** | 影响现场执行或导致错误命名 | 立即修复 / 不超过 1 周 |
| **P1** | 影响检索 / Obsidian 双链 / 文件归属 | W21 Phase 1 处理 |
| **P2** | 影响长期治理但不阻塞业务 | W22 / W23 Phase 2 处理 |
| **P3** | 历史遗留 / 可暂缓 | W24+ / 视情况处理 |
