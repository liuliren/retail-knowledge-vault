## 三者互链建议（不本轮执行）
```
晟果新零售 Wiki(入口)
   ├─ 学方法论 → 科学零售知识树 MOC（全景树/学习路径）
   ├─ 查概念定义 → 核心概念主定义索引（SSOT）
   └─ 找具体文件 → 知识库总索引（全库台账）
三者顶部各加一行"姊妹导航"互链，明确分工，不重复内容。
```

## 关键建议
1. **核心概念主定义索引 = 概念 SSOT**：保持现行，作术语/概念唯一权威。
2. **知识库总索引 = 台账**：刷新（v1.5→新版）或降 deprecated；不与 MOC 抢"方法论入口"。
3. **知识树 MOC = 方法论全景**：补全、升 active；学习路径与知识地图归它。
4. 三者**顶部互链**（姊妹导航），但**职责不重叠**。
5. 执行另开 **P1-GOV-Nav-Hub-Fix-001**（本轮不改三文件正文）。
