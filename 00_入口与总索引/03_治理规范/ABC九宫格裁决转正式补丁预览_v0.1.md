## 1. 九宫格全表（销售贡献 A/B/C × 毛利额贡献 甲/乙/丙）
| 销售贡献 | 甲（毛利贡献高） | 乙（毛利贡献中） | 丙（毛利贡献低） |
|---|---|---|---|
| **A**（销额累计≤60%） | 核心引擎 ✅ | 核心引擎 ✅ | 流量品 ✅ |
| **B**（60-90%） | 潜力利润品 🆕 | 常规品 🆕 | 流量补充·控利 🆕 |
| **C**（>90%） | 利润品 ✅ | 长尾利润·待裁决 🆕 | 双低 ✅ |

✅=原 §3.1 已确认（与 abc_classifier.py 一致）；🆕=本补丁新增（B 行 3 格 + C+乙）。

> ⚠ **甲/乙/丙 = 毛利额累计贡献分档，不等同于毛利率高/中/低**（ABC-Rule-Text-Correction-001）。判断毛利率异常须走毛利率字段 / 负毛利·清仓异常字段，不可用甲/乙/丙代替。

## 2. 每格业务解释 + SKU 动作建议映射
| 格 | 标签 | 业务含义 | SKU 动作建议（落动作表） | 需人工复核? |
|---|---|---|---|:--:|
| A+甲 | 核心引擎 | 高销 + 毛利贡献高，店之根基 | 死保 + 黄金位 + 不准断货 + 安全水位 | 否 |
| A+乙 | 核心引擎 | 高销 + 毛利贡献中，主力走量 | 死保 + 黄金位 + 盯毛利不下滑 | 否 |
| A+丙 | 流量品 | 高销 + 毛利贡献低，带客流 | 保留**控排面**，找高毛利款替代/搭售带利 | 否 |
| B+甲 | **潜力利润品** | 中销 + 毛利贡献高，有成长空间的赚钱品 | **提曝光 / 扩排面试拉量**，盯转化 | 否 |
| B+乙 | **常规品** | 中销 + 毛利贡献中，稳健走量基本盘 | **保持标准排面**，常规补货 | 否 |
| B+丙 | **流量补充·控利** | 中销 + 毛利贡献低，帮带客流但别堆资源（一线：能卖但利润薄，别给太多资源）| **控排面 / 搭高毛利款**，别扩 | 否 |
| C+甲 | 利润品 | 低销 + 毛利贡献高，常被误杀的金矿（需排除异常进价/售价/退货/联营/一次性销售）| **不准砍**，上移 / 提曝光 / 促销拉动 | 否 |
| C+乙 | **长尾利润·待裁决** | 低销 + 毛利贡献中，可能小众刚需（一线：卖得少但还有利润，先查原因再决定）| **进误判防线复核**，过则留 | **是** |
| C+丙 | 双低 | 低销 + 毛利贡献低 | 进误判 4 问，过了才砍 | 否 |

## 3. C+乙 人工复核要求
- **`需复核` 字段**：默认仅 **C+乙（长尾利润·待裁决）= 是**；其余 8 格有明确动作 = 否。
- **复核中间态**：C+乙 进 [[2026-06-19_RetailOS_v1.1_五步SOP接入版]] 误判 4 问 / 六桶复核后，落「留 / 汰」**最终裁决**；**不长期停留「观察 / 待裁决」**。
- 复核字段**服务误判防线，不替代裁决**。
