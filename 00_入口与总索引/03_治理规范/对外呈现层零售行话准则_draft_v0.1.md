# 对外呈现层零售行话准则 v0.3

> **总方向（六哥 2026-07-08 二裁定调）**：**不是"去专业化"，是"专业词分场景使用"**——表达通俗直接；保留体现专业性的行业通用词；行话少量且只在助于理解处用；内部 IT 系统名/字段/编码对客户一律不露。目标：专业性与理解度并得。
> **地位**：核心任务持续推进；**三轮审议全部完成并已签字升 stable**（第1轮盘点+改写实证 / 第2轮魔鬼代言人 184a2aa / 第3轮行业steelman 4051180 / 签字 2026-07-08）。核心原则：**专业词分场景使用，不是去专业化**。
> **适用范围与唯一例外**（同 v0.1）：一切对外呈现层；例外=内部系统开发场景；灰区=方法论页正文可留系统语言、其对外表达段须合规。

## 机制一 · 受众三档（取代单一替换表）

| 档 | 受众 | 规则 |
|---|---|---|
| **对客层** | 老板（销售/汇报/报告） | 通俗直接；白名单 A 档词直接用；行话仅在助解处少量用 |
| **课堂层** | 店长/采购（训练/过品） | **术语双写制**：首现"术语（人话解释）"，此后放开用术语——训练的目的之一就是教会行业词 |
| **内部/同行层** | 同行/专业场合/内部 | 术语为主不强制翻译；同行场合专业质感=产品的一部分 |

**边界硬规则（第3轮裁定）**：**客户组织内任何人都不是同行**——客户总部人员（如好家源总部）归**课堂层**（总部需要学习方法体系；且报告会向下转发给店长，裸 C 档词会在客户组织内失控）。同行层仅适用于无客户关系的专业交流（同业顾问/行业演讲/供应商总部谈判）。

## 机制二 · 保留词白名单（禁止替换）

- **A1 档·行业普遍理解词（免释直用）**：毛利率 / 客单价 / 客流 / 库存 / 周转 / 价格带 / 排面 / 陈列 / 条码 / 畅销·平销·滞销 / **动销率** / 账期 / 进价·售价 / 黄金位 / 堆头
- **A2 档·专业但首次需解释（首现"专业词+简短解释"，后续直用）**：**诊断**（给门店做经营体检；标题·产品名照用）/ ABC 分类（按销售额把商品分成头部/中部/尾部）/ 盘点差异率（账和实点差多少）/ 生鲜损耗 / 损益表（赚钱亏钱一张账；个体户多无此表——问数改问"营业额、进货额、费用三笔"）/ 汰换（把卖不动的下架换新品进来）/ 端架（货架两端那一节）/ 周转天数（一批货平均几天卖完；防与资金周转歧义）
- **B 档·课堂层双写后放开**：SKU / 负库存 / 条码匹配率 / 坪效 / 前台·后台毛利 / 盘点差异率
  - SKU 标准双写句式（六哥定版）：首现写"**SKU，也就是系统里的一条商品记录 / 一个可单独管理的商品规格**"，后续直接用 SKU；**不得机械替换成"支"或"单品"**
- **C 档·仅同行/专业场合**：P25·中位数·P75 / 长尾 / GMROI / 证据分层概念 / 内部方法名

## 机制三 · 术语双写制 + 五载体规范（v0.3 定版）
基本格式：`专业词（简短解释）`，首现一次，后续直用专业词。已实证范例：训练包"防误杀（断货不是卖不动）"。

| 载体 | 规范 |
|---|---|
| 咨询报告 | 首现双写，之后用专业词；报告末附"口径说明"小节收录本报告 A2/B 档词与阈值算法 |
| PPT | 首现双写+页脚术语行；**标题不得全部白话化**（专业质感）；每页新术语≤1；内部编码零露出 |
| 培训教材 | 专业词+教学解释——目的是**培养行业语言能力**；文末词汇表（术语/人话/本店例子），下次课抽问 |
| 合同/验收件 | 正文=客户理解语言（不用括注）；附录=术语定义与口径表，**验收以口径表为准** |
| 门店操作手册 | 首现解释，后续统一专业词；步骤句内直接给动作定义 |

## 机制四 · 禁止错译清单（第2轮审议实证）

| 禁止 | 原因 | 正确表达 |
|---|---|---|
| 长尾 → ~~尾货~~ | "尾货"零售业实指处理货/清仓货，错译会让采购清错货；**尾部商品≠尾货须显式防混** | 定版：首现双写「**长尾商品（尾部商品）**」；专业报告用"长尾商品"，对老板解释用"长期卖得少、但还占着货架和库存的商品" |
| 量词"支"作 SKU 通用量词 | 不合超市语感且与件数歧义 | 款 / 个（SKU 按 B 档双写） |
| 动销率 → ~~"卖得动"~~ | 丢失可量化的"率"，KPI 不可比 | 动销率（A 档保留）；口头可补"卖得动的占比" |
| 诊断 → ~~体检/把脉~~（正式件） | 行业通用词+产品名 | 诊断（A 档）；"体检"仅限销售话术比喻 |
| P25/P75 → ~~"便宜那头/贵那头"~~ | "那头"=端点语义，四分位≠端点，会误读 | 低价段/主流价/高价段（各约占四分之一销售）；同行场合保留 P25/P75 |

## 机制五 · 数字阈值禁意译（硬规则）
**任何数字阈值/比率必须保留原数字，只能补解释，不得改成模糊量词。**
- 正例：`条码匹配率低于 95%（每 20 个条码约有 1 个对不上）时，相关数字先不用于结论`
- 反例（第2轮抓获并废止）：~~"账和单对不上超一成"~~（95% 线≠10%，错一倍）
- 连带纪律："账和X对不上"类短语一事一译：库存账实=账和货 / 单据核对=账和单 / 条码匹配=销售表和商品档案对得上——禁混用

## 机制六 · 合同/验收件双层制
正文用客户能理解的话；**附录必附"口径定义表"**（术语+算法口径+基准日），**验收以口径表为准**。合同附件 A 行话化列入下次合同修订签字批执行本制。
