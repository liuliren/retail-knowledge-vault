### 服从原则

以下事项**必须服从** P1-P3 三项核心目标，**不得反向占用核心目标的精力与节奏**：

- 知识库建设（含治理修复 / 索引同步 / 审计 backlog 处理）
- 方法论沉淀（含 01_方法论文件升级 / 主定义补全）
- AI 协作（含 Claude Code 治理 / 提示词优化）
- 选品系统卷宗提取（卷宗 §02 PRD / §03 战略评估 / §04 商业评估 / §05 行动方案）

**判断口径：** 任何拟启动的非 P1-P3 任务，必须先回答"它如何服务花厅坊样板 / 新客户开拓 / 现金流？"——无法直接回答 = 暂缓。
