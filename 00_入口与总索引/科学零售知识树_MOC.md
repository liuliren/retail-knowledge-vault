## 🔄 自动列件（装 Dataview 插件后生效）

> 以下查询自动列出全库对应类别的件，**新增文件自动出现**——这是治孤儿的核心机制。

### 全部 active 方法论
```dataview
TABLE version, status FROM "01_科学零售方法论"
WHERE source_type = "method" OR source_type = "methodology"
SORT file.name ASC
```

### 全部 M-DEC 决策记忆
```dataview
TABLE version, status FROM "16_客户与战役档案"
WHERE contains(file.name, "M-DEC")
SORT file.name ASC
```

### 全部 SOP 与交付件
```dataview
TABLE version, status, client_safety FROM "01_科学零售方法论/SOP" OR "10_咨询交付模板"
SORT file.name ASC
```

### ⚠ 治理告警：active 但未签字的件（应为空）
```dataview
TABLE status FROM ""
WHERE status = "active" AND !signoff AND !signed AND !signed_off
SORT file.folder ASC
```
