# 三回归仪表

vault 是工具，不是目的；战役是路径，不是终点。本仪表是"事业级指南针"，**每周一短扫（10 分钟）+ 月初深扫（30 分钟）**，用三回归反"假忙碌"。

> 架构定位：7 层架构最底层（参 [[2026-05-04_知识库重构方案_v0.2_完整作战方案]] §3.1 / §4.5.3 / §6 / §10.2）。
> 三仪表合奏：[[战役指挥板]]（战役怎么走）+ [[Vault健康仪表]]（vault 是否健康）+ 本仪表（是不是假忙碌） = PM 控制层完整画面。

## 三个永远的拷问（每次有疑虑时问）

参 `CLAUDE.md` §20。

1. 这件事跟客户交付、签约、保留有什么直接关系？

**答得清，干；答不清，停。**

## 六、本仪表的使用纪律

1. **不可空填**：每周一短扫必须填本期数据，不填 = 等于本周 vault 完全空跑。
2. **不可装饰**：客户方变得更好的人数是几就是几，不允许"技术上"算成 1 (e.g. "我自己也算客户")。
3. **红线即停**：连续 2 周客户接触 0 = 必须停 vault 推进。
4. **历史不抹**：W18 数据建立后，W19 的填写不应覆盖 W18，应在新章节追加（按周累积）。
