# 00_入口与总索引

## 目录作用

这是整个知识库的**导航台、总控台、治理中心和作战指挥台**。

所有人进入知识库，首先看这个目录。所有治理规范、索引文件、知识地图、检查清单都在这里集中管理。它不存储业务内容，只负责让整个知识库保持有序、可检索、可治理。

---

## 适合存放

- 知识库总索引（全库文件入口）
- 核心概念主定义索引（防止概念冲突）
- 科学零售知识地图（方法论全景）
- 当前重点项目看板（项目状态）
- 常用模板入口（快速跳转）
- 标签体系说明（标签分类规范）
- 文件命名规范（命名规则）
- 版本更新记录（变更历史）
- 知识库治理规范（治理总则）
- 反幻觉检查清单（操作级检查表）
- 逻辑一致性检查清单（跨文件一致性）
- 文档工程化标准（Markdown 模板与 frontmatter 规范）
- 变更影响检查清单（修改前必查）
- 知识库测试用例（规则验证）

---

## 不适合存放

- 原始照片、原始 Excel
- 单个门店的过程资料
- 未整理的临时想法
- 方法论正文（属于 `01_科学零售方法论`）
- 长篇行业研究正文（属于 `14_外部案例与行业研究`）
- 任何带有"待确认"状态的一次性内容
