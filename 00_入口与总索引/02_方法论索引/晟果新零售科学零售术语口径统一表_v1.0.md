# 晟果新零售科学零售术语口径统一表

## 0. 使用边界

- 本表是**晟果新零售科学零售知识体系**在 v0.5 架构下的术语口径权威清单，隶属 [[晟果新零售科学零售Wiki知识体系治理方案_v1.0]]（§6 命名 / §7 frontmatter / §9 client_safety）。
- 本表**只统一口径、不重定义方法论**：各术语的方法内容以「主定义位置」列指向的文件为准，本表与主定义冲突时以主定义为准。
- 本表为 **internal_only**：表中"对内用法/对外用法"区分仅供顾问参照，对外文档须按"对外用法"列改写。
- 本表**不批量改全库**：登记标准口径后，存量文件的口径对齐属后续分步专项（治理方案 §15 B-03/B-06），逐批确认。

### 1.4 「生活性零售」取消使用

「生活性零售」**不再作为晟果新零售科学零售知识体系中的正式术语使用，也不再作为业态 / 场景描述使用**。

后续如需表达客户业态，应根据具体对象使用「社区超市」「社区生活超市」「生活超市」「中小连锁超市」「社区零售门店」「生活服务型社区门店」「生鲜加强型社区超市」「500-5000 平社区型商超」等更清晰说法。

历史文件或历史版本记录中如出现「生活性零售」，**仅作为历史留痕，不作为当前有效口径**。本表主表不收录「生活性零售」为正向术语，仅在 §4 禁用 / 慎用叫法清单中登记。
