## 核心论点

零售数学不是为了好看，而是把经营现象**拆开、量化、还原**，定位问题在哪一层级，才能选对工具。**先用RM-01确定问题层（客数/客单价），再用RM-04/RM-06/IM-04深挖原因，最后选商品工具处方。**

## RM-01 销售额公式树（业绩诊断根工具）

```
销售额 = 客数 × 客单价
              │           │
         ┌────┘     ┌─────┘
    既存客来店      1客点数 × 1点单价
    × 来店频率      (买几样)  (每样多少钱)
    + 新增客数
```

### 业绩下滑诊断逻辑

| 诊断结果 | 问题层 | 解决方向 |
|---|---|---|
| 客数↓ · 客单价平 | 获客/留客 | 营销/服务/选址因素 |
| 客数平 · 客单价↓ | 商品结构/价格 | 商品配置/陈列/促销 |
| 1客点数↓ | 关联购买弱化 | 关联陈列/搭配促销 |
| 1点单价↓ | 高价品(M域)没卖动 | 独特品/偏好品配置 |

### 实战案例

```
现状：日均客数820(↑)，客单价42元(↓16%)，日销售3.4万(↓)
诊断：客数问题 ✅ 排除，问题在客单价
进一步：1客点数4.2(↓)·1点单价10元(↓)
→ 关联陈列弱化 + M域SKU占比从35%降至20%
整改：① 进口零食撤端架→还给本地烘焙Preferred品
      ② 收银台恢复连带促销
      ③ M域SKU补回30%以上
```

## RM-04 毛利管理体系

```
毛利额 = 销售额 × 综合毛利率
综合毛利率 = Σ(品类销售占比 × 品类毛利率)
```

### 两条提升路径

| 路径 | 方法 | 风险 |
|---|---|---|
| **A路径**（推荐）| 提高高毛利品类的**销售占比** | 安全，不丢客流 |
| **B路径** | 直接提高某品类毛利率（涨价/换品）| 容易丢客流 |

### 四大实战工具

1. **调整品类销售占比**：增加高毛利品类陈列资源；引进Preferred品（M域，毛利率高）
2. **商品结构调整（IPC）**：M域SKU占比提升5% → 综合毛利率通常提升1-2个百分点
3. **促销节奏管理**：正价期毛利率23-28%；促销期15-20%；全月加权20-25%（社区超市基准）
4. **损耗管理**：生鲜损耗率控制在3-5%；工业品<1%；损耗每降1% → 综合毛利率涨约0.3%

## RM-06 库存管理五大指标

| 指标 | 公式 | 目标方向 | 含义 |
|---|---|---|---|
| **周转率** | 销售成本/平均库存 | 越高越好 | 资金使用效率 |
| **周转天数** | 365/周转率 | <账期天数 | 否则现金流断裂 |
| **交差率** | 毛利率×周转率 | 越高越好 | 库存投资回报率 |
| **库存坪效** | 月销售额/卖场面积 | 越高越好 | 卖场资源效率 |
| **账面库存** | 期末库存金额 | 接近目标库存 | 月度对账核心 |

### 大湾区社区超市健康基准

| 品类 | 周转天数 | 毛利率 |
|---|---|---|
| 生鲜（叶菜）| 2-3天 | 15-20% |
| 生鲜（肉禽）| 3-5天 | 12-18% |
| 熟食烘焙 | 1-2天 | 25-35% |
| 工业食品 | 15-30天 | 18-25% |
| 酒水饮料 | 10-25天 | 12-18% |

**铁律**：综合周转天数必须 < 结算周期（通常30-45天），否则资金链紧张。

## IM-04 ABC-Z 单品分析

### 分类规则

| 类别 | 条件 | 行动 |
|---|---|---|
| **A类** | 累计销售额前70%的SKU | ⭐ 永远不能断货；最佳陈列；可扩深度 |
| **B类** | 70-90% | ✅ 维持现状；定期检视 |
| **C类** | 90-100% | ⚠️ 评估战略意义（新品试销？CDT占位？）否则淘汰 |
| **Z类** | 3个月销售=0 | ❌ 立即清货下架（除非是季节品/新品） |

### IM-05 ABC × 陈列矩阵（深度诊断）

```
         陈列饰面多   陈列饰面少
A类      ✅ 健康       ⚠️ 加强陈列（机会损失）
B类      ⚠️ 减少陈列   ✅ 合理
C/Z类    ❌ 立即调整   ⚠️ 评估去留

→ 错配核心：饰面从"B/C类陈列多"转移到"A类陈列少"
```

### IM-09 删品决策框架

**数据触发条件**（满足任一则进入评估）：
- 连续3个月销售≤2件 → 候选删除
- 连续2个月销售=0 → 强制删除
- 库存>3个月销量 → 候选删除
- 毛利率<10%且非引流品 → 候选删除

**战略评估**（保留条件，满足任一可暂保）：
- 在CDT中占据独特位置且无替代品
- 是新品（上架<3个月）→ 再观察1-2个月
- 是季节品 → 等旺季再评估
- 是品牌战略品（厂商谈判筹码）

**关键认知**：删品不是目的，**迭代结构才是**。没有删品的门店，三年后货架上一半都是滞销品。

> - **格效提升预测**：退出尾部 40–60 款后，TOP20 款陈列面数可提升 30–50%，格效理论提升 15–20%

## 失效条件

门店刚完成SKU大规模调改后30天内，RM-01客单价↓的诊断结果可能只是新品适应期摩擦而非真实问题——按框架立即执行整改动作会过早干预正常磨合期。保护期规则：调改后首30天内，客单价↓幅度<15%不触发整改；>15%才视为真实信号。
