## 核心论点

**品牌墙的本质是把陈列形象层（L1）资源向"销售/品牌势能"倾斜**——强势品牌占形象输出，弱势品牌让出形象资源保留商品资源；适用于目的性/偏好性品类，是把品类心智锚定到具体品牌的视觉武器。

## 一、什么是品牌墙

**品牌墙**指货架形象层（L1）专门留给某个强势品牌做集中视觉输出的设计——通过品牌堆头、品牌色块、品牌灯箱、品牌包装一致排列等手法，让顾客从远处一眼识别"这家店有这个品牌"。

它不是单纯的"陈列好看"——是**把品类心智锚定到具体品牌的视觉武器**。

---

## 二、为什么需要品牌墙

### 商业逻辑

社区生活超市面临一个核心矛盾：**销售毛利贡献集中在前 20% 单品/品牌，但陈列资源是均匀分布的**。这种均匀陈列让强势品牌的优势被稀释，弱势品牌占据了不该占的视觉资源。

品牌墙的本质是**把陈列资源向"销售/品牌势能"倾斜**——让强势品牌做形象输出，弱势品牌让出形象资源、保留商品资源。

### 顾客体验逻辑

顾客在快速购物时，**靠品牌识别快速决策**。"乐事薯片"比"原味马铃薯片"更快被识别——因为前者是品牌锚点，后者是品类描述。

品牌墙降低顾客的决策成本，提高时间效率（与时间效率主轴 A6 同向）。

### 差异化逻辑

某些品类的差异化锚点本质上是**品牌锚点**——比如大湾区凉果带 = 顺宝品牌墙，膨化主力 = 乐事/可比克品牌墙。把"差异化锚点"从概念落地为品牌级形象，是把方法论翻译为具体陈列动作的关键一步。

---

## 三、什么品类适合做品牌墙

### 必做品牌墙的情况

满足**三条**全部时，必须做品牌墙：

1. 该 L2 是**目的性 / 偏好性**角色
2. 该 L2 内有**销售/品牌势能集中**的强势品牌（TOP1 品牌占该 L2 销售 ≥ 15%）
3. 该 L2 在该面位的**形象层（L1）资源足够**（至少占 1 个完整子格的 L1）

### 不必做品牌墙的情况

- **偶发性 / 便利性 / 季节性品类**：形象资源稀缺，不必做品牌墙
- **L2 内销售/品牌分散**：如果 TOP1 品牌占比 < 10%，没有强势品牌，做品牌墙无效
- **客户敏感/价格敏感品类**：做品牌墙容易让顾客产生"贵"的心理，反而拒绝

### 灰色地带

- **L2 内有 2 个旗鼓相当的品牌**：可以做"双品牌墙"，两个品牌并列形象输出
- **某品牌虽然销售大但顾客认知度低**：单独做品牌墙效果差，可以做"主力品组合墙"

## 四、品牌墙的设计要素

### 视觉要素

| 要素 | 说明 |
|---|---|
| 品牌主色 | 用品牌官方主色作为色块背景 |
| 品牌 LOGO | 大尺寸 LOGO 在 L1 中央或顶端 |
| 包装一致性 | 同品牌的多个 SKU 包装并列时呈现整齐感 |
| 品牌口号 / 主张 | 品牌自带的 slogan 或主张展示 |
| 品牌灯箱 | 有条件时增加灯箱强化视觉 |

### 商品要素

| 要素 | 说明 |
|---|---|
| 该品牌主力 SKU 全集 | 不只放 1-2 款，而是把品牌的主力 SKU 都放上来 |
| 包装规格梯度 | 小包装 + 标准装 + 家庭装的全规格 |
| 价格梯度 | 低价引流款 + 主力款 + 高端款 |

### 空间要素

| 要素 | 说明 |
|---|---|
| 至少占 1 个完整子格的 L1 | 不能只占半格 |
| L1 与下层（L2-L3）联动 | L1 是形象，L2-L3 是该品牌主力品的销售层 |
| 端架联动 | 主货架的品牌墙 + 端架的品牌堆头 = 双重视觉强化 |

## 五、品牌墙与品类墙的区别

**品牌墙**（推荐）：以单一强势品牌为核心，强化品牌识别。适用于品牌势能集中的品类。

**品类墙**（替代）：以品类为核心，整齐陈列同品类商品（不强化具体品牌）。适用于品牌势能分散但品类需要强化识别度的情况。

**形象墙**（保底）：既不突出品牌也不突出品类，只是排列美观。适用于不重要的面位或边缘格位。

调改时优先做品牌墙，做不了再降级到品类墙、形象墙。

## 六、案例：花厅坊的两个品牌墙

> **client_safety**: 以下案例涉及客户门店，仅供内部方法论参考，不得对外发布。

### 案例 1：G1-A 第 6 格"顺宝品牌墙"（凉果带）

**为什么是品牌墙**：
- 蜜饯果干/凉果是目的性角色（A2 判断）
- TOP20 中至少 4 款顺宝产品（话梅皇 / 香脆橄榄 / 盐津陈皮条 / 九制陈皮）
- 顺宝是大湾区凉果第一品牌，品牌势能远高于其他凉果品牌
- 形象资源：G1-A 第 6 格 L1 完整一格

**设计要点**：
- 品牌主色（顺宝品牌色）作 L1 背景
- 顺宝 LOGO 居中
- L2-L3 放顺宝主力 SKU 全集（话梅皇、香脆橄榄、盐津陈皮条、九制陈皮、话梅肉等）
- L4-L6 配置非顺宝凉果（珍妮小熊、农夫山庄等）作为补充层
- 端架联动：建议在 G1-A 与生鲜区交界端架放顺宝品牌堆头 + "煲汤主题"

**预期效果**：
- 强化"花厅坊凉果就找顺宝"的心智
- 与竞品形成差异化（竞品不会单独为大湾区品牌做形象）
- 早高峰老人路径上形成视觉锚点

### 案例 2：G1-B 第 1 格"乐事 / 可比克品牌墙"（薯片带）

**为什么是品牌墙**：
- 膨化是偏好性角色（A2 判断）
- TOP20 中乐事 3 款 + 可比克 2 款
- 这 5 款单品合计销售约占膨化总销售 15%
- 形象资源：G1-B 第 1 格 L1 完整一格

**设计要点**：
- 双品牌并列形象（乐事左、可比克右，各占半格）
- 品牌色块 + LOGO + 主力包装一致排列
- L2 放乐事原味 / 番茄味 / 黄瓜味 + 可比克番茄味（主力）
- 与水饮区联动：端架做"薯片 + 可乐"组合促销

**预期效果**：
- 降低顾客在膨化面前的决策成本
- 强化"花厅坊薯片就找乐事/可比克"的简单心智
- 不试图跟竞品的薯片宽度比，但保住主力品牌势能

## 七、品牌墙的常见错误

### 错误 1：把"形象层放好看的商品"当品牌墙

形象层放最好看的包装 ≠ 品牌墙。**品牌墙的核心是品牌识别，不是包装美观**。如果 L1 上是好几个不同品牌的产品并列，那是形象墙而非品牌墙。

### 错误 2：选错品牌

把销售大但顾客认知度低的品牌做品牌墙——顾客认不出，等于没做。

**纠偏标准**：选品牌时三看
- 销售排名（前 1-2）
- 顾客认知度（顾客提到该品类时第一个想到的品牌）
- 品牌物料供给（厂家是否能配合提供品牌主色、灯箱等物料）

### 错误 3：品牌墙太多

整个区域里做了 5-6 个品牌墙——每个都做，相当于都没做。

**纠偏标准**：一个区域内**通常只做 2-3 个品牌墙**——服务于该区域 2-3 个差异化锚点。

### 错误 4：品牌墙占据黄金销售位

把品牌 LOGO/物料放在 L2 或 L3——这两层是销售核心层，不能让形象占据。

**纠偏**：品牌墙的形象部分**只在 L1**。L2-L3 是该品牌的主力 SKU 销售层。

## 八、品牌墙落地的工作流

```
1. 用 A2 五分法识别该面位的目的性/偏好性 L2
2. 拉销售数据，识别 L2 内的强势品牌（TOP1 品牌占该 L2 销售 ≥ 15%）
3. 评估形象资源（L1 是否有完整一格）
4. 确认品牌物料供给（厂家是否能配合）
5. 设计品牌墙（视觉 + 商品 + 空间三要素）
6. 联动端架（如有空位）
7. 落地后做形象审计（每 90 天一次）
```

---

## 九、品牌墙与端架的联动

**单独的品牌墙**只是主货架上的形象输出。**主货架品牌墙 + 端架品牌堆头**才是完整的品牌强化。

联动设计：

| 主货架品牌墙 | 端架联动方案 |
|---|---|
| 顺宝品牌墙（凉果） | 顺宝品牌堆头 + 煲汤主题（连接生鲜区） |
| 乐事/可比克品牌墙（膨化） | 薯片 + 可乐组合堆头（连接水饮区） |
| 卤味升级品牌墙（如周黑鸭） | 卤味 + 啤酒夜宵堆头（连接水饮区） |

端架放大主货架品牌墙的视觉权重，是品牌墙发挥最大效力的关键。

## 花厅坊直接调用

> 花厅坊 G1-A 第 6 格（顺宝凉果品牌墙）和 G1-B 第 1 格（乐事/可比克薯片品牌墙）已作为本页两个验证案例（见§六）。
> **进店快判品牌墙是否有效**：站在货架 3 米处，5 秒内能否认出「这里有顺宝」或「这里是乐事」？能认出 = 品牌墙形象输出正常；看不出 = 形象资源分散，需重新集中 L1。
> **升级路径**：还需 1 个额外案例完成三案例验证。候选：卤味 / 广州烧腊品类（若花厅坊卤味升级推进，可作为案例 3）。

## 失效条件（张力）

| 失效场景 | 原因 |
|---|---|
| 偶发性 / 季节性品类 | 无高频顾客认知，品牌识别无复购价值，做墙是浪费 L1 |
| L2 内无强势品牌（TOP1 < 10%）| 没有可锚定的品牌势能，做墙等于均匀陈列换了个名字 |
| 面位资源极度紧张（<2 个子格）| 挤出 1 整格给 L1 形象会严重压缩 L2–L3 销售层 |
| 目标客群价格极度敏感（城中村 / 工厂区）| 品牌墙传递"贵"信号，反而降低购买意愿 |
| 品牌物料供给断链 | 厂家无法提供主色/灯箱/物料，品牌墙沦为"贴标签" |
