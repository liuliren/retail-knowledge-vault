# M4 陈列货位簇

> ✅ **已吸收 v0.1**。深度笔记 → [[2026-06-19_零售老刘体系吸收_04_陈列货位簇_v0.1]]

## 1. 主治痛点
**买不便 / 找货累**(顾客)+ 黄金位错配(经营)。

## 2. 武器清单(速查)
| 武器 | 一句话 |
|---|---|
| **人货场关联** ⭐ | 商品分析必叠加 人+场+顾客需求;脱离场不公平 |
| **假A类** | 断货致低销量≠差,"畅销的才是缺货的";卖家思维≠顾客思维 |
| **货位空间系数** ⭐ | 按动线位置赋系数,把"黄金位"变成可计算值(空间=不变/时间=变)|
| 棚格图表 | Plan-o-gram 排面规划,联动 DisplayMap(二遍)|

## 3. 骨架映射
- 货位空间系数/棚格图 → **运营力·陈列规划** + 联动 06_陈列规划与DisplayMap
- 人货场/假A类/场成本 → 运营力 + 反哺商品力(真实商品力)

## 4. 花厅坊落地 + M-DEC 候选
- 陈列改造 → 休食货位赋空间系数,查黄金位是否给了对的商品
- **M-DEC 已落候选**:[[M-DEC-011_黄金位与ABC错配诊断_v0.1_候选]](黄金位×ABC错配三筛)

## 6. 二遍补强(2026-06-19 / G 簇 3 篇精读:04/06/15)
> 增量,不推翻 §2。深度 → [[2026-06-19_零售老刘体系二遍吸收_G_陈列货位与空间系数簇_v0.1]]

| 补强 | 口径 |
|---|---|
| 三层关系 | 货位配置(放什么/几个)→ 空间系数(货位固有权重,不变)→ 陈列系数=空间×时间 |
| 配置分配 | 货位数/单品数 = 销售占比 × 毛利占比(交叉比)|
| 货位内占比 | 该货号销额/该货位合计(非全店);占比高却非黄金层 → 层内上移 |
| 空间系数 space_score | 派生自 位置等级×可见性×客流量;把"黄金位"从4级定性升为连续量化 |
| 错配四型 | ①高系数喂低ABC ②高ABC窝低系数 ③高系数低坪效 ④层位错配 |

> 对 M-DEC-011:把"黄金位"升级为 `space_score` 连续权重 + 增④层位错配型,**不改临界值**。`space_score`=表E派生计算列(不增POS字段、不改position_grade定义)。
