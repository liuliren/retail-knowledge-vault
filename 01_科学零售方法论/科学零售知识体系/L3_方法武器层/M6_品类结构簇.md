# M6 品类结构簇

> ✅ **已吸收 v0.1**。深度笔记 → [[2026-06-19_零售老刘体系吸收_06_品类结构簇_v0.1]]

## 1. 主治痛点
**买不对**(顾客)/ 品类角色错配 → 毛利/客流双失。

## 2. 武器清单(速查)
| 武器 | 一句话 |
|---|---|
| **小类=顾客需求** ⭐ | 商品管理根基;对小类不重视=对需求不重视=战略问题;小类数不照搬大卖场 |
| **类别 RFM** | 顾客RFM迁移到类别,识别各小类需求强度 |
| **经营属性聚类** | 3模式:金额分群/**增长率分群(±10%稳/−20%降/+20%升)**/最高额固定 |

## 3. 骨架映射
- 小类=需求 / 类别RFM / 经营属性聚类 → **品类管理**(角色与结构,直接增强)
- 增长率分群 → 品类管理 + 数据底座
