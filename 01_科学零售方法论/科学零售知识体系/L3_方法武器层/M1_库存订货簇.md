# M1 库存订货簇

> ✅ **已吸收**。深度笔记见 → [[2026-06-18_零售老刘体系吸收_01_库存订货簇_v0.1]](`14_外部案例/零售老刘体系/`)

## 1. 主治痛点
**缺货**(经营)/ **买不到**(顾客)。核心反常识:追加式订货 = 越畅销越缺货。

## 2. 武器清单(公式卡速查)

| 武器 | 表达式 |
|---|---|
| 标准订货公式 | Q = d×(T+L) + Z·σ − I |
| 合理库存 | 日均销量 × 1.5 |
| 内仓零库存 | 陈列量 = 1.5 × 订货单位 |
| 有货率→Z | 95%→1.64 / 99%→2.33 |
| 售罄率盈亏平衡 | X = 毛利率 / (1−费用率) |
| 周转硬约束 | 周转天数 < 月结账期(30天)|
| 损耗红线 | 0.4% |

## 3. 骨架映射
- 订货公式/安全库存/DMS → **数量管理**
- 周转率/售罄率/动销率 → **商品力·库存周转**
- IDSM/周转<账期/损耗 → **金额管理·现金流**
- 闭环需求计划 → **组织力·流程闭环**

## 4. 花厅坊落地 + M-DEC 候选
- 采购矛盾 → 还原真实 DMS + 闭环需求计划
- 老品汰换 → 售罄率达 50% 天数排名
- **M-DEC 已落候选**:[[M-DEC-009_休食老品汰换时间维度防误判_v0.1_候选]](售罄率/假需求 防误杀)

## 6. 二遍补强(2026-06-19 / A 簇 6 篇精读)
> 增量,不推翻 §2。深度 → [[2026-06-19_零售老刘体系二遍吸收_A_库存订货与安全库存簇_v0.1]]

| 补强武器 | 表达式/口径 |
|---|---|
| 安全库存全式 | **SS = Z·σ·√VP**(VP=提前期);Z(95%)=1.65 |
| IR 库存系数 | **IR = INV/S = 12×MC%/ITO**;日更库存健康(与滞销标准v1.0周转维**互补不替换**)|
| EOQ 经济订货量 | √(2·D·i·S/V),D=365×DMS |
| 三条库存线 | 安全库存线/再订购点/库存上下限 → 触发(低于安全=紧急/低于再订购点=订单/高于最大=清库)|
| 售罄率正确用法 | 必带时间维度;用"达50%售罄速度"判畅销;平衡点46.7% |
