## 核心论点

顾客购买行为不能用"品类"解释，必须用**购买动机任务**解释——同一 SKU 可同时承载多个任务，正确的切分方式是"1主任务+1-2次任务+权重"的多标签结构，而非单任务硬归类。休闲食品9任务+水饮7任务是大湾区500-2000㎡社区生鲜超市选品系统的最小决策单元。

## 一、设计原则（前置约定）

**1. 任务不是互斥分类，是"购买动机标签"**
9 + 7 个任务在多个维度上正交切分（人数 / 关系 / 场景 / 时机 / 人群），SKU 必然多任务承载。AI 打标统一采用 **1 主任务 + 1-2 次任务 + 权重** 的多标签结构，禁止单任务硬归类。

**2. 任务由"SKU 特征族"反推，不是凭空切分**
每个任务必须能映射到一组**结构化区分的 SKU 特征**（包装 / 品牌 / 价格 / 品类 / 口味），否则就不是有效任务，是营销概念。

**3. 任务是"门店画像激活权重"的最小决策单元**
不同门店画像下同一任务的"激活强度"和"档位偏好"不同。例：花厅坊有小学 → 儿童零食/儿童饮品权重 0.9；纯白领商圈门店 → 这两个任务权重 0.3。

**4. 大湾区本地化必须显性嵌入**
凉果陈皮、拜山祭祀、煲汤备货、儿童早餐奶、配辣凉茶等本地特征，必须出现在对应任务的 SKU 特征族中。

## 二、休闲食品 9 任务

### 任务 1：解馋独享

| 项 | 内容 |
|---|---|
| 核心场景 | 1 人即时消费，情绪驱动（嘴巴寂寞、压力大、追剧、加班、单身夜宵） |
| 人数 | 1 人 |
| 时机 | 即时消费、计划性低 |
| 动机 | 情绪 / 感官 driven |
| 包装规格 | 单包小规格（30-80g），独立小包 |
| 品牌敏感度 | 低 |
| 价格带 | 3-10 元 |
| 品类倾向 | 辣条、魔芋爽、小面筋、薯片小包、素肉 |
| 口味倾向 | **强口味**（辣、酸辣、麻辣、咸香） |

**典型 SKU**：卫龙大面筋 26g、麻辣魔芋爽 15g、乐事小袋装、卫龙辣条、素鸡素肉系列

**边界规则**：
- vs 办公室零食：解馋独享是**即时一次性冲动**，办公室零食是**计划性囤货**
- vs 郊游：解馋独享是室内/工位场景；郊游强调便携不易碎

**大湾区本地化**：辣味偏好不如川渝，但年轻群体的网红爆款承接（辣条、魔芋）依然成立。

### 任务 2：办公室零食

| 项 | 内容 |
|---|---|
| 核心场景 | 白领工位囤货、抽屉常备、下午茶补能 |
| 人数 | 1 人为主，偶尔同事分享 |
| 时机 | 计划购买，预期消费多次 |
| 动机 | 抗饿 / 解闷 / 补能，要求"不打扰" |
| 包装规格 | 中等规格 50-150g，独立小包数量分装 |
| 品牌敏感度 | 中（信任品牌、关注配方） |
| 价格带 | 8-25 元 |
| 品类倾向 | 坚果、能量棒、蛋黄派、蔬果干、轻食饼干 |
| 关键属性 | **无气味、不掉渣、耐放、可分次吃** |

**典型 SKU**：三只松鼠每日坚果（独立装）、好丽友派盒装、比比赞蛋黄酥、沃隆混合坚果、谷物棒系列

**边界规则**：
- vs 解馋独享：动机不同——办公室是抗饿/补能（理性），解馋是情绪释放（感性）
- vs 家庭时光：办公室是个人计划，家庭时光是家人共享

---

### 任务 3：聚会（HOSTING）

| 项 | 内容 |
|---|---|
| 核心场景 | 5+ 人非家庭聚会（同学/同事聚餐、朋友过夜、世界杯、生日趴）；**你是组织者/招待者** |
| 人数 | 5+ 人 |
| 关系 | 非家人 |
| 动机 | **招待场面 + 面子货** |
| 包装规格 | **大桶 / 大盒 / 大包**（200g+） |
| 品牌敏感度 | **高**（信任品牌+大牌效应） |
| 价格带 | 15-50 元 / 单品 |
| 品类倾向 | 罐装薯片、大盒夹心饼、品牌坚果罐、糖果大袋 |
| 关键属性 | **上桌好看**，分给客人不掉价 |

**典型 SKU**：乐事/可比克罐装薯片、好丽友派大盒装、奥利奥分享装、不二家糖果罐、三只松鼠坚果罐 500g

**边界规则**：
- vs 朋友分享：聚会 = **HOSTING**（你招待别人）；朋友分享 = **CO-DISCOVERING**（共同尝鲜）
- vs 家庭时光：聚会是非家人面子驱动；家庭时光是家人信任驱动

### 任务 4：朋友分享（CO-DISCOVERING）

| 项 | 内容 |
|---|---|
| 核心场景 | 2-4 人小范围（闺蜜下午茶、周末小聚、同事下班小酌、新人尝鲜）；**话题驱动** |
| 人数 | 2-4 人 |
| 关系 | 非家人，平等关系 |
| 动机 | **新奇尝鲜 + 话题谈资** |
| 包装规格 | 中等规格（80-200g） |
| 品牌敏感度 | 中（更看重新奇度） |
| 价格带 | 10-30 元 |
| 品类倾向 | **网红爆款、地方特产、新口味、小红书/抖音爆款** |
| 关键属性 | **有话题、可讨论、可拍照** |

**典型 SKU**：网红爆款（每月迭代）、进口小众零食（韩/日/泰）、新口味产品（咸蛋黄/芝士麻辣等）、地方特色（柳州螺蛳粉味薯片等）

**边界规则**：
- vs 聚会：朋友分享是新奇度驱动+小规格；聚会是面子驱动+大包装
- vs 解馋独享：朋友分享是**共享体验**，解馋独享是**自我满足**

**大湾区本地化**：奶茶店周边、港式怀旧零食、本地网红品牌。

### 任务 5：家庭时光（CO-LIVING）

| 项 | 内容 |
|---|---|
| 核心场景 | 家人围坐场景（追剧、电影夜、节日团聚、周末家庭日）；**家人共享** |
| 人数 | 3-5 人家庭单位 |
| 关系 | 家人 |
| 动机 | **家人共享、各取所需** |
| 包装规格 | **家庭装、混合大礼包**（300g+） |
| 品牌敏感度 | 低-中（信任度优先于新奇） |
| 价格带 | 20-80 元 |
| 品类倾向 | 综合零食礼包、混合干果、家庭装薯片、爆米花桶、瓜子坚果罐 |
| 关键属性 | **多样化、覆盖不同口味偏好** |

**典型 SKU**：旺旺大礼包、沃隆每日坚果（家庭装）、三只松鼠综合礼包、大桶爆米花、瓜子+坚果家庭装混合

**边界规则**：
- vs 聚会：家庭时光是家人间信任，不需要"面子货"
- vs 年货送礼：家庭时光是**自家消费**，年货是**送给他人**

### 任务 6：郊游 / 户外

| 项 | 内容 |
|---|---|
| 核心场景 | 远足、野餐、车载、出游携带、登山；**便携消费** |
| 人数 | 1-多人 |
| 时机 | 户外、移动中 |
| 动机 | **便携、抗折损、能补能量** |
| 包装规格 | **便携、不易碎、密封、轻量** |
| 品牌敏感度 | 中 |
| 价格带 | 5-25 元 |
| 品类倾向 | 能量棒、压缩饼干、肉脯、独立坚果、巧克力派、糖果 |
| 关键属性 | **水分少、不融化、不怕颠、儿童友好** |

**典型 SKU**：牛肉粒/肉脯独立装、士力架/玛氏巧克力（耐高温款）、独立包装坚果、能量棒、压缩饼干

**边界规则**：
- vs 解馋独享：户外强调**便携属性**（不易碎、不融化）
- vs 聚会：户外是移动场景；聚会是固定场所

**大湾区本地化**：周末徒步、自驾游频率高，全年都有需求。

### 任务 7：年货送礼

| 项 | 内容 |
|---|---|
| 核心场景 | 春节、中秋、端午、回老家伴手、商务送礼 |
| 人数 | 送给他人 |
| 时机 | 节日强季节性 |
| 动机 | **面子 + 心意 + 地方特色** |
| 包装规格 | **精品礼盒装、地方特色礼盒** |
| 品牌敏感度 | **极高** |
| 价格带 | 50-300 元+ |
| 品类倾向 | 礼盒装坚果、传统糖点礼盒、地方特产礼盒 |
| 关键属性 | **包装精美、价格带明显高、地方文化感** |

**典型 SKU**：三只松鼠坚果礼盒、奥利奥精品礼盒、香港大班冰皮月饼、老婆饼/凤梨酥礼盒（广东本地）、老字号年货礼盒

**边界规则**：
- vs 家庭时光：年货送礼是**送给他人**；家庭时光是**自家消费**
- vs 聚会：年货送礼是个人**手礼**，聚会是现场**招待**

**大湾区本地化**：春节年货市场、清明饼礼盒、中秋月饼礼盒——节日峰极强。

### 任务 8：儿童零食（花厅坊专项）

| 项 | 内容 |
|---|---|
| 核心场景 | 小学生放学路过自购、家长备货、节日给孩子 |
| 人数 | 儿童为主 |
| 关系 | 儿童自购 / 家长代购 |
| 动机 | **口感偏甜 + IP/卡通驱动 + 家长认证（健康）** |
| 时机 | 放学时段、周末家庭采购 |
| 包装规格 | **小规格、卡通包装、亮色** |
| 品牌敏感度 | 中（IP驱动 > 品牌驱动） |
| 价格带 | 1-15 元（学生自购倾向 1-5 元） |
| 品类倾向 | 含奶饼干、儿童糖果、卡通棒棒糖、果冻、巧克力豆、儿童派 |
| 关键属性 | **含奶量、低盐、营养标识、IP 形象** |

**典型 SKU**：旺仔小馒头、喜之郎果冻、旺旺仙贝、卡通棒棒糖、奥利奥小包装、蓝胖子

**边界规则**：
- vs 解馋独享：儿童零食的**包装/口感/IP**完全针对儿童
- vs 家庭时光：儿童零食是**专为儿童**，家庭时光是**全家共享**

**大湾区本地化**：花厅坊有小学 → 该任务在花厅坊门店权重显著高于纯白领商圈门店。

### 任务 9：中老年常备 / 佐茶

> **大湾区差异化任务**：大湾区社区超市的中老年客群有非常明确的、与年轻群体完全不同的购买行为。这些 SKU 在前 8 任务里都放不进去——不是冲动（≠解馋）、不是 occasion-driven（≠家庭时光）、不在户外（≠郊游）。独立成任务后，凉果陈皮在打标系统里有了正确的任务级承载，避免在中老年客群门店给出错误建议。

| 项 | 内容 |
|---|---|
| 核心场景 | 老年人日常嗑食 / 佐茶习惯 / 在家慢消费 |
| 人数 | 1 人为主 |
| 时机 | 日常、高频、计划购买 |
| 动机 | **习惯性嗑食 / 佐茶 / 怀旧** |
| 包装规格 | 中等规格（100-300g）、自封袋 |
| 品牌敏感度 | 低（认产品类型不认品牌） |
| 价格带 | 5-25 元 |
| 品类倾向 | **凉果陈皮、话梅、嘉应子、瓜子（日嗑）、山楂条、九制陈皮** |
| 关键属性 | **慢消费、耐放、咸甜复合口感** |

**典型 SKU**：化州橘红凉果、九制陈皮、话梅、嘉应子、袋装瓜子（200g+）、山楂条、果丹皮

**边界规则**：
- vs 解馋独享：动机不同——中老年是**日常习惯**（高频低值），解馋是**情绪冲动**（一次性）
- vs 家庭时光：中老年是**个人慢消费**，家庭时光是 occasion-driven 共享

**与方法论的关联**：花厅坊调改时把"凉果陈皮"作为差异化锚点之一——这一任务就是这个锚点的**任务级承载**。

## 三、水饮 7 任务

### 任务 1：日常解渴

- 即时饮用、单瓶、最高频
- 包装：350-550ml 单瓶
- 价格带：2-6 元
- 典型：怡宝矿泉水、康师傅冰红茶、农夫山泉、维他柠檬茶

### 任务 2：佐餐场景

- 配早餐 / 午餐 / 外卖、配辣
- 包装：单瓶 / 盒 / 利乐包
- 价格带：3-8 元
- 典型：早餐奶（蒙牛/伊利）、维他奶、王老吉/加多宝（配辣）、果汁
- **大湾区本地化**：配辣 → 凉茶（王老吉 = 大湾区配辣首选）；早茶 → 普洱/铁观音瓶装

### 任务 3：功能性

- 运动后、熬夜、解辣、护胃
- 包装：单瓶
- 价格带：4-10 元
- 典型：脉动、宝矿力、红牛、东鹏特饮、苏打水

### 任务 4：聚会饮品

- 多人聚餐、围坐、餐厅外带
- 包装：**1.25L+ 大瓶 / 多瓶组合**
- 价格带：8-25 元
- 典型：大瓶可乐雪碧、芬达、大瓶果汁、苏打水组合

### 任务 5：户外偶发

- 登山、拜山、郊游、自驾
- 包装：**整箱、便携多瓶**
- 价格带：15-40 元 / 箱
- 典型：整箱矿泉水、整箱运动饮料、整箱椰子水
- **大湾区本地化**：清明拜山祭祀（瓶装水、利乐包果汁/牛奶作为祭品+消费品）；周末徒步整箱水

### 任务 6：送礼 / 节日

- 礼盒装伴手
- 包装：礼盒、盒装多瓶
- 价格带：50-200 元
- 典型：蒙牛特仑苏礼盒、椰汁礼盒、维他奶礼盒、燕窝水礼盒

### 任务 7：儿童饮品（花厅坊专项）

- 学生自购 / 家长备货
- 包装：小瓶 / 利乐包 / 多支装
- 价格带：1-8 元
- 典型：旺仔牛奶、优酸乳、爽歪歪、AD 钙奶、阳光美味果汁、小瓶豆奶
- **vs 佐餐**：儿童饮品是**专给儿童**（卡通包装、儿童口味），佐餐是**配餐成人**

## 四、多标签政策（打标规则）

### 主任务 + 次任务 + 权重

每个 SKU 至少 1 个主任务（权重 ≥ 0.5），可附 1-2 个次任务（权重 0.1-0.4）。权重之和 = 1.0。

### 打标示例

| SKU | 主任务 | 次任务 1 | 次任务 2 |
|---|---|---|---|
| 卫龙大面筋 26g | 解馋独享 (0.7) | 朋友分享 (0.2) | 儿童零食 (0.1) |
| 三只松鼠坚果礼盒 | 年货送礼 (0.8) | 家庭时光 (0.2) | — |
| 旺仔牛奶 125ml × 12 | 儿童饮品 (0.7) | 家庭时光 (0.2) | 日常解渴 (0.1) |
| 化州橘红凉果 200g | 中老年常备 (0.8) | 解馋独享 (0.2) | — |
| 1.25L 可口可乐 | 聚会饮品 (0.7) | 家庭时光 (0.3) | — |
| 整箱怡宝 555ml × 24 | 户外偶发 (0.5) | 日常解渴 (0.4) | 聚会饮品 (0.1) |
| 大桶爆米花 500g | 家庭时光 (0.6) | 聚会 (0.3) | 郊游 (0.1) |
| 沃隆每日坚果 25 包 | 办公室零食 (0.6) | 家庭时光 (0.3) | 郊游 (0.1) |

### AI 打标 → 校准工作流

1. AI 输出（主任务 + 次任务 + 权重 + **推理说明** + 置信度）
2. 六哥校准（修改 / 补充 / 接受）
3. **每次校准必须记录"修改理由"** → 沉淀进《打标判断手册》
4. 累积语料反哺 AI；AI 准确率随校准次数变化曲线作为关键看板

## 花厅坊直接调用

> T03 饼干 = 偏好性购物任务（计划性冲动购买：顾客走到饼干区才决定买，不是专程来）。
> **陈列位置含义**：T03 面位不应放在入口处引流（那是目的性品类的位置），而应在顾客已进入休食区后的中段位置。
> **高峰时段**：偏好性品类购买高峰 = 下午 17:00–19:30 下班顺路，这个时段休食区应有促销主推或视觉吸引点。
> **任务打标验证**：若门店有 POS 数据按时段分析，17:00–19:30 的饼干销售占比应高于早市——若低于早市，说明下午时段陈列引导未到位。
