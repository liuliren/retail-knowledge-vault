# 11_系统产品与PRD

## 目录作用

本目录沉淀**数字化产品设计、AI 工具规划、小程序/PC 后台需求、PRD、字段字典和开发提示词**，是咨询业务产品化的核心库。

这里的内容连接咨询方法论与可开发系统，是"让咨询工作效率更高、可规模化复制"的关键环节。

---

## 适合存放

- 社区超市商品诊断系统 PRD
- 30天商品提效咨询作业系统 PRD
- 微信小程序需求池
- PC 后台页面规划
- 字段字典（所有业务字段的统一定义）
- 实体关系说明
- 数据导入规则
- 字段映射模板
- 异常报表规则
- Claude Code 开发提示词
- Hermes 工具配置说明

---

## 不适合存放

- 已上线系统的运营数据（属于 `08_经营分析与数据看板`）
- 方法论文件（属于 `01_科学零售方法论`）
- Excel 模板（属于 `13_数据分析与工具脚本`）
