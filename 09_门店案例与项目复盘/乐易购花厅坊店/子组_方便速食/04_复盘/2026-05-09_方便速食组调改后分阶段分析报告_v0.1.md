# 2026-05-09 方便速食组调改后分阶段分析报告 v0.1（A+B+C 组合）

> **本报告聚焦"分阶段"看调改效果**，不再做单点结论。3 个维度组合：
> - **A 时间窗口渐进**：2025 → 调改前 60 → 调改前 17 → 调改后 13 / 4 阶段
> - **B 子类目分阶段**：方便面 / 方便粉 / 即食类 / 方便食品 / 4 子类
> - **C 附录 ABCD 等级**：调改后 30 天 SKU 销额累计分布
>
> ⚠️ **关键发现**：整体 +23% 毛利**掩盖了内部巨大分化** — 方便面毛利率猛跌 -7.5pp（警告 ⚠️）/ 方便粉销额翻倍 +108% ⭐ / 即食类毛利率猛涨 +12.6pp ⭐。**单点结论会误导决策**。
>
> ⚠️ **数据精度声明**（§13.17 推断标注）：所有调改后数字基于差值法估算 / 真实精度 ±10-15% / 不可作对外硬证据。

### A.2 阶段间跃迁

#### A1 → A2（2025 年 → 调改前 60 天 / 同比"年同期"）

| 指标 | A1 | A2 | 变化 |
|---|---|---|---|
| 销额 | 356.44 | 357.66 | **+0.3%** |
| 销量 | 42.57 | 53.88 | **+26.6%** ⭐ |
| 毛利 | 70.79 | 68.30 | **-3.5%** ⚠️ |
| SKU 数 | 280 | 239 | -41（-14.6%）|

**[推断]** 这意味着 2025-2026 早期门店已做"低价高频"结构调整：销量上升 / 但单件毛利率轻微下滑 / SKU 净减 41 个。**这不是 4/25 调改的效果 / 是更早期的累积**。

#### A2 → A4（调改前 60 天 → 调改后 13 天 / 真正的"调改前 vs 调改后"⭐）

| 指标 | A2 | A4 | 变化 |
|---|---|---|---|
| 销额 | 357.66 | 463.73 | **+29.7%** 🟢 |
| 销量 | 53.88 | 81.00 | **+50.3%** 🟢⭐ |
| 毛利 | 68.30 | 84.00 | **+23.0%** 🟢 |
| SKU 数 | 239 | 199 | -40（-16.7%）|

**洞察**：调改的本质 = **再砍 40 个 SKU + 销额仍涨 30% / 毛利涨 23%**。
- 销量增速（+50%） > 销额增速（+30%）= 单价下降
- 销额增速（+30%） > 毛利增速（+23%）= **整体毛利率从 19.1% → 18.1% / -1pp**
- 看似"轻微毛利率下滑" — 但 §B 子类目分析揭示**真相完全不同**

#### A1 → A4（2025 年基线 → 调改后 13 天 / 跨年总跃迁）

| 指标 | A1 | A4 | 倍数 |
|---|---|---|---|
| 销额 | 356.44 | 463.73 | **1.30x** |
| 销量 | 42.57 | 81.00 | **1.90x** ⭐⭐ |
| 毛利 | 70.79 | 84.00 | **1.19x** |
| SKU 数 | 280 | 199 | **0.71x**（净减 81） |

**关键判断**：跨年看（2025 → 调改后 13 天 / 估算）/ **方便速食组整体经历两次"做减法 + 提销量"调改**：① 2025-2026 年初（SKU -41 / 销量 +27%）② 2026-04-25（SKU -40 / 销量 +50%）。**调改不是单点动作 / 是连续优化的延续**。

## §B. 子类目分阶段（4 子类）⭐⭐⭐

整体 +23% 毛利**掩盖了内部巨大分化**。逐子类拆开看：

### B.1 4 子类目调改前后对比表

| 子类目 | SKU 演化（2025/90/30）| A2→A4 销额 | A2→A4 销量 | A2→A4 毛利 | A2→A4 毛利率 | 评级 |
|---|---|---|---|---|---|---|
| **方便面** | 199→173→143 | +9.4% | +27.5% | **-32.5%** ⚠️ | 19.7% → **12.1%**（-7.5pp）⚠️⚠️ | 🔴 警告 |
| **方便粉** | 50→33→27 | **+108.5%** ⭐ | **+163.7%** ⭐⭐ | **+135.6%** ⭐⭐ | 24.5% → **27.7%**（+3.2pp）| 🟢⭐⭐ 大赢 |
| **即食类** | 31→29→26 | +63.7% | +85.6% | **+210.1%** ⭐⭐ | 14.1% → **26.6%**（+12.6pp）⭐ | 🟢⭐⭐ 结构升级 |
| **方便食品** | 0→4→3 | +22.1% | +138.5% | -17.9% | 27.1% → 18.2%（-8.9pp）| 🟡 微小占比 |

### B.2 方便面 — 警告 ⚠️

**事实**：销额日均 253.08 → 276.82（+9.4%）/ 销量 39.08 → 49.81 件（+27.5%）/ 毛利 49.74 → 33.56（**-32.5%**）/ 毛利率 19.7% → **12.1%**（-7.5pp）。

**[推断]** 可能原因：
1. 调改时引入了**低毛利大牌**（统一红烧牛肉桶 / 康师傅经典等）/ 这些品牌强势但毛利率 10-11%
2. 调改时**做了降价 / 促销**（回应竞争）
3. 高毛利方便面 SKU 流失（SKU 199→143 / 净减 56 / 可能流失了高毛利但低销量品种）

**Top 3 销售方便面（调改后）**：
- 统一红烧牛肉桶（145 件 / 销额 567 / 毛利率 10.5%）
- 统一红油老坛酸菜牛肉桶（144 件 / 565 / 10.9%）
- 统一香辣牛肉面桶（116 件 / 456 / 11.1%）
**→ Top 3 全是统一桶面 / 毛利率全部 10-11%** = 数据印证假设 1（引入低毛利大牌）

**🔴 警告**：方便面虽然销量+27%，但毛利绝对额-32.5% / **不是健康的调改方向**。需要启明 confirm 这是有意为之（引流款牺牲毛利）/ 还是无意识的负面后果。

### B.3 方便粉 — 大赢 ⭐⭐

**事实**：销额日均 35.97 → 74.98（**+108.5%**）/ 销量 4.70 → 12.39 件（+163.7%）/ 毛利 8.80 → 20.74（+135.6%）/ 毛利率 24.5% → **27.7%**（+3.2pp）。

**Top 1 爆款 = 臭宝爆爆浓汤柳州螺蛳粉 258g**（90 件 / 销额 801 / 毛利率 30%）⭐⭐
- 单 SKU 占整个方便速食组销额的 **6.6%**（801/12108）
- 单 SKU 占整个方便粉子类销额的 **50.5%**（801/1586）

**[推断]** 调改成功原因：
1. 螺蛳粉网红爆款 / 客群匹配（年轻女性 / 美食爱好者）
2. 单价 8.9 元 / 在 8-10 元价格带是甜区
3. 黄金视线层增排 / 调位（推断 / 待 client_told 确认）
4. 子类 SKU 净减 23 个（50→27）/ 集中资源到爆款

**🟢⭐⭐ 大赢**：方便粉子类**所有维度全面大涨** / 是调改最成功的子类 / 应做为后续调改的范本案例。

### B.4 即食类 — 结构升级 ⭐⭐

**事实**：销额日均 67.70 → 110.80（+63.7%）/ 销量 10.02 → 18.59 件（+85.6%）/ 毛利 9.51 → 29.50（**+210.1%**）/ 毛利率 14.1% → **26.6%**（+12.6pp）⭐。

**毛利率从 14.1% 跃升到 26.6%** = 即食类的"结构升级" / 不是销量驱动 / 是**结构驱动**。

**Top 3 即食类爆款**：
- 一号果仓大瓶黄桃罐头 880g（45 件 / 544 / 毛利率 29.7% ⭐）
- 王老吉龟苓膏 220g（39 件 / 386 / 22.6%）
- 喜茶黄桃罐头 958g（20 件 / 199 / 14.9%）

**[推断]** 调改逻辑：
1. **黄桃罐头家族**（一号果仓 + 喜茶 + 一号果仓 510g）3 款进 Top 20 = 子类爆品集群化
2. **泰奇八宝粥**多款（清润型 / 原味 / 红豆香沙）= 即食粥子集群
3. SKU 几乎不动（31→29→26 / 净减 5 / 微调）/ 销量翻倍 = **结构升级（高毛利单品冒头）**

**🟢⭐⭐ 结构升级**：即食类是**毛利率提升最显著**的子类 / 调改对其结构产生了深刻影响。

### B.5 方便食品 — 微小子类（不重要）

3 SKU / 30 元销额 / 占整组 0.25% / 数据噪声 / 不做单独决策。

## §B 总结：分化决策矩阵

| 子类目  | 销额变化          | 毛利变化          | 决策                                      | 优先级      |
| ---- | ------------- | ------------- | --------------------------------------- | -------- |
| 方便面  | +9.4%         | **-32.5%** ⚠️ | **警告 / 启明 confirm 是否引流款牺牲毛利策略 / 否则需调整** | 🔴 P0    |
| 方便粉  | **+108.5%** ⭐ | **+135.6%** ⭐ | **保持 + 复盘成功原因 + 复制到其他子类**               | 🟢 P0 范本 |
| 即食类  | +63.7%        | **+210.1%** ⭐ | **保持 + 黄桃罐头家族扩品 + 八宝粥子集群**              | 🟢 P1    |
| 方便食品 | +22%          | -18%          | 不做决策（微小占比）                              | ⚪ 监控     |

**核心反幻觉警告**：单看整体 +23% 毛利会**误判调改全面成功**。实际上：
- 方便面是**毛利大输家**（-32.5% / 7.5pp 毛利率下滑）
- 方便粉 + 即食类是**真正赢家**
- 调改的"成功"主要靠**方便粉 + 即食类**拉起来 / 不是方便面

→ M-DEC-007 §S5 调整动作决策的实战经验：**方便面引入大牌引流款 = 销量驱动但毛利杀手** / 需谨慎使用 / 不可推广到所有子类。
