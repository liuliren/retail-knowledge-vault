## 13. 库存处理（5/23-5/24）

### 13.1 库存处理矩阵

| 库存状态 | 销售状态 | 处理 |
|---|---|---|
| 库存金额高 + 末次销售 ≤ 30 天 | 正常动销 | 保留 / 持续观察 |
| 库存金额高 + 末次销售 30-60 天 | 半滞销 | 调位 + 促销试销 |
| 库存金额高 + 末次销售 > 60 天 | **重度滞销** | 促销消化 / 后续淘汰 |
| 库存金额低 + 末次销售 > 60 天 | 轻滞销 | 直接淘汰 |
| 库存金额 = 0 + 销售有 | 缺货 | 补货 / 加供应商 |
| 库存金额 > 0 + 销售 = 0 + 引进 > 30 天 | 引进失败 | 退换货 / 直接报损 |
