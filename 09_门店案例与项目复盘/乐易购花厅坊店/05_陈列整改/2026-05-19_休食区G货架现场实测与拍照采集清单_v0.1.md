## 3. 每组货架拍照标准

### 3.1 必拍 8 张（每货架最低）

| # | 视角 | 距离 / 设置 |
|:---:|---|---|
| 1 | 正面全景（眼平 / 顾客视角）| 距离 ≥ 2m / 横拍 / 全幅入画 |
| 2 | 左 45 度 | 距离 1.5m / 含上下层 |
| 3 | 右 45 度 | 同上 |
| 4 | 上半区特写（黄金层 1-2）| 距离 0.8m |
| 5 | 下半区特写（中层 + 下层 + 底层）| 同上 |
| 6 | 价格签细节（典型 5-10 SKU）| 平视 / 不俯拍 |
| 7 | 异常点特写（缺货 / 错位 / 模糊价签）| 现场标注 |
| 8 | 端架 / 侧挂 / 堆头关联照（如有）| — |

### 3.2 特殊补拍要求

- **G3-B 方便速食**：**调改后复拍**（同机位 vs 5/12 17 张）→ v0.6 升级关键证据
- **G4-A 冲调**：补拍**层板与价格签细节**（5/13 W20-Day4 已有部分 / 5/19 强化补 6 机位 vs 4/24 30 张）
- **G4-B 水饮**：**必须补齐全套**（6 机位 + 全层板 + 价格签 + 端架关联）→ 当前最大盲点
