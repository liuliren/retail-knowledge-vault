# 花厅坊门店通道命名表 v0.1

## 1. 文件定位

本文件用于沉淀**花厅坊门店内部通道的标准命名**，作为后续 DisplayMap、动线分析、商品诊断、陈列整改、首访整理时引用通道名称的**唯一依据**。

> 本表为 v0.1 草案，**全部内容来自现有 74 张原始照片文件名的反向提取**，未经实地动线复核。等同性判断、通道实际数量与位置关系，**待人工实地核实后升 v1.0**。

## 2. 适用场景

- 编写门店首访记录、商圈动线分析、商品诊断、陈列整改方案时，统一通道命名口径
- 后续命名规范文件（`文件命名规范.md` v1.x）正式立项时引用
- 为 DisplayMap 草图、卖场空间信息采集表提供通道标识参考

## 4. 命名结构规则（基于现有照片实证）

### 4.1 三段式（拍摄起点 + 朝向）

```
花厅坊_[拍摄起点区/通道]_[朝向区/通道]视角_[序号].扩展名
```

示例：`花厅坊_称重区_入门主通道视角_01.jpg`

### 4.2 四段式（通道 + 左/右侧 + 朝向）

```
花厅坊_[通道]_[左侧/右侧]_[朝向区]视角_[序号].扩展名
```

示例：`花厅坊_垂直主通道01_右侧_水果区及入口主通道位置视角_01.jpg`

### 4.3 拍摄主体（特例）

```
花厅坊_[区域]_[拍摄主体]_[序号].扩展名
```

示例：`花厅坊_休食区_水饮货架_01.jpg`

## 5. 分析判断（基于命名相似度的疑似归并）

> 以下为**基于命名相似度的推断**，未经实地核实。**不得直接作为结论引用**，必须实地核实后才能合并或拆分。

### 5.1 高度疑似同一通道（不同叫法）

| 命名 A | 命名 B | 推断 | 待核实点 |
|---|---|---|---|
| 进门主通道 | 入门主通道 | 疑同一通道 | 是否物理上同一条？还是"进门"指入口段、"入门"指刚进店段？ |
| 出口主通道 | 出口通道 | 疑同一通道 | 是否物理上同一条？或"主通道"是更宽的总通道、"通道"是收银前细通道？ |
| 垂直中央主通道 | 垂直中央通道 | 疑同一通道 | "垂直中央通道"仅出现 1 次，疑为漏字或别名 |
| 水饮风幕柜主道 | 水饮风幕柜通道 | 疑同一通道 | "主道"是否等同于"主通道"的简写？还是不同概念？ |
| 风幕柜通道 | 水饮风幕柜通道 | 疑同一通道 | 是否为简写？ |

### 5.2 同一通道族内的"无侧标"与"带侧标"关系

`垂直主通道02_水果区视角_01.jpg` 与 `垂直主通道02_右侧_水果区视角_01.jpg` 同时存在：

- 推断 A：无侧标的是从"通道正中"拍摄
- 推断 B：无侧标的是早期拍摄、未做侧标区分
- 推断 C：拍摄角度差异已用序号 _01/_02/_03/_04 区分

**待核实**：建议下次现场核实"侧标"的物理含义后写入 v1.0。

## 6. 操作规则（v0.1 临时约定，待 v1.0 正式立项）

1. **新拍摄照片暂时使用现有命名结构**：三段式 / 四段式 / 拍摄主体特例
2. **无侧标 vs 带侧标**：暂保持现有照片命名不变，下次实地拜访时统一确认
3. **"垂直主通道00出口"语义**：留待后续核实是否为独立通道
4. **5.1 节疑似归并的命名**：在正式整理时，**先按现有原始名引用**，不擅自合并
