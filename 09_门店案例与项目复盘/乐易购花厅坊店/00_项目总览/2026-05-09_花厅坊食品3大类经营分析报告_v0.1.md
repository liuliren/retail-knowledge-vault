## §F. 关键发现 + 调改建议

### F.1 5 大关键发现

#### 发现 1: 大湾区地方品牌占主导 ⭐
- 粤盐家族（基础调味 / 3 SKU 进 Top 5）
- 甘竹+鹰金钱（豆豉鲮鱼 / 酱菜类双爆款）
- 益力多+燕塘（低温白奶 / 大湾区两大乳品本地品牌）
- 致美斋（醋 / Top）

→ **战略意义**：与赵一鸣硬折扣错位的关键 / 启明侧应继续深挖大湾区特色品牌。

#### 发现 2: 乳品毛利率拖累严重 ⚠️
- 水饮料 L1 毛利率 16.5%（最低）/ 主要是低温奶 10.6% + 常温奶 12.1%
- 益力多多个 SKU 毛利率 1-5% = 高客流低毛利引流款
- **如果益力多+燕塘合计销额可削减 10% / 毛利率提升 2-3pp**

→ **建议**：评估益力多客流贡献度（是否真的不可替代引流款）/ 如有替代品则可适度调整。

#### 发现 3: 调味品 L2 毛利贡献第一 ⭐
- 调味品 L2 毛利 13,241 元 / 占 L2 总毛利 24.9%
- 毛利率 23.4% / 健康
- 粤盐家族 + 海天 + 千禾 + 致美斋 = 头部品牌强势

→ **建议**：调味组维持 / 不动结构 / 保留头部品牌权重。

#### 发现 4: D 类长尾过长（1,012 SKU / 33.6% / 仅贡献 5%）⚠️
- D + 真流失 272 = **1,284 SKU 待清理候选**
- 库存资金回笼估算（待精算）

→ **建议**：W19-W22 分批淘汰（详见 `02_采购淘汰品清单.csv`）。

#### 发现 5: 范围外 23.4% 销额值得关注（用户决定后续处置）
- 冷冻类 330 SKU / 4.15 万（雪糕+面点+水饺+汤圆等）
- 酒精类 276 SKU / 3.54 万（啤酒+白酒+红酒等）
- 火腿肠 69 SKU / 1.0 万
- 茶叶 4 SKU / 109 元

→ **建议**：用户后续决定这 4 类是否纳入 V4.0 体系 / 还是用原 cat_name 单独管理。
