```

横向 [表 E 货位表 v1.0]  (5 段制 ZTGSL / 货位单源)
```

## §5. 使用 SOP

### 5.1 复制使用流程

```
Step 1: 拷出本交付包目录到目标位置
Step 2: 按表头创建 Excel（vault .md 字段定义 → Excel 列）
Step 3: POS 数据导出 + 现场实测 → 装载到 Excel
Step 4: 联动键 VLOOKUP 实现表间关联
Step 5: 运行 M-DEC-007 7 步 SOP（数据准备 → 六维交叉 → ABCD+Z → 归因 → 决策 → sign off → 闭环）
```

## §7. 反幻觉红线（[[CLAUDE.md]] §13 / 跨 6 表）

1. ❌ 不允许仅凭销量定 ABCD 等级（必须 7 维交叉）
2. ❌ 不允许 ABCD + Z 异常标记缺失（V4 ④ 标准）
3. ❌ 不允许销量驱动单一标记"促销期"（必须 4 类区分）
4. ❌ 不允许货架字段（25-28 + location_code）凭印象填（必须现场实测）
5. ❌ 不允许 retest 节点跳过（必走 D7/D15/D30/D60 4 节点）
6. ❌ 不允许淘汰决策 sign off 凭证缺失
7. ❌ 不允许新品引进 `composite_score < 6.0` 强引（除非战略例外）
8. ❌ 不允许品类角色无依据判定（必须 3 维：销售贡献 / 客流 / 毛利）
