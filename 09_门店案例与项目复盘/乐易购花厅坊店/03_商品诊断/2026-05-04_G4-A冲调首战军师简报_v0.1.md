# G4-A 冲调首战军师简报（5/6 调改方向初判）

> **数据源**：`90_/00_客户原始材料/花厅坊/2026-05-04_AI数据武装_v0.1/G4-A_冲调74SKU_数据基线_v0.1.csv`
>
> **数据窗口**：4 月（30 天日均）/ 库存截止 2026-04-24
>
> **范围**：G4-A 冲调正面货架 6 组位 74 SKU
>
> **算法**：SKU 名称→规格提取算法 92.8% 完全命中（leave-one-out 验证）；SKU 主数据补全 v0.1（待 5/5 二访 + 5/8 实战回流后升 v0.5）
>
> ⚠️ **反幻觉硬约束**（[[CLAUDE.md]] §13.3 / §13.13）：本简报为**调改前数据武装**，不是诊断结论。最终调改决策必须经 5/5 二访现场核实 + 5/6 老板对话确认 + 包装尺寸/层位/动线判断后定。

## 0. 一句话总判（先看这一句）

**G4-A 6 组位 74 SKU 当前结构是"中央肥大 + 两端真空"**：燕麦谷物（G4-A-01）放了 34 SKU 占 46% 但贡献销售 49%；营养粉/代餐（G4-A-02）完全空白；早餐功能补充（G4-A-06）3 SKU 全死。**整体不动销率 81%**（60/74 SKU 月销量 < 3 件）—— 第一次调改的核心动作就是**砍肥大 + 填真空**。

## 1. 整体战况

| 指标 | 数值 | 解读 |
|---|---|---|
| 总 SKU 数 | 74 | G4 排 A 面 6 组位（按 48 组位编码总表）|
| 4 月销售总额 | **¥2,673** | 平均每天 ¥89 / 单 SKU 月均 ¥36 |
| 库存成本占用 | **¥4,722** | 库存周转 ≈ **53 天**（远超社区超市 30 天健康线）|
| 不动销 SKU | **60 个 / 81%** | 月销量 < 3 件（日均 < 0.1）|
| 负库存 SKU | 3 个 / 4% | 收货录入差错的实证（参 [[CLAUDE.md]] §13 反幻觉规则）|
| 滞销 SKU（预可售 >180 天）| 8 个 / 11% | 半年以上库存周转不动 |

**核心问题**：销售规模 < 库存成本 → **冲调正面货架在亏现金流**。每天 ¥89 销售，但占用 ¥4722 库存 + 1 面 G4 货架的陈列资源。

## 3. 各组位调改方向初判（5/6 老板对话用）

### 3.1 G4-A-01 燕麦谷物（34 SKU → 砍 50%）

**问题**：34 SKU 占 G4-A 接近一半但 26 个不动销，是占陈列资源最贵的"长尾"。品牌池：桂格 8 / 皇麦世家 5 / 金味 4 / 西麦 4 / 智力 3。

**畅销 TOP 3**（保留）：
- 顺天缘五谷杂粮 16g ¥1.0（日均 0.43 — **唯一高频**）
- 金味原味麦片 600g ¥22.5（日均 0.23）
- 阿华田即食燕麦片 450g ¥9.9（日均 0.13）

**建议下架优先级（不动销 + 高库存成本）**：
- 桂格 5 黑麦片 518g ¥20 库存 1 / 桂格快煮燕麦片 1000g 库存 0 / 桂格即食燕麦片 1000g 库存 3
- 阿华田黄金大麦 180g（**库存 -1 = 负库存，先盘点**）
- 丹顿核桃牛奶燕麦片 490g ¥18（210 天滞销）
- 金味即食纯生燕麦片 1000g（330 天滞销）
- 金味中老年 7+1 纤维麦片 484g（240 天滞销）
- 南国椰子粉 170g（**840 天滞销 + 库存 28 = 占 ¥? 现金**）

**调改后目标**：17 SKU（砍 50%）/ 4 大品牌覆盖即可（桂格 / 西麦 / 金味 / 皇麦世家） / 价格带 5-25 元主销带

### 3.2 G4-A-02 营养粉/代餐（0 → 6-8 SKU，必须补）

**问题**：完全空白。这是当下增长最快的早餐场景（健康 / 健身 / 减脂 / 中老年营养），社区客群在便利店/电商已经习惯，超市没货 = 客流外流。

**5/6 与老板对话核心**：
> "您冲调正面 G4-A-02 营养粉/代餐这个位是空的。这是 2024-2026 年增长最快的细分场景，您隔壁的百欢有没有？罗森有没有？我们补上 6-8 个 SKU 这块至少能拿回 ¥500-1500/月销售。"

**建议候选品类**（待 5/5 二访询问供应商）：
- 蛋白粉（如汤臣倍健 / 安利系 / Myprotein 国产线）
- 代餐谷物粉（如桂格 / 雀巢系 / 王饱饱）
- 中老年营养粉（如雀巢怡养 / 三元）
- 健身 / 体重管理类

### 3.3 G4-A-03 奶粉/豆奶粉（5 → 8 SKU）

**现状**：雀巢 2 + 伊利 2 + 维维 1 — 大品牌覆盖了 / 但 SKU 太薄。

**畅销**：维维维他型豆奶粉 360g ¥15（日均 0.10 — G4-A-03 唯一动销）

**问题**：
- 雀巢 2 SKU 都不动销（怡养羊奶粉 ¥258 / 高钙营养奶粉 ¥99 — 价格带太高 / 与社区客群不匹配）
- 伊利 2 SKU 不动销（全脂甜 / 中老年鲜奶粉 — 可能跟乳粉竞争）

**建议**：
- 撤雀巢羊奶粉 / 高钙营养奶粉（¥250+ 价位不适合社区）
- 补 2-3 个中端奶粉（¥30-60 价位 / 蒙牛 / 伊利金典 / 飞鹤系入门款）
- 维维豆奶粉补 2-3 个规格（小包装 / 大包装家庭装）

### 3.4 G4-A-04 芝麻糊/核桃粉（21 SKU 整顿，目标 12-15）

**畅销 TOP 3**：
- 雀巢原味炼奶 185g ¥13（日均 0.17 — **G4-A-04 单品冠军**）
- 南方原味黑芝麻糊 480g ¥18.8（日均 0.13）
- 南方红枣黑芝麻糊 480g ¥21.8（日均 0.10）

**问题**：丰衣足食系列 3 SKU（包装幼砂糖 / 罐装粗砂糖 / 罐装幼砂糖 — **疑似不属于"传统冲饮"L4**，可能是混类）

**建议**：
- 砍丰衣足食 3 SKU（移到糖类货架）
- 砍皇麦世家高铁高钙优麦 600g ¥25（不动销）
- 砍怀滋露洋槐蜂蜜 450g ¥25（库存 0 但仍占 SKU 位）
- 留南方系列 + 太古红糖 + 雀巢炼奶（这几个有动销）

### 3.5 G4-A-05 咖啡/奶茶（11 SKU 但 91% 不动销 → 大整顿）

**问题最严重**：11 SKU 中只有 1 个动销（雀巢咖啡无蔗糖 ¥10.2 / 日均 0.10）。

**雀巢一家独大 8 SKU 但 7 个不动销**：
- 雀巢醇品咖啡玻璃瓶 90g ¥35（不动销）
- 雀巢醇香原味咖啡 7 杯 ¥12.5（不动销）
- 雀巢厚乳丝滑拿铁咖啡 95g ¥14（**库存 14 / 210 天滞销**）
- 雀巢奶香拿铁咖啡 7 杯 ¥11（不动销）
- 雀巢桃桃拿铁风味咖啡 95g（**库存 11 / 330 天滞销**）

**温莎威尔 2 SKU 都负库存**（生椰拿铁 -7 / 原味 -8 = **典型收货差错** — 5/5 现场必须盘点）

**建议**：
- 留 1 个雀巢主销（无蔗糖）+ 撤 5 个不动销
- 撤所有温莎威尔（盘点后下架，2 SKU 都是负库存）
- 旧街场白咖啡 1 SKU 留（差异化 / 价位 ¥29.9 是利润带）
- 补 2-3 个本地化奶茶 / 三合一咖啡（¥3-8 价位 / 引流带）

### 3.6 G4-A-06 早餐功能补充（3 SKU 全死，重做）

**3 SKU 全部不动销**（九阳豆浆粉 / 维维豆奶粉 350g / 维维豆奶粉 680g）

**建议**：
- 这 3 SKU 与 G4-A-03 奶粉/豆奶粉重复 / 建议挪过去
- G4-A-06 改做 "早餐功能 + 联动"：方便速食大类（碗装小米粥 / 营养早餐快食）+ G2-A-05 早餐/吐司替代联动

## 5. 反幻觉自检（数据局限明示）

| 局限 | 影响 | 应对 |
|---|---|---|
| **30 天数据窗口** | 4 月不是冲调旺季（旺季是冬季 11-2 月），不动销率 81% 可能含季节因素 | 5/8 后回 vault 拉 1 月 / 11 月数据对比 |
| **L4 → 组位算法是启发式** | G4-A-02 营养粉/代餐归类 0 SKU 可能是因为"营养粉/代餐"在 L4 命名上不是这字面 | 5/5 现场对照 SKU 实物核 |
| **库存截止 2026-04-24** | 距今 11 天数据 / 5/4-5/14 之间动销变化未反映 | 5/5 现场补 5/4 库存快照（B.6） |
| **品牌列已知有脏数据** | 已发现"农夫山庄"被填成"金磨坊"等异常 | 调改方案不依赖品牌字段，改用 SKU 名 + 售价 + 销量交叉验证 |
| **门店实际分类 ≠ 学术分类** | 该 74 SKU 是按学术 L2="冲调早餐/冲饮" 筛选；但门店货架"G4-A 一面"实际可能含别的品类 | 5/5 现场用 §A.3.1 SKU 全摸排表核实"实际在架 SKU"vs"系统显示 G4-A SKU" |
