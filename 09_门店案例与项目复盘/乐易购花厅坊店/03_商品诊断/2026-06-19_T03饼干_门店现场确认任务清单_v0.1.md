# T03 饼干门店现场确认任务清单 v0.1

> 📋 带去门店逐项勾选 · 可打印 / 微信查看 · **不是淘汰清单**

## 1. 使用说明
- 本清单用于**六哥到花厅坊门店现场核查** T03 饼干。
- **不是淘汰清单、不是最终调改方案。**
- 现场**只做：确认、拍照、记录**。
- 最终动作要回到"**数据表现 + 库存状态 + 陈列状态 + 店长确认 + 需求点判断**"之后再定。

## 2. 本次现场要完成的 4 件事
- [ ] 核查 **4 个"有货无销"候选**商品；
- [ ] 核查 **10 个库存=0** 商品是 售罄 / 停购 / 替换 / 缺货；
- [ ] 请**店长确认 T03 饼干需求点**；
- [ ] 拍摄 **饼干货架 / 层板 / 排面 / 重点商品**照片。

## 3. 4 个"有货无销"候选商品核查表
> 不含条码。现场对照货架逐个找（凭"商品类型/价格/特征"识别）。

| 候选 | 商品类型/特征 | 当前问题 | 现场看什么 | 问店长什么 | 拍照 | 动作选项 | 现场结论 |
|---|---|---|---|---|---|---|---|
| **①** | 低价小饼干(约1元) / **负毛利·疑特价清仓** | 有货、>60天没卖、卖价<进价 | 见下"8看" | 见下"7问" | 见 §6 | 保留/替换/清仓/调陈列/观察/待补数 | |
| **②** | 造型/动物饼干(约9.9) / **儿童造型向** | 有货(约4)、>60天没卖 | 同上 | 同上+儿童需求 | 同上 | 同上 | |
| **③** | 中价饼干(约8.8) / **近售罄(只剩1)** | 只剩1个、70天没动 | 同上 | 同上 | 同上 | 同上 | |
| **④** | 威化饼干(约1元) / **负毛利·同质多** | 只剩1、91天没动 | 同上 | 同上+替代品 | 同上 | 同上 | |

**每个候选「现场 8 看」**（勾）：
- [ ] 是否还在货架上 - [ ] 是否在正常饼干区 - [ ] 是否被遮挡 - [ ] 排面是否过少
- [ ] 位置是否太低/太高/太角落 - [ ] 包装是否旧/临期/破损 - [ ] 是否有明显替代品 - [ ] 是否实际已准备清掉

**每个候选「问店长 7 问」**：
- [ ] 这个还准备继续卖吗？ - [ ] 是否已停购？ - [ ] 是不是特价/清仓？
- [ ] 有没有顾客点名买？ - [ ] 是否属于 儿童/健康老人/本地口味/节日送礼 需求？
- [ ] 有替代品吗？ - [ ] 店长建议：保留 / 替换 / 清仓 / 观察？

> ⚠️ 动作选项**不含"直接淘汰"**——现场只到"保留/替换/清仓/调陈列/观察/待补数"。

## 5. T03 饼干 需求点 店长确认表
> 目的：把当前 `confirmed_by_user`(六哥判断)的需求点，看能否升级为 `confirmed_by_store_manager`。**现场确认后才升级,本清单不替店长确认。**

| 需求点 | 当前判断 | 问店长 | 店长确认 | 证据 | 后续 |
|---|---|---|---|---|---|
| 1 健康/老人偏好 | confirmed_by_user | 周边老人会买健康型饼干吗?偏 低糖/粗粮/苏打/消化/无糖?有固定人群? | □确认存在 □不明显 □不确定 | | |
| 2 本地口味/老人 | confirmed_by_user | 老人偏好 广式/传统/本地口味?有顾客点名找? | □确认存在 □不明显 □不确定 | | |
| 3 儿童饼干 | 数据上不缺,待确认真实需求 | 真的有人买吗?家长买还是孩子挑?集中在某品牌/造型/小包装? | □确认存在 □不明显 □不确定 | | |
| 4 送礼/礼盒 | seasonality_pending | 节假日前有送礼需求?平时有人买礼盒?只在 春节/中秋/走亲 明显? | □节假日明显 □平时也有 □不明显 □不确定 | | |
| 5 中高端/品质型 | seasonality_pending | 顾客会买20元以上饼干?日常自用还是节日送礼?卖不动是没需求还是没合适商品? | □有需求 □节假日有 □不明显 □不确定 | | |
| 6 办公代餐/饱腹 | 待确认 | 周边上班人群会买饼干当早餐/代餐?哪些规格合适? | □确认存在 □不明显 □不确定 | | |
| 7 家庭分享 | 待确认 | 大包装/分享装好卖吗?家庭客会买? | □确认存在 □不明显 □不确定 | | |
| 8 休闲解馋 | 待确认 | 夹心/曲奇/威化 是主要消费?是否已太多/太重复? | □确认存在 □过多 □不确定 | | |
