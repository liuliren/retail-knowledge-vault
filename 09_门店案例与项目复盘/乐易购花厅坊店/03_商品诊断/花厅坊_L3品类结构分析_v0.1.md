## 2. 三大结构性问题

### 2.1 白酒独占库存 1/3——最大资金黑洞

| | 数值 |
|---|---|
| 白酒库存成本 | **¥166,613（占全店 33.3%）** |
| 白酒 SKU | 69（全部标记为滞销） |
| 白酒期间销售额 | ¥2,323（销转比 1.4%） |
| A/B/C 分布 | 6 / 11 / 52 |

- **诊断**：52 个 C 类白酒占用 80%+ 的白酒库存。这是一个 500–2000㎡ 社区生鲜超市，白酒应是配角而非主角。
- **参照**：正常社区超市白酒库存占比 ≤5%；此处 33% 意味着**每 3 元库存就有 1 元压在白酒**。
- **直接动作**：C 类白酒（52 个）启动清货定价；高价位进口烈酒类（存量大、转化低）优先处置。

## 5. 诊断建议（直接可执行，按优先级）

### P0 · 立即处置（资金回收）

1. **白酒 C 类（52 SKU）清货定价**：52 个 C 类白酒启动阶梯让利清货，目标回收 ¥80,000+ 库存成本；高价进口烈酒按到期/滞销天数优先出清。
2. **葡萄酒（16 SKU 全滞销）**：考虑整批促销处置或退货谈判；此品类在社区超市复购率极低。

### P1 · 本月完成（SKU 瘦身）

3. **餐具从 188 → 40 SKU**：保留高频消耗品（筷子/汤勺/碗碟主流规格），淘汰异形/高价/重复款。
4. **烹饪工具从 128 → 30 SKU**：同上，保留锅铲/菜板/剪刀等刚需，清退装饰性/低周转款。
5. **卫生巾从 76 → 25 SKU**：按品牌×型号重新选品，保留头部 3 品牌（护舒宝/苏菲/七度空间）主流型号。

### P2 · 本季调整（品类优化）

6. **常温乳品锁定主力**：14 SKU 中 3 个 A 类是核心，围绕这 3 个深化陈列和库存，其余 9 个进入自然淘汰。
7. **酒水总占比从 39% → 目标 15% 以下**：减少白酒/葡萄酒深度，啤酒保留 10–15 个主流流通款。
