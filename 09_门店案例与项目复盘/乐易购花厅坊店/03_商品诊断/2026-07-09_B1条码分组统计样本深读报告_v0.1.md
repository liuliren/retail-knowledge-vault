# B1条码分组统计样本深读报告 v0.1

> **结论先行**：判 **B档——只深读与已裁决动作相关的关键类别/窗口文件，不做118个文件全量深读**。样本证实B1具备SKU级"销售/退货/进价/库存"完整字段，能直接对已裁决的方便食品/巧克力汰换动作做数量级核对，且核对中发现一处销量数字与原决策单不一致，需六哥人工确认口径。

## 2. 字段能力

全部样本文件字段结构完全一致(17个有效列)：

`货号 / 品名 / 单位 / 销售数量 / 销售金额 / 零售价 / 售价金额 / 退货数量 / 退货金额 / 主供应商 / 赠送金额 / 数量小计 / 金额小计 / 金额占比 / 类别 / 类别名称 / 进价 / 库存`

**SKU级颗粒度，退货字段确认真实可用**(非全零占位)：

| 文件 | 总SKU行 | 退货数量>0的行数 |
|---|---:|---:|
| 2025年酒水 | 1405 | 109 |
| 90天方便食品 | 239 | 6 |
| 120天休闲 | 1164 | 11 |

## 3. 独有价值

- **多窗口覆盖**：同一品类有30天/90天/120天/2025全年四种窗口版本，可做时间序列对比(其他数据批次多为单一窗口)。
- **退货字段**：D1(库存进价表)、D3(日客单序列)均无退货明细，B1是全库唯一能算"退货率"的数据源。
- **进价+库存同列**：可直接算单SKU毛利率、库存周转，颗粒度比D1(全店口径)更细。

## 4. 与D1/D3数据的互补关系

- D1(7/3库存进价表)是**单点快照**(某一天的库存)，B1是**期间流量**(某窗口内的销售+退货)——两者结合可以做"进销存"闭环，D1单独做不到。
- D3(409天日客单序列)是**店级汇总**，B1是**SKU级明细**——D3回答"店整体好不好"，B1回答"哪个SKU好不好"。

## 5. 可验证的已裁决动作(重点发现)

抽样核对了方便食品90天文件中4个已裁决SKU：

| SKU | 原裁决(2026-06-26决策单·近3月) | B1本轮核对(90天窗口) | 一致性 |
|---|---|---|---|
| 康师傅韩式火鸡五连包(汰) | 销量4.0/动销3天 | 销量3件，库存5 | 基本一致 |
| 大汤王酸辣豚骨汤面(汰) | 销量4.0/动销3天 | **销量10件**，库存0 | **不一致**，B1显示销量是原决策单的2.5倍 |
| 家乡人板面酸汤牛肉味158g(汰) | 销量4.0/动销3天 | 销量3件，库存16 | 基本一致 |
| 康师傅好汤面老母鸡汤面(观察，未汰) | 销量4.0/动销4天 | **销量18件**，库存16 | **不一致**，B1显示销量是原决策单的4.5倍 |

**⚠️需六哥核对的问题**：原汰换决策单标注的"近3月销量"字段，与B1同期窗口(90天≈3个月)算出的销量数字对不上，尤其"康师傅好汤面老母鸡汤面"这个当时判"观察"(未汰)的SKU，B1显示销量其实不低(18件/90天)——**如果B1数字准确，这个"观察"判断本身是稳妥的，但也说明原决策单的销量字段来源口径需要核实**(可能是不同数据源/不同截止日期/不同门店口径导致的差异，不代表任何一方错误，需人工核对数据来源)。

## 6. 不能回答的问题

- 单纯用B1(供应商口径为主的销售汇总)**不能算真实毛利率**——"进价"字段的准确性此前已知有C级可信度限制(部分品类进价可能不含最新采购价)。
- B1没有陈列位置/货架层级字段，不能做陈列侧分析。
- B1是汇总表，不含每日逐笔明细，不能做"哪一天开始滞销"这类时点分析(D3的日客单序列更适合这类问题，但D3是店级非SKU级)。

## 7. 是否建议全量深读

**判定：B档**——只深读关键品类/关键文件，不做119个文件全量深读。

理由：
1. 118个文件本质是"同一份POS报表按30天/90天/120天/2025全年四种窗口 × 约30个品类"的排列组合，边际信息增量主要来自"品类"维度，不来自"窗口"维度(同品类不同窗口高度重叠，长窗口包含短窗口的大部分SKU)。
2. 与花厅坊已裁决/待验证的品类(方便食品/巧克力所在的休闲/白酒所在的酒水)相关的文件已抽样验证，具备直接可用性。
3. 尚未验证的品类(如粮油/清洁/个护等)与当前经营验证阶段的重点案例无直接关联，暂不需要投入。

## 8. 若全量深读，建议范围和优先级

若六哥认为需要更全面的SKU级验证，建议优先顺序：

| 优先级 | 对象 | 理由 |
|---|---|---|
| 高 | 90天_方便食品、120天_休闲(找巧克力子类)、2025年_酒水 | 直接对应已裁决/待核实的3个M-DEC案例(方便食品/巧克力/白酒) |
| 中 | 90天_食品_供应商口径 | 可能提供供应商维度的交叉视角 |
| 低 | 其余约25个品类文件(粮油/清洁/个护/纸品等) | 与当前经营验证案例无直接关联，暂不建议投入 |

**优先级排序原则：跟着已裁决动作走，不跟着"文件还没读过"走。**
