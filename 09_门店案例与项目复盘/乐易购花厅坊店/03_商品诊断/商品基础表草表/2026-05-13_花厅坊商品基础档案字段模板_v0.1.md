# 2026-05-13_花厅坊商品基础档案字段模板_v0.1

## 1. 文件定位

- 本文件是**花厅坊商品基础档案字段模板**。
- 用于后续：① 商品主数据整理 ② 动销分析 ③ 商品结构分析 ④ 淘汰候选 ⑤ 供应商分析 ⑥ BI 流程。
- **不直接清洗数据**。
- **不输出具体商品结论**。
- 上游：[[2026-05-13_花厅坊商品数据底座与动销分析模块设计稿_v0.1]]（v0.1 提交于 2026-05-13）。
- 下游：v0.5（W21）字段模板落 Excel + 数据清洗 SOP / v1.0（W22）跑通三表合并 BI 流程。

---

## 2. 字段设计原则

| # | 原则 | 含义 |
|---|---|---|
| 1 | **条码是主键** | 全 vault 商品数据唯一标识 / 不允许任何字段替代 |
| 2 | **商品名不能替代条码** | 同名不同条码 / 同条码不同名 / 用名 join 必错 |
| 3 | **系统字段和标准字段分开** | 系统大类 / 系统中类 / 系统小类 vs 标准 L1/L2/L3/L4 / 双轨保留 |
| 4 | **原始字段和清洗后字段分开** | 原始字段保留 audit-log / 清洗字段供分析 |
| 5 | **不确定字段标记待核验** | `_待核验` 后缀 / `is_pending_check` 字段标记 |
| 6 | **品类字段对接 V4.0 L1/L2/L3/L4** | [[2026-05-09_品类表治理决议_v1.0]] 权威 |
| 7 | **供应商字段必须保留** | 系统可导出（5/10 六哥 confirm / fact_layer: client_told）/ 5/12 实际拉数验证完整性（参 [[2026-05-10_花厅坊_供应商与进价字段口径confirm_v0.1]]）|
| 8 | **进价 / 售价 / 毛利率必须可回算** | 毛利率 = （售价 - 进价）/ 售价 / 自动校验 |
| 9 | **库存 / 动销 / 陈列位置应能后续关联** | 通过条码主键 join |
| 10 | **字段命名服务后续 BI** | snake_case + 中英对照 + Excel 列宽友好 |
