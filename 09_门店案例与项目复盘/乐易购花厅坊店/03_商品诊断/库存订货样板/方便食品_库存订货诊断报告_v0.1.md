# 花厅坊 · 方便食品 库存订货诊断报告 v0.1
> 🔒 client_confidential·已剔负库存/异常。进价不准→毛利/周转档仅参考。**本报告=[[社区超市库存订货优化包_主定义_v0.1]] 首个实战样板**,验证数量管理(周转/呆滞/ABC订货)。客户版(0术语脱敏)待六哥签发。

## 1. 一句话结论
**方便食品不缺货、反而普遍压货——52 款里 48 款卖得比备的慢(可销天数>90 天)、17 款已呆滞、压住约 5827 元库存成本。该做的不是补货,是清呆滞 + 收紧 C 类订货。**

## 2. 健康红黄绿表
| 维度 | 实测 | 灯 | 判读(数量管理判据) |
|---|---|---|---|
| **缺货/有货** | 当前库存=0 款 **0**;期间无断货 | 🟢 | 不缺货(甚至偏多)——有货率非本品类痛点 |
| **周转** | **48/52 款可销天数>90 天**;总积压成本≈5827 元 | 🔴 | 周转过慢(§6.6 周转率/DOS)——钱压在货上 |
| **呆滞** | **17 款**最近销售距今>30 天且有货,呆滞积压≈812 元 | 🔴 | 真呆滞(库存充足→非断货假滞销,SKU 防误杀已排除) |
| **ABC 结构** | A 11 / B 10 / **C 31** | 🟡 | 长尾 C 类 31 款偏多(呼应单品类诊断长尾肥大) |

## 3. SKU 结构与关键发现(52 款)
- **呆滞 TOP**:统一香菇炖鸡桶(库存25·售前35天)、陈村系列多款(米线/河粉/麻酱宽面·售前36-82天)、康师傅好汤面老母鸡(售前45天·DOS376)、优选香辣牛肉面(售前112天)——**集中在 C 类小众口味**,与单品类诊断"统一独大、长尾肥大"同源:长尾不仅卖得少,还在压库存。
- **慢周转普遍**:48 款 DOS>90 天,其中多款 94 天销售流水里无记录(强滞销信号)。
- **真滞销非假滞销**:这些款库存充足却卖不动→**不是断货致假滞销**(§数量管理 追加式订货防误杀),是真需求弱,可放心清。

## 4. 三步调改动作(挂 So-What)
**① 清 —— 17 款呆滞(C 类为主)分批清理**(降价促清/退供/汰换)。
- So-What:腾出约 800+ 元呆滞现金 + 货架;**先逐款现场确认非季节/引流角色**(过五角色闸),再清。

**② 收 —— C 类 31 款订货系数收紧(−25%)、A 类 11 款保订货**。
- So-What:**ABC 分级订货**(A+20%/B−10%/C−25%·非一刀切)——别再对慢动 C 类追加补货,把订货向 A 类倾斜;A 类不缺货是底线。

**③ 跟 —— 上周转看板,周跟踪 DOS/呆滞/积压额**。
- So-What:把"压货"从一次诊断变成可对账的周复盘,防反弹。

## 5. 数据边界(诚实声明)
- 库存快照=2026-05-08,销售流水=03-11~06-12,有时差(周转/呆滞为近似);
- **进价不准**→毛利/周转成本档仅参考;
- 无日度缺货数据→有货率用"库存=0"代理(本批 0 款,故缺货结论稳健,有货率维度待进销存补全);
- 剔除:负库存 0 行(方便食品干净)、空/异常 1 行;**生鲜负库存问题待花厅坊加工后单跑**。
