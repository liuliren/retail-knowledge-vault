## 4. 使用规则（11 条硬约束）

1. **条码为主键**：所有 sheet 之间通过 barcode join。
2. **商品名不能替代条码**：同名不同条码 / 同条码不同名 / 用名 join 必错。
3. **供应商字段系统可导出**（5/10 六哥 confirm / fact_layer: client_told）/ 但需 5/12 实际拉数验证完整性。
4. **进价字段系统可导出** / 但**生鲜类进价未入系统**（非生鲜诊断主线优先用系统进价）。
5. **不确定字段标记待核验**（is_pending_check / dq_* / 待核验原因 / 等）。
6. **本模板不直接生成淘汰结论**（09_商品动作字段表仅定义动作字段）。
7. **本模板不直接生成 SKU 调整清单**。
8. **照片进入系统必须通过 07_照片索引表**。
9. **外部照片如暂不重命名**：必须在 07_照片索引表 记录原始文件名 + 标准文件名 双轨。
10. **照片命名遵循**：字段内部用 `-` / 字段之间用 `_`（参 [[2026-05-06_门店陈列照片命名规范_v0.3]] v0.4 §7.2）。
11. **fact_layer 4 元字段**：observed / client_told / inferred / pending（参 CLAUDE.md §13.16-17）。

## 5. 数据导入前准备（5/12 启明协调拉数前清单）

| # | 准备项 | 验证方式 |
|---|---|---|
| 1 | 确认商品档案是否包含**条码 / 商品名 / 售价 / 进价 / 供应商 / 品类** | 字段名 + 字段填充率 ≥ 95% |
| 2 | 确认销售明细是否包含**日期 / 条码 / 销量 / 销售额 / 毛利额** | 字段名 + 时段覆盖（建议 90 天）|
| 3 | 确认库存表是否包含**条码 / 库存数量 / 库存金额** | 字段名 + 库存金额 = 进价 × 库存数量 校验 |
| 4 | 确认 90 天销售明细是否可导出 | POS 系统 / 启明协调 |
| 5 | 确认活动标记是否可导出 | POS 系统 / 启明协调 |
| 6 | 确认生鲜类是否进入本轮诊断 | 默认 ❌ 不进 / 生鲜类单独例外 |
| 7 | 确认导出文件来源与时段 | 在每个 sheet "数据来源" 字段记录 |

**5/12 拉数后字段完整性验证清单（参 [[2026-05-10_花厅坊_供应商与进价字段口径confirm_v0.1]] §4.2）**：

| # | 字段 | 验证方式 |
|---|---|---|
| 1 | 供应商（primary_supplier）| 拉数后填充率（≥ 95% OK / ≥ 70% 可用 / < 70% 异常）|
| 2 | 进价（cost）| 同上 / 排除生鲜类后计算 |
| 3 | 库存金额（stock_amount）| 校验 = 进价 × 库存数量 |
| 4 | 90 天销售明细（sales_amount_90d）| 时段覆盖 |
| 5 | 活动标记（is_promotion）| 字段是否存在 + 填充率 |

## 6. 数据导入后检查（10 项）

按 10_数据质量检查表 字段顺序执行：

| # | 检查项 | 字段 | 处置 |
|---|---|---|---|
| 1 | 条码缺失检查 | dq_missing_barcode | true → 散称 / 临时录入 / 异常清单 |
| 2 | 重复条码检查 | dq_duplicate_barcode | true → 同条码多记录 / 合并或单独 |
| 3 | 商品名冲突检查 | dq_name_conflict | true → 同条码不同名 / 优先用最新 |
| 4 | 进价缺失检查 | dq_cost_missing | true → **本次导出缺失**（非系统能力）/ 生鲜单独 |
| 5 | 供应商缺失检查 | dq_supplier_missing | true → **本次导出缺失**（非系统能力）|
| 6 | 负库存检查 | dq_negative_stock | true → 单独处理 / 不参与正常周转 |
| 7 | 毛利异常检查 | dq_margin_anomaly | 毛利率 > 100% / < -50% → 异常 |
| 8 | 售价异常检查 | dq_price_anomaly | 售价 = 0 / ≤ 进价 → 异常 |
| 9 | 品类缺失检查 | dq_category_missing | L1-L4 缺失 |
| 10 | 活动 / 大盘影响标记 | is_promotion / 客流字段 | 区分自然动销 vs 促销 |
