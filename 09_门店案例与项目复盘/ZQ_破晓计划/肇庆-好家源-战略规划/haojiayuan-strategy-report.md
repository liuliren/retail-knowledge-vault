**战略方向 · 门店策划 · 供应链研判**

> 编号 RPT-2026-07-HJY　|　状态：研讨草案 · 待实盘校准　|　出品：六哥零售 · 晟果新零售
> Slogan：科学零售,成果为证　|　承诺：每一个建议,都敢签字

## 阅读须知

本报告从"两个引擎(零售+批发)都不赚钱"的现状出发,先研判战略方向、再落到店型策略、再回到"当下开哪家店",最后论证供应链能否与零售打通赋能,形成一个可决策的闭环。

**本报告不替你编数字。** 凡涉及行业数值(毛利率区间、账期规律等)均为行业基准,用于推演逻辑,须以好家源实盘校准;凡涉及好家源自身经营的数字,一律标注【待验证】,不猜测填充。报告给的是**决策框架与止损线**,不是"一定可行"的结论——这正是"敢签字"的前提:算不实的结论,不签。

分析逻辑(闭环):**现状分析 → 研判定方向 → 店型策略 → 开哪家店 → 供应链研判 → 合力与商业模型。**

## 模块一 · 现状分析

### 1.1 两个引擎,而且两个都不赚钱

好家源实际上是两个引擎并行：

- **引擎一 · 批发/供应链公司**：已在肇庆经营多年,供货触达乡镇、多个县城,甚至周边城市的县域。状态是**打平/微利**——本质是商品批发,无差异化。
- **引擎二 · 零售门店(约 10 家)**：社区超市,分布在鼎湖及周边。状态是**持续亏损**,一直往里贴补。

**现状定性：** 一个只能打平的批发盘子,养着一个持续失血的零售盘子——交叉补贴正在流血,这就是迫切性的来源。

### 1.2 零售竞争场：已经拥挤,而且有人做得很好

结合实地调研与地图 POI：

- **强对手(基准)**：龙福购物超市,距坑口约 757 米,**日销约 7 万**,位置极好、经营在线。这是你要越过的门槛。
- **折扣势力**：赵一鸣零食,约 1 公里,2306 人去过。零食折扣业态,信号有二——本地存在折扣需求;零食品类已被它占据。
- **弱对手**：家乐福(富力·尚悦居店),258 米,评分 3.4、跑不动。它不是缺口。
- **底盘密度**：立洋、英姐、容容等士多/便利店高密度分布。

**更正一个此前的错误判断**：曾把弱店(家乐福)当成缺口,这是只看评分、不看经营实况的错误。真实基准是龙福 7 万。含义是——市场不是空的等你占,而是已被强对手卡位;**差异化不是加分项,是活下去的前提。**

### 1.3 批发盘子：有县域触达,却只能打平

第一性原理诊断为什么只能打平：

- **薄毛利定律**：通用品类分销毛利通常 5–12%。没有自有品牌、没有品类专精,就是搬箱子,只能打平。
- **路径与账期**：县域触达=路径长、密度低;下游要账期=你在垫钱。规模不带来利润,只带来营运资本压力。

**但这是一项资产。** 这套盘子一旦被强零售拉力激活(规模买手 + 自有品牌),就能从"打平"变成"成本武器"。关键判断：批发只能打平,不是因为盘子小,而是因为**没有被一个能创造需求拉力的零售端激活**——这正好指向"门店×供应链"的合力问题。

### 1.4 坑口点位画像(实地照片)

- **门店现状**：住宅底商大开间,物业自有、可切分。【切分性待现场核实】
- **幼儿园 200 米**：年轻家庭密度信号 + 早晚接送客流窗口。
- **交通/环境**：新区宽路、在建、住宅底商为主;客流需"养",非现成。【客流待实测】

综合判断：坑口是"家庭型、需培育、可切分物业"的点位。幼儿园+住宅=生鲜与家庭客群的天然锚;但新区=培育期,不能靠"开大店走量"赌现成客流——这会直接影响"坑口 vs 老城区"的选址判断。

### 1.5 现状的核心矛盾

好家源手里是**两个都没有护城河的商品生意**：一个无差异化的批发,一个无差异化的零售。它们各自为战,所以一个打平、一个亏损。真正缺的,是一个把两者拧成一台低成本机器的系统设计。所以下一步不是"再开一家店"或"再扩批发",而是先回答：**我们到底要成为哪一种生意?**

## 模块二 · 战略与方向(先定方向)

### 2.1 定方向 = 回答一道分叉题

"研判定方向"是整个闭环的枢纽。方向不是一句口号,而是一道二选一的分叉题——一旦选定,店型/选址/品类/盈利模型/供应链角色全都随之确定。**两条路不能既要又要。**

- **模型甲 · 精品社区生鲜店**：靠生鲜品质与体验差异化,较高毛利;供应链服务门店。
- **模型乙 · 区域硬折扣系统**：靠低价 + 高周转 + 自有品牌,供应链是成本引擎;回老城区高流量地段。

### 2.2 两条路逐维对比

| 维度 | 模型甲 · 精品社区生鲜店 | 模型乙 · 区域硬折扣系统 |
|---|---|---|
| 靠什么赚钱 | 生鲜品质 + 熟食现制毛利 + 到家/会员 | 高周转 × 供应链低成本价差 × 自有品牌 + 返利 |
| 生鲜角色 | 品质差异化(赚毛利) | 低价引流(亏本换流量) |
| 供应链角色 | 服务门店(附带) | 成本引擎(核心) |
| 选址 | 坑口/湖景(社区、自有物业) | 老城区核心繁华地段(高流量) |
| SKU 策略 | 宽而精,生鲜熟食深 | 窄而狠,精简收敛,高动销 |
| 你的直觉指向 | 部分 | 大部分(低价店/回老城区/生鲜引流/供应链赋能) |

### 2.3 模型甲详解 · 精品社区生鲜店

- **定位与店型**：社区生鲜精品店,生鲜为核心、熟食现制为特色、家庭刚需为基本盘;面积中等,避免面积负担;落在坑口/湖景社区点位。
- **靠什么赚钱**：生鲜品质溢价 + 熟食现制高毛利 + 到家/会员复购;不打价格战。
- **供应链角色**：后台支撑,不是利润中心——即"附带能力"。
- **风险**：在龙福的社区主场正面竞争;新区客流需培育;毛利结构偏单一。

### 2.4 模型乙详解 · 区域硬折扣系统

- **定位与店型**：硬折扣低价店,窄 SKU、高周转、自有品牌、精简费用;生鲜低价引流;落在老城区核心高流量地段。
- **靠什么赚钱**：不靠单品毛利,靠周转速度 × 供应链买价优势 × 自有品牌 + 规模返利;单品薄利,规模取胜。
- **供应链角色**：成本引擎(核心)。已有的县域批发盘子在这里被激活——统采压价→低价→放量→更大采购→更低价。
- **风险**：硬折扣是运营纪律 + 规模 + 自有品牌的游戏,执行极难;要真砍掉"大而全"的老习惯。

### 2.5 一个结构判断："门店第一 vs 供应链附带",其实就是在选甲/乙

这是给"门店第一"较真的一页,也是修正(修得更准,不是否定)：

- "门店第一"在**甲**下完全成立;但你补充的事实(已有县域批发 + 低价店直觉 + 生鲜引流)指向**乙**——而乙里,供应链不是"附带",是**被门店激活的成本引擎**。
- 更准的说法：**门店是锚和拉力**(没有强零售拉力,批发就只能是打平的商品批发,这正是现状);**供应链是被门店激活的引擎**。两者共生,但确实从"先做出一家能赚钱的店"开始。
- 结论：**"先门店后供应链"的次序你说得对;但"供应链是附带品"的定性要改成"共生引擎"。** 这不是文字游戏——它决定了要不要为门店专门设计供应链。

### 2.6 证据指向哪边

现有事实,大部分指向乙——但要用数据落锤：

- 已有触达县域的批发盘子 → 乙的成本引擎已有雏形;甲用不上这个资产。
- 你的直觉:先搞低价店、回老城区核心地段 → 硬折扣的选址与定价逻辑。
- 生鲜低价引流(水果蔬菜放低价) → 折扣店打法,不是精品店的毛利打法。
- 赵一鸣折扣店跑得动 → 本地折扣需求被验证。

**押乙之前必须验证：** ① 批发盘子是否有可量化的买价优势,幅度多少;② 自有品牌能力(有没有、能不能做);③ 老城区目标点位的租金/客流/竞争。

### 2.7 关于"整省"愿景

"整个广东省的好家源未来长什么样"是值得画的愿景,但可复制性是把单店模型 + 供应链飞轮跑通之后的自然结果,不是起手式。把"趋势/愿景成立"误读成"现在就能扩",是这类项目最贵的错误。次序：先(0–12 月)跑通单店模型 → 中(1–2 年)证明飞轮 → 后(2 年+)复制 → 整省。

## 模块三 · 现有业务调整思路

### 3.1 零售：逐店分诊(留/关/并)

10 家店不是"保或不保",是逐店分诊：

- **留**：赚钱店——保留,总结其赚钱本质(流量?品类?点位?),提炼成可复制要素。
- **观察/并**：打平店——能否靠新店型/新供应链改造转正;不能,则并入或降本。
- **关/转**：亏损店——逐店评估关停/整合。释放现金 + 管理带宽,反哺新模型。

这块"不是重点",但它是现金与注意力的来源。关掉拖累项,才有资源把新店做成。这是"合力"的第一步。【各档真实数量/盈亏待验证】

### 3.2 批发：从"打平的搬箱子"→"被零售激活的成本引擎"

1. **保留骨架**：县域触达、车队、关系是资产,不动。
2. **找差异化(脱离打平)**：三选一或组合——自有品牌 / 品类专精(有真实货源优势的品类)/ 用零售规模做集中采购,压低买价。
3. **与零售打通**：新店的采购走批发盘子 → 采购量上升 → 买价下降 → 门店价格更低/毛利更稳。这就是"赋能"。

这一路径的成立度,取决于方向选甲还是乙。选乙,它是主线;选甲,它是辅线。

### 3.3 斩断交叉补贴盲区：量化"亏损时钟"

批发只能打平、零售持续亏损——那么是谁在补贴零售?补贴速度多快?这个数决定改革的时间窗口,现在是个盲区,必须先点亮(配套 Excel 测算表)：

- 零售月度净亏损合计 =【待填】
- 批发月度净利(或净亏)=【待填】
- 集团月度净现金流 =【待填】
- 现金储备可支撑月数 =【待填】← 这是改革窗口

**为什么重要**：窗口短,就先止血(关店)再造血;窗口长,可以更从容地试新模型。次序由这个数定。

### 3.4 调整总原则：形成合力而非拖累

四个"共"，把两个盘子拧成一股绳：**共采**(集中采购,放大买手规模)、**共牌**(一套自有品牌,门店与批发下游共用)、**共仓**(共享仓/DC,坑口物业后场可复用面积)、**共数**(一套数据看板,选品补货去经验化)。拖累的根源是"各干各的";合力的机制是"共用一套底层能力"。无论甲乙都成立,差别只在权重。

## 模块四 · 门店策划(重点)

### 4.1 为什么"门店第一"

没有一家跑通的、能赚钱的门店,供应链就没有拉力、没有放量的出口,只能永远打平——这正是现状。所以门店是**锚**：它创造需求拉力、验证盈利模型、给供应链一个放量的理由。先把一家店做成,再谈复制、再谈飞轮。

但"门店第一"是起点次序,不等于"门店单打独斗"。这家店从第一天起,就要按"能被供应链赋能"的方式来设计(选址、SKU、价格带都与供应链耦合)。

### 4.2 店型设计九维度

| 维度 | 模型甲取向 | 模型乙取向 |
|---|---|---|
| 面积 | 中等(避免面积负担) | 紧凑高效,坪效优先 |
| SKU 数 | 宽而精 | 窄而狠(收敛) |
| 生鲜占比/角色 | 高 · 品质差异化 | 高 · 低价引流 |
| 熟食 | 现制特色(重) | 轻或不做【论证中】 |
| 价格带 | 中 | 低 |
| 毛利结构 | 生鲜+熟食毛利 | 薄利多销+自有品牌+返利 |
| 客群 | 社区家庭 | 价格敏感大众 |
| 选址 | 社区(坑口/湖景) | 老城区核心地段 |

### 4.3 选址策略：老城区核心 vs 鼎湖坑口

选址跟随模型,不能反过来：

- **鼎湖坑口/湖景**：自有物业、社区、需培育。适合模型甲的社区生鲜店;或作为乙的后场共享仓/DC + 首个低价社区试点。自有物业=改造自由、无租金压力。
- **老城区核心地段**：高流量、价格敏感、你的直觉。适合模型乙的硬折扣低价店:现成客流、价格敏感人群;租金高,但周转快可覆盖。

**判断**：若押乙,首店进老城区核心地段做低价店,坑口物业转为后场仓配/社区试点;若押甲,坑口做社区生鲜精品店。选址是方向的下游,不是独立决策。

### 4.4 品类战略：谁引流、谁赚钱、谁走量、谁不做

品类不是"摆什么货",是"用什么赚、用什么引、放弃什么"：

- **引流**：生鲜(果蔬)——放低价、做锐度,负责把人拉进店。大概率确认。
- **论证中**：熟食现制——做不做还在论证;甲下是利润与特色,乙下偏轻。【待定】
- **基本盘**：高频刚需(米面粮油、日百)——走量、走供应链买价。
- **放弃**：零食——让给赵一鸣,不正面打它的强项。

品类战略的核心问题不是"全不全",是"每个品类担什么角色"。这套角色分工,由方向(甲/乙)决定。

### 4.5 靠什么赚钱(最重要)

两套答案,先选一套：

- **模型甲 · 毛利型盈利**：利润来源=生鲜品质溢价 + 熟食现制高毛利;放大器=到家 + 会员复购;成败关键=做出别人抄不动的"鲜"和"现制"。
- **模型乙 · 周转型盈利**：利润来源=供应链低买价 × 高周转的价差累积;放大器=自有品牌毛利 + 规模返利;成败关键=SKU 收敛 + 成本纪律 + 自有品牌。

注意：乙的生鲜是**亏本引流**,钱不在生鲜上赚,在周转和后端赚。这与"生鲜低价引流"的直觉完全吻合——也是为什么证据偏向乙。【盈利模型待推敲落定】

### 4.6 低价店店型原型(若选乙)

- **SKU 精简收敛**：每个子类只留高动销少数 SKU,砍长尾(用 H-Score / IssueTag 逻辑)。
- **价格**：全场低价心智,生鲜天天低价引流,标品对标折扣。
- **自有品牌**：走量品类做自有品牌,毛利与差异化一起来。
- **费用**：极简运营,陈列简化、人效优先、堆头周转,压低单位费用。

难点诚实说：硬折扣是"纪律+规模+自有品牌"的游戏,不是把价格标低就行。它要求真正砍掉"大而全"的旧习惯——这是执行上最难的一关。

### 4.7 先开哪家

| 候选 | 优势 | 成立条件 |
|---|---|---|
| 坑口(自有) | 无租金、可切分、社区+幼儿园 | 选甲;或选乙做"后场仓+社区试点" |
| 湖景(自有) | 自有物业、社区 | 同坑口,取决于点位客流【待比较】 |
| 老城区核心 | 现成高流量、价格敏感客群 | 选乙;需拿到合适租金点位 |

**决策顺序**：先定甲/乙 → 定选址逻辑 → 用真实客流/租金/改造成本给 3 个候选打分 → 选首店。不要在没定方向前先定点位。

## 模块五 · 供应链研判

### 5.1 先把批发盘子拆成三个问题

- **① 内部赋能 · 供应链服务自己的门店**：统采压价、共享仓降损耗、数据看板。无争议该做,是"合力"的地基,也是乙的成本引擎。
- **② 现有外部批发升级 · 从打平到盈利**：盘子已存在。问题是能否靠自有品牌/品类专精/零售规模买手,把它从打平做到盈利。
- **③ 扩张外部分销 · 做更大的县域/城市**：设卡后置。只有在①②跑通、单位经济学转正后再放大,默认不激进扩张。

### 5.2 批发为什么只能打平,以及三条脱困路径

**诊断(第一性原理)**：通用品类商品批发=薄毛利、无护城河、账期垫资。规模不带来利润,只带来营运资本压力。要脱离打平,必须造出差异化。

- **路径 1 · 自有品牌**：在走量品类做自有品牌,把通用品的薄毛利换成品牌毛利。门店 + 批发下游共用。
- **路径 2 · 品类专精**：只在有真实货源/买价优势的品类(如米面粮油?)做深,别处不碰。【待验证优势】
- **路径 3 · 零售规模买手**：新店 + 留存店采购集中,放大采购量→压低买价→零售批发同时受益。这条最依赖"门店做成"。

### 5.3 打通赋能：成本飞轮

一句话：门店创造放量,放量压低买价,低买价换来低价/稳毛利,吸引更多客流,再放量。

- 这台飞轮里,**门店是启动杆**(所以"门店第一"是对的)。
- **供应链是飞轮本体**——没有它,门店压不下成本,只能拼毛利硬碰龙福。
- 现状之所以两个都不赚钱,正是因为**飞轮从没转起来**：批发没被门店放量激活,门店没被批发降本赋能。

### 5.4 扩张分销的门槛与止损

**放大前必须为真**：单位经济学(毛利率×客单 − 单均履约 − 坏账)> 0;账期总额 ≤ 可承受营运资本上限;路径密度达到成本可控阈值;①内部赋能已跑通(否则先内后外)。

**止损线**：坏账率 > 阈值【待定】→ 停;账期占用 > 现金储备阈值【待定】→ 停;单位经济学连续 3 月为负 → 停。

现有盘子已到县域,不代表"再扩更多县域"就对。先把已有盘子做到盈利,再谈扩张——否则是把打平的问题放大成亏损的问题。

## 模块六 · 合力与发展

### 6.1 合力 = 门店 × 供应链

合力不是"又开店又做批发",是让两者互为杠杆：

- **门店给供应链**：需求拉力 + 放量。门店把商品卖出去,给批发盘子一个放量的出口——采购量上去,买价才下得来。
- **供应链给门店**：成本武器 + 差异化。低买价 + 自有品牌,让门店能打低价而不亏、或稳毛利而不贵。

现状是拖累：两个盘子各自为战,一个打平养着一个亏损。目标是合力：一套采购、一套自有品牌、一套仓配、一套数据,拧成一台低成本机器。

### 6.2 发展次序(别跳步)

1. **跑通一家能赚钱的店**：验证盈利模型(甲或乙),这是地基。
2. **证明飞轮**：这家店的采购拉动批发降本、批发降本反哺门店价格——用数据证实闭环。
3. **复制(县域/城市)**：模型标准化后,沿已有批发触达的县域/城市复制门店。
4. **整省愿景**：当"门店×供应链"成为可复制的标准件,整省才是水到渠成。

### 6.3 三年路径图

| 阶段 | 零售 | 供应链 | 里程碑 |
|---|---|---|---|
| 0–6 月 | 止血(关/并弱店)+ 首店筹备 | 内部统采 + 共享仓 + 数据看板 | 亏损时钟点亮 · 方向定案 |
| 6–12 月 | 首店开业,跑通盈利模型 | 自有品牌试点 · 规模买手 | 一家店转正 · 飞轮初现 |
| 12–24 月 | 2–3 店复制 | 批发从打平→盈利 | 飞轮被数据证实 |
| 24 月+ | 沿县域/城市扩 | 供应链能力强化 | 模型标准化 · 迈向整省 |
