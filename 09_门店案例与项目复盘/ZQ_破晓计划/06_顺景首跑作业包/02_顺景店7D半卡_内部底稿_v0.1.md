# 顺景店 7D 半卡 · D1/D2/D4 远程预判（内部底稿 v0.1）

> ⚠️ **内部工具层底稿**：7D 是 N5 节点内部评分工具（KB-DX-FULLSTORE-001 §0.5），**客户版必须转四模块口径**（商品力 ← D1+D2；运营力 ← D3+D4+D7；经营底盘 ← D5；组织力 ← D6，**2026-07-07 六哥裁"全店经营总诊断"对客户改名"经营底盘"**），7D/D 编号永不出现在客户面前。
> ⚠️ 依据为 **draft v0.1 框架**（§9 六项裁决未定）；本卡仅 D1/D2/D4 三维可远程 POS 预判，D3/D6/D7 须现场，D5 受表五费用缺口压制。
> ⚠️ **面积特别提示**：顺景 3200㎡+外租 600 超出方法论 500–2000㎡ 锚定区间（KB-DX-FRAMEWORK-001 失效条件 §2），SKU 目标数/特化基准套用前须先校准，本卡评分暂沿用统一权重，**待六哥裁是否需拆两套基准**。
> ⚠️ **证据纪律**：POS 单源按 7D 语义 = **C 级**，以下全部得分为"待验证预判"，禁作独立结论写入客户报告；到店交叉核验后方可升 B。

## D1 商品力（品类结构）｜权重 20 · 预判得分 9/20（得分率 45% · C 级）

| 指标 | 观测值 | 判定 |
|---|---|---|
| D1-1 ABCZ 结构 | A 1,407 / B 2,012 / C 6,164 / Z 4,586；Z 占在册 **34%（三店最低占比，但绝对量三店最大）** | 🟡 占比最优但绝对量最重 |
| D1-5 空销率（90 天口径代理） | 动销率 66.5% → 零动销 33.5%（框架红线 <30%） | 🔴 仍超红线，但幅度小于富城（41.8%） |
| C 类尾部 | 90 天净额<20 元的 C 类 3,160 个（占 C 类 51%）；<50 元 4,282 个 | 🔴 长尾冗余绝对量三店最大 |
| 结构性异常 | 家居用品柜 1,563 动销 SKU 仅 7.54 万（三店最大非食长尾）/ 熟食柜≈0 | 🔴 非食长尾+熟食空缺 |
| D1-3 价格带完整度 | 店级 P25≈4~4.5 / 中位 8~9 / P75 15~16.5（三店同构低价民生盘） | 🟡 柜组级三档待拆 |
| D1-4 CR5 品牌集中度 | 未算（需品牌字段清洗） | ⏸ pending |

**证据**：POS 单源 C 级。**待到店补**：① A 类是否占黄金位；② 3200㎡ 大店五态宽度空缺是否比富城/金沙咀更严重（面积大理论应更宽）；③ 家居用品柜 1,563 SKU 现场排面占比核实；④ 94.8% 匹配缺口成因优先核（散称码/外租区）——**此项不核实，D1 全部计数可能失真**。

## D2 生鲜运营｜权重 25 · 预判得分 16/25（得分率 64% · C 级）

| 指标 | 观测值 | 判定 |
|---|---|---|
| D2-1 生鲜占比 f | ≈57.3%（8 生鲜柜口径 405.2 万/706.8 万，inferred） | ✅ 三店最高，超稳固档 |
| 生鲜头柜 | 猪肉 20.4% 毛利（全店第一大柜）+ 水果 12.5% + 叶菜 11.2% | 🟡 叶菜反低于水果，与富城相反，疑口径/损耗差异 |
| 散装柜毛利 | **9.7%（三店最低）** | 🔴 定价/损耗疑点，优先查 |
| 大湾区特化 | **熟食柜≈0（2 个动销 SKU）**；3200㎡ 大店客流承载力更强，缺位机会成本更高 | 🔴 特化缺位候选，优先级高于富城 |
| D2-2 日清 / D2-4 损耗 | 无台账数据 | ⏸ 全靠现场 |

**f 校准提示（§8 公式）**：f≈57.3% 时 D2_final≈min(45, 25×0.573/0.42)≈34.1（触顶）、D1_final≈20×0.435/0.665≈13.1——**校准后 D2 权重顶格、D1 降**，顺景生鲜话语权最强，但也意味着散装/水果毛利偏低的绝对影响被放大。
**待到店补**：肉类日清机制、叶菜品种数≥10、猪肉三档、清远鸡必备、散装称重与定价现场核实、熟食是否外租/联营。

## D4 客流人效｜权重 15 · 预判得分 10/15（得分率 67% · C 级）

| 指标 | 观测值 | 判定 |
|---|---|---|
| D4-1/2 增长归因 | 客流 **+71.5%（三店最强）** / 客单价 **−4.4%**（三店居中）/ 日均销售 +63.9% | 🟡 客数驱动、客单漏水幅度小于富城，但绝对客流量最大，漏水总额需核算 |
| 客单结构 | 客单 29.50 元（三店最高档）；件数×件均价拆分未做 | ⏸ 需小票行级 |
| D4-3 时段分布 | 三店共性=早市型；顺景 9 点占全天 15%（三店中唯一有单点数据） | 🟡 高峰承接能力需现场核 |
| D4-4 时段人效 | 50 人 · ≈1,480 元/人/日（三店最高） | ⏸ 需排班表 |

**证据**：POS 单源 C 级；客流为 POS 客单数代理（**非真实进店人次**）。**待到店补**：① 早市 8-10 点峰值收银排队目测（50 人排班是否够）；② 外租 600㎡ 是否计入客流/客单口径（可能拉高分母）；③ 客单 −4.4% 的现场归因；④ 老板/店长数据意识。
