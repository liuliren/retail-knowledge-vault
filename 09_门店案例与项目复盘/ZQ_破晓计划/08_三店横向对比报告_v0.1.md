# 好家源三店 · 横向对比报告 v0.1（2026-07-07 · draft）

> ⚠️ **证据纪律**：本报告数字口径全部为 POS 单源汇总（`01_清洗输出/` 标准件），按 7D 框架语义为 **C 级**（数据单项，未经现场交叉验证），任何结论均为"待验证预判"，不得作为客户最终交付定论；须经六哥首访三店后升 B/A 级方可对外定论。
> 正文 0 条码 / 0 进价裸值 / 0 供应商真名，数字全部汇总口径。

## 一、结论先行：三店画像与撬动性排序

- **顺景**——"体量担当、毛利担当反面"：净销 706.8 万（占三店 69.6%）、流量最强（+71.5%），但毛利率三店最低（13.4%）、面积超方法论锚定区间、条码匹配率三店唯一跌破防呆线（94.8%）——**数字可信度本身待修**。
- **富城**——"最弱二店"：净销 225.4 万（22.2%），动销率三店最低（58.2%）、Z 类占比三店最高（42%）、客单价降幅三店最深（−8.3%）——三个"三店最差"集中在一店，是最典型的"漏水样板"。
- **金沙咀**——"小而美"：净销仅 83.2 万（8.2%），却是三店毛利率最高（18.1%）、客单降幅最浅（−1.9%）、坪效最高（≈29.0 元/㎡/日）的店——体量最小但健康度最高，是三店中唯一值得"反向萃取经验"而非"优先急救"的店。

**撬动性排序（先打哪家 / 学哪家）**：

1. **先打富城**——问题最集中、面积（1000㎡）唯一落在方法论 500–2000㎡ 锚定区间内，诊断基准可直接套用，不需先解决顺景的口径信任问题，六哥 C2 已裁定切入，**单店样板性价比最高**。
2. **同步核顺景的数据信任门**——94.8% 条码匹配率缺口不解决，顺景任何 Z 类/ABCZ 数字都不能进入最终裁决；因体量占三店近七成，一旦核实清楚，顺景是撬动金额最大的店，但**门禁在前，动作在后**。
3. **学金沙咀**——客单价漏水控制最好（−1.9%）、生鲜毛利结构三店最健康，其连带承接与生鲜定价打法是本报告要萃取并回输富城/顺景的"小而美"因子（见 §三②）。
4. **金沙咀本身暂缓单独立项**——300㎡ 长尾候选池绝对规模三店最小（C 类净额仅 10.2 万），撬动性有限，汰换动作可后置或与老板店内巡检合并。

## 三、差异化诊断

### ① 顺景——最大但毛利最低+条码匹配率跌破防呆线

- **散装柜毛利倒挂三店最深**：66.6万净销，毛利仅9.7%（富城/金沙咀散装柜均14.0%）——疑称重/定价/记账问题，**优先怀疑散称码/外租区商品混入库存表口径**，与94.8%匹配缺口高度相关（498个未匹配条码疑似散称码或外租区商品）。So-What：先核实这498个条码去向，再决定顺景的Z类/ABCZ数字能否用于最终裁决。
- **面积超出方法论锚定区间**（3200㎡+外租600 vs 500–2000㎡）：SKU目标数、特化基准套用前须六哥裁是否拆两套诊断基准（KB-DX-FRAMEWORK-001失效条件§2）。So-What：诊断卡填写前先过"是否需要独立基准"这道门，不可直接套用富城口径的权重公式。
- **熟食柜近乎空白**（仅2个动销SKU，净销≈0）：3200㎡大店客流承载力最强，缺位机会成本高于富城——若确认非外租/联营遗漏，是三店中优先级最高的品类补位候选。

### ② 富城——客流涨客单跌，接客能力弱

- **客单价降幅三店最深（−8.3%）**，同时动销率三店最低（58.2%）、Z类占比三店最高（42%）——三个"三店最差"叠加在一店，说明富城不是单点问题，而是"档案含水量+连带承接"复合型漏水。
- **休闲小食柜431个动销SKU仅卖2.6万（占比1.2%）**——连带区宽而不产出，与客单价−8.3%直接互证。So-What：走店时优先核休食柜排面陈列与动线位置，而非急着砍SKU。
- **家居用品柜647个在册SKU（全店最多）仅卖2.4万，C类275个100%全部90天净额<20元**——非食长尾重灾区，缩排面第一候选。

### ③ 金沙咀——体量最小但毛利最高动销最健康，萃取"小而美"成功因子

- **客单价降幅三店最浅（−1.9%）**：小店品类宽度窄、动线短，顾客更容易被引导到连带位——这是三店中唯一"反向学习"而非"优先救火"的样板点。
- **生鲜毛利结构三店最健康**：猪肉毛利29.7%（三店最高）、叶菜23.1%（三店最高）、水果17.5%（三店唯一健康，顺景/富城水果毛利均仅12.5%）。So-What：现场须核实这是"定价策略"还是"成本更低"，若是定价打法，可直接萃取话术/陈列动作输出给富城/顺景的水果柜。
- **坪效三店最高（≈29.0元/㎡/日）**：面积小、周转集中，是三店中"每平米产出效率"的标杆，可作为富城/顺景排面精简后的对照基准。

## 四、分店打法建议（每店3条 · 挂So-What）

### 富城（首站样板）
1. **客单价漏水诊断先行**：核实休食/糖饼连带区陈列与动线错配 → So-What：走店时先看收银台前区与连带陈列位，再决定是否需要重新组合大单品，直接对照客单−8.3%这条最深降幅。
2. **Z类42%档案清理**：4,086个零动销条码，花厅坊211汰换打法可直接复制 → So-What：先清档案再谈品类调改，低风险见效快，作为三店"第一役"的实操载体。
3. **熟食/烧腊特化缺位核查**：现场确认是否外租/联营未入POS → So-What：若确属结构性空缺，1000㎡社区店理应补位大湾区烧腊特化，若已外租则数字口径需修正，两种情形动作完全不同，必须先分清。

### 顺景（体量最大·待解锁）
1. **条码匹配率94.8%缺口优先核实**：498个未匹配条码疑似散称码/外租区混入 → So-What：不核实此项，顺景任何Z类/ABCZ/汰换裁定都不可执行，这是一切后续动作的前置门。
2. **散装柜毛利倒挂立案**：9.7%毛利率为三店最低，且大概率与匹配缺口同源 → So-What：与①合并核查，现场核实称重记账流程，一次现场动作可能同时解决两个数据信任问题。
3. **家居用品柜长尾缩排面**：1,563个动销SKU仅卖7.54万（三店该问题绝对规模最大）→ So-What：待①②核实完成后，作为顺景第一批可执行的汰换候选，优先级低于口径修复但高于其他品类调整。

### 金沙咀（后置·经验输出源）
1. **萃取客单价稳定因子**：现场记录收银连带话术/陈列细节 → So-What：形成可复制清单，反向输出给富城/顺景的休食/糖饼连带区改造。
2. **烟酒柜占比核查（10.1%，高于水产+副食品+面点+酱料总和）**：确认是烟草证经营刚需还是品类结构失衡 → So-What：若为失衡，释放的排面可补生鲜或连带区宽度。
3. **汰换动作后置合并处理**：C类净额仅10.2万，撬动性有限 → So-What：不单独立项，与老板店内例行巡检合并执行，节省专项诊断资源投向富城/顺景。

## 五、三店共性病灶（跨店对比才能看见的规律）

> 这是单店诊断看不到的价值：以下问题在三店中重复出现，说明它们大概率是好家源**连锁层面的系统性问题**，而非个别门店的管理失误——诊断优先级应高于任何单店局部问题。

1. **客单价三店齐降（−1.9%~−8.3%），且降幅与店面规模无单调关系**：三店增长全部由客流驱动，客单价无一例外下滑。降幅从小到大为金沙咀(−1.9%)<顺景(−4.4%)<富城(−8.3%)——与净销规模、动销率均无线性关系,说明客单漏水**不是"店大了自然稀释"，而是连带区管理能力的差异**。So-What：这是全连锁诊断的第一优先级，建议以金沙咀的连带承接经验作为标杆，逐店复盘为什么规模居中的富城反而漏水最深。
2. **水果柜毛利在大店系统性偏低（顺景/富城均12.5%，反低于叶菜），仅金沙咀健康（17.5%）**：两个大店同一品类同一异常值，排除巧合，指向水果柜的**定价或损耗记账口径存在连锁性共性问题**（而非单店偶发）。So-What：三店一起核水果柜定价流程与损耗台账，很可能是同一套操作习惯在不同店重复出现，修复一次可复制到全部门店。
3. **家居用品柜（非食长尾）在三店均是C类全尾部重灾区**：富城647个在册（275个C类100%尾部）、顺景1,563个（949个C尾）、金沙咀204个（84个C尾）——绝对规模随店面积单调递增，说明这是**连锁统一选品/铺货逻辑的系统性冗余**，而非某店个别陈列问题。So-What：建议不按店逐个诊断，而是先在总部/供应链层面核这条非食长尾的选品标准，一次性调整三店铺货基准，比逐店汰换更省力。
4. **熟食/烧腊柜在顺景、富城均近乎空白（≈0销售），金沙咀结构性未设**：两个大店"有柜组但不产出"性质相同，与金沙咀"从未设置"不同——指向连锁**熟食品类战略缺位**（可能是统一未布局烧腊/熟食供应链，而非各店独立决策）。So-What：这是唯一值得从"连锁战略"层面（而非单店调改）讨论的品类空白，若确认非外租/联营遗漏，建议作为好家源全案的一个独立品类拓展议题上报。
5. **Z类占在册比例三店均落34–42%区间，无一店低于三成**：万级僵尸档案不是个别店的疏于清理，而是**连锁档案维护流程的系统性缺口**（如条码变更未清、季节品未定期清档）。So-What：建议设计一套连锁统一的档案清理SOP（而非三次独立执行的汰换项目），一次流程修复可持续压低三店Z类增速。

## 七、建议诊断/改造优先级排期

1. **第一站：富城**（本周内）——面积唯一落方法论锚定区间，问题最集中（三个"三店最差"叠加），诊断基准可直接套用，无需先解决数据信任问题，六哥C2已裁定切入，走店验证D1/D2/D4预判并现场采集D3/D6/D7。
2. **并行/第二站：顺景条码口径核查**（可与富城走店同步或紧随其后）——94.8%匹配缺口是"数据能否使用"的前置门，不涉及深度走店，可先由后台快速核实498个条码去向，为后续顺景深度诊断清障；若核实结果良好，顺景因体量最大（占三店69.6%）应成为第二个深度诊断站。
3. **第三站/后置：金沙咀**——健康度最高、问题绝对规模最小，此次走店的核心任务不是"找问题"而是"萃取客单价与生鲜定价的成功因子"，可与富城/顺景走店穿插进行（例如富城/顺景现场诊断间隙顺访金沙咀半日），不必单独排专项档期。
