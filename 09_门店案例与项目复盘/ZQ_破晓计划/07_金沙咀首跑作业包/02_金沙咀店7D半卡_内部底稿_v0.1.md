# 金沙咀店 7D 半卡 · D1/D2/D4 远程预判（内部底稿 v0.1）

> ⚠️ **内部工具层底稿**：7D 是 N5 节点内部评分工具（KB-DX-FULLSTORE-001 §0.5），**客户版必须转四模块口径**（商品力 ← D1+D2；运营力 ← D3+D4+D7；经营底盘 ← D5；组织力 ← D6，**2026-07-07 六哥裁"全店经营总诊断"对客户改名"经营底盘"**），7D/D 编号永不出现在客户面前。
> ⚠️ 依据为 **draft v0.1 框架**（§9 六项裁决未定）；本卡仅 D1/D2/D4 三维可远程 POS 预判，D3/D6/D7 须现场，D5 受表五费用缺口压制。
> ⚠️ **面积特别提示**：金沙咀 300㎡ 低于方法论 500–2000㎡ 锚定区间下限（与顺景超上限对称），SKU 目标数/特化基准套用前须先校准，**部分品类（熟食/内衣袜）结构性不存在，不宜按"缺位"逻辑评分**。
> ⚠️ **证据纪律**：POS 单源按 7D 语义 = **C 级**，以下全部得分为"待验证预判"，禁作独立结论写入客户报告；到店交叉核验后方可升 B。

## D1 商品力（品类结构）｜权重 20 · 预判得分 10/20（得分率 50% · C 级）

| 指标 | 观测值 | 判定 |
|---|---|---|
| D1-1 ABCZ 结构 | A 667 / B 685 / C 1,497 / Z 1,497；Z 占在册 35%（居中） | 🟡 与顺景/富城同量级，绝对量三店最小 |
| D1-5 空销率（90 天口径代理） | 动销率 64.7% → 零动销 35.3%（框架红线 <30%） | 🔴 超红线，但幅度介于顺景（33.5%）与富城（41.8%）之间 |
| C 类尾部 | 90 天净额<20 元的 C 类 718 个（占 C 类 48%）；<50 元 1,085 个 | 🔴 长尾比例居三店中位 |
| 结构性异常 | 家居用品柜 204 动销 SKU 仅 0.58 万；熟食/内衣袜柜结构性不存在（非缺位） | 🟡 非食长尾绝对规模小，判读需区分"不该有"vs"该有未有" |
| D1-3 价格带完整度 | 店级 P25≈4~4.5 / 中位 8~9 / P75 15~16.5（三店同构低价民生盘） | 🟡 柜组级三档待拆 |
| D1-4 CR5 品牌集中度 | 未算（需品牌字段清洗） | ⏸ pending |

**证据**：POS 单源 C 级。**待到店补**：① 300㎡ 店的 A 类是否占黄金位（小店动线短，理论上更容易做到）；② 五态宽度空缺盘点是否受限于面积而非管理问题；③ 家居用品柜排面占比是否已是最小可行宽度；④ Z 类 35% 现场核因。

## D2 生鲜运营｜权重 25 · 预判得分 18/25（得分率 72% · C 级）

| 指标 | 观测值 | 判定 |
|---|---|---|
| D2-1 生鲜占比 f | ≈51.4%（6 生鲜柜口径 42.8 万/83.2 万，inferred） | ✅ 三店最低但仍稳固档（≥40%） |
| 生鲜头柜 | 猪肉 29.7% 毛利（三店最高）+ 叶菜 23.1%（三店最高）+ 水果 17.5%（三店唯一健康） | ✅ 三店中生鲜毛利结构最健康 |
| 散装柜毛利 | 14.0%，与富城持平，优于顺景（9.7%） | ✅ 健康 |
| 大湾区特化 | 熟食柜无记录（结构性不存在，非"缺位"）；土猪柜仅 2 SKU 0 销售、三鸟柜 0.85 万偏弱 | 🟡 特化候选价值低于富城/顺景（体量太小，投入产出比需先算） |
| D2-2 日清 / D2-4 损耗 | 无台账数据 | ⏸ 全靠现场 |

**f 校准提示（§8 公式）**：f≈51.4% 时 D2_final≈min(45, 25×0.514/0.42)≈30.6（触顶）、D1_final≈20×0.486/0.647≈15.0——校准后 D2 权重顶格，且金沙咀生鲜毛利结构三店中最健康，D2 实际表现应优于其他两店。
**待到店补**：肉类日清机制、叶菜品种数、猪肉/水果毛利健康背后是否为定价策略（而非成本更低）、烟酒占比偏高是否挤占生鲜陈列空间。

## D4 客流人效｜权重 15 · 预判得分 11/15（得分率 73% · C 级）

| 指标 | 观测值 | 判定 |
|---|---|---|
| D4-1/2 增长归因 | 客流 +20.3%（三店最弱）/ 客单价 **−1.9%（三店最浅）**/ 日均销售 +18.0% | ✅ 增长质量三店最好，客单漏水最少 |
| 客单结构 | 客单 19.09 元（三店最低档，符合小店客单结构）；件数×件均价拆分未做 | ⏸ 需小票行级 |
