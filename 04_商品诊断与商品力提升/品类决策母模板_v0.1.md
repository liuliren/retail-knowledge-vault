# 品类决策母模板 v0.1

> **这是什么**:把单品类诊断升格为**可复用母版**——后续所有 SKU/ABC/陈列/汰换决策"填这张母模板"即可(系统自我复制起点)。是 [[单品类诊断]] skill + 方便食品诊断卡结构的升格,**非新结构**。
> **90/10**:AI 出母模板框架 + 机器候选(90%);**每个品类的最终取舍 = 六哥 Z(红线·见 §0)**。

## §0 红线取舍槽(六哥 Z·不可代签)
每实例必须六哥拍三样,机器只出候选:① **净可汰款最终取舍**(机器给候选·过五角色闸·六哥定砍哪些)② **补哪些锚点品牌/SKU** ③ 发店长前脱敏复核签。

## 母模板四锚点(每品类诊断按此填)
| 锚点 | 定什么 | 判据(方法论源) |
|---|---|---|
| **① SKU结构锚** | 头部/腰部/长尾占比·失真型(品牌遮蔽/价格带遮蔽/SKU冗余…) | [[复杂类分层计算引擎_v1.0]] · 多品类压测6类失真库 |
| **② ABC锚** | 小类内ABC分级(A扛销额·C长尾)·假A类还原 | [[数量管理]]§6.6G ABC · [[SKU角色层与目的品保护机制_v0.1]] 假A类/小类内ABC |
| **③ 陈列锚** | 数量×位置(陈列=解决商品在卖场的数量和位置)·黄金位给目的品 | [[数量管理]]§0 数量为先 · [[分层到陈列一致性模型_v1.0]] |
| **④ 汰换规则锚** | 净可汰判据 + 防误杀闸 | 动销差+慢周转+呆滞+非保护角色→候选;**五角色闸(引流/利润/形象/补充/季节)+假滞销还原(追加式订货防误杀)→豁免** |

## 复制流程(后续任意品类)
1. 跑 [[单品类诊断]] skill(脱敏→指标→增强SKU表)→ 自动填①②③ + ④的候选;
2. 六哥过 §0 三拍(取舍/锚点/签);
3. 出诊断卡(套方便食品 5 段结构)→ 调改 → 复盘回流(机制B/M-DEC)。
