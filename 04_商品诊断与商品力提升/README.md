# 04_商品诊断与商品力提升

## 目录作用

本目录沉淀 **SKU 诊断、单品管理和商品力提升**的标准方法，是"30天商品提效"类咨询项目的核心工具库。

商品诊断是从"哪些 SKU 有问题"到"如何处置"的完整判断链路。这里存放的是通用方法和规则，具体门店的商品诊断结论存放于 `09_门店案例与项目复盘` 对应门店的 `03_商品诊断` 子目录。

---

## 适合存放

- 商品诊断标准流程（数据准备 → 分类 → 判断 → 处置）
- SKU ABCD 分类规则（基于销售额、毛利额、动销率等维度）
- ABC-Z 模型应用规则
- 商品力评价模型
- 动销分析方法
- 毛利贡献分析方法
- 库存周转诊断方法
- 滞销品淘汰规则
- 新品引进规则
- 候选商品池管理规则
- 缺货补充清单模板
- SKU 保留清单模板 / 淘汰清单模板
- 商品诊断报告模板
- 商品诊断反幻觉规则

---

## 不适合存放

- 某家门店的具体 SKU 诊断数据（属于门店案例子目录）
- 品类规划层面的工作（属于 `05_品类管理与商品规划`）
- 陈列方面的诊断（属于 `06_陈列规划与DisplayMap`）
- 原始 Excel 数据（先进 `90_素材暂存与待整理`）
