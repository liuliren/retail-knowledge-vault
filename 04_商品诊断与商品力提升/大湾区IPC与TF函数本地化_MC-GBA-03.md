## 一、节点定义

**大湾区 IPC 与 P/M域收入校准公式（原称"TF函数"）本地化**是指:
在 MC-06(IPC 商品配置图)和 MC-08(TF 函数)通用框架基础上,
结合广州、佛山等大湾区城市的**真实收入水平、消费心理、价格敏感度**,
对 IPC 价格带划分、P 域/M 域分界点、SKU 数集中区域等关键参数进行本地化校准,
使 IPC 诊断的输出能真正反映大湾区社区超市目标客群的支付意愿。

## 二、为什么需要本地化(三个不可妥协的现实)

### 2.1 大湾区社区超市的"双低悖论"

```
表面看(对外宣传):
  广州/佛山是一线/新一线城市
  人均 GDP 排全国前列
  → 应该有较高的支付能力

实际上(社区超市真实客群):
  ① 核心客群是 45~65 岁本地家庭主妇/主夫
     - 退休工资 + 子女补贴,日可支配 50~100 元
  ② 次级客群是制造业外来务工人员
     - 月可支配 2000~4000 元,家庭采购预算紧
  ③ 双职工小家庭只占 20% 左右
     - 即便他们可支配高,也以"周末大采购"为主,工作日不进社区超市

→ 真实的"主力价格带"远低于一线城市平均水平
→ 用全国统一 TF 函数会高估 P 域上限,导致选品偏贵
```

### 2.2 大湾区消费心理的特殊价格敏感性

```
"讲究" × "精打细算" 的复合心理:

  对生鲜:
    - 新鲜度不妥协(可以多花 10~20%)
    - 但同等新鲜度下,差 5 毛钱也会换店

  对工业品:
    - 品牌忠诚度极高(海天/厨邦/亚太)
    - 但同品牌下,会精确到分对比包装规格的单价

  对调味品:
    - 蒸鱼豉油 / 蚝油是刚需,即便贵也要买
    - 但花生油 / 大米这种主粮,对价格极度敏感

→ IPC 需要按品类大类分别校准,不能一刀切
```

### 2.3 通用 TF 函数 vs 大湾区实际

```
渥美原版 TF 函数:
  P/M 分界点 = a × e^(b × 人均税后日薪)

通用值(全国平均):
  人均税后日薪 ≈ 200~300 元
  P/M 分界点 ≈ 30~50 元

大湾区社区超市实际:
  目标客群人均日可支配(非月薪) ≈ 60~120 元
  → 用日薪代入会得到 P 域上限 ≈ 15~25 元
  → 这个数字才是真实的大湾区社区超市 P/M 分界点
```

## 三、大湾区 IPC 价格带的本地化划分

### 3.1 标准价格带(适用于普通商品)

| 价格带 | 价格区间 | 顾客感知 | 占比建议 | SKU 数指引 |
|---|---|---|---|---|
| **P 域入门** | <8 元 | "便宜" | 25~35% | 货架最显眼,创造"有便宜货"印象 |
| **P 域主力** | 8~25 元 | "正常" | 40~50% | 主力价格带,A 类商品集中区 |
| **M 域入口** | 25~50 元 | "稍好" | 12~18% | 次要陈列,Preferred 品类区 |
| **M 域主力** | 50~100 元 | "好东西" | 5~8% | 礼盒区/进口区 |
| **高端区** | >100 元 | "高档" | 2~5% | 节庆礼盒/进口精品 |

```
理想 IPC 形态(大湾区社区超市):
  价格带 →   <8    8~25   25~50   50~100   >100
  SKU 数 →  ▓▓▓▓▓ ▓▓▓▓▓▓▓▓ ▓▓▓ ▓▓ ▓
            30%   45%      15%   7%   3%
  → 完美的"右下倾斜"
```

### 3.2 关键品类的特化价格带

#### 生鲜类(蔬菜/水果/肉禽)

```
P 域入门:< 5 元/份(零散叶菜、便宜根茎)
P 域主力:5~15 元/份  ← 核心 ★
M 域入口:15~30 元/份(进口水果、精品水产)
M 域以上:>30 元/份(礼盒/活鲜)

实战提醒:
  生鲜的"便宜对照物"必须在 < 5 元的入门段保证有 SKU
  否则顾客一进生鲜区就觉得"全店都贵"
```

#### 调味品类(海湾区文化重镇)

```
P 域入门:< 10 元(小瓶装、入门款)
P 域主力:10~25 元/瓶  ← 核心 ★
                     (海天/厨邦的 750ml 主力规格)
M 域入口:25~50 元/瓶(蒸鱼豉油大瓶、特色酱料)
M 域以上:>50 元/瓶(进口/有机/精品)

实战提醒:
  蒸鱼豉油是 Destination 品类,即便定价偏 M 域也必须配齐
  老抽/生抽是 Routine,严格守在 P 域主力价格带
```

#### 米面油类

```
P 域入门:< 30 元/包(2.5~5kg 入门米;1L 入门油)
P 域主力:30~80 元  ← 核心 ★
                  (5kg 主力米;5L 主力油)
M 域入口:80~150 元(10kg 优质米;5L 进口油)
M 域以上:>150 元(精品米/橄榄油)

实战提醒:
  米和油的 P 域必须坚守,这是社区超市的"价格记忆锚点"
  这两个品类一旦定价偏高,顾客会得出"全店贵"的总判断
```

#### 烘焙/熟食/腊味类

```
腊味(季节性):
  P 域入门:< 20 元/份(散装腊肠 100g 装)
  P 域主力:20~60 元  ← 核心 ★
  M 域入口:60~150 元(精装礼盒)
  M 域以上:>150 元(高档礼盒/送礼)

烧腊(全年):
  P 域:< 40 元/份(白切鸡 1/4 只;烧鸭 1/4 只)
  M 域:40~100 元(整只;特色品)

实战提醒:
  烧腊熟食区是社区超市的"差异化能力中心",
  允许 M 域占比稍高(20~25%),但必须有 P 域托底
```

## 四、大湾区 P/M域收入校准公式（原称"TF函数"，见文首术语澄清）

### 4.1 通用 TF 函数(MC-08 原版)

```
P 域上限 = a × e^(b × x)
其中:
  x = 当地人均税后日薪
  a, b = 经验参数(通用值 a=1.5, b=0.012)

例:
  上海 x = 350,P 域上限 ≈ 1.5 × e^4.2 ≈ 100 元
  北京 x = 320,P 域上限 ≈ 1.5 × e^3.84 ≈ 70 元
```

### 4.2 大湾区社区超市校准版

```
P 域上限(社区超市)= a' × e^(b' × x')

参数校准(基于乐易购周边实地调研经验):
  x' = 不是人均日薪,而是"目标客群日可支配额度"
       (退休金/家庭月度补贴/外来务工日工资 的中位数)
  a' = 0.8(社区超市更保守)
  b' = 0.015(消费心理对价格更敏感)

广州番禺社区超市估算:
  x' ≈ 80~100 元/天
  P 域上限 = 0.8 × e^(0.015 × 90) ≈ 0.8 × 3.86 ≈ 3.1
  
  ⚠️ 这个数字看起来不对——因为 TF 函数的输出单位不是"元",
     而是"P 域上限相对系数",需要乘以品类基准价后才是绝对价格
```

### 4.3 实操简化版(推荐使用)

> 严谨版的 TF 函数用起来麻烦,实战中用下面这个简化版即可:

```
大湾区社区超市 P/M 域分界经验值(2026 年):

  生鲜类:        P 域 ≤ 15 元/份,M 域 > 15 元
  调味品类:      P 域 ≤ 25 元/瓶,M 域 > 25 元
  米面油:        P 域 ≤ 80 元/件,M 域 > 80 元
  日用品:        P 域 ≤ 30 元/件,M 域 > 30 元
  零食饮料:      P 域 ≤ 15 元/件,M 域 > 15 元
  腊味熟食:      P 域 ≤ 60 元/份,M 域 > 60 元
  非食家品:      P 域 ≤ 50 元/件,M 域 > 50 元

⚠️ 这是 2026 年中段的经验值,
   每年通胀 3~5%,需要每年校准一次
```

## 五、IPC 诊断的大湾区特化判断标准

### 5.1 形状判断

```
✅ 健康(右下倾斜):
   < 8元   8~25元   25~50元   50~100元   >100元
    ▓▓▓▓▓  ▓▓▓▓▓▓▓  ▓▓▓     ▓▓        ▓
    
   特征:P 域 SKU 数最多,逐级下降
   含义:"这家店有便宜的也有好的,但便宜的多"

⚠️ 警告(右上倾斜):
   < 8元   8~25元   25~50元   50~100元   >100元
    ▓▓     ▓▓▓     ▓▓▓▓▓    ▓▓▓▓       ▓▓
    
   特征:M 域 SKU 数比 P 域还多
   含义:顾客感知"这家店贵",会失去价格敏感型顾客
   常见于:从大卖场转型社区店但没调整选品的门店

❌ 危险(双峰):
   < 8元   8~25元   25~50元   50~100元   >100元
    ▓▓▓▓   ▓▓      ▓▓▓▓     ▓▓        ▓
    
   特征:中间 8~25 元主力价格带反而最少
   含义:严重错位,客群定位混乱
   行动:立即重做商品配置
```

### 5.2 占比判断

```
合格线:
  □ P 域(< 25 元主力)SKU 数占比 ≥ 65%
  □ M 域 SKU 数占比 ≤ 25%
  □ 高端区(>100 元)SKU 数占比 ≤ 5%

胖东来对照值(教科书级):
  P 域占比 84.2%
  M 域占比 15.8%
  → 大湾区社区超市可以做到 P 域 70~80%

危险信号:
  ❌ P 域占比 < 50% → 严重偏离社区超市定位
  ❌ M 域占比 > 35% → 顾客流失风险
  ❌ 高端区 > 10% → 选品方向错了
```

### 5.3 周转天数(TR)判断

```
大湾区社区超市的 TR 红线(全店综合):
  生鲜:        ≤ 3 天
  调味品/米面油: ≤ 45 天
  日用品:       ≤ 60 天
  全店综合:     ≤ 30 天(优秀)/ ≤ 45 天(合格)

诊断:
  □ TR-sku(按 SKU 数计) > 60 天 → 商品结构有问题
  □ TR-stk(按金额计)   > 90 天 → M 域定价过高,资金占用过重
  
注:大湾区生鲜的"日次"特性使 TR-sku 受影响明显,
   做诊断时建议生鲜与非生鲜分开看
```

## 六、操作步骤:大湾区 IPC 现场诊断

```
Step 1:确定品类大类
  □ 是生鲜?调味品?米面油?根据品类调用 §3.2 的特化价格带

Step 2:导出全部 SKU 数据
  □ SKU 名称 / 规格 / 吊牌价(必须用吊牌价,不是促销价)/ 90 天销售额

Step 3:按 §3.1 / §3.2 的价格带分桶,统计 SKU 数和销售额
  □ 在 Excel 中做透视表

Step 4:画 IPC 图(横轴价格带,纵轴 SKU 数)
  □ 同时画"销售额 IPC"作为对照(看 SKU 数和销售额是否一致集中)

Step 5:用 §5.1 形状判断、§5.2 占比判断,诊断
  □ 形状是否右下倾斜?
  □ P 域占比是否 ≥ 65%?
  □ 是否有危险信号?

Step 6:用 §5.3 周转天数,判断商品结构可持续性
  □ TR-sku 是否 ≤ 45 天?
  □ TR-stk 是否 ≤ 90 天?

Step 7:输出诊断结论
  □ 形状评分(0~3 分)
  □ 占比评分(0~3 分)
  □ 周转评分(0~3 分)
  □ 综合评分(0~9 分)
  □ 行动建议
```

## 七、实战示例:乐易购休食区 IPC 大湾区校准

> 假数据用于演示(实际数据请以乐易购真实数据为准)

```
休食区现状(假设):
  SKU 总数:280 个
  
  价格带分布:
    < 8 元:     SKU 数 70 个(25%)
    8~15 元:   SKU 数 90 个(32%)
    15~25 元:  SKU 数 65 个(23%)
    25~50 元:  SKU 数 35 个(13%)
    >50 元:    SKU 数 20 个(7%)

按大湾区零食饮料价格带校准(P 域 ≤ 15 元):
  P 域:160 个(57%)  ⚠️ 偏低(应 ≥ 65%)
  M 域:100 个(36%)  ⚠️ 偏高(应 ≤ 25%)
  高端:20 个(7%)    ⚠️ 偏高(应 ≤ 5%)

形状判断:
  P 域占比偏低,M 域偏高
  → 接近"右上倾斜"边缘,有危险倾向

诊断结论:
  形状评分:1/3(临界右上)
  占比评分:1/3(三段都偏离)
  综合得分:2/6(警告)

行动建议:
  ① 删除高端区 20 个 SKU 的一半(节庆/试销品保留 10 个)
  ② 压缩 25~50 元 M 域 SKU 数(保留 25 个)
  ③ 增加 < 8 元入门段 SKU(目标 90 个)
  ④ 增加 8~15 元主力段 SKU(目标 110 个)
  → 调改后 P 域占比预计 ≥ 70%,形状回归右下倾斜
```

## 八、注意事项与局限性

### 8.1 数据敏感性

```
本节点的所有"经验值"是基于:
  ① DavidLiu 在乐易购的实地观察
  ② 广州番禺社区超市的同行数据交流
  ③ 2024~2026 年的物价水平

不适用于:
  □ 一线核心商圈(天河 CBD/珠江新城)的精品超市
  □ 城中村/城乡结合部的 100㎡ 以下小店
  □ 高端会员店(Costco/山姆)
  □ 折扣店(奥特乐/好特卖)
  
对于以上业态,需要重做 TF 函数校准
```

### 8.2 时效性

```
经验值会随通胀失效:
  - 2024~2026 年:本节点数值适用
  - 2027 年起:每年通胀 +3~5%,P 域上限相应上调
  
建议:每年 3 月做一次校准(春节后,清明前)
```

### 8.3 个体差异

```
即使同样在大湾区,不同社区的:
  - 居民结构(本地 vs 外来)
  - 收入水平(老城区 vs 新城区)
  - 家庭结构(三代同堂 vs 双职工)
都会影响真实的 P/M 域分界

→ 本节点是"大湾区平均值",
  实际门店要在此基础上结合自身商圈做 ±10~20% 微调
```
