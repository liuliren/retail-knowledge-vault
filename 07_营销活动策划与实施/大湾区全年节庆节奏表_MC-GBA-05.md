## 一、节点定义

**大湾区全年节庆节奏表**是指:
将 MC-GBA-01 中分散提到的大湾区时节信息,系统化整理为可执行的全年节奏表,
包括每个节点的**预热期、爆发期、清仓期、备货指引、品类组合**,
让社区超市采购和店长能按表落地,不再凭感觉上货。

---

## 二、为什么大湾区需要专门的节奏表

```
全国通用节奏表的问题:
  ① 把"中秋"当作大节,但大湾区中秋不如冬至
  ② 把"春节"作为最大节,但大湾区"冬至大过年"是文化共识
  ③ 忽略大湾区独有的"煲汤季"(秋季)和"腊味季"(初冬)
  ④ 端午节在大湾区的强度远超北方(粽子自制率高)

大湾区独有的节奏特征:
  ✓ 冬至(12月)是全年第一大节,商业意义 > 春节
  ✓ 腊味季(10月~2月)长达 4 个月,贯穿冬季
  ✓ 煲汤季(9月~11月)滋补品销售爆发
  ✓ 凉茶/消暑(6月~9月)是"日常季",而非节庆
  ✓ 清明扫墓的食材消费(烧猪/水果)是隐性大节

→ 必须有大湾区专版节奏表才能踩准节拍
```

### 卡 1:冬至(12 月 21~22 日前后)— 大湾区第一大节

```
═══════════════════════════════════════
  作战卡 1:冬至 — 全年最大销售高峰
═══════════════════════════════════════

【商业意义】
  大湾区"冬至大过年"是文化共识
  全年单日销售额 TOP 3 通常出现在冬至前 3 天
  烧腊/腊味/糯米食材的爆发节点
```

#### 备货预热(11 月初~12 月 15 日)

| 周次 | 任务 | 重点 SKU |
|---|---|---|
| 11/1~11/7 | 腊味预订,礼盒首批入仓 | 皇上皇腊肠 / 荣华腊肉 / 自制腊味区启动 |
| 11/8~11/30 | 礼盒陈列上架,腊味散装区扩面 | 礼盒(中端 60~150 元主力)/ 散装腊味 |
| 12/1~12/14 | 烧腊预订系统启动,糯米食材上货 | 整只烧鸭/白切鸡预订;糯米/腐皮/虾米 |
| 12/15~12/20 | 全店冬至主题陈列就位 | 端架 13 张全部冬至化 |

#### 爆发执行(12 月 21~22 日)

```
冬至前 3 天(12/19~12/21):
  □ 烧腊区:配置 3 倍日常人手
  □ 腊味:全品类不允许断货
  □ 糯米/糕点:增加 2 倍备货
  □ 餐桌主菜(海鲜/优质猪肉):备货 1.5 倍
  □ 礼盒:做最后一波收尾促销

冬至当天(12/22):
  □ 烧腊从早上 6 点开始切配,持续到下午 5 点
  □ 收银台应配 5 人以上
  □ 售罄品类立即换 SKU(不要让顾客看到空货架)
```

#### 清仓收尾(12 月 23~31 日)

```
冬至后立即:
  □ 烧腊预订转为现做现卖
  □ 冬至专属礼盒:80 折快速清仓 → 70 折 → 50 折(不超过 7 天)
  □ 端架立即转为春节预热(腊味礼盒延续,加春节糖果年货)

12 月 31 日复盘:
  □ 实际销售 vs 备货量(目标:售罄率 90~95%)
  □ 烧腊营业额 vs 去年同期
  □ 礼盒残货金额(目标 < 总进货 5%)
```

### 卡 2:春节(1~2 月)— 全国大节的大湾区特化

```
═══════════════════════════════════════
  作战卡 2:春节 — 礼盒+年货+腊味延续
═══════════════════════════════════════

【商业意义】
  大湾区春节 ≠ 全国春节
  - 礼盒销售强度高(送礼 + 拜年)
  - 年货囤购周期短(初三后销量急降)
  - 腊味销售延续(从冬至持续到正月)
```

#### 备货预热(12 月初~春节前 10 天)

```
12 月初:春节礼盒首批进货
  □ 礼盒主力价格带(60~200 元)备齐
  □ 高端礼盒(200~500 元)精选 5~8 款

12 月中:糖果/坚果/糕点上架
  □ 糖果:大湾区偏好"传统"(冰糖/麦芽糖)+"现代"(奶糖/巧克力)各占 50%
  □ 瓜子/花生:必备
  □ 糕点:广式年糕、马蹄糕、萝卜糕

春节前 10 天:进入冲刺期
  □ 海鲜/腊味/烧腊:进入备货高峰
  □ 春联/红包/挥春:配齐
```

#### 爆发执行(春节前 5 天~初三)

```
关键时间点:
  农历廿八(春节前 2 天):全年第二个销售高峰
    → 团圆饭食材 + 拜年礼盒最后一波
  
  农历廿九/除夕:
    → 烧腊井喷,年夜饭食材爆发
  
  正月初一~初三:
    → 销售断崖式下降(80% 商铺关门,但社区超市开)
    → 拜年礼盒补充购买为主
    → 此期间是"售罄清仓"的最佳窗口

售罄管理:
  - 海鲜/烧腊:除夕前必须清光
  - 糖果/坚果:初三后开始 9 折→8 折
  - 礼盒:初五后 7 折→5 折
```

#### 清仓收尾(初三~正月十五)

```
初五~十五:
  □ 春节残货积极清仓(损失永远比下一年再卖小)
  □ 元宵食材(汤圆/糯米粉)逐渐占据端架

农历元月十五(元宵):
  □ 春节销售完整结束
  □ 当天复盘整个春节档期
```

### 卡 3:清明(4 月 4~6 日)— 隐性大节

```
═══════════════════════════════════════
  作战卡 3:清明 — 祭祀+踏青+春季食材
═══════════════════════════════════════

【商业意义】
  大湾区清明祭祖文化浓厚,易被外地经营者低估
  - 烧猪是核心刚需
  - 鸡蛋(尤其鸽蛋)消费爆发
  - 水果按"偶数包装"出售
  - 香烛冥纸需求(部分门店经营)
```

#### 备货预热(3 月中~4 月初)

```
3 月中:清明用品上架准备
  □ 烧猪:与本地烧腊厂商提前预订
  □ 鸡蛋:增加进货量(目标 +50%)
  □ 水果:切换为"4 个/8 个"包装(忌单数)
  □ 艾草/糯米粉(青团原料)

4 月初:进入清明前 1 周
  □ 端架 5 张转为"清明专题"
  □ 香烛/冥纸上架(如门店经营)
  □ 烧猪预订系统开放
```

#### 爆发执行(清明节前 3 天)

```
关键时段:
  清明前 1~3 天:祭祖准备购物高峰
    → 烧猪取货(预订)/ 现购烧腊
    → 水果(必须双数)
    → 鸡蛋大量购买
    → 香烛/纸钱

  清明当天:扫墓 + 踏青食材
    → 户外野餐用品
    → 即食食品/便携饮料
```

#### 清仓收尾(清明后 3 天)

```
清明后:
  □ 香烛/纸钱:清明专属包装快速清仓
  □ 双数水果包装:转为单数包装重新陈列
  □ 端架立即换为"端午预热"
```

  □ 广式咸肉粽(主流)

#### 爆发执行(端午前 5 天)

```
关键时段:

#### 清仓收尾(端午后)

```
端午后 1 周:
  □ 粽叶等限定原料快速清仓(损失永远比浪费货架小)
  □ 礼盒残货:6 折快速处理
  □ 端架转为"凉茶/消暑"主题
```

```
═══════════════════════════════════════
  作战卡 6:腊味季 — 横跨 5 个月
═══════════════════════════════════════

【商业意义】
  大湾区腊味季是社区超市最长的销售旺季
  从 10 月启动 → 12 月冬至高峰 → 2 月元宵收尾
  涉及散装/包装/礼盒/自制 多种业态形式
```

#### 备货预热(9 月初~10 月初)

```
9 月初:腊味供应商对接
  □ 大品牌(皇上皇/荣华/长城)预订首批
  □ 自制腊味区供应商确定(本地作坊)
  □ 礼盒厂商对接

10 月初:首批上架
  □ 散装腊肠(广式甜味/咸味两种)
  □ 腊肉/腊鸭(整条卖,根据顾客需求斩件)

```
分阶段重点:

10 月中~11 月底:腊味散装/小包装高峰
  □ 主力价格带(20~60 元)是核心

12 月初~12 月中:礼盒高峰前夜
  □ 全店端架转为礼盒(13 张里 8~10 张)
  □ 礼盒主力(60~200 元)放最显眼位置
  □ 高端礼盒(200~500 元)精选位置

12 月 21~22 日:冬至(同卡 1)
  □ 烧腊+腊味双爆发

1 月:春节延续(同卡 2)

## 五、跨节点的关键提醒

### 5.1 大湾区节庆的"三不妥协"

```
① 时机不妥协
   节点早 2 天上货 = 占库存
   节点晚 1 天上货 = 错过高峰
   误差不允许超过 ±3 天

② 品类不妥协
   节点专属品类(腊味/烧猪/粽子)必须本地优先
   外地品牌即便便宜也难替代本地认知

③ 清仓不妥协
   节后积压 = 重大损耗
   宁可节前售罄缺货,也不要节后积压
```

### 5.2 节庆陈列的本地化原则

```
① 视觉冲击力优先
   节庆期间,顾客心智被各类信息淹没
   陈列必须让人"一眼看到"是节庆

② 场景化关联陈列
   不要按"品类"陈列,要按"用途"陈列
   例:粽子区 = 粽子 + 粽叶 + 糯米 + 咸蛋黄 + 五花肉 + 蘸料

③ 价格梯度清晰
   每个节庆都要有"入门款 + 主力款 + 高端款"
   入门款守 P 域,礼盒主力守 M 域,高端款做形象
```

### 5.3 节庆采购的纪律

```
□ 每个节庆都做"备货 vs 销售" Excel 跟踪
□ 每个节庆结束 1 周内复盘(售罄率 / 残货率 / 销售额)
□ 每个节庆复盘归档,3 年后形成自己的"节庆采购数据库"
□ 不要用单一供应商(节点期一旦断货就崩)
□ 不要为了"低价拿货"而牺牲品质(节庆品质感是关键)
```

```
