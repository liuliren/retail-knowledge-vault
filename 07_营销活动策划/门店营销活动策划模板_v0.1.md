# 门店营销活动策划单 v0.1（draft · 可直接复制填写）

> **这是什么**：一张策划一次门店营销活动的逐栏填写单。填完即是一份可执行的活动方案（S06 DoD-1）。
> **装配纪律**：本模板不定义任何方法。战术选型看 [[KB-METHOD-4TACTICS-001_四大战术动作总框架_v0.1]] 与 [[KB-INSTORE-MKT-001_场内营销体系_v0.1]]，活动组织看 [[KB-EVENT-ACTIVATION-001_活动组织与拉动_v0.1]]，指标口径查 [[KB-METRIC-DICT-001_经营指标字典_v0.2]]，交付流程按 [[S06_营销活动配套]]。
> **诊断为纲**：没有诊断短板挂钩的活动不立项（不为活动而活动）。第 1 栏填不出来 → 本单作废。

---

## 使用方法

复制下方「策划单正文」到新文件（命名：`活动策划单_[门店代号]_[活动名]_[YYYYMMDD].md`），逐栏填写。每栏含【填写指引】和【示例·虚构】——示例仅示范颗粒度，数字均为虚构，不作基准。

### 1. 活动目标（挂哪条诊断短板）★ 立项门槛

**填写指引**：写清本活动响应的是哪条诊断结论——来自 7D 诊断哪个维度（如 D4 客流客单）、品类状态机哪个🟡、或哪份诊断卡哪条发现。目标只允许挂一个北极星：ATV（客单价）或品类渗透率（口径见 [[KB-INSTORE-MKT-001_场内营销体系_v0.1]] §1）。注意：若短板是商品结构问题（价格带空缺/缺品），按 KB-INSTORE-MKT-001 命题5 应先调品，不该走活动。

**示例·虚构**：`响应 2026-06 诊断卡发现"休食区连带率低于全店基线"；北极星=品类渗透率（休食×生鲜同篮比例）；诊断出处：T03 诊断卡 §3。`

| 诊断短板出处 | 北极星（二选一） | 目标值（基线→目标） |
|---|---|---|
| | ☐ ATV ☐ 品类渗透率 | |

### 2. 活动主题

**填写指引**：一句顾客听得懂的话（广府语境优先，见 KB-INSTORE-MKT-001 框架3 关联标牌语言规则）；不写内部术语。主题须与档期节令呼应（查 [[大湾区社区生鲜节令营销日历_v0.2]]）。

**示例·虚构**：`「秋风起·煲靓汤」— 汤料一次买齐`

### 3. 档期

**填写指引**：起止日期 + 三阶段划分（预热/高峰/收尾，阶段动作规范见 KB-INSTORE-MKT-001 §6 接口2）。档期避开/借力节令峰，查节令日历的备货提前期倒推筹备启动日。

| 阶段 | 日期 | 关键动作 |
|---|---|---|
| 预热期（前3–7天） | | |
| 高峰期 | | |
| 收尾期（后3天） | | |

### 4. 选品逻辑（回指品类角色）

**填写指引**：每个活动商品标注其品类角色（口径见 [[KB-CAT-ROLE-5WAY-001_五分法品类角色分配方法_v0.1]]，大湾区特化见 [[KB-CAT-ROLE-GBA-001_大湾区品类角色特化_v0.1]]）与在本活动中的任务（引流/连带/利润/形象）。禁止把 Z 类滞销品当主推（清库存活动除外，须注明）。关联组合须满足篮分析阈值或标注为「待验证假设」（阈值见 KB-INSTORE-MKT-001 框架4）。

**示例·虚构**：

| 商品 | 品类角色 | 活动任务 | 选择理由 |
|---|---|---|---|
| 猪骨（示例） | 目的性/引流 | 引流品 | 煲汤季刚需，带客进店 |
| 陈皮/淮山/眉豆（示例） | 连带 | 连带品 | 与猪骨同篮组合（待本店篮数据验证） |

### 5. 价格与让利结构

**填写指引**：逐品写正常价→活动价→让利来源（自让/供应商承担/费用置换）。让利深度与增量利润逻辑参照 [[KB-RETAIL-PROMO-001_商品促销策略_v0.1]]；限时折扣纪律（截止时间锚点、折扣下限授权）见 KB-INSTORE-MKT-001 §8。**总让利额填入第 9 栏预算。**

**示例·虚构**：

| 商品 | 正常价 | 活动价 | 让利承担方 | 限时/限量条件 |
|---|---|---|---|---|
| 猪骨（示例） | 22.8元/斤 | 19.8元/斤 | 门店自让 | 每日限量·当日18点前 |

### 6. 陈列联动

**填写指引**：写清动哪些位（堆头/端架/关联副展台/黄金视线区调整），执行规则按 KB-INSTORE-MKT-001 框架3 VMD checklist（关联品不占主品陈列面超25%、黄金层只放A类/高毛利/主推）。附平面示意或货架编号。

**示例·虚构**：`入口堆头=猪骨+汤料组合；G2端架=活动POP+组合价签；主品货架面不动。`
