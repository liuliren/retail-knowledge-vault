# vault 商品库 字段定义 v1.0

> **single source of truth**：所有商品库 CSV 字段一律以本文件为准 / 05_/商品库索引仅引用 / 不重复定义。
>
> **Oracle ACID 纪律**：FK 完整 / 主键唯一 / 字段类型严格。

---

## §1. 标准 SKU 字段（A_core_sku.csv / B_category_skeleton.csv / C_client_specific/*.csv 共用）

### A. 主键 + 关联（4 字段）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 1 | `sku_id` | STRING PK | ✅ | vault 内部唯一标识（格式：`<layer>-<seq>` 如 `B-00001` / `A-00001` / `C-hcp-00001`）|
| 2 | `barcode` | STRING | ⏸ | 条码（13 位 EAN-13 或 8 位）/ 可空（如散装 / 现场加工）|
| 3 | `category_l4` | STRING FK→V4.0 | ✅ | V4.0 大湾区版 L4 编码（如 `10103` 调味品/酱油/生抽）|
| 4 | `data_source` | ENUM | ✅ | `off` / `v4_skeleton` / `client_<id>` / `gs1` / `manual` |

### B. 标准品名（5 字段 / Layer A 完整 / Layer B 简化）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 5 | `sku_name_standard` | STRING | ✅ | 标准品名（中文 / vault 锁定 / 如 "海天生抽 500ml 玻璃瓶"）|
| 6 | `brand` | STRING | ✅ | 品牌（如 "海天"）|
| 7 | `brand_alias_id` | STRING FK | ⏸ | FK → `_maintenance/brand_alias.csv`（如 `BA-001` 海天=HADAY）|
| 8 | `sub_brand` | STRING | ⏸ | 子品牌（如 "金标" / "金字"）|
| 9 | `product_line` | STRING | ⏸ | 产品线（如 "生抽" / "老抽" / "蒸鱼豉油"）|

### C. 规格 + 包装（5 字段）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 10 | `package_size` | STRING | ✅ | 规格（如 "500ml" / "1.5kg"）|
| 11 | `package_size_std` | DECIMAL | ✅ | 标准化规格数值（如 500.0 / 1500.0）|
| 12 | `package_unit` | STRING FK | ✅ | 单位（如 "ml" / "g" / "kg" / "L" / FK → `_maintenance/package_unit.csv`）|
| 13 | `package_type` | ENUM | ⏸ | 包装类型：`bottle_glass` / `bottle_plastic` / `bag` / `box` / `can` / `bulk` / `other` |
| 14 | `package_count` | INTEGER | ⏸ | 整箱数量（如 1 / 12 / 24）|

### D. 价格 + 经营（4 字段）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 15 | `suggested_retail_price` | DECIMAL | ⏸ | 建议零售价（参考 / 元）|
| 16 | `price_band` | ENUM | ⏸ | 价格带：`low` / `mid` / `high` / `premium` |
| 17 | `is_strategic` | BOOLEAN | ⏸ | 战略 SKU（品牌形象款 / 老板认可款）|
| 18 | `is_seasonal` | BOOLEAN | ⏸ | 季节性 SKU（如冷饮 / 火锅）|

### E. Layer C 专属（3 字段 / 仅 client_specific 文件用）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 19 | `client_id` | STRING | ⏸ | 客户标识（如 `hcp` 花厅坊 / `lyg` 乐易购 / `zq` 肇庆破晓）|
| 20 | `client_sku_name_raw` | STRING | ⏸ | 客户原始品名（未清洗 / 如 "海天 500"）|
| 21 | `client_internal_code` | STRING | ⏸ | 客户内部编码（如客户 POS 系统的内码）|

### F. 元数据（5 字段）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 22 | `fact_layer` | ENUM | ✅ | `observed` / `client_told` / `inferred`（与 vault §13.16-19 协同）|
| 23 | `created_date` | DATE | ✅ | 创建日期 |
| 24 | `last_updated_date` | DATE | ✅ | 最后更新日期 |
| 25 | `lifecycle_status` | ENUM | ✅ | `active` / `dormant` / `archived`（与 Memory §17.20+ 协同）|
| 26 | `embedding_id` | STRING FK | ⏸ | FK → `_embedding/`（W26+ 升级）|

## §2. brand_alias.csv 字段（_maintenance/）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 1 | `brand_alias_id` | STRING PK | ✅ | 主键（格式：`BA-NNN`）|
| 2 | `brand_canonical` | STRING | ✅ | 标准品牌名（中文 / 如 "海天"）|
| 3 | `aliases` | STRING | ✅ | 别名（pipe 分隔 / 如 `HADAY|hisha|海天味业|HADAY GROUP`）|
| 4 | `country_origin` | STRING | ⏸ | 原产地（中国 / 中国香港 / 美国 / 等）|
| 5 | `parent_company` | STRING | ⏸ | 母公司（如 "佛山市海天调味食品股份有限公司"）|

## §3. package_unit.csv 字段（_maintenance/）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 1 | `unit_canonical` | STRING PK | ✅ | 标准单位（如 "ml" / "g" / "kg" / "L"）|
| 2 | `aliases` | STRING | ✅ | 别名（pipe 分隔 / 如 `毫升|ML|cc|c.c.`）|
| 3 | `category` | ENUM | ✅ | `volume` 体积 / `weight` 重量 / `count` 数量 / `length` 长度 |
| 4 | `convert_to_base` | DECIMAL | ✅ | 转换为基础单位的系数（如 ml→ml=1.0 / L→ml=1000.0 / kg→g=1000.0）|

## §4. 类型规约（Oracle ACID）

```sql
-- 数据契约 / 严格类型
sku_id              VARCHAR(20)   PRIMARY KEY    -- 格式 <layer>-<id>-<seq>
barcode             VARCHAR(13)   NULLABLE       -- EAN-13 / 8
category_l4         VARCHAR(10)   FOREIGN KEY → V4.0
package_size_std    DECIMAL(10,3) NOT NULL       -- 标准化数值
package_unit        VARCHAR(10)   FOREIGN KEY → package_unit
fact_layer          ENUM('observed','client_told','inferred')
lifecycle_status    ENUM('active','dormant','archived')
created_date        DATE          NOT NULL
last_updated_date   DATE          NOT NULL DEFAULT CURRENT_DATE
```
