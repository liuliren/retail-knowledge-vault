# 13_数据分析与工具脚本

## 目录作用

本目录沉淀 **Excel 模板说明、Python 脚本、数据清洗规则、条码匹配规则和自动化报告工具**，是提升咨询作业效率的工具箱。

这里的工具让"商品诊断、经营分析、DisplayMap 制作"等工作可以更快速、更准确地完成。

---

## 适合存放

- Excel 诊断模板说明（使用方法、字段说明）
- 销售数据清洗规则
- 库存数据清洗规则
- 商品档案匹配规则
- 条码匹配规则（不同系统导出的条码格式统一规则）
- Python 脚本说明文档
- 数据分析脚本（.py 文件或 Jupyter Notebook）
- DisplayMap 模板说明
- PowerQuery 规则
- 自动化报告生成说明

---

## 不适合存放

- 某家门店的原始 Excel 数据（先进 `90_素材暂存与待整理`）
- 方法论分析框架（属于 `01_科学零售方法论` 或 `08_经营分析与数据看板`）
- 系统 PRD（属于 `11_系统产品与PRD`）
