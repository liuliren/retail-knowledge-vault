### §2.2 失败文件清单

```
1. 90天_3个月_食品商品动销明细数据_20260508-1.xlsx
2. 90天_3个月_食品商品动销明细数据_20260508-2.xlsx
```

[推断] 这 4 个文件是客户方导出**特殊视图**（明细 / 供应商口径）/ 命名带"-1 / -2"或"双下划线"/ 与标准格式不符 / 但内容应可读。

### §2.3 v2.0 改进 1: file read 健壮性

```python
def read_v2(fp):
    try:
        # ... 原 v2.0.1 逻辑
    except Exception as e:
        log_error(fp, e)
        return None  # 不中断 batch / 失败文件单独 log
```

→ 防止 1 个 corrupt 文件中断全店 batch / 失败文件单独 log 后人工处理。

## §4. 轮 3 测试 — 参数敏感性

### §4.1 流失 threshold 敏感性（方便食品 40 流失 SKU）

| threshold | 真流失 | 噪音 | 真流失占比 |
|---|---|---|---|
| 3 件 | 30 | 10 | 75% |
| **5 件 ⭐** | **16** | **24** | **40%** |
| 8 件 | 9 | 31 | 22.5% |
| 10 件 | 6 | 34 | 15% |
| 15 件 | 3 | 37 | 7.5% |

[推断] threshold = 5 件**最稳**（与方便食品案例 16/40 = 40% 吻合 / 既不太松也不太严）。

### §4.2 ABCD 阈值敏感性

| 阈值 | A | B | C | D | 评估 |
|---|---|---|---|---|---|
| (0.5, 0.8, 0.95) | 22 | 56 | 65 | 56 | 标准帕累托 / 大组合理 |
| (0.5, 0.85, 0.95) | 22 | 71 | 50 | 56 | A 不变 / B 偏大 |
| **(0.6, 0.85, 0.95)** | **35** | **58** | **50** | **56** | 中等组（100-300 SKU）合理 |
| (0.6, 0.85, 0.97) | 35 | 58 | 66 | 40 | C 偏大 |
| **(0.7, 0.9, 0.97)** | **53** | **61** | **45** | **40** | 小组（<100 SKU）合理 |

### §4.3 v2.0 改进 5: 动态 ABCD 阈值

```python
def get_abcd_thresholds(sku_count):
    if sku_count < 100:
        return (0.7, 0.9, 0.97)  # 小组
    elif sku_count < 300:
        return (0.6, 0.85, 0.95)  # 中等组
    else:
        return (0.5, 0.8, 0.95)  # 大组（标准）
```

### §4.4 季节性比例敏感性

| seasonal_ratio | 识别季节性 | 占流失% |
|---|---|---|
| **2** | **2** | **5.0%** |
| 3 | 2 | 5.0% |
| 5 | 1 | 2.5% |
| 8 | 0 | 0% |

[推断] **ratio = 2** 最合理（不能太严 / 否则错过夏季淡季品类如火锅底料）。
